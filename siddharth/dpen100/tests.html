DPEN100 · Assessment Prep
Tests & Mock Exam
ENGG100 is assessed by lab work, a programming exam (30%) and a dynamics exam (20%), plus assignments. Below are practice versions written to match that split — try them closed-book and timed, then reveal the full worked solutions.
Treat these like the real thing. The Programming Theory Test drills MATLAB syntax, data types, control flow and functions. The Dynamics Quiz covers kinematics. The Mock Final samples the whole subject in one paper. Write code by hand, show every step in the dynamics working, and check your method — not just the answer — against the solution.

Test 1 · Programming
Programming Theory Test
MATLAB fundamentals, control flow, functions, recursion and the Python transition. Closed book.
10 questions~50 min50 marksGuides 1-3
Q1State what each of these produces: (a) x = 5  (b) x = 5;  (c) 5 + 2 with no assignment.3 marks
▶ Show solution
(a) Assigns 5 to x and echoes x = 5 to the Command Window. (b) Same assignment but the semicolon suppresses the output. (c) Computes 7 and stores it in the default variable ans.
Q2A program needs whole numbers 0-255 only. Which integer type is the best fit, and what happens to uint8(300)?4 marks
▶ Show solution
uint8 (unsigned 8-bit, range 0-255) is ideal. uint8(300) saturates to 255 — MATLAB integers clamp at the limit rather than overflowing/wrapping.
Q3For a = [2 4 6] and b = [1 2 3], give the result of (a) a .* b and (b) a ./ b.4 marks
▶ Show solution
(a) [2 8 18] (element-wise multiply). (b) [2 2 2] (element-wise divide). The dot makes the operation happen position-by-position.
Q4Write one line that builds a vector of the squares of 1 to 5, i.e. [1 4 9 16 25].4 marks
▶ Show solution
sq = (1:5).^2
The colon makes [1 2 3 4 5]; .^2 squares each element. (Plain ^2 would attempt a matrix power and error.)
Q5For M = [10 20 30; 40 50 60], give (a) M(2,1), (b) M(:,3), (c) size(M).5 marks
▶ Show solution
(a) 40 (row 2, col 1). (b) [30; 60] (all rows of column 3). (c) [2 3] (2 rows, 3 columns). Indexing is 1-based and ordered (row, column).
Q6Write an if/elseif/else block that classifies a number n as 'positive', 'negative' or 'zero'.5 marks
▶ Show solution
if n > 0 disp('positive') elseif n < 0 disp('negative') else disp('zero') end
Order matters: the first true branch runs; else catches everything left (here, exactly zero).
Q7Write a for loop that sums the integers 1 to 100 into a variable total. What value results?5 marks
▶ Show solution
total = 0; for k = 1:100 total = total + k; end
total = 5050 (\(\tfrac{100\cdot101}{2}\)). The accumulator starts at 0 and adds each k. (One-liner check: sum(1:100).)
Q8Write a function kineticEnergy(m, v) returning \(\tfrac12 m v^2\). What rule governs the file name?5 marks
▶ Show solution
function KE = kineticEnergy(m, v) KE = 0.5 * m * v.^2; end
The file must be named kineticEnergy.m — the file name must match the function name. Inputs m, v and output KE are local to the function.
Q9Define recursion. Write a recursive function for the \(n\)-th Fibonacci number and name its base cases.6 marks
▶ Show solution
Recursion = a function that calls itself on a smaller input until a base case stops it.
function f = fib(n) if n <= 2 f = 1; % base cases: fib(1)=fib(2)=1 else f = fib(n-1) + fib(n-2); end end
Base cases: \(n=1\) and \(n=2\) both return 1. Without them the calls never terminate.
Q10Give three differences between MATLAB and Python syntax, and name the Python libraries that replace MATLAB's array maths and plotting.4 marks
▶ Show solution
Differences: indexing from 1 (MATLAB) vs 0 (Python); blocks end with end vs indentation; comments % vs # (also ^ vs ** for power). Libraries: numpy for arrays/matrix maths, matplotlib for plotting.
Test 2 · Dynamics
Dynamics Practice Quiz — Kinematics
Rectilinear and curvilinear motion. Calculator allowed; show all working. Use \(g=9.81\,\text{m/s}^2\).
8 questions~40 min40 marksGuide 4
Q1Define velocity and acceleration as derivatives of position, and write the time-free relation linking \(a\), \(v\) and \(s\).4 marks
▶ Show solution
\(v=\dfrac{ds}{dt}\), \(a=\dfrac{dv}{dt}\), and the time-free relation \(a\,ds=v\,dv\) (used when time is not given).
Q2A train starts from rest and accelerates uniformly at \(0.5\,\text{m/s}^2\). Find its speed and distance after 20 s.5 marks
▶ Show solution
\(v=10\,\text{m/s}\); \(s=100\,\text{m}\). \(v=u+at=0+0.5(20)=10\); \(s=ut+\tfrac12at^2=0+\tfrac12(0.5)(400)=100\,\text{m}\).
Q3A car travelling at 30 m/s brakes at \(6\,\text{m/s}^2\). How far does it travel before stopping?5 marks
▶ Show solution
75 m. Use \(v^2=u^2+2as\) with \(v=0\): \(0=30^2+2(-6)s\Rightarrow s=\dfrac{900}{12}=75\,\text{m}\).
Q4A ball is thrown straight up at 25 m/s. Find (a) the time to the highest point and (b) the maximum height.5 marks
▶ Show solution
(a) At the top \(v=0\): \(t=\dfrac{u}{g}=\dfrac{25}{9.81}=2.55\,\text{s}\). (b) \(h=\dfrac{u^2}{2g}=\dfrac{625}{19.62}=31.9\,\text{m}\).
Q5A particle moves so that \(s(t)=t^3-6t^2+9t\) (m). Find its velocity and acceleration functions, and the times when it is momentarily at rest.6 marks
▶ Show solution
\(v=3t^2-12t+9\), \(a=6t-12\). At rest \(v=0\): \(3(t^2-4t+3)=0\Rightarrow(t-1)(t-3)=0\), so \(t=1\,\text{s}\) and \(t=3\,\text{s}\). (The particle reverses direction between these — relevant for total distance.)
Q6Explain the difference between distance travelled and displacement, and how you compute total distance for the particle in Q5 between \(t=0\) and \(t=4\).5 marks
▶ Show solution
Displacement = net change in position; distance sums every metre regardless of direction. Split at the turning points \(t=1,3\): \(s(0)=0,\;s(1)=4,\;s(3)=0,\;s(4)=4\). Legs: \(|4-0|+|0-4|+|4-0|=4+4+4=\)12 m total distance (vs net displacement of 4 m).
Q7A stone is kicked horizontally at 12 m/s from a 20 m cliff. Find the time to land and the horizontal range.5 marks
▶ Show solution
\(t=2.02\,\text{s}\); range ≈ 24.2 m. Vertical: \(20=\tfrac12gt^2\Rightarrow t=\sqrt{40/9.81}=2.02\,\text{s}\). Horizontal (constant): \(x=u_x t=12\times2.02=24.2\,\text{m}\).
Q8A projectile is launched at 40 m/s, \(\theta=35^\circ\) on level ground. Find the time of flight and range.5 marks
▶ Show solution
\(t=4.68\,\text{s}\); range ≈ 153 m. \(u_y=40\sin35^\circ=22.9\,\text{m/s}\), \(u_x=40\cos35^\circ=32.8\,\text{m/s}\). \(t=\dfrac{2u_y}{g}=\dfrac{45.9}{9.81}=4.68\,\text{s}\); \(R=u_x t=32.8\times4.68\approx153\,\text{m}\).
Mock Final · Whole subject
Mock Final Exam
Twelve questions sampling programming and dynamics, weighted ~60% programming / 40% dynamics to mirror the ENGG100 exam split. 100 marks. ~2 hours, closed book.
12 questions~2 hours100 marksGuides 1-4
Section A — Programming (60 marks)
Q1(a) Why is double MATLAB's default numeric type? (b) Why should you never test two computed floating-point values for exact equality, and what should you do instead?6 marks
▶ Show solution
(a) double gives ~15-16 significant digits and a huge range, suitable for general engineering maths, so MATLAB uses it unless told otherwise. (b) Many decimals can't be stored exactly in binary, so rounding errors (~1e-16) make a==b unreliable. Test abs(a-b) < tol with a small tolerance instead.
Q2Given v = [5 -2 8 -1 4], write single lines to (a) get the elements greater than 0, (b) count how many are negative, (c) replace negatives with 0.6 marks
▶ Show solution
% (a) pos = v(v > 0); % [5 8 4] % (b) nNeg = sum(v < 0); % 2 % (c) v(v < 0) = 0; % [5 0 8 0 4]
Logical masks (v>0 etc.) both select and assign.
Q3Write a script that uses a while loop to find the smallest integer \(n\) such that \(2^n > 1000\). State the danger of a while loop.8 marks
▶ Show solution
n = 0; while 2^n <= 1000 n = n + 1; end disp(n) % 10 (2^10 = 1024)
Danger: if nothing inside the loop changes the test variable, it never terminates (infinite loop). Here n = n + 1 guarantees progress.
Q4Write a function stats(v) that returns the mean and the maximum of a vector v as two outputs. Show how it is called.8 marks
▶ Show solution
function [m, mx] = stats(v) m = mean(v); mx = max(v); end
Call with two outputs: [a, b] = stats([3 7 2]); giving a = 4, b = 7. Multiple outputs go in square brackets on both sides.
Q5Trace this code and give the printed output:
total = 0; for k = 1:5 if mod(k,2) == 0 total = total + k; end end disp(total)
6 marks
▶ Show solution
6. mod(k,2)==0 is true only for even k (2 and 4). It adds 2 + 4 = 6; the odd values are skipped.
Q6Write the MATLAB to plot \(y=e^{-x}\cos(4x)\) for \(x\) from 0 to 5 with 200 points, a red line, axis labels and a grid.6 marks
▶ Show solution
x = linspace(0, 5, 200); y = exp(-x) .* cos(4*x); plot(x, y, 'r') xlabel('x'); ylabel('y') grid on
Note the element-wise .* — both exp(-x) and cos(4x) are vectors of equal length.
Q7Name the MATLAB functions to (a) read a CSV into a table and (b) save variables to a .mat file. (c) Give the three steps of low-level file access with fopen.6 marks
▶ Show solution
(a) readtable('file.csv'). (b) save('data.mat') (load back with load). (c) open with fopen → read/write (e.g. fscanf/fprintf) → close with fclose. Always close the file.
Q8Rewrite this MATLAB loop in Python (printing 1 to 5):
for i = 1:5 disp(i) end
8 marks
▶ Show solution
for i in range(1, 6): print(i)
Key differences shown: range(1,6) excludes the upper bound (so it stops at 5); the block is defined by indentation (no end); and the comment/print syntax differs (#, print).
Section B — Engineering Dynamics (40 marks)
Q9A motorcycle accelerates from 8 m/s to 28 m/s over a distance of 180 m. Find (a) the acceleration and (b) the time taken.8 marks
▶ Show solution
(a) \(v^2=u^2+2as\Rightarrow a=\dfrac{28^2-8^2}{2(180)}=\dfrac{784-64}{360}=2\,\text{m/s}^2\). (b) \(v=u+at\Rightarrow t=\dfrac{28-8}{2}=10\,\text{s}\).
Q10A ball is dropped from a tower and takes 3.5 s to reach the ground. Find (a) the tower height and (b) the impact speed.8 marks
▶ Show solution
(a) \(h=\tfrac12gt^2=\tfrac12(9.81)(3.5^2)=\tfrac12(9.81)(12.25)=60.1\,\text{m}\). (b) \(v=gt=9.81\times3.5=34.3\,\text{m/s}\).
Q11A particle's velocity is \(v(t)=12-4t\) (m/s). Find (a) when it is momentarily at rest, (b) its acceleration, and (c) the total distance travelled from \(t=0\) to \(t=5\,\text{s}\).12 marks
▶ Show solution
(a) \(v=0\Rightarrow t=3\,\text{s}\). (b) \(a=\dfrac{dv}{dt}=-4\,\text{m/s}^2\) (constant). (c) Displacement = \(\int v\,dt = 12t-2t^2\). From 0→3: \(36-18=18\,\text{m}\). From 3→5: \((60-50)-(36-18)=10-18=-8\,\text{m}\) (so 8 m back). Total distance = \(18+8=\)26 m (net displacement only 10 m).
Q12A projectile is fired at 50 m/s, \(\theta=40^\circ\) above the horizontal on level ground. Find (a) \(u_x\) and \(u_y\), (b) the maximum height, (c) the time of flight, and (d) the range.12 marks
▶ Show solution
(a) \(u_x=50\cos40^\circ=38.3\,\text{m/s}\), \(u_y=50\sin40^\circ=32.1\,\text{m/s}\). (b) \(H=\dfrac{u_y^2}{2g}=\dfrac{32.1^2}{19.62}=52.6\,\text{m}\). (c) \(t=\dfrac{2u_y}{g}=\dfrac{64.3}{9.81}=6.55\,\text{s}\). (d) \(R=u_x t=38.3\times6.55\approx251\,\text{m}\).
Marking yourself: 50-64 = developing, 65-74 = sound, 75-84 = strong, 85+ = excellent. Re-attempt any question where you missed the method, not just the arithmetic.
