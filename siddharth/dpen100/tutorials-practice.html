Tutorial 1 · Week 1
MATLAB Basics
The environment, numeric data types and variables — getting comfortable in the Command Window.
Maps to Guide 1, Topics 1-3MATLAB Onramp
Problems 1-8Tap to reveal solutions
1What does a semicolon ; at the end of a MATLAB statement do?▶
It suppresses output. The statement still runs and the variable is still assigned — MATLAB just doesn't echo the result to the Command Window. Without it, MATLAB prints the answer (or stores it in ans if there is no left-hand side).
2After typing 5 + 3 with no variable, where is the result stored?▶
In the built-in variable ans. Any expression without an explicit assignment is placed in ans, which is overwritten by the next un-assigned result.
3Name MATLAB's default numeric type and give its range/precision class.▶
double — 64-bit floating point. Every number is a double unless you explicitly cast it (e.g. int8, single). It carries ~15-16 significant decimal digits.
4What is the largest value an int8 can hold, and what happens if you exceed it?▶
127 (range −128…127). MATLAB integers saturate: int8(200) returns 127 rather than wrapping around. The smallest is −128.
5Predict the output: z = 3 + 4i; abs(z)▶
5. abs of a complex number is its magnitude \(\sqrt{3^2+4^2}=5\). Related: real(z)=3, imag(z)=4, angle(z)=0.927\,\text{rad}\).
6Which of these are valid variable names: 2nd, my_var, end, Speed1?▶
Valid: my_var, Speed1. 2nd starts with a digit (illegal); end is a reserved keyword. Names must start with a letter, then letters/digits/underscores, and are case-sensitive.
7How do you check the data type (class) of a variable x?▶
class(x) returns the type as a string, e.g. 'double'. Use isa(x,'double') to test a specific class, or whos to list all variables with their types and sizes.
8Why can 0.1 + 0.2 == 0.3 return false in MATLAB?▶
Floating-point rounding. 0.1, 0.2 and 0.3 cannot be stored exactly in binary, so the sum differs from 0.3 by ~1e-17. Never test floats for exact equality — use abs(a-b) < 1e-9 instead.
Tutorial 2 · Week 2
Operators & Logic
Arithmetic and precedence, element-wise (dot) operators, relational and logical operators.
Maps to Guide 1, Topics 4-6Tables 2.1 / 2.3
Problems 1-8Tap to reveal solutions
1Evaluate by precedence: 2 + 3 * 4 ^ 2▶
50. Power first: \(4^2=16\); then multiply \(3\times16=48\); then add \(2+48=50\). Order is: () → ^ → * / → + −.
2For a = [1 2 3], b = [4 5 6], what is a .* b?▶
[4 10 18]. The dot operator .* multiplies element-by-element. Plain a * b would error (inner dimensions don't match for matrix multiply).
3What is the difference between / and ./ for arrays?▶
./ divides element-wise; / is matrix right-division (solves a linear system). For two equal-size vectors you almost always want ./. Same idea for .* vs * and .^ vs ^.
4Evaluate mod(17, 5) and rem(17, 5).▶
Both 2 here. \(17 = 3\times5 + 2\). They differ only for negative operands: mod follows the sign of the divisor, rem follows the sign of the dividend.
5What does a relational operator like x > 3 return?▶
A logical value (1 = true, 0 = false). Applied to an array it returns a logical array of the same size, which can then be used for logical indexing, e.g. x(x>3).
6Difference between & and && in MATLAB?▶
& is element-wise (works on arrays); && is scalar short-circuit. && stops evaluating as soon as the result is known (if the left side is false it never checks the right), so use it for scalar conditions in if statements.
7Predict: ~(5 > 2) and (3 == 3) | (1 > 9).▶
0 (false) and 1 (true). ~ is logical NOT, so NOT(true)=false. The OR is true because the first operand is true.
8Write one line giving the elements of v = [3 8 1 9 4] that are > 4.▶
v(v > 4) → [8 9]. The mask v>4 is [0 1 0 1 0] (logical); indexing with it keeps only the true positions.
Tutorial 3 · Week 3
Arrays & Matrices
Creating vectors and matrices, the colon operator, indexing and slicing, and cell arrays.
Maps to Guide 2, Topics 1-21-based indexing
Problems 1-8Tap to reveal solutions
1What does 1:2:9 produce?▶
[1 3 5 7 9]. The colon form is start:step:stop. With no step, 1:5 gives [1 2 3 4 5] (step = 1).
2Difference between linspace(0,10,5) and 0:10?▶
linspace fixes the count; the colon fixes the step. linspace(0,10,5) = [0 2.5 5 7.5 10] (5 points). 0:10 = 11 integers. Use linspace when you need an exact number of samples.
3For v = [10 20 30 40], what is v(2) and v(end)?▶
v(2)=20, v(end)=40. MATLAB indexing starts at 1 (not 0). end always refers to the last element.
4Given A = [1 2 3; 4 5 6], what does A(2,3) return, and what is size(A)?▶
A(2,3)=6; size(A)=[2 3] (2 rows, 3 columns). Indexing is (row, column). The semicolon inside [ ] starts a new row.
5How do you select the entire second column of A above?▶
A(:,2) → [2; 5]. A lone colon means "all" in that dimension: A(:,2) = all rows, column 2; A(1,:) = row 1, all columns.
6Why use a cell array {...} instead of a normal array [...]?▶
Cells hold mixed types and different-sized contents. e.g. c = {'Tom', 25, [1 2 3]} stores a string, a number and a vector together. A normal array must be all the same type and rectangular.
7For the cell c = {'Tom', 25}, how do you get the number 25 itself (not a 1×1 cell)?▶
c{2} (curly braces). Curly braces {} extract the content; round braces c(2) return a 1×1 cell still wrapping the value.
8Write a vector of all even numbers from 2 to 20, then sum it.▶
e = 2:2:20; total = sum(e) % = 110
The colon builds [2 4 6 … 20] (10 values); their sum is 110.
Tutorial 4 · Weeks 4-5
Control Flow & Functions
Conditionals, loops, writing functions and recursion — the heart of the programming exam.
Maps to Guide 2, Topics 3-6Exam: 30% programming
Problems 1-8Tap to reveal solutions
1Write an if/elseif/else that prints "pass" if mark >= 50, else "fail".▶
if mark >= 50 disp('pass') else disp('fail') end
Every if block must close with end. The condition is any expression that evaluates to true/false.
2When is a switch statement clearer than a long if/elseif chain?▶
When you compare one variable against several discrete constant values. e.g. switching on a menu choice 1/2/3 or a day name. otherwise handles unmatched cases (like a final else).
3Trace: how many times does this loop run, and what is s at the end?
s = 0; for k = 1:4 s = s + k; end
▶
Runs 4 times; s = 10. k takes 1, 2, 3, 4 and the accumulator builds 1+2+3+4 = 10. This "running total" pattern is extremely common.
4When should you use a while loop instead of a for loop?▶
When you don't know in advance how many iterations are needed — you loop until a condition becomes false (e.g. "keep asking until the user enters a valid number"). Use for when the count is known.
5What is the danger with a while loop, and how do you avoid it?▶
An infinite loop if the condition never becomes false. Always make sure something inside the loop changes the variable being tested (e.g. i = i + 1;), so it eventually exits. Ctrl+C breaks a runaway loop.
6Write a function circle_area that returns the area of a circle of radius r.▶
function A = circle_area(r) A = pi * r.^2; end
Save as circle_area.m (file name must match). The output variable A is whatever the function returns. Call it with circle_area(5).
7What does "variable scope is local" mean for a function?▶
Variables created inside a function exist only inside it. They are not visible to the script that called it (and vice-versa). The only data that crosses the boundary is via the input arguments and the returned outputs — this keeps functions self-contained and reusable.
8Define a recursive factorial and name its two essential parts.▶
function f = fact(n) if n <= 1 f = 1; % base case else f = n * fact(n-1); % recursive case end end
The two parts are the base case (stops the recursion) and the recursive case (calls itself on a smaller input). Without a base case it recurses forever.
Tutorial 5 · Weeks 6-13
Plotting, File I/O & Python
Visualising data, reading and writing files, and the jump from MATLAB to Python.
Maps to Guide 3plot · readtable · numpy
Problems 1-8Tap to reveal solutions
1Write the lines to plot \(y = \sin x\) over 0 to 2π with axis labels and a title.▶
x = linspace(0, 2*pi, 100); plot(x, sin(x)) xlabel('x'); ylabel('sin x') title('Sine curve')
Use ~100 points so the curve looks smooth. Always label axes in engineering plots.
2What does hold on do?▶
It keeps the current plot so the next plot draws on top instead of replacing it — used to overlay several curves on one set of axes. hold off returns to the default (each new plot clears the old).
3What does plot(x, y, 'r--o') specify?▶
Red colour (r), dashed line (--), circle markers (o). The line-spec string combines colour + line style + marker. e.g. 'b-' = blue solid, 'g:' = green dotted.
4What does subplot(2,1,1) create?▶
A grid of 2 rows × 1 column, and selects the 1st (top) tile to draw in. The pattern is subplot(rows, cols, index). Call it again with index 2 before the second plot.
5Which function reads a spreadsheet/CSV into a table, and which writes one back?▶
readtable('data.csv') reads; writetable(T,'out.csv') writes. For raw numeric matrices use readmatrix/writematrix; for images use imread/imwrite.
6Name the three steps of low-level file reading with fopen.▶
Open → read → close. fid = fopen('f.txt','r'); then loop reading (e.g. fscanf/fgetl until feof(fid)), then always fclose(fid); to release the file. Forgetting fclose can lock the file.
7Give three syntax differences between MATLAB and Python.▶
(1) Indexing: MATLAB starts at 1, Python at 0. (2) Blocks: MATLAB uses end; Python uses indentation. (3) Comments: % in MATLAB, # in Python. Also power: ^ vs **, and files .m vs .py.
8Which Python libraries replace MATLAB's matrix maths and plotting?▶
numpy for arrays/matrix maths and matplotlib (pyplot) for plotting. A MATLAB plot(x,y) becomes plt.plot(x, y) after import matplotlib.pyplot as plt. tkinter covers GUIs.
Tutorial 6 · Weeks 8-9
Engineering Dynamics & Kinematics
Position-velocity-acceleration, SUVAT, erratic motion and projectiles — the 20% dynamics exam.
Maps to Guide 4Exam: 20% dynamics
Problems 1-8Tap to reveal solutions
1State the calculus links between position \(s\), velocity \(v\) and acceleration \(a\).▶
\(v=\dfrac{ds}{dt}\), \(a=\dfrac{dv}{dt}=\dfrac{d^2s}{dt^2}\), and the time-free form \(a\,ds = v\,dv\). Velocity is the slope of the position graph; acceleration is the slope of the velocity graph.
2List the four constant-acceleration (SUVAT) equations.▶
\(v=u+at\); \(s=ut+\tfrac12at^2\); \(v^2=u^2+2as\); \(s=\tfrac12(u+v)t\). They only apply when acceleration is constant. Pick the one that contains the three knowns + the unknown.
3A car accelerates from 10 m/s at 2 m/s² for 5 s. Find final speed and distance.▶
\(v=20\,\text{m/s}\), \(s=75\,\text{m}\). \(v=u+at=10+2(5)=20\); \(s=ut+\tfrac12at^2=10(5)+\tfrac12(2)(25)=50+25=75\,\text{m}\).
4A ball is dropped from rest. How far has it fallen after 3 s? (g = 9.81)▶
≈ 44.1 m. \(s=ut+\tfrac12gt^2=0+\tfrac12(9.81)(9)=44.1\,\text{m}\). Speed then is \(v=gt=29.4\,\text{m/s}\).
5For variable motion given \(s(t)\), how do you find the times of maximum/minimum velocity?▶
Set acceleration to zero: differentiate twice and solve \(a=\dfrac{dv}{dt}=0\). The velocity is stationary (max or min) at those instants. (Likewise the particle is momentarily at rest when \(v=ds/dt=0\).)
6Why can total distance differ from displacement in erratic motion?▶
Because the particle can reverse direction. Displacement is the net change in position; distance adds up every metre travelled regardless of direction. Find the turning points (where \(v=0\)), compute each leg separately, and sum their magnitudes for total distance.
7A projectile is launched at \(u=20\,\text{m/s}\), \(\theta=30^\circ\). Find the initial \(u_x\) and \(u_y\).▶
\(u_x=u\cos\theta=20\cos30^\circ=17.3\,\text{m/s}\); \(u_y=u\sin\theta=20\sin30^\circ=10\,\text{m/s}\). Horizontal velocity stays constant (\(a_x=0\)); vertical obeys \(a_y=-g\).
8For that projectile (level ground), find the time of flight and horizontal range.▶
\(t\approx2.04\,\text{s}\), range ≈ 35.3 m. Time of flight \(t=\dfrac{2u_y}{g}=\dfrac{2(10)}{9.81}=2.04\,\text{s}\); range \(R=u_x\,t=17.3\times2.04\approx35.3\,\text{m}\).
