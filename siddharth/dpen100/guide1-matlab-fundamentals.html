Topic 1 · Week 1
The MATLAB Environment
Where you type, where your data lives, and how to save your work
MATLAB means MATrix LABoratory — an environment built for engineering calculation. Before learning any syntax you need to know where things happen: the Command Window for quick experiments, the Workspace that remembers your variables, and script files (.m) that store reusable programs. Getting comfortable here removes 90% of beginner confusion.
DefinitionThe four main panels▶
1
Command Window — type a command after the >> prompt and press Enter for an instant result. Best for quick tests.
2
Workspace — lists every variable currently in memory, with its name, value, size and class.
3
Editor — where you write .m script and function files so a whole program can be re-run with one click.
4
Current Folder — the directory MATLAB looks in for your files. Your script must be on this path to run.
PatternSemicolons, ans and comments▶

Three habits every MATLAB user needs from day one:

Command Window
5 + 3 % no semicolon → result is shown ans = 8 % MATLAB stores unnamed results in 'ans' x = 5 + 3; % semicolon SUPPRESSES the output % nothing is printed, but x now holds 8 % Anything after a % is a comment — ignored by MATLAB
Rule of thumb: end every line with ; in scripts so you are not flooded with output, and remove it only when you deliberately want to see a value.
PatternScript vs Command Window▶

A script is just a list of commands saved in a .m file. Running it is the same as typing those commands one by one — but you can edit, save and re-run instantly.

area_of_circle.m
% Script: compute area of a circle r = 5; area = pi * r^2; disp(area) % prints 78.5398
Watch out: a script file's name must be a valid variable name — start with a letter, no spaces, e.g. lab1.m not 1lab.m or my lab.m.
ExampleUseful housekeeping commands▶
CommandEffect
clcClears the Command Window screen
clearRemoves all variables from the Workspace
clear xRemoves just variable x
who / whosList variables (whos = with details)
help sinShows documentation for a function
format longShow more decimal places

Topic 2 · Week 2
Data Types & Numbers
Integers, floating-point reals, and complex numbers
A computer must know what kind of number it is storing to choose how many bytes to use and how to do arithmetic. Choosing the right type matters: integers are exact but limited in range; double reals cover a huge range but carry tiny rounding errors; complex numbers let you solve problems (like AC circuits and roots of negatives) that real numbers cannot.
DefinitionInteger types▶

Integers store whole numbers only. The number after the name is how many bits of storage — more bits means a wider range.

TypeBitsSigned range
int88\(-128\) to \(127\)
int1616\(\pm 3.27\times10^{4}\)
int3232\(\pm 2.15\times10^{9}\)
int6464\(\pm 9.22\times10^{18}\)

Each has an unsigned twin (uint8…uint64) that drops negatives to double the positive range.

a = int8(100); % fits b = int8(200); % TOO BIG → saturates to 127
Saturation: if a value exceeds an integer type's range, MATLAB clamps it to the maximum (not "wrap-around"). int8(200) becomes 127.
DefinitionFloating-point: single vs double▶

Real (decimal) numbers are stored in floating-point form. MATLAB's default for every number you type is double.

TypeBytesLargest magnitude
single4\(\approx 3.4028\times10^{38}\)
double8\(\approx 1.7977\times10^{308}\)

double has about 15–16 significant digits; single only ~7 but uses half the memory.

Why it matters: floating-point cannot store every decimal exactly. 0.1 + 0.2 is not exactly 0.3 — it is 0.30000000000000004. Never test floats with ==; check abs(a-b) < 1e-9 instead.
DefinitionComplex numbers▶

A complex number has a real and an imaginary part. MATLAB uses i or j for \(\sqrt{-1}\):

\[ z = a + bj \qquad |z| = \sqrt{a^2+b^2} \qquad \theta = \tan^{-1}\!\left(\tfrac{b}{a}\right) \]
FunctionReturns
real(z)Real part \(a\)
imag(z)Imaginary part \(b\)
abs(z)Magnitude \(|z|\)
angle(z)Phase angle (radians)
conj(z)Conjugate \(a-bj\)
z = 3 + 4j; abs(z) % 5 (since √(9+16)=5) real(z) % 3 conj(z) % 3 - 4j
ExampleTwo worked examples▶
Example 1 — Magnitude of a complex number

Find the magnitude of \(z = 6 + 8i\).

\[ |z| = \sqrt{6^2+8^2} = \sqrt{36+64} = \sqrt{100} = 10 \]
ANSWER abs(6+8i) = 10
Example 2 — Will it fit in int8?

Does the value 130 fit in an int8?

1
int8 range is \(-128\) to \(127\).
2
130 > 127, so it does not fit and saturates to 127.
ANSWER No — int8(130) = 127

Topic 3 · Week 2
Variables & Assignment
Naming, storing and converting values
A variable is a named box that holds a value. The = sign is not "equals" — it is assignment: "take the value on the right and store it in the name on the left." Good variable names make a program readable; understanding type and casting prevents silent errors.
DefinitionAssignment and naming rules▶
speed = 60; % store 60 in 'speed' speed = speed + 5; % read 60, add 5, store 65 back

Naming rules:

1
Must start with a letter.
2
May contain letters, digits and underscores — no spaces or symbols.
3
Case-sensitive: Speed, speed and SPEED are three different variables.
4
Avoid overwriting built-ins like pi, sin, i.
PatternChecking and changing type▶

Use class() to ask "what type is this?" and a cast function to convert.

x = 7.9; class(x) % 'double' y = int32(x); % cast → rounds to 8 (NOT 7!) class(y) % 'int32'
Watch out: casting a float to an integer rounds to nearest, it does not truncate. int32(7.9) is 8. Use floor, ceil or fix if you want a specific rounding direction.
ExampleWorked example — swapping two values▶
Example — Swap a and b

Given a = 3, b = 8, swap their values using a temporary variable.

temp = a; % temp = 3 a = b; % a = 8 b = temp; % b = 3
ANSWER a = 8, b = 3

Topic 4 · Week 2
Operators & Expressions
Arithmetic, precedence, and the all-important element-wise dot operators
Operators are the verbs of MATLAB. The big idea — and the #1 source of beginner bugs — is the difference between matrix operations (* / ^) and element-wise operations (.* ./ .^). When working with whole arrays, you almost always want the dotted versions.
DefinitionScalar arithmetic operators▶
OperatorOperationExample
+Addition5 + 2 → 7
-Subtraction5 - 2 → 3
*Multiplication5 * 2 → 10
/Division5 / 2 → 2.5
^Power5 ^ 2 → 25
\Left division5 \ 10 → 2
PatternOperator precedence (order of operations)▶

MATLAB follows standard maths precedence:

\[ \text{( )} \;\to\; \text{power} \;\to\; \times,\, / \;\to\; +,\, - \]
2 + 3 * 4 % = 14 (× before +) (2 + 3) * 4 % = 20 (brackets first) 2 ^ 3 ^ 2 % = 64 ... power is left-to-right here
Tip: when in doubt, add brackets. They cost nothing and make intent obvious.
PatternElement-wise (dot) operators — the key idea▶

When you operate on whole arrays, the dot tells MATLAB to act element-by-element rather than as matrix algebra.

Element-wiseMeaning
.*Multiply matching elements
./Divide matching elements
.\Left divide element-wise
.^Raise each element to a power
a = [1 2 3]; b = [4 5 6]; a .* b % [4 10 18] element-by-element a .^ 2 % [1 4 9] square each element a * b % ERROR — inner dimensions must match
The classic exam trap: to square every element of a vector, you need a.^2. Writing a^2 attempts matrix power and errors on a non-square array.
ExampleWorked example — evaluating an expression▶
Example — Evaluate by precedence

What is 10 - 2 ^ 2 * 3?

1
Power first: 2^2 = 4 → 10 - 4 * 3.
2
Multiply: 4 * 3 = 12 → 10 - 12.
3
Subtract: 10 - 12 = -2.
ANSWER −2
Quick Practice — Topic 4Click to reveal
A

B

▶
v.^2 — the element-wise power operator squares each element.
C

▶

Topic 5 · Week 2
Logical & Relational Values
true, false, and the operators that produce them
Every decision a program makes — every if and every while in Guide 2 — boils down to a single logical value: true (1) or false (0). Relational operators compare values to produce logicals; logical operators combine them. Master this and control flow becomes easy.
DefinitionLogical values▶

A logical is the simplest type: it is either true (displayed as 1) or false (displayed as 0). In arithmetic, true behaves as 1 and false as 0.

true + true % = 2 (1 + 1) sum([1 0 1 1 0]) % = 3 (counts the trues)
DefinitionRelational operators▶
OperatorMeaningExample → result
==Equal to3 == 3 → 1
~=Not equal to3 ~= 4 → 1
<Less than2 < 5 → 1
>Greater than2 > 5 → 0
<=Less than or equal5 <= 5 → 1
>=Greater than or equal4 >= 5 → 0
Watch out: = is assignment, == is comparison. Writing if x = 5 is an error; you mean if x == 5.
PatternLogical operators & short-circuiting▶
OperatorMeaning
&&AND (scalar, short-circuit)
||OR (scalar, short-circuit)
~NOT
& / |Element-wise AND / OR (for arrays)

Short-circuit means MATLAB stops as soon as the answer is known:

% If x is 0, never evaluate 1/x → avoids divide error if x ~= 0 && 1/x > 2 ... end
Truth tables: AND is true only if both are true; OR is true if at least one is true; NOT flips the value.
ExampleWorked example — combining conditions▶
Example — Evaluate a logical expression

Given age = 20, evaluate (age >= 18) && (age < 65).

1
age >= 18 → 20 >= 18 → true (1).
2
age < 65 → 20 < 65 → true (1).
3
true && true → true (1).
ANSWER 1 (true)
Quick Practice — Topic 5Click to reveal
A
What does 5 ~= 5 return?
▶
0 (false) — 5 is equal to 5, so "not equal" is false.
B
Evaluate (3 > 2) || (10 < 1).
▶
true OR false → 1 (true). OR needs only one true.
C
Why use && rather than & in an if?
▶
&& short-circuits: if the first part is false it skips the second, which is faster and prevents errors (e.g. avoiding a divide-by-zero).
D
What is sum([1 0 0 1 1]) and why is it useful?
▶
3 — since true=1, summing a logical array counts how many elements are true.
Topic 6 · Week 2
Strings & Characters
Storing and manipulating text
Programs do not only crunch numbers — they label outputs, read filenames and build messages. MATLAB has two text types: the older char array (single quotes) and the modern string (double quotes). Knowing how to join text and convert between numbers and text is essential for readable output.
Definitionchar vs string▶
c = 'hello'; % char array (single quotes) s = "hello"; % string (double quotes, newer) class(c) % 'char' class(s) % 'string'

A char array is really a row of characters — c(1) is 'h'. Strings are more convenient for whole-text operations.

PatternJoining and converting text▶
TaskFunction
Join char arraysstrcat('a','b') or ['a' 'b']
Number → textnum2str(42) → '42'
Text → numberstr2num('42') → 42
Length of textlength(c) or strlength(s)
Formatted printfprintf('x = %d\n', x)
name = 'Sid'; msg = ['Hello ', name]; % 'Hello Sid' fprintf('Area = %.2f m^2\n', 78.539); % Area = 78.54 m^2
ExampleWorked example — building output▶
Example — Report a result as a sentence

You have r = 5 and area = 78.54. Print "Circle radius 5 has area 78.54".

fprintf('Circle radius %d has area %.2f\n', r, area);
OUTPUT Circle radius 5 has area 78.54
