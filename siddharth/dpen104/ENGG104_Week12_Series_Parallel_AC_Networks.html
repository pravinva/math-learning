1 Series-parallel impedance reduction

Impedances combine using the same series/parallel structure as resistors, series impedances simply add, parallel impedances combine via reciprocals, but because \(Z=R\pm jX\) carries both magnitude and phase, the arithmetic must be done as complex numbers, not plain real numbers.

Key idea

Series: \(Z_t=Z_1+Z_2\) (add real parts and imaginary parts separately). Parallel: convert each branch to admittance \(Y=1/Z\), sum the admittances, then invert back: \(Z_{parallel}=1/(Y_1+Y_2)\).

Worked example

\(R_1=10\ \Omega\) in series with a parallel combination of \(R_2=20\ \Omega\) and a pure inductive branch \(X_L=15\ \Omega\). Find the parallel section's equivalent impedance, then the total.

\(Y_{R2}=1/20=0.05\text{ S}\). \(Y_L=1/(j15)=-j0.0667\text{ S}\). \(Y_{total}=0.05-j0.0667\text{ S}\).

\(|Y|=\sqrt{0.05^2+0.0667^2}=0.0834\text{ S}\). \(|Z_{parallel}|=1/0.0834=12.0\ \Omega\), at angle \(+53.13^\circ\) (angle flips sign when inverting \(Y\to Z\)).

In rectangular form: \(Z_{parallel}=12(\cos53.13^\circ+j\sin53.13^\circ)=7.2+j9.6\ \Omega\).

\(Z_{total}=R_1+Z_{parallel}=10+7.2+j9.6=17.2+j9.6\ \Omega\). \(|Z_{total}|=\sqrt{17.2^2+9.6^2}=19.7\ \Omega\), \(\theta=\tan^{-1}(9.6/17.2)=29.2^\circ\).

Practice 1

A parallel section has equivalent impedance \(Z_p=8+j6\ \Omega\), in series with \(R_1=5\ \Omega\). Find the total impedance (rectangular form) and its magnitude.

Solution

\(Z_{total}=5+8+j6=13+j6\ \Omega\). \(|Z_{total}|=\sqrt{169+36}=14.32\ \Omega\).

For Practice 1, find the total phase angle \(\theta\).

Solution

\(\theta=\tan^{-1}(6/13)=24.78^\circ\).

Practice 3

A parallel combination of \(R=30\ \Omega\) and a pure inductive branch \(X_L=40\ \Omega\). Using the admittance method, find \(|Z_{parallel}|\).

Solution

\(Y_R=1/30=0.0333\text{ S}\). \(Y_L=-j/40=-j0.025\text{ S}\). \(|Y|=\sqrt{0.0333^2+0.025^2}=0.04166\text{ S}\). \(|Z_{parallel}|=1/0.04166=24.0\ \Omega\).

Practice 4

Explain why you cannot simply average or directly add two parallel branches' impedances the way you might expect from DC parallel resistor intuition.

Solution

Impedances have both magnitude AND phase (they're complex quantities), a resistive branch and a reactive branch don't align in the same "direction." Combining them requires proper complex (vector) arithmetic, via admittances, summed then inverted, not simple real-number parallel formulas, since the reactive branch's opposition is 90° phase-shifted from the resistive branch's.

Practice 5

Outline the general workflow for solving a series-parallel AC network's total impedance.

Solution

1. Express each branch as a complex impedance \(R\pm jX\). 2. For any parallel sections, convert to admittances (\(Y=1/Z\)), sum them, then invert back to get that section's equivalent impedance. 3. Add that result in series with any remaining series elements (impedances in series simply add). 4. The final total \(Z\) gives both magnitude \(|Z|\) and phase angle \(\theta\) for the whole network.

2 Full network analysis: current, voltage, power

Once the total impedance is known, current, power factor, and power follow directly, then you can work backward to any individual branch.

Worked example

Using Section 1's network (\(Z_{total}=17.2+j9.6\ \Omega\), \(|Z_{total}|=19.7\ \Omega\), \(\theta=29.2^\circ\)), with \(V_{rms}=120\text{ V}\), find \(I_{total}\), \(PF\), and average power.

\(I_{total}=120/19.7=6.09\text{ A}\). \(PF=\cos(29.2^\circ)=0.873\).

\(P=120(6.09)(0.873)=637.8\text{ W}\).

Practice 6

A network has \(|Z_{total}|=25\ \Omega\), \(\theta=36.87^\circ\), \(V_{rms}=100\text{ V}\). Find \(I_{total}\), \(PF\), and \(P\).

Solution

\(I=100/25=4\text{ A}\). \(PF=\cos(36.87^\circ)=0.8\). \(P=100(4)(0.8)=320\text{ W}\).

Practice 7

A network has \(|Z_{total}|=50\ \Omega\), \(\theta=53.13^\circ\), \(V_{rms}=200\text{ V}\). Find \(I_{total}\), \(PF\), and \(P\).

Solution

\(I=200/50=4\text{ A}\). \(PF=\cos(53.13^\circ)=0.6\). \(P=200(4)(0.6)=480\text{ W}\).

Practice 8

Once \(I_{total}\) is known for a series-parallel network, explain how you'd find the voltage across the parallel section specifically.

Solution

Multiply the total current by the parallel section's own equivalent impedance magnitude: \(V_{parallel}=I_{total}\times|Z_{parallel}|\), since the parallel block, once reduced, behaves like a single series element carrying the same total current.

Practice 9

Once the voltage across a parallel section is known, explain how you'd find the current in one specific branch of that section.

Solution

Divide that voltage by the specific branch's own impedance: \(I_{branch}=V_{parallel}/Z_{branch}\), Ohm's Law applied to that branch alone, since all parallel branches share the same voltage.

Practice 10

Summarise why series-parallel AC analysis is really just the DC series-parallel techniques from Weeks 2–4, applied with complex numbers instead of plain real ones.

Solution

Every rule stays the same in structure, series impedances add, parallel impedances combine via reciprocals (admittances), Ohm's Law relates \(V\), \(I\), and \(Z\). Only now \(R\) is replaced by the complex impedance \(Z=R\pm jX\), which carries both magnitude and phase. The underlying logic of reducing a network step by step is identical to Weeks 2–4's DC circuit analysis.
