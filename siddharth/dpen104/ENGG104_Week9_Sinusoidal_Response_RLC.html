Sinusoidal Response of R, L, and C

How each of the three basic components individually responds to AC: a resistor keeps voltage and current perfectly in step, while an inductor and a capacitor each introduce a 90° phase shift, in opposite directions.

1 Resistor response to AC

A resistor doesn't care whether the signal is DC or AC, or how fast it's changing, Ohm's Law applies instant by instant.

Key idea

For a resistor, voltage and current are always exactly in phase (0° difference), at any frequency. \(V=IR\) holds using peak, RMS, or instantaneous values consistently.

Worked example 1

A 100 Ω resistor has \(v(t)=50\sin(\omega t)\text{ V}\) applied. Find \(i(t)\).

\(i(t)=v(t)/R=50\sin(\omega t)/100=0.5\sin(\omega t)\text{ A}\), same phase angle as \(v(t)\).

Worked example 2

Using RMS values, \(V_{rms}=35.36\text{ V}\) (i.e. \(V_m=50\text{ V}\)), \(R=100\ \Omega\). Find \(I_{rms}\).

\(I_{rms}=35.36/100=0.3536\text{ A}\).

Practice 1

\(R=50\ \Omega\), \(v(t)=100\sin(\omega t)\). Find \(i(t)\).

Solution

\(i(t)=2\sin(\omega t)\text{ A}\).

Practice 2

\(R=200\ \Omega\), \(V_{rms}=141.4\text{ V}\). Find \(I_{rms}\).

Solution

\(I_{rms}=141.4/200=0.707\text{ A}\).

Practice 3

Explain why a resistor's current and voltage are always exactly in phase, regardless of frequency.

Solution

\(v=iR\) is an instantaneous relationship, no time-derivative or integral is involved, so whatever the voltage does at any instant, the current directly mirrors it (scaled by \(1/R\)) at that SAME instant, there's no mechanism for a time delay to develop.

Practice 4

A resistor sees \(v(t)=80\sin(\omega t+30^\circ)\). Write \(i(t)\) if \(R=40\ \Omega\).

Solution

\(i(t)=2\sin(\omega t+30^\circ)\text{ A}\), the same phase angle as the voltage.

Practice 5

Does a resistor's behaviour in an AC circuit change with frequency?

Solution

No. Resistance is frequency-independent, a resistor's opposition to current stays the same no matter how fast or slow the AC signal oscillates.

2 Inductor response to AC

An inductor's opposition to AC grows with frequency, and it shifts current behind voltage by exactly a quarter cycle.

Key idea

Inductive reactance: $$X_L=\omega L=2\pi fL \ (\Omega)$$

Current lags voltage by exactly 90°. \(V_{rms}=I_{rms}\times X_L\).

Worked example 1

An inductor \(L=0.1\text{ H}\) is in a 50 Hz circuit. Find \(X_L\).

\(X_L=2\pi(50)(0.1)=31.42\ \Omega\).

Worked example 2

For that inductor, \(V_{rms}=100\text{ V}\). Find \(I_{rms}\), and state the phase relationship.

\(I_{rms}=100/31.42=3.183\text{ A}\). Current lags voltage by 90°.

Practice 6

\(L=0.2\text{ H}\), \(f=60\text{ Hz}\). Find \(X_L\).

Solution

\(X_L=2\pi(60)(0.2)=75.4\ \Omega\).

Practice 7

\(X_L=50\ \Omega\), \(V_{rms}=120\text{ V}\). Find \(I_{rms}\).

Solution

\(I_{rms}=120/50=2.4\text{ A}\).

Practice 8

\(L=50\text{ mH}\), \(f=1000\text{ Hz}\). Find \(X_L\).

Solution

\(X_L=2\pi(1000)(0.05)=314.16\ \Omega\).

Practice 9

Explain why \(X_L\) increases with frequency, connecting this to \(V_L=L(di/dt)\).

Solution

Higher frequency means current changes more rapidly (faster \(di/dt\) for the same amplitude), so the induced opposing voltage is larger for the same current, the inductor "resists" AC more strongly at higher frequencies, hence \(X_L=\omega L\) grows with frequency.

Practice 10

An inductor has \(X_L=0\) at DC (\(f=0\)). Explain what this means physically.

Solution

At DC, \(X_L=2\pi fL=0\), an ideal inductor offers NO opposition to steady current at all, consistent with \(V_L=L(di/dt)=0\) when current is constant.

Practice 11

State, in one sentence, the phase relationship between voltage and current in a pure inductor.

Solution

Current lags voltage by exactly 90° in a pure inductor.

3 Capacitor response to AC

A capacitor behaves exactly opposite to an inductor: its opposition shrinks with frequency, and it shifts current ahead of voltage.

Key idea

Capacitive reactance: $$X_C=\frac1{\omega C}=\frac1{2\pi fC} \ (\Omega)$$

Current leads voltage by exactly 90°.

Worked example 1

A capacitor \(C=10\ \mu\text{F}\) is in a 50 Hz circuit. Find \(X_C\).

\(X_C=1/(2\pi\times50\times10\times10^{-6})=1/0.003142=318.3\ \Omega\).

Worked example 2

For that capacitor, \(V_{rms}=100\text{ V}\). Find \(I_{rms}\), and state the phase relationship.

\(I_{rms}=100/318.3=0.3142\text{ A}\). Current leads voltage by 90°.

Practice 12

\(C=20\ \mu\text{F}\), \(f=60\text{ Hz}\). Find \(X_C\).

Solution

\(X_C=1/(2\pi\times60\times20\times10^{-6})=1/0.007540=132.6\ \Omega\).

Practice 13

\(X_C=100\ \Omega\), \(V_{rms}=50\text{ V}\). Find \(I_{rms}\).

Solution

\(I_{rms}=50/100=0.5\text{ A}\).

Practice 14

\(C=5\ \mu\text{F}\), \(f=1000\text{ Hz}\). Find \(X_C\).

Solution

\(X_C=1/(2\pi\times1000\times5\times10^{-6})=1/0.03142=31.83\ \Omega\).

Practice 15

Explain why \(X_C\) DECREASES with increasing frequency, opposite to \(X_L\).

Solution

At higher frequency, the capacitor's voltage doesn't have time to build up as much before the current reverses direction again, so less opposition builds up. \(X_C=1/(2\pi fC)\) has \(f\) in the denominator, so higher \(f\) means lower \(X_C\).

Practice 16

A capacitor has \(X_C\to\infty\) as \(f\to0\) (DC). Explain what this means physically.

Solution

At DC, a fully-charged capacitor blocks current entirely (acts like an open circuit) once steady-state is reached, consistent with \(X_C\) becoming infinite as frequency approaches zero.

Practice 17

State the phase relationship in a pure capacitor, and contrast it with an inductor.

Solution

Current LEADS voltage by 90° in a pure capacitor, exactly opposite to an inductor, where current LAGS voltage by 90°.
