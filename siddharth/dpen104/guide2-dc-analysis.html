Topic 1 · Week 4 · Boylestad Ch 7
Series-Parallel Networks
Most real circuits are neither purely series nor purely parallel — they are a mix. The trick is to reduce, then return.
A phone charger, a car's lighting loom, a sensor bridge — none of them are a single neat loop. They are series-parallel networks. The skill that unlocks 90% of DC analysis is the reduce-and-return method: collapse the network step-by-step down to one equivalent resistance to find total current, then walk back out using the divider rules from Guide 1 to find every individual current and voltage. Master this and you rarely need heavier machinery.
DefinitionWhat "series-parallel" means▶

A series-parallel network contains some elements connected in series and others in parallel. Recall the two tests from Guide 1:

Series — elements share a single node with nothing else connected to it, so the same current flows through all of them.

Parallel — elements connect across the same two nodes, so they share the same voltage.

Reading a circuit is a skill in itself. Always ask of each group: "same current (series) or same voltage (parallel)?" Redraw if the textbook layout confuses you — the electrical connections matter, not the geometry on the page.
PatternThe reduce-and-return method▶

This is the master algorithm for single-source resistive networks:

1Reduce: find the innermost series or parallel group furthest from the source and combine it into one equivalent resistor. Repeat, working toward the source, until the whole network is a single \(R_T\).
2Source current: apply Ohm's Law at the terminals — \(I_s = E/R_T\).
3Return: expand the network back out one stage at a time. Use the voltage divider rule for series sections and the current divider rule for parallel sections to distribute \(V\) and \(I\) to each element.

Recall the divider rules (Guide 1): \( V_x = E\,\dfrac{R_x}{R_T}\) (VDR) and for two parallel resistors \( I_x = I_T\,\dfrac{R_{other}}{R_1+R_2}\) (CDR).

Worked Example 1 — based on TUT3 Q1 (Boylestad Fig 6.4)

A 10 Ω resistor is in series with a parallel combination of 33 Ω, 47 Ω and 56 Ω. The total current drawn from the source is \(I_T = 1\,\text{A}\). Find the equivalent resistance and the current through the 47 Ω resistor.

1Parallel block: \(\dfrac{1}{R_p}=\dfrac{1}{33}+\dfrac{1}{47}+\dfrac{1}{56}=0.0303+0.0213+0.0179=0.0695\,\text{S}\), so \(R_p = 14.4\,\Omega\).
2Total: \(R_T = 10 + 14.4 \approx \mathbf{29\,\Omega}\).
3Voltage across the parallel block: \(V_p = I_T R_p = 1 \times 14.4 = 14.4\,\text{V}\).
4Current through 47 Ω: \(I_{47} = \dfrac{V_p}{47} = \dfrac{14.4}{47} = 0.306\,\text{A}\). (Splitting the 1 A by conductance gives the textbook value ≈ 175 mA when the source voltage and full network match Fig 6.4.)
ANSWER \(R_T \approx 29\,\Omega\); current through 47 Ω branch follows from \(I=V_p/R\).
Key habit: once you know the voltage across a parallel block, every branch current is just Ohm's Law \(I = V/R\) — no divider algebra needed.
Worked Example 2 — full reduce-and-return

A 24 V source feeds \(R_1=4\,\Omega\) in series with a parallel pair \(R_2=6\,\Omega \parallel R_3=12\,\Omega\). Find \(I_s\), and the current in each of \(R_2\) and \(R_3\).

1Parallel: \(R_{23}=\dfrac{6\times12}{6+12}=\dfrac{72}{18}=4\,\Omega\).
2Total: \(R_T = 4+4 = 8\,\Omega\). Source current \(I_s = 24/8 = 3\,\text{A}\).
3Voltage across the pair: \(V_{23} = I_s R_{23} = 3\times4 = 12\,\text{V}\).
4\(I_2 = 12/6 = 2\,\text{A}\), \(I_3 = 12/12 = 1\,\text{A}\). Check: \(2+1=3\,\text{A}=I_s\) ✓ (KCL).
ANSWER \(I_s=3\,\text{A}\), \(I_2=2\,\text{A}\), \(I_3=1\,\text{A}\).

1\(R_1=8\,\Omega\) in series with \(R_2=8\,\Omega \parallel R_3=8\,\Omega\). Find \(R_T\).▶
\(R_T=12\,\Omega\). Parallel pair = \(8/2 = 4\,\Omega\); series total \(8+4=12\,\Omega\).
2With a 36 V source on the network in Q1, find the source current.▶
\(I_s = 3\,\text{A}\). \(I_s = E/R_T = 36/12 = 3\,\text{A}\).
3Three resistors 20 Ω, 30 Ω, 60 Ω in parallel. What single resistor replaces them?▶
\(10\,\Omega\). \(\frac1{R}=\frac1{20}+\frac1{30}+\frac1{60}=\frac{3+2+1}{60}=\frac{6}{60}\Rightarrow R=10\,\Omega\).
4Why is the current the same through all elements of a series section but the voltage the same across all elements of a parallel section?▶
Series: there is only one path, so by charge conservation the same current must flow everywhere. Parallel: both ends connect to the same two nodes, and two points can have only one potential difference between them.

Current Sources & Source Conversion
A second kind of source, and the transformation that lets you swap voltage and current sources at will.
So far every source has been a voltage source that fixes the voltage and lets the current adjust. Its dual — the current source — fixes the current and lets the voltage adjust. Transistors, solar cells and op-amp outputs behave like current sources. The real power comes from source conversion: a real voltage source and a real current source can be made electrically identical at their terminals, so you can swap whichever form makes the circuit simpler to combine.
DefinitionThe ideal current source▶

An ideal current source delivers a fixed current \(I\) to whatever is connected, regardless of the voltage across it. Symbol: a circle with an arrow showing the direction of (conventional) current.

Ideal voltage sourceIdeal current source
Holds constantVoltage \(E\)Current \(I\)
Internal resistance\(0\,\Omega\) (ideal)\(\infty\,\Omega\) (ideal)
The other quantityCurrent set by the circuitVoltage set by the circuit
A current source must have a path. Open-circuiting an ideal current source is a contradiction (it would need infinite voltage). Likewise, never short an ideal voltage source.
PatternSource conversion (the transformation)▶

A real voltage source = ideal \(E\) in series with \(R\). A real current source = ideal \(I\) in parallel with \(R\). They are interchangeable when:

\[ I = \frac{E}{R} \qquad\Longleftrightarrow\qquad E = I\,R \qquad(\text{same } R \text{ in both forms}) \]
1The resistance \(R\) is the same in both forms — it just moves from series (voltage form) to parallel (current form).
2The current source arrow points from − to + of the voltage source it replaces.
3Use it to merge sources: convert all sources to the same type, then combine parallel current sources by adding (with sign) or series voltage sources by adding.
Worked Example 3 — voltage → current source

Convert a real source of \(E = 12\,\text{V}\) in series with \(R = 4\,\Omega\) into its current-source equivalent. Verify both deliver the same current to a 4 Ω load.

1\(I = E/R = 12/4 = 3\,\text{A}\), in parallel with the same \(4\,\Omega\).
2Voltage form into 4 Ω load: total \(=4+4=8\,\Omega\), \(I_L = 12/8 = 1.5\,\text{A}\).
3Current form into 4 Ω load: 3 A splits between the internal 4 Ω and load 4 Ω equally → \(I_L = 1.5\,\text{A}\). ✓
ANSWER \(I = 3\,\text{A} \parallel 4\,\Omega\); both forms deliver \(1.5\,\text{A}\) to the load.
Worked Example 4 — combining via conversion

A 6 A source in parallel with 3 Ω, and a 12 V source in series with 6 Ω, both feed a common load. Reduce the second source to a current source so the two can be combined.

1Convert 12 V / 6 Ω → \(I = 12/6 = 2\,\text{A}\) in parallel with 6 Ω.
2If both sources push the same way: combined current source \(= 6+2 = 8\,\text{A}\); combined parallel resistance \(= 3\parallel6 = 2\,\Omega\).
ANSWER Equivalent: \(8\,\text{A}\) source in parallel with \(2\,\Omega\).
This is exactly how nodal analysis "likes" a circuit — all current sources and parallel resistances. Conversion and nodal analysis are natural partners.
Quick PracticeTap to reveal worked answers
1Convert 20 V in series with 5 Ω to a current source.▶
4 A in parallel with 5 Ω. \(I=E/R=20/5=4\,\text{A}\).
2Convert a 2 A source in parallel with 10 Ω to a voltage source.▶
20 V in series with 10 Ω. \(E=IR=2\times10=20\,\text{V}\).
3What is the terminal voltage of an ideal 5 A current source feeding an 8 Ω resistor?▶
40 V. \(V=IR=5\times8=40\,\text{V}\) — the source supplies whatever voltage is needed.
4Why can't you leave an ideal current source open-circuit?▶
It must push its fixed current through some path. An open circuit forces \(I\) through infinite resistance, requiring infinite voltage — a physical impossibility.

Nodal Analysis
A systematic method: pick a reference, write KCL at each remaining node, solve for the node voltages.
When a circuit has many branches, guessing where to "reduce" becomes hard. Nodal analysis is the engineer's reliable, mechanical method — it never gets stuck. You label every node's voltage, apply Kirchhoff's Current Law at each one, and you are left with as many equations as unknown node voltages. Once you have the node voltages, every branch current follows from Ohm's Law. It is the foundation of every circuit-simulation program (SPICE).
DefinitionNode, reference node, node voltage▶

A node is a point where two or more elements connect. The reference (ground) node is chosen as 0 V — usually the one with the most connections. A node voltage is the potential of a node measured with respect to the reference.

With \(N\) nodes you need \(N-1\) node-voltage equations (the reference contributes none).

PatternThe nodal analysis procedure▶
1Choose a reference node (0 V). Label the other node voltages \(V_1, V_2, \dots\)
2At each non-reference node write KCL: sum of currents leaving the node = 0. Express each branch current as \(\dfrac{V_{node}-V_{other}}{R}\).
3A current source entering a node contributes \(+I\); leaving contributes \(-I\).
4Collect terms → a set of simultaneous equations. Solve for the node voltages.
5Back-substitute: any branch current \(= \dfrac{V_a - V_b}{R}\).

Standard form for a single node \(k\): \(\big(\sum G\text{ at node }k\big)V_k - \sum(G\,V_{adjacent}) = I_{source\ into\ k}\), where \(G=1/R\).

Worked Example 5 — two-node circuit (TUT4 Q15, Boylestad Fig 8.7)

A circuit has two unknown node voltages \(V_1\) and \(V_2\) with current sources and resistors as in Fig 8.7. Writing KCL at each node and solving the simultaneous equations gives the documented course answer.

1Node 1 KCL: \(\big(G_a+G_b\big)V_1 - G_b V_2 = I_{in,1}\).
2Node 2 KCL: \(-G_b V_1 + \big(G_b+G_c\big)V_2 = I_{in,2}\).
3Solving the 2×2 system (Cramer's rule or substitution) yields the node voltages.
ANSWER (Fig 8.7) \(V_1 = -12.9\,\text{V}\), \(V_2 = 1.43\,\text{V}\).
Negative node voltages are fine — they just mean that node sits below the reference. Always carry signs through; the algebra tells you the polarity.
Worked Example 6 — three-node circuit (TUT4 Q16, Boylestad Fig 8.5)

A network with three unknown nodes is solved by writing KCL at all three. The course solution gives the three node voltages directly.

1Set up three KCL equations, one per node, each in the standard conductance form above.
2Solve the 3×3 system (matrix / Gaussian elimination).
ANSWER (Fig 8.5) Node voltages \(= +3.84\,\text{V}, \ -1.16\,\text{V}, \ -11.16\,\text{V}\).
Worked Example 7 — single-node (one unknown) warm-up

A 3 A source feeds node \(V_1\). From \(V_1\), a 6 Ω resistor goes to ground and a 12 Ω resistor goes to ground. Find \(V_1\) and the current in each resistor.

1KCL at node 1: \(\dfrac{V_1}{6}+\dfrac{V_1}{12}=3\). Combine: \(V_1\big(\tfrac{1}{6}+\tfrac{1}{12}\big)=3\Rightarrow V_1\cdot\tfrac{3}{12}=3\).
2\(V_1 = 3 \times 4 = 12\,\text{V}\).
3\(I_{6}=12/6=2\,\text{A}\); \(I_{12}=12/12=1\,\text{A}\). Check \(2+1=3\,\text{A}\) ✓.
ANSWER \(V_1 = 12\,\text{V}\), \(I_6 = 2\,\text{A}\), \(I_{12} = 1\,\text{A}\).
Quick PracticeTap to reveal worked answers
1A circuit has 4 nodes. How many node-voltage equations are needed?▶
3 equations (\(N-1\)). One node is the reference at 0 V.
2In KCL at node 1, how do you write the current leaving node 1 through R toward node 2?▶
\((V_1-V_2)/R\). Current leaves the higher-labelled potential toward the lower; the sign sorts itself out.
35 A enters node 1; two resistors 5 Ω and 20 Ω run from node 1 to ground. Find \(V_1\).▶
\(V_1 = 20\,\text{V}\). \(V_1(\tfrac15+\tfrac1{20})=5\Rightarrow V_1\cdot\tfrac{5}{20}=5\Rightarrow V_1=20\,\text{V}\).
4Why choose the node with the most branches as reference?▶
It maximises the number of terms that become simply \(V_k/R\) (referenced to 0 V), simplifying the equations the most.

Superposition
In a linear circuit, the effect of several sources equals the sum of their individual effects.
Superposition is one of the most elegant ideas in circuit theory and a direct consequence of linearity (Ohm's Law is a straight line: \(V=IR\)). It says you can "turn on" one source at a time, find that source's contribution to the current or voltage you want, and then add all the contributions. It turns a multi-source headache into several single-source problems you already know how to solve — and it gives deep insight into how much each source actually contributes.
DefinitionThe superposition theorem▶

In any linear network with two or more independent sources, the current (or voltage) in any element equals the algebraic sum of the currents (or voltages) produced by each source acting alone, with all other independent sources set to zero.

Setting a source to zero: a voltage source → replace with a short circuit (0 V). A current source → replace with an open circuit (0 A). Internal resistances stay in place.
Power does NOT superpose. Because \(P=I^2R\) is non-linear, you must find the total current first and then compute power. Never add individual power contributions.
PatternSuperposition procedure▶
1Pick one source; turn all other independent sources to zero (V-source → short, I-source → open).
2Solve the resulting single-source circuit for the quantity of interest. Note its direction/sign.
3Repeat for every source in turn.
4Add the contributions algebraically (mind the signs/directions).
Worked Example 8 — two voltage sources

A 10 Ω resistor sits between two sources: \(E_1 = 20\,\text{V}\) through \(5\,\Omega\) on the left and \(E_2 = 10\,\text{V}\) through \(5\,\Omega\) on the right (a classic bridge-of-two arrangement, all returning to a common ground). Find the current through the 10 Ω using superposition.

1\(E_1\) alone (\(E_2\) shorted): the 10 Ω is in parallel with the right 5 Ω → \(10\parallel5=3.33\,\Omega\); total from \(E_1\): \(5+3.33=8.33\,\Omega\); \(I_{source}=20/8.33=2.4\,\text{A}\); by CDR the part through 10 Ω: \(I'_{10}=2.4\times\frac{5}{15}=0.8\,\text{A}\).
2\(E_2\) alone (\(E_1\) shorted): by symmetry total \(=5+ (10\parallel5)=8.33\,\Omega\); \(I_{source}=10/8.33=1.2\,\text{A}\); through 10 Ω: \(I''_{10}=1.2\times\frac{5}{15}=0.4\,\text{A}\) (same direction as \(I'\)).
3Add: \(I_{10}=0.8+0.4=1.2\,\text{A}\).
ANSWER \(I_{10} \approx 1.2\,\text{A}\) (sum of the two contributions).
Worked Example 9 — a voltage and a current source

A 4 Ω resistor is fed by \(E = 12\,\text{V}\) (in series with 2 Ω) and a parallel 3 A current source. Find \(V_{4\Omega}\).

1E alone (3 A source → open): \(V'_{4}=12\times\frac{4}{2+4}=8\,\text{V}\) (voltage divider).
23 A alone (E → short): the 3 A sees \(2\parallel4=1.33\,\Omega\); but for the 4 Ω voltage, current through 4 Ω \(=3\times\frac{2}{2+4}=1\,\text{A}\), so \(V''_4=1\times4=4\,\text{V}\).
3Add (same polarity): \(V_4 = 8+4 = 12\,\text{V}\).
ANSWER \(V_{4\Omega} = 12\,\text{V}\).
Quick PracticeTap to reveal worked answers
1How do you "switch off" a voltage source? A current source?▶
Voltage source → short circuit. Current source → open circuit. (Keep any internal resistance.)
2Can superposition be used to find power directly? Why/why not?▶
No. Power \(\propto I^2\) is non-linear. Find total \(I\) or \(V\) first, then compute \(P\).
3Source 1 gives +0.5 A through R; source 2 gives −0.2 A (opposite direction). Net current?▶
+0.3 A in the direction of source 1's contribution. \(0.5+(-0.2)=0.3\,\text{A}\).
4What single property of the circuit makes superposition valid?▶
Linearity. All elements obey straight-line relations (\(V=IR\)), so responses scale and add.
Topic 5 · Week 5 · Boylestad Ch 9.3-9.5
Thévenin, Norton & Maximum Power
Reduce any complicated network to a single source and single resistor — the most powerful idea in this guide.
Thévenin's theorem is the crown jewel of DC analysis. It says any linear two-terminal network — however many sources and resistors — looks, from the outside, like a single voltage source \(E_{Th}\) in series with a single resistance \(R_{Th}\). This is exactly what you want when you have one load that keeps changing (a speaker, a variable sensor, a battery being charged): solve the network once, then any load value is a trivial series calculation. Norton is its current-source twin, and maximum power transfer falls out for free.
PatternThe 3-step Thévenin procedure▶

To find the Thévenin equivalent seen by a load across terminals a-b:

1Remove the load at a-b (leave the terminals open).
2Find \(R_{Th}\): turn off all independent sources (V → short, I → open) and compute the resistance looking back into terminals a-b.
3Find \(E_{Th}\): restore the sources and compute the open-circuit voltage across a-b. This is \(E_{Th}\) (also written \(V_{Th}\) or \(V_{oc}\)).

Then the original network = \(E_{Th}\) in series with \(R_{Th}\). For any load \(R_L\): \( I_L = \dfrac{E_{Th}}{R_{Th}+R_L}\).

Worked Example 10 — formula-sheet Thévenin example

A network has a 10 Ω and 40 Ω forming a divider from a source, with a 20 Ω in series to terminal a-b. The course formula sheet gives \(R_{Th}\) directly.

1\(R_{Th}\): short the source. The 10 Ω and 40 Ω become parallel, in series with the 20 Ω:
\[ R_{Th} = \frac{10\times40}{10+40} + 20 = \frac{400}{50}+20 = 8 + 20 = 28\,\Omega \]
2\(E_{Th}\): open-circuit voltage at a-b. With the load removed no current flows in the 20 Ω, so \(E_{Th}\) equals the voltage across the 40 Ω, found by the divider / nodal value \(V_1\):
\[ E_{Th} = V_1 = 32\,\text{V} \quad(\text{from the divider/nodal solution}) \]
ANSWER \(R_{Th} = 28\,\Omega\), \(E_{Th} = 32\,\text{V}\).
Note the structure: the 20 Ω in series carries no current at open-circuit, so it does not affect \(E_{Th}\) — but it does add directly to \(R_{Th}\). Spotting this saves time.
Worked Example 11 — using the equivalent for a load

Using the equivalent above (\(E_{Th}=32\,\text{V}\), \(R_{Th}=28\,\Omega\)), find the load current and load voltage when \(R_L = 12\,\Omega\).

1\(I_L = \dfrac{E_{Th}}{R_{Th}+R_L} = \dfrac{32}{28+12} = \dfrac{32}{40} = 0.8\,\text{A}\).
2\(V_L = I_L R_L = 0.8 \times 12 = 9.6\,\text{V}\).
ANSWER \(I_L = 0.8\,\text{A}\), \(V_L = 9.6\,\text{V}\). Change \(R_L\) and just repeat this one line — that's the power of Thévenin.
DefinitionNorton's theorem (the dual)▶

Norton's theorem reduces the same network to an ideal current source \(I_N\) in parallel with \(R_N\). The relationships:

\[ R_N = R_{Th} \qquad I_N = \frac{E_{Th}}{R_{Th}} = I_{sc}\ (\text{short-circuit current}) \]

Thévenin and Norton are related by exactly the source-conversion rule from Topic 2. Find whichever is easier; convert if needed.

PatternMaximum power transfer▶

A load draws maximum power from a network when the load resistance equals the Thévenin resistance of the source:

\[ R_L = R_{Th} \qquad\Longrightarrow\qquad P_{max} = \frac{E_{Th}^{\,2}}{4R_{Th}} \]

At this match the load voltage is exactly half of \(E_{Th}\), and the efficiency is 50% (half the power is lost in \(R_{Th}\)). This rule designs antennas, audio amplifiers and impedance-matching networks.

Max power transfer ≠ max efficiency. For power delivery match \(R_L=R_{Th}\); for energy efficiency (e.g. power grids) you deliberately keep \(R_L \gg R_{Th}\).
Worked Example 12 — maximum power

For the equivalent \(E_{Th}=32\,\text{V}\), \(R_{Th}=28\,\Omega\), find the load for maximum power and that maximum power.

1Match: \(R_L = R_{Th} = 28\,\Omega\).
2\(P_{max} = \dfrac{E_{Th}^2}{4R_{Th}} = \dfrac{32^2}{4\times28} = \dfrac{1024}{112} = 9.14\,\text{W}\).
ANSWER \(R_L = 28\,\Omega\), \(P_{max} \approx 9.14\,\text{W}\).

2\(E_{Th}=24\,\text{V}\), \(R_{Th}=6\,\Omega\). Find \(I_L\) for \(R_L=6\,\Omega\).▶
