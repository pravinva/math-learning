Numerical Integration: The Trapezoidal Rule & Simpson's Rule

Not every integrand has an elementary antiderivative. Numerical integration approximates the area under a curve directly — first with straight-line segments (trapezoids), then with a dramatically more accurate parabola-fitting approach (Simpson's Rule). Exam-level worked examples and practice throughout. Solutions stay hidden until you click.

Lay of the land — this week's toolkit
  • Trapezoidal Rule: $T_n=\dfrac{h}{2}\big[f(x_0)+2f(x_1)+2f(x_2)+\cdots+2f(x_{n-1})+f(x_n)\big]$, $h=\dfrac{b-a}{n}$.
  • Trapezoidal error bound: $|E_T|\le\dfrac{(b-a)h^2}{12}\max|f''(x)|$ — error $\propto h^2$.
  • Simpson's Rule (requires $n$ even): $S_n=\dfrac{h}{3}\big[f(x_0)+4f(x_1)+2f(x_2)+4f(x_3)+\cdots+4f(x_{n-1})+f(x_n)\big]$.
  • Simpson's error bound: $|E_S|\le\dfrac{(b-a)h^4}{180}\max|f^{(4)}(x)|$ — error $\propto h^4$, so halving $h$ cuts the error by $16\times$, not just $4\times$.
  • Key fact: Simpson's Rule is exact for any polynomial of degree $\le3$ (its fourth derivative is zero), which is why it usually massively outperforms the trapezoidal rule for the same $n$.

0 The Trapezoidal Rule & Error Estimation

Split $[a,b]$ into $n$ equal subintervals of width $h=\dfrac{b-a}{n}$. On each subinterval, approximate the area under the curve by a trapezoid (rather than a rectangle) — using the straight line connecting $(x_{i-1},f(x_{i-1}))$ and $(x_i,f(x_i))$ as the top edge. A single trapezoid's area is $\dfrac{h}{2}\big(f(x_{i-1})+f(x_i)\big)$.

Summing over all $n$ trapezoids, each interior function value gets counted twice (once as the right edge of one trapezoid, once as the left edge of the next), while the two endpoints are counted once each — giving the $1,2,2,\dots,2,1$ coefficient pattern.

Error. Since each trapezoid replaces a curved top with a straight line, the error depends on how much the function curves (its second derivative). For a concave-up function, the chord sits above the curve, so the trapezoidal rule systematically overestimates such integrals; for concave-down, it underestimates.

Key idea

$$T_n = \frac{h}{2}\Big[f(x_0)+2f(x_1)+\cdots+2f(x_{n-1})+f(x_n)\Big], \qquad |E_T|\le\frac{(b-a)h^2}{12}\max|f''(x)|$$

Worked example

Example. Estimate $\displaystyle\int_0^2 x^2\,dx$ using $n=4$ trapezoids, and bound the error (given $f''(x)=2$).

$h=0.5$; $x_0,\dots,x_4=0,0.5,1,1.5,2$; $f$-values: $0,0.25,1,2.25,4$.

$T_4=\dfrac{0.5}{2}\big[0+2(0.25)+2(1)+2(2.25)+4\big]=0.25(11)=2.75$ (true value $\tfrac83\approx2.667$).

$|E_T|\le\dfrac{2(0.5)^2(2)}{12}=\dfrac{1}{12}\approx0.083$ (actual error was $2.75-2.667=0.083$ — matches almost exactly).

Practice 1

Estimate $\displaystyle\int_0^{\pi} \sin x\,dx$ using the Trapezoidal Rule with $n=6$, and compare with the exact value $2$.

Solution

$h=\pi/6\approx0.5236$; $x_i=0,\pi/6,\pi/3,\pi/2,2\pi/3,5\pi/6,\pi$.

$f$-values: $0, 0.5, 0.8660, 1, 0.8660, 0.5, 0$.

$T_6=\dfrac{h}{2}\big[0+2(0.5+0.8660+1+0.8660+0.5)+0\big]=\dfrac{0.5236}{2}\big[2(3.732)\big]=\dfrac{0.5236}{2}(7.464)=1.954$.

Answer: $T_6\approx1.954$, slightly below the exact value of $2$ (as expected: $\sin x$ is concave-down on this interval, so the trapezoidal rule underestimates).

Practice 2

A function $f$ has $\max|f''(x)|=8$ on $[1,5]$. Find the minimum number of subintervals $n$ (even count not required here) needed to guarantee the Trapezoidal Rule error is less than $0.001$.

Solution

$|E_T|\le\dfrac{(b-a)h^2}{12}\max|f''|=\dfrac{4h^2(8)}{12}=\dfrac{32h^2}{12}=\dfrac{8h^2}{3}$.

Require $\dfrac{8h^2}{3}<0.001 \implies h^2<\dfrac{0.003}{8}=3.75\times10^{-4} \implies h<0.01936$.

Since $h=(b-a)/n=4/n$: $n>4/0.01936\approx206.6$, so $n=207$ (or round up further for a convenient value).

Answer: $n\ge207$.

Practice 3

Estimate $\displaystyle\int_1^3 \ln x\,dx$ using $n=4$ trapezoids (to 4 decimal places), and compare with the exact value $3\ln3-2\approx1.2958$.

Solution

$h=0.5$; $x_i=1,1.5,2,2.5,3$; $f$-values: $0, 0.4055, 0.6931, 0.9163, 1.0986$.

$T_4=\dfrac{0.5}{2}\big[0+2(0.4055+0.6931+0.9163)+1.0986\big]=0.25\big[2(2.0149)+1.0986\big]=0.25(5.1284)=1.2821$.

Answer: $T_4\approx1.2821$, slightly under the exact value $1.2958$ (consistent with $\ln x$ being concave-down, so trapezoidal underestimates).

1 Simpson's Rule & Error Estimation

The trapezoidal rule fits a straight line across each pair of points. Simpson's Rule instead fits a parabola through each set of three consecutive points, using its area as the approximation instead of two trapezoids — this requires $n$ to be even (so the points pair up completely).

The parabola-fitting calculation produces the weighting pattern $1,4,2,4,2,\dots,4,1$ — odd-indexed points get weight $4$, even-indexed interior points get weight $2$.

Error. Since Simpson's Rule matches any parabola exactly, the error depends on the fourth derivative rather than the second. Because the error bound uses $h^4$ rather than $h^2$, halving $h$ shrinks the error by a factor of $16$, not just $4$ — this is why Simpson's Rule usually converges far faster than the trapezoidal rule.

Key idea

$$S_n = \frac{h}{3}\Big[f(x_0)+4f(x_1)+2f(x_2)+4f(x_3)+\cdots+4f(x_{n-1})+f(x_n)\Big], \quad n\text{ even}$$ $$|E_S|\le\frac{(b-a)h^4}{180}\max|f^{(4)}(x)|$$

Worked example

Example. Estimate $\displaystyle\int_0^2 x^2\,dx$ using Simpson's Rule with $n=4$ (same data as the trapezoidal example).

$h=0.5$; $f$-values: $0,0.25,1,2.25,4$; weights: $1,4,2,4,1$.

$S_4=\dfrac{0.5}{3}\big[0+4(0.25)+2(1)+4(2.25)+4\big]=\dfrac{0.5}{3}(16)=\dfrac83$ — exactly matches the true value, since $x^2$ is itself a parabola (Simpson's Rule is exact for polynomials up to degree 3).

Practice 4

Estimate $\displaystyle\int_0^{\pi} \sin x\,dx$ using Simpson's Rule with $n=6$ (same nodes as the trapezoidal practice question above), and compare with the exact value $2$.

Solution

$h=\pi/6$; $f$-values: $0, 0.5, 0.8660, 1, 0.8660, 0.5, 0$; weights: $1,4,2,4,2,4,1$.

$S_6=\dfrac{\pi/6}{3}\big[0+4(0.5)+2(0.8660)+4(1)+2(0.8660)+4(0.5)+0\big]$

$=\dfrac{\pi}{18}\big[2+1.732+4+1.732+2\big]=\dfrac{\pi}{18}(11.464)=2.0002$.

Answer: $S_6\approx2.0002$ — dramatically closer to the exact value $2$ than the trapezoidal estimate ($1.954$), consistent with Simpson's Rule's much smaller error for the same $n$.

Practice 5

A function $f$ satisfies $\max|f^{(4)}(x)|=16$ on $[0,4]$. Compare the Simpson's Rule error bound for $n=4$ against $n=8$, and verify the factor-of-16 improvement.

Solution

For $n=4$: $h=1$. $|E_S|\le\dfrac{4(1)^4(16)}{180}=\dfrac{64}{180}=0.3556$.

For $n=8$: $h=0.5$. $|E_S|\le\dfrac{4(0.5)^4(16)}{180}=\dfrac{4(0.0625)(16)}{180}=\dfrac{4}{180}=0.0222$.

Ratio: $0.3556/0.0222=16.0$ — exactly the factor of $16$ predicted, since halving $h$ divides $h^4$ by $2^4=16$.

Answer: error bounds are $0.3556$ ($n=4$) and $0.0222$ ($n=8$); ratio $=16$, confirming the $h^4$ scaling.

Practice 6

Estimate $\displaystyle\int_1^3 \ln x\,dx$ using Simpson's Rule with $n=4$ (same nodes as the earlier trapezoidal question), and compare both estimates against the exact value $3\ln3-2\approx1.2958$.

Solution

$h=0.5$; $f$-values: $0, 0.4055, 0.6931, 0.9163, 1.0986$; weights: $1,4,2,4,1$.

$S_4=\dfrac{0.5}{3}\big[0+4(0.4055)+2(0.6931)+4(0.9163)+1.0986\big]$

$=\dfrac{0.5}{3}\big[1.622+1.3862+3.6652+1.0986\big]=\dfrac{0.5}{3}(7.772)=1.2953$.

Answer: $S_4\approx1.2953$, compared with the trapezoidal $T_4\approx1.2821$ and exact value $1.2958$ — Simpson's estimate is roughly $14\times$ closer to the true value than the trapezoidal estimate, despite using the same 5 function evaluations.

Practice 7

Explain why applying Simpson's Rule to $n=5$ subintervals is not directly possible, and describe two valid workarounds.

Solution

Simpson's Rule pairs subintervals into groups of two, fitting one parabola across each pair — this requires the total count $n$ to be even so every subinterval belongs to exactly one pair. With $n=5$ (odd), one subinterval is left over with no partner.

Workaround 1: use $n=6$ instead (round up to the nearest even number), recomputing function values at the extra node.

Workaround 2: apply Simpson's Rule to the first 4 (even) subintervals, then use the Trapezoidal Rule (or another method) on the single remaining subinterval, combining the two partial results — a valid but less commonly used hybrid approach.

Practice 8

Use the Trapezoidal Rule with $n=8$ to estimate $\displaystyle\int_0^1 e^{-x^2}\,dx$.

Solution

$h=0.125$, nodes $x_i=i/8$.

Using $f(x)=e^{-x^2}$ values, $T_8\approx0.7462$.

Answer: $T_8\approx0.7462$.

Practice 9

Use Simpson's Rule with $n=8$ to estimate $\displaystyle\int_0^1 e^{-x^2}\,dx$, then compare with Practice 8.

Solution

Same nodes as Practice 8, Simpson weights $1,4,2,\dots,4,1$.

$S_8\approx0.7468$, closer than $T_8$ for same node set.

Answer: $S_8\approx0.7468$ (better than $T_8\approx0.7462$).

Practice 10

Find the smallest even $n$ guaranteeing Simpson error below $10^{-4}$ for $\int_0^2 \ln(1+x)\,dx$, given $\max|f^{(4)}(x)|=\frac{6}{(1+x)^4}\le 6$.

Solution

$|E_S|\le \dfrac{(b-a)h^4}{180}M=\dfrac{2h^4}{180}\cdot6=\dfrac{h^4}{15}$.

Need $h^4/15<10^{-4}\Rightarrow h<0.1968$.

$h=2/n$, so $n>10.16$; smallest even $n$ is $12$.

Practice 11

Apply Simpson's Rule with $n=4$ to $\displaystyle\int_0^1 \frac{1}{1+x^2}\,dx$ and compare with exact $\arctan(1)=\pi/4$.

Solution

$h=0.25$, values at $0,0.25,0.5,0.75,1$: $1,0.9412,0.8,0.64,0.5$.

$S_4=\frac{0.25}{3}[1+4(0.9412)+2(0.8)+4(0.64)+0.5]\approx0.7854$.

Exact is $\pi/4\approx0.7854$; extremely close.

Practice 12

Explain, with one concrete polynomial example, why Simpson's Rule is exact for degree 3 but generally not for degree 4.

Solution

Error term contains $f^{(4)}$. For cubic, $f^{(4)}=0$, so error bound is zero.

Example: $f(x)=x^3$ gives exact Simpson value. But for $f(x)=x^4$, $f^{(4)}=24\neq0$, so nonzero error generally occurs.

Practice 13

Use Trapezoidal Rule with $n=10$ to estimate $\int_0^2 \frac{1}{1+x}\,dx$, and state whether it over- or underestimates.

Solution

$h=0.2$, compute table values $f(x_i)=1/(1+x_i)$.

Numerically $T_{10}\approx1.1029$.

Since $f''(x)=2/(1+x)^3>0$ (concave up), trapezoidal overestimates. Exact is $\ln3\approx1.0986$.

Practice 14

Use Simpson's Rule with $n=10$ for the same integral in Practice 13 and compare errors.

Solution

Using same nodes and Simpson weights gives $S_{10}\approx1.0986$ (to 4 d.p.).

Error is far smaller than trapezoidal, consistent with $h^4$ vs $h^2$ scaling.

Practice 15

An engineer has function values at 7 equally spaced nodes. Which rule (Trapezoidal/Simpson) can be applied directly over the full interval? Justify using subinterval count.

Solution

7 nodes means 6 subintervals. Simpson needs even $n$; here $n=6$ so Simpson applies directly.

Trapezoidal always applies too, but Simpson is usually preferred for accuracy at same nodes.

Practice 16

Compute $T_4$ and $S_4$ for $\int_0^2 x^3\,dx$, and comment on exactness.

Solution

Exact integral $=4$.

$T_4=4.25$ (not exact). $S_4=4.00$ exactly.

Confirms Simpson exactness for cubic polynomials.

Practice 17

Given $|E_T|\le \frac{K}{n^2}$ and current trapezoidal error tolerance uses $n=40$, estimate required $n$ to make tolerance 9 times tighter.

Solution

Error scales like $1/n^2$. To reduce error by factor 9, multiply $n$ by $\sqrt{9}=3$.

Required $n\approx120$.

Practice 18

Given $|E_S|\le \frac{C}{n^4}$ and Simpson currently uses $n=8$, estimate required $n$ to make tolerance 16 times tighter.

Solution

Need error factor $16$, so $n$ scales by $16^{1/4}=2$.

Required $n=16$ (and still even).

Practice 19

Estimate $\int_0^1 \sqrt{1+x}\,dx$ with $T_6$, then use concavity to state whether the estimate is above or below the exact value.

Solution

Using $h=1/6$, computation gives $T_6\approx1.2199$.

$f''(x)=-\frac{1}{4}(1+x)^{-3/2}<0$ so concave down; trapezoidal underestimates.

Practice 20

A student applies Simpson with weights $1,2,4,2,\dots,4,1$. Explain the structural mistake and consequence.

Solution

Simpson weights must alternate $4,2,4,2,\dots$ on interior nodes starting with $4$ at $x_1$.

Swapping the first interior weight breaks the quadratic interpolation derivation and produces a biased estimate.

Practice 21

Challenge: For fixed node count, why is Simpson often preferred over trapezoidal for smooth functions, and when might trapezoidal still be chosen in practice?

Solution

Simpson has fourth-order error ($h^4$) versus second-order ($h^2$), so accuracy improves much faster as grid is refined.

Trapezoidal may still be chosen for streaming data, odd subinterval counts, simpler implementation, or when function noise makes higher-order fitting less reliable.
