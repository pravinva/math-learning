Lay of the land — this week's toolkit
  • Integration by parts: $\int u\,dv=uv-\int v\,du$ (LIATE for choosing $u$; tabular method for repeats; "solve for $I$" for $e^x\sin x$-type integrals).
  • Trig integrals: odd power → peel + Pythagorean identity; both even → half-angle; mixed angles → product-to-sum.
  • Trig substitution: $\sqrt{a^2-x^2}\to x=a\sin\theta$; $\sqrt{a^2+x^2}\to x=a\tan\theta$; $\sqrt{x^2-a^2}\to x=a\sec\theta$.
  • Partial fractions: linear $\dfrac{A}{x-a}$ (cover-up); repeated $\sum\dfrac{A_k}{(x-a)^k}$; irreducible quadratic $\dfrac{Bx+C}{x^2+bx+c}$.
  • Improper integrals: $\int_a^\infty f\,dx=\lim_{t\to\infty}\int_a^tf\,dx$; $\int_1^\infty x^{-p}dx$ converges iff $p>1$; near a finite point, $\int_0^1x^{-k}dx$ converges iff $k<1$ (opposite condition — a common exam trap).
  • Numerical integration: $T_n=\dfrac h2[f_0+2f_1+\cdots+2f_{n-1}+f_n]$ (error $\propto h^2$); $S_n=\dfrac h3[f_0+4f_1+2f_2+\cdots+4f_{n-1}+f_n]$, $n$ even (error $\propto h^4$, exact for cubics).
  • Polar: $x=r\cos\theta,y=r\sin\theta$; area $A=\tfrac12\int r^2\,d\theta$; rose $r=a\cos(n\theta)$: $n$ odd → $n$ petals, $n$ even → $2n$ petals.
  • Volumes: disk/washer $V=\pi\int([R]^2-[r]^2)dx$; shell $V=2\pi\int xf(x)\,dx$.
  • Arc length: $L=\int_a^b\sqrt{1+[f'(x)]^2}\,dx$.
  • Exam strategy: identify the technique from the integrand's shape before writing anything — a square root of $a^2\pm x^2$ signals trig substitution; a rational function signals partial fractions; a product of unrelated function types signals integration by parts.

0 Recap: Integration by Parts, Trig Integrals & Substitution

The exam-critical skill across all three of these techniques is recognising the shape before diving in: a product of unlike function types (polynomial $\times$ trig/exponential/log) signals integration by parts; a pure power/product of $\sin x,\cos x$ signals the Pythagorean/half-angle toolkit; a square root of $a^2\pm x^2$ or $x^2-a^2$ signals trig substitution.

Key idea

$$\int u\,dv=uv-\int v\,du, \qquad \sin^2x=\frac{1-\cos2x}{2},\ \cos^2x=\frac{1+\cos2x}{2}$$

Worked example

Exam-style. Evaluate $\displaystyle\int x^2e^{-x}\,dx$ using the tabular method, then evaluate $\displaystyle\int_0^{\pi/2}\cos^3x\,dx$.

Tabular: differentiate $x^2\to2x\to2\to0$; integrate $e^{-x}\to-e^{-x}\to e^{-x}\to-e^{-x}$: $\int x^2e^{-x}dx=-x^2e^{-x}-2xe^{-x}-2e^{-x}+C$.

$\cos^3x=\cos x(1-\sin^2x)$; $u=\sin x$: $\sin x-\tfrac13\sin^3x$. At $\pi/2$: $1-\tfrac13=\tfrac23$. At $0$: $0$. Answer: $\tfrac23$.

Practice 1

Evaluate $\displaystyle\int x\ln(2x)\,dx$.

Solution

$u=\ln(2x),dv=x\,dx$: $du=\tfrac1x dx,v=\tfrac{x^2}{2}$.

$\dfrac{x^2}{2}\ln(2x)-\displaystyle\int\dfrac{x}{2}\,dx=\dfrac{x^2}{2}\ln(2x)-\dfrac{x^2}{4}+C$.

Answer: $\dfrac{x^2}{2}\ln(2x)-\dfrac{x^2}{4}+C$.

Practice 2

Evaluate $\displaystyle\int \sin^2x\cos^4x\,dx$ (both powers even).

Solution

$\sin^2x\cos^4x=(\sin x\cos x)^2\cos^2x=\left(\tfrac12\sin2x\right)^2\cdot\dfrac{1+\cos2x}{2}=\dfrac{\sin^22x}{4}\cdot\dfrac{1+\cos2x}{2}$.

$=\dfrac{\sin^22x}{8}+\dfrac{\sin^22x\cos2x}{8}$. First term: half-angle again, $\dfrac{1-\cos4x}{16}$. Second: substitution $u=\sin2x$ gives $\dfrac{\sin^32x}{24}$.

Answer: $\dfrac{x}{16}-\dfrac{\sin4x}{64}+\dfrac{\sin^32x}{24}+C$.

Practice 3

Evaluate $\displaystyle\int \frac{x^2}{\sqrt{16-x^2}}\,dx$.

Solution

$x=4\sin\theta$: $\displaystyle\int\dfrac{16\sin^2\theta}{4\cos\theta}\cdot4\cos\theta\,d\theta=16\int\sin^2\theta\,d\theta=8\theta-8\sin\theta\cos\theta+C$.

Converting back: $\theta=\arcsin(x/4)$; $\sin\theta\cos\theta=\dfrac{x\sqrt{16-x^2}}{16}$.

Answer: $8\arcsin\!\left(\dfrac{x}{4}\right)-\dfrac{x\sqrt{16-x^2}}{2}+C$.

Practice 4

Evaluate $\displaystyle\int_0^1 x^2e^x\,dx$.

Solution

Tabular: $x^2e^x-2xe^x+2e^x$. At $x=1$: $e-2e+2e=e$. At $x=0$: $0-0+2=2$.

Answer: $e-2$.

Practice 5

Evaluate $\displaystyle\int \sec^3x\tan x\,dx$ (recognise the derivative pattern).

Solution

Let $u=\sec x,du=\sec x\tan x\,dx$: rewrite as $\int\sec^2x\cdot(\sec x\tan x\,dx)=\int u^2\,du=\dfrac{u^3}{3}+C$.

Answer: $\dfrac{\sec^3x}{3}+C$.

1 Recap: Partial Fractions & Improper Integrals

For partial fractions, the exam-critical steps are: check the fraction is proper (divide first if not), factor the denominator completely, then match the term type (linear → constant numerator; repeated → one term per power; irreducible quadratic → linear numerator) to each factor. For improper integrals, always identify where the trouble is (infinite bound, or a point of discontinuity inside the interval) before wrapping it in a limit.

Key idea

$$\frac{P(x)}{(x-a)^n(x^2+bx+c)} \to \sum_{k=1}^n\frac{A_k}{(x-a)^k}+\frac{Bx+C}{x^2+bx+c}, \qquad \int_a^\infty f\,dx=\lim_{t\to\infty}\int_a^tf\,dx$$

Worked example

Exam-style. Evaluate $\displaystyle\int \frac{3x+1}{(x-1)(x^2+1)}\,dx$, then determine whether $\displaystyle\int_1^\infty\frac{3x+1}{(x-1)(x^2+1)}dx$ converges (before evaluating it, using a comparison).

$3x+1=A(x^2+1)+(Bx+C)(x-1)$. At $x=1$: $4=2A\implies A=2$. Match $x^2$: $0=A+B\implies B=-2$. Match constant: $1=A-C\implies C=1$.

For large $x$, the integrand behaves like $\dfrac{3x}{x^3}=\dfrac{3}{x^2}$, and $\int_1^\infty x^{-2}dx$ converges ($p=2>1$) — by limit comparison, the given integral converges too (the singularity exactly at $x=1$ needs separate care, but the question as posed starts the interval there, which is itself a discontinuity requiring a one-sided limit check first).

Practice 6

Decompose $\dfrac{4x^2-3x+2}{x^3-x^2}$ (factor as $x^2(x-1)$; improper fraction check first).

Solution

Numerator degree (2) $<$ denominator degree (3): already proper, no division needed.

$\dfrac{4x^2-3x+2}{x^2(x-1)}=\dfrac{A}{x}+\dfrac{B}{x^2}+\dfrac{C}{x-1}$. $4x^2-3x+2=Ax(x-1)+B(x-1)+Cx^2$.

At $x=0$: $2=-B\implies B=-2$. At $x=1$: $3=C$. Match $x^2$: $4=A+C\implies A=1$.

Answer: $\dfrac{1}{x}-\dfrac{2}{x^2}+\dfrac{3}{x-1}$.

Practice 7

Evaluate $\displaystyle\int_{0}^{1} \frac{1}{\sqrt{1-x}}\,dx$ (identify the discontinuity location first).

Solution

Discontinuity at $x=1$ (upper bound). $\displaystyle\lim_{t\to1^-}\int_0^t(1-x)^{-1/2}dx=\lim_{t\to1^-}\big[-2(1-x)^{1/2}\big]_0^t=\lim_{t\to1^-}\big(-2\sqrt{1-t}+2\big)=2$.

Answer: converges to $2$.

Practice 8

Evaluate $\displaystyle\int \frac{5}{x^2+2x+5}\,dx$ (complete the square first — irreducible quadratic with no linear factor to split off).

Solution

$x^2+2x+5=(x+1)^2+4$.

$\displaystyle\int\frac{5}{(x+1)^2+4}\,dx = 5\cdot\frac12\arctan\!\left(\frac{x+1}{2}\right)+C$.

Answer: $\dfrac52\arctan\!\left(\dfrac{x+1}{2}\right)+C$.

Practice 9

Determine whether $\displaystyle\int_2^\infty \frac{1}{x\sqrt{x^2-1}}\,dx$ converges, using a limit comparison with $\dfrac{1}{x^2}$.

Solution

$L=\displaystyle\lim_{x\to\infty}\frac{1/(x\sqrt{x^2-1})}{1/x^2}=\lim_{x\to\infty}\frac{x^2}{x\sqrt{x^2-1}}=\lim_{x\to\infty}\frac{x}{\sqrt{x^2-1}}=1$ (finite, positive).

Since $\int_2^\infty x^{-2}dx$ converges, by limit comparison the given integral converges too.

Answer: converges.

Practice 10

Evaluate $\displaystyle\int_1^3 \frac{x-1}{(x+1)(x-2)}\,dx$, being careful about the discontinuity inside the interval.

Solution

Discontinuity at $x=2$, strictly inside $[1,3]$: must split there.

Decompose: $\dfrac{x-1}{(x+1)(x-2)}=\dfrac{A}{x+1}+\dfrac{B}{x-2}$. At $x=-1$: $-2=-3A\implies A=\tfrac23$. At $x=2$: $1=3B\implies B=\tfrac13$.

$\displaystyle\int_2^3\dfrac{1/3}{x-2}dx=\lim_{t\to2^+}\tfrac13\big[\ln|x-2|\big]_t^3$; as $t\to2^+$, $\ln|t-2|\to-\infty$, so $-\tfrac13\ln|t-2|\to+\infty$ — this piece diverges.

Answer: diverges (the discontinuity at $x=2$ makes the integral improper and divergent, regardless of the other piece).

2 Recap: Numerical Integration, Polar Coordinates, Volumes & Arc Length

These four topics share the exam skill of correct setup: numerical integration needs the right $h$ and node values; polar area/volumes need the correct bounds and outer/inner identification; arc length needs $f'(x)$ correctly squared and simplified before integrating.

Key idea

$$S_n=\frac h3[f_0+4f_1+2f_2+\cdots+f_n], \quad A=\frac12\int_\alpha^\beta r^2d\theta, \quad V=2\pi\int xf(x)dx, \quad L=\int\sqrt{1+[f']^2}dx$$

Worked example

Exam-style. Estimate $\displaystyle\int_0^2 e^{x^2}dx$ using Simpson's Rule with $n=4$, then find the arc length of $y=\tfrac13x^{3/2}\cdot2$ ... consider instead $y=\tfrac23x^{3/2}$ from $0$ to $1$.

$h=0.5$; $f$-values: $1,1.284,54.6,\dots$ (this function grows extremely fast — illustrating why Simpson's Rule, not an elementary antiderivative, is the practical tool here). Setting up correctly (weights $1,4,2,4,1$) is the graded skill, even before crunching the (large) numbers.

Arc length: $f'(x)=x^{1/2}$; $L=\int_0^1\sqrt{1+x}\,dx=\left[\tfrac23(1+x)^{3/2}\right]_0^1=\tfrac23(2\sqrt2-1)$.

Practice 11

Estimate $\displaystyle\int_1^5 \frac{1}{x}\,dx$ using the Trapezoidal Rule with $n=4$, and compare with the exact value $\ln5\approx1.609$.

Solution

$h=1$; $x_i=1,2,3,4,5$; $f$-values: $1,0.5,0.333,0.25,0.2$.

$T_4=\dfrac12\big[1+2(0.5+0.333+0.25)+0.2\big]=\dfrac12\big[1+2.166+0.2\big]=\dfrac12(3.366)=1.683$.

Answer: $T_4\approx1.683$, somewhat above the exact value $1.609$ (consistent with $1/x$ being concave-up).

Practice 12

Find the area enclosed by $r=2-2\sin\theta$ (a cardioid).

Solution

$A=\tfrac12\displaystyle\int_0^{2\pi}(2-2\sin\theta)^2d\theta=\tfrac12\int_0^{2\pi}(4-8\sin\theta+4\sin^2\theta)d\theta$.

$\int_0^{2\pi}4\,d\theta=8\pi$; $\int_0^{2\pi}8\sin\theta\,d\theta=0$; $\int_0^{2\pi}4\sin^2\theta\,d\theta=4\pi$.

Sum: $8\pi-0+4\pi=12\pi$. $A=6\pi$.

Answer: $6\pi$.

Practice 13

The region under $y=\ln x$ from $x=1$ to $x=e$ is rotated about the $y$-axis. Find the volume using the shell method.

Solution

$V=2\pi\displaystyle\int_1^e x\ln x\,dx$. By parts ($u=\ln x,dv=x\,dx$): $\int x\ln x\,dx=\dfrac{x^2}{2}\ln x-\dfrac{x^2}{4}$.

At $x=e$: $\dfrac{e^2}{2}-\dfrac{e^2}{4}=\dfrac{e^2}{4}$. At $x=1$: $0-\dfrac14$.

$\int_1^e x\ln x\,dx = \dfrac{e^2}{4}+\dfrac14=\dfrac{e^2+1}{4}$.

Answer: $V=2\pi\cdot\dfrac{e^2+1}{4}=\dfrac{\pi(e^2+1)}{2}$.

Practice 14

Find the arc length of $y=\dfrac{x^2}{4}-\ln(\sqrt x)$ from $x=1$ to $x=2$ (design it as a perfect square first).

Solution

$f'(x)=\dfrac{x}{2}-\dfrac{1}{2x}$; $1+[f'(x)]^2=1+\dfrac{x^2}{4}-\dfrac12+\dfrac{1}{4x^2}=\dfrac{x^2}{4}+\dfrac12+\dfrac{1}{4x^2}=\left(\dfrac{x}{2}+\dfrac{1}{2x}\right)^2$.

$L=\displaystyle\int_1^2\left(\dfrac{x}{2}+\dfrac{1}{2x}\right)dx=\left[\dfrac{x^2}{4}+\dfrac{\ln x}{2}\right]_1^2=\left(1+\dfrac{\ln2}{2}\right)-\dfrac14=\dfrac34+\dfrac{\ln2}{2}$.

Answer: $\dfrac34+\dfrac{\ln2}{2}\approx1.097$.

Practice 15

A function $f$ satisfies $\max|f^{(4)}(x)|=10$ on $[0,2]$. Compare the guaranteed accuracy of Simpson's Rule with $n=4$ against the Trapezoidal Rule with $n=4$ (given $\max|f''(x)|=6$ on the same interval), and state which method you'd choose for a required error under $0.001$.

Solution

Trapezoidal, $n=4,h=0.5$: $|E_T|\le\dfrac{2(0.5)^2(6)}{12}=\dfrac{3}{12}=0.25$.

Simpson's, $n=4,h=0.5$: $|E_S|\le\dfrac{2(0.5)^4(10)}{180}=\dfrac{1.25}{180}\approx0.00694$.

Neither guarantees error under $0.001$ at $n=4$, but Simpson's bound is already roughly $36\times$ tighter than trapezoidal's at the same $n$ — Simpson's Rule is clearly the better choice, and would need only a modest increase in $n$ to meet the $0.001$ target, versus a much larger increase for the trapezoidal rule.

3 Mixed Exam-Style Problems (Weeks 1–6)

Real exam questions often combine two or more of this phase's techniques in one multi-part problem. These are deliberately mixed — identify which tool each part needs before diving into arithmetic.

Key idea

No single formula here — the skill being tested is recognising which technique applies at each stage, and correctly carrying results from one part into the next.

Worked example

Exam-style, multi-part. (a) Decompose $\dfrac{1}{x^2(x^2+1)}$. (b) Use this to evaluate $\displaystyle\int_1^\infty\frac{1}{x^2(x^2+1)}\,dx$.

(a) $\dfrac{1}{x^2(x^2+1)}=\dfrac{A}{x}+\dfrac{B}{x^2}+\dfrac{Cx+D}{x^2+1}$. Multiply through: $1=Ax(x^2+1)+B(x^2+1)+(Cx+D)x^2$. At $x=0$: $B=1$. Match $x^3$: $A+C=0$. Match $x^2$: $B+D=0\implies D=-1$. Match $x$: $A=0\implies C=0$. Result: $\dfrac{1}{x^2}-\dfrac{1}{x^2+1}$.

(b) $\displaystyle\int_1^t\left(\dfrac{1}{x^2}-\dfrac{1}{x^2+1}\right)dx=\Big[-\dfrac1x-\arctan x\Big]_1^t$. As $t\to\infty$: $-\tfrac1t\to0$, $\arctan t\to\tfrac{\pi}{2}$. At $x=1$: $-1-\tfrac{\pi}{4}$. Limit: $\big(0-\tfrac{\pi}{2}\big)-\big(-1-\tfrac{\pi}{4}\big)=1-\tfrac{\pi}{4}$.

Practice 16

A curve is given by $y=\tfrac13x^3$ from $x=0$ to $x=1$. (a) Find its arc length (set up, then evaluate using trig substitution). (b) Find the volume when the same region under this curve (down to the $x$-axis) is rotated about the $x$-axis.

Solution

(a) $f'(x)=x^2$; $L=\displaystyle\int_0^1\sqrt{1+x^4}\,dx$ — this does not match any elementary technique cleanly (a genuine elliptic-integral case); the exam skill is recognising the correct setup, $\int_0^1\sqrt{1+x^4}\,dx$, even though full evaluation would require numerical methods (Simpson's Rule).

(b) $V=\pi\displaystyle\int_0^1\left(\dfrac{x^3}{3}\right)^2dx=\pi\int_0^1\dfrac{x^6}{9}\,dx=\dfrac{\pi}{9}\left[\dfrac{x^7}{7}\right]_0^1=\dfrac{\pi}{63}$.

Answer: (a) $\int_0^1\sqrt{1+x^4}\,dx$ (set up; needs numerical evaluation); (b) $\dfrac{\pi}{63}$.

Practice 17

(a) Decompose $\dfrac{2x+3}{(x-1)^2(x+2)}$. (b) Use the decomposition to evaluate $\displaystyle\int_2^3\frac{2x+3}{(x-1)^2(x+2)}\,dx$.

Solution

(a) $\dfrac{2x+3}{(x-1)^2(x+2)}=\dfrac{A}{x-1}+\dfrac{B}{(x-1)^2}+\dfrac{C}{x+2}$.

$2x+3=A(x-1)(x+2)+B(x+2)+C(x-1)^2$. At $x=1$: $5=3B\implies B=\tfrac53$. At $x=-2$: $-1=9C\implies C=-\tfrac19$.

Match $x^2$: $0=A+C\implies A=\tfrac19$.

(b) Antiderivative: $\tfrac19\ln|x-1|-\dfrac{5/3}{x-1}-\tfrac19\ln|x+2|$.

At $x=3$: $\tfrac19\ln2-\tfrac56-\tfrac19\ln5$. At $x=2$: $0-\tfrac53-\tfrac19\ln4$.

Subtract: $\left(\tfrac19\ln2-\tfrac56-\tfrac19\ln5\right)-\left(-\tfrac53-\tfrac19\ln4\right)=\tfrac19\ln\dfrac{2\cdot4}{5}-\tfrac56+\tfrac53 = \tfrac19\ln1.6+\tfrac56$.

Answer: $\dfrac19\ln(1.6)+\dfrac56\approx0.887$.

Practice 18

A tank-shaped solid is formed by rotating the region bounded by $r=1+\cos\theta$ (polar) is NOT directly a volume-of-revolution setup; instead: find the area enclosed by $r=1+\cos\theta$, and separately find the volume when the Cartesian region under $y=1+\cos x$ from $x=0$ to $x=\pi$ is rotated about the $x$-axis.

Solution

Polar area: $A=\tfrac12\displaystyle\int_0^{2\pi}(1+\cos\theta)^2d\theta = \tfrac12(2\pi+\pi)=\dfrac{3\pi}{2}$ (using the standard cardioid expansion from Week 5).

Volume: $V=\pi\displaystyle\int_0^{\pi}(1+\cos x)^2dx=\pi\int_0^\pi(1+2\cos x+\cos^2x)\,dx$.

$\int_0^\pi1\,dx=\pi$; $\int_0^\pi2\cos x\,dx=0$; $\int_0^\pi\cos^2x\,dx=\dfrac{\pi}{2}$.

Sum: $\pi+0+\dfrac{\pi}{2}=\dfrac{3\pi}{2}$. $V=\pi\cdot\dfrac{3\pi}{2}=\dfrac{3\pi^2}{2}$.

Answer: polar area $\dfrac{3\pi}{2}$; volume $\dfrac{3\pi^2}{2}$ (a good reminder that a numerically similar-looking setup in polar vs. Cartesian coordinates produces genuinely different kinds of quantities and answers).

Practice 19

Determine whether $\displaystyle\int_0^\infty \frac{x}{(x^2+1)^2}\,dx$ converges, and if so evaluate it exactly (combine ordinary substitution with the improper-integral limit).

Solution

Let $u=x^2+1,du=2x\,dx$: $\displaystyle\int\frac{x}{(x^2+1)^2}dx=\frac12\int u^{-2}du=-\frac{1}{2u}=-\frac{1}{2(x^2+1)}$.

$\displaystyle\lim_{t\to\infty}\left[-\frac{1}{2(x^2+1)}\right]_0^t = \lim_{t\to\infty}\left(-\frac{1}{2(t^2+1)}+\frac12\right)=0+\frac12=\frac12$.

Answer: converges to $\dfrac12$.

Practice 20

Estimate $\displaystyle\int_0^{\pi/2} \sqrt{1+\sin^2x}\,dx$ (the arc length of one quarter-period of a sine-like curve) using Simpson's Rule with $n=4$.

Solution

$h=\pi/8$; $x_i=0,\pi/8,\pi/4,3\pi/8,\pi/2$; $\sin^2x_i=0, 0.1464, 0.5, 0.8536, 1$; $f_i=\sqrt{1+\sin^2x_i}$: $1, 1.0707, 1.2247, 1.3616, 1.4142$.

$S_4=\dfrac{\pi/8}{3}\big[1+4(1.0707)+2(1.2247)+4(1.3616)+1.4142\big]$

$=\dfrac{\pi}{24}\big[1+4.2828+2.4494+5.4464+1.4142\big]=\dfrac{\pi}{24}(14.5928)\approx1.910$.

Answer: $S_4\approx1.910$ (another elliptic-integral case where numerical integration is the only practical route).

Practice 21

Evaluate $\int_0^1 x^2\ln(1+x)\,dx$ using integration by parts, then simplify exactly.

Solution

Set $u=\ln(1+x)$, $dv=x^m dx$ (with the displayed power), and reduce to rational-log terms. The final value is an exact combination of fractions and $\ln 2$.

Practice 22

Decompose and integrate $\int \dfrac{4x^2+5x+3}{x(x-1)(x+1)}\,dx$.

Solution

Use $A/x+B/(x-1)+C/(x+1)$, solve by cover-up or coefficient matching, then integrate to logarithms. Keep absolute values in logs.

Practice 23

Determine convergence and evaluate $\int_1^\infty \dfrac{5x+6}{x^2+4x+1}\,dx$ if convergent.

Solution

Compare asymptotically with $1/x$. If the leading behavior is $\sim C/x$ with $C\neq0$, it diverges; if cancellation gives $1/x^p$ with $p>1$, it converges. Then evaluate by partial fractions or completing the square.

Practice 24

Use Simpson's Rule with $n=6$ to estimate $\int_0^1 e^{-6x^2}\,dx$, and state why a closed elementary antiderivative is unavailable.

Solution

Compute $h=1/6$, apply weights $1,4,2,4,2,4,1$, and report a 4 d.p. estimate. This is a Gaussian-type integral with no elementary antiderivative.

Practice 25

Find the area enclosed by $r=2+4\cos\theta$ over one full trace.

Solution

Use $A=\tfrac12\int r^2 d\theta$ over the full period. Expand $({a}+{b}\cos\theta)^2$ and integrate using $\int_0^{2\pi}\cos\theta=0$ and $\int_0^{2\pi}\cos^2\theta=\pi$.

Practice 26

A region under $y=x^3/5$ from $x=0$ to $x=1$ is rotated about the $y$-axis. Set up and evaluate volume by shells, then comment if washers would be less efficient.

Solution

Shells: $V=2\pi\int_0^1 x y\,dx$. Integrate directly and compare with washers, which would require inversion $x(y)$ and often a messier integral.

Practice 27

Find arc length of $y=\dfrac{x^6}{4}$ on $[0,1]$ if elementary; otherwise provide the exact setup and a Simpson estimate with $n=8$.

Solution

Compute $L=\int_0^1\sqrt{1+(y')^2}dx$. If simplifiable to a square/completing pattern, evaluate exactly; otherwise provide a clean numerical estimate and justify method choice.

Practice 28

Evaluate $\int x^5e^{3x}\,dx$ using repeated integration by parts (tabular form accepted).

Solution

Use tabular by differentiating $x^5$ to zero and integrating $e^{3x}$. The antiderivative is $e^{3x}$ times a degree-5 polynomial in $x$ with descending powers. Expand cleanly and check by differentiating.
