Core Symbols + Standard Form

In mathematical logic and set theory, "for every" is expressed using the universal quantifier \(\forall\). It translates to "for all," "for each," or "for any". When applied to a specific set, it is combined with the set membership symbol \(\in\) to restrict the domain.

The Core Symbols
\(\forall\): The universal quantifier. It asserts that the following statement is true for every element in the given domain.
\(\in\): Set membership. It indicates that an element belongs to a specified set.
\(\subset\) or \(\subseteq\): Subset. Indicates that all elements of one set are completely contained within another.

How to Write "For Every" in a Set
To state that a condition is true for every element in a specific set, you combine the universal quantifier with the set membership symbol.

Standard Form: \(\forall x \in S, P(x)\)

u-Substitution
The chain rule in reverse. When the integrand contains a composite function and its inner derivative is present (or can be introduced), substitution converts it to a standard form.
The Substitution Method

If the integrand has the form \(f(g(x))\cdot g'(x)\,dx\), set \(u=g(x)\), then \(du=g'(x)\,dx\) and the integral becomes \(\int f(u)\,du\) ŌĆö a standard form.

Full procedure ŌĆö every step is required
  • Identify \(u\) ŌĆö the inner function of a composite; often the expression inside brackets, under a root, or in an exponent
  • Find \(du\) ŌĆö differentiate \(u\) w.r.t. \(x\) and write \(du = u'\,dx\)
  • Substitute completely ŌĆö replace every \(x\)-expression and \(dx\); no \(x\) should remain
  • Integrate in \(u\) ŌĆö apply a standard form
  • Back-substitute ŌĆö replace \(u\) with the original expression, add \(C\)
Missing constant factor: if \(du = 5x\,dx\) but only \(x\,dx\) appears, write \(x\,dx = \tfrac{1}{5}\,du\). You may adjust for a missing constant factor ŌĆö never for a missing variable factor.

Definite Integrals & the Fundamental Theorem
The bridge between antiderivatives and area. The FTC means you rarely need Riemann sums ŌĆö find any antiderivative and evaluate it at the limits.
The Fundamental Theorem of Calculus

Part 1 ŌĆö differentiation undoes integration:

\[ \frac{d}{dx}\left[\int_a^x f(t)\,dt\right] = f(x) \]

Part 2 ŌĆö evaluate definite integrals via antiderivatives:

\[ \int_a^b f(x)\,dx = F(b) - F(a) \quad\text{where } F' = f \]
Key properties of definite integrals
  • Signed area: regions below the \(x\)-axis contribute negative area
  • \(\displaystyle\int_a^a f\,dx = 0\); \(\quad\int_a^b f\,dx = -\int_b^a f\,dx\)
  • Splitting: \(\displaystyle\int_a^b f\,dx = \int_a^c f\,dx + \int_c^b f\,dx\) for any \(c\in[a,b]\)
  • Even function, symmetric limits: \(\displaystyle\int_{-a}^a f\,dx = 2\int_0^a f\,dx\)
  • Odd function, symmetric limits: \(\displaystyle\int_{-a}^a f\,dx = 0\)
For u-substitution in a definite integral ŌĆö change the limits to \(u\)-values when you substitute. Then you never need to back-substitute.

Area & Engineering Applications
Integration's physical meaning: signed area, total distance, accumulated quantity. Each problem type has a standard setup ŌĆö learn the setup as rigorously as the calculation.
Three Core Applications
Area ŌĆö setup rules
  • Area under \(f\): \(\displaystyle\int_a^b |f(x)|\,dx\) ŌĆö split at zeros if \(f\) changes sign
  • Area between \(f\) and \(g\): \(\displaystyle\int_a^b [f(x)-g(x)]\,dx\) where \(f\geq g\) on \([a,b]\) ŌĆö find intersections first
  • Displacement: \(\displaystyle\int_{t_1}^{t_2} v(t)\,dt\) ŌĆö net, can be negative
  • Total distance: \(\displaystyle\int_{t_1}^{t_2}|v(t)|\,dt\) ŌĆö split at \(v=0\), always positive
  • Accumulated quantity: \(Q(T)=Q_0+\displaystyle\int_0^T r(t)\,dt\) where \(r\) is the rate
