Differentiation Formula Sheet (DPEN22)

Rules, derivatives, and trigger map for choosing the right rule quickly.

1) Core Rules

\[ \frac{d}{dx}(c)=0,\quad \frac{d}{dx}(x^n)=nx^{n-1},\quad \frac{d}{dx}[cf(x)] = c f'(x) \]
\[ (f\pm g)'=f'\pm g',\quad (fg)'=f'g+fg',\quad \left(\frac{f}{g}\right)'=\frac{f'g-fg'}{g^2} \]
\[ \frac{d}{dx}f(g(x))=f'(g(x))g'(x)\quad\text{(chain rule)} \]

3) Composite Forms (Most Exam Questions)

\[ \frac{d}{dx}\sin(u)=\cos(u)\,u',\quad \frac{d}{dx}\cos(u)=-\sin(u)\,u' \]
\[ \frac{d}{dx}e^{u}=e^{u}u',\quad \frac{d}{dx}\ln(u)=\frac{u'}{u} \]
Trigger: if you see brackets/exponents with inner expressions, chain rule is almost always involved.

4) Useful Application Formulas

\[ y' = 0\ \text{at stationary points},\qquad y''>0\ \text{local min},\ y''<0\ \text{local max} \]
\[ \text{Point-slope tangent: } y-y_1 = m(x-x_1),\quad m=f'(x_1) \]
\[ \text{Normal slope } m_n=-\frac{1}{m_t} \]
