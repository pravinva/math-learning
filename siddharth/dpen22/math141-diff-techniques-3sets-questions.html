Set A - Core Rules + Chain/Product/Quotient

  1. Differentiate \(y=5x^4-3x^2+7\).
  2. Differentiate \(y=\dfrac{2}{x^3}\).
  3. Differentiate \(y=(x^2+1)(x^3-2x)\).
  4. Differentiate \(y=\dfrac{x^2+3x}{x-1}\).
  5. Differentiate \(y=(3x-2)^5\).
  6. Differentiate \(y=\sqrt{4x+1}\).
  7. Differentiate \(y=(x^2+1)e^x\).
  8. Differentiate \(y=e^{x^2-3x}\).
  9. Differentiate \(y=\ln(2x^2+1)\).
  10. Differentiate \(y=\ln\!\left(\dfrac{x+1}{x-1}\right)\).
  11. Differentiate \(y=\sin(4x)\cos x\).
  12. Differentiate \(y=\tan(3x-1)\).
  13. Differentiate \(y=\sec^2(2x)\).
  14. Differentiate \(y=\arcsin x\).
  15. Differentiate \(y=\arctan(2x)\).
  16. Differentiate implicitly: \(x^2+y^2=16\).
  17. Differentiate implicitly: \(xy+y^2=5\).
  18. For \(x=t^2,\ y=t^3-1\), find \(\dfrac{dy}{dx}\).
  19. For \(x=\cos t,\ y=\sin t\), find \(\dfrac{dy}{dx}\).
  20. Find tangent line to \(y=x^3-3x\) at \(x=1\).

Set B - Logs, Inverse, Implicit, Parametric

  1. Differentiate \(y=x^x\) using logarithmic differentiation.
  2. Differentiate \(y=\ln(\sin x)\).
  3. Differentiate \(y=\log_{10}(x^2+1)\).
  4. Differentiate \(y=(\arctan x)^2\).
  5. Differentiate \(y=\arccos(3x)\).
  6. Differentiate \(y=\operatorname{arccot}(x)\) using \(d(\arccot x)/dx=-1/(1+x^2)\).
  7. Differentiate \(y=\sinh(2x)\).
  8. Differentiate \(y=\cosh(3x-2)\).
  9. Differentiate \(y=\tanh(4x)\).
  10. Differentiate \(y=\operatorname{sech}(x)\).
  11. Differentiate implicitly: \(x^2+xy+y^2=7\).
  12. Differentiate implicitly: \(\sin(xy)=x+y\).
  13. Differentiate implicitly: \(e^{xy}=x^2+y\).
  14. For \(x=t^3+1,\ y=t^2-t\), find \(\dfrac{dy}{dx}\).
  15. For \(x=t-\sin t,\ y=1-\cos t\), find \(\dfrac{dy}{dx}\).
  16. For \(x=t^2,\ y=t^4\), find \(\dfrac{d^2y}{dx^2}\).
  17. Find stationary points of \(f(x)=x^3-6x^2+9x+1\).
  18. Find intervals where \(g(x)=x^3-3x\) is increasing/decreasing.
  19. Find inflection point(s) of \(h(x)=x^4-4x^3\).
  20. Find normal line to \(y=x^2\) at \(x=2\).

Set C - Exam Mix

  1. Differentiate \(y=\dfrac{\ln x}{x}\).
  2. Differentiate \(y=x^2e^{-x}\).
  3. Differentiate \(y=(x^2+1)^{3/2}\).
  4. Differentiate \(y=\dfrac{e^x}{x^2+1}\).
  5. Differentiate \(y=\arcsin(x^2)\).
  6. Differentiate \(y=\arctan\!\left(\dfrac1x\right)\), \(x\ne0\).
  7. Differentiate \(y=\ln\!\left(x+\sqrt{x^2+1}\right)\).
  8. Differentiate \(y=\cosh^2 x\).
  9. Differentiate \(y=\sinh x\cosh x\).
  10. Differentiate \(y=\tanh^{-1}(x)\) for \(|x|<1\).
  11. Differentiate implicitly: \(x^3+y^3=6xy\).
  12. Differentiate implicitly: \(\ln(xy)=x-y\).
  13. For \(x=2\cos t,\ y=2\sin t\), find slope at \(t=\pi/3\).
  14. For \(x=t^2+1,\ y=t^3\), find \(\dfrac{d^2y}{dx^2}\).
  15. Find stationary points and classify for \(f(x)=2x^3-15x^2+24x\).
  16. Find concavity intervals for \(g(x)=x^4-2x^3+x\).
  17. Tangent line to \(y=\ln x\) at \(x=e\).
  18. Normal line to \(y=e^x\) at \(x=0\).
  19. If \(s(t)=t^3-6t^2+9t\), find velocity and acceleration.
  20. For \(C(q)=q^3-9q^2+24q+40\), find where cost is increasing/decreasing (\(q\ge0\)).
