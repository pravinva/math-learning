DPEN22 → MATH141 Preparedness Pack — Questions

Five mixed sets to bridge from DPEN22 calculus into MATH141 readiness (calculus + vectors + basic matrices).
Format: 5 sets × 10 questions each.
Use: attempt timed (35-40 minutes per set), then check the separate worked solutions.

Set 1 — Core Derivative + Integral Fluency

  1. Find \(\lim_{x\to2}\dfrac{x^2-4}{x-2}\).
  2. Differentiate \(y=(x^2+1)e^x\).
  3. Differentiate \(y=\ln(3x^2+1)\).
  4. For \(f(x)=x^3-6x^2+9x+1\), find stationary points and classify them.
  5. Find intervals of concavity and inflection points for \(g(x)=x^4-4x^3+2\).
  6. Evaluate \(\int (6x^2-4x+1)\,dx\).
  7. Evaluate \(\int_0^2 (3x^2+1)\,dx\).
  8. Evaluate \(\int x e^x\,dx\).
  9. Find the area between \(y=2x\) and \(y=x^2\) on \(0\le x\le2\).
  10. Find the average value of \(f(x)=x^2+2x\) on \([0,4]\).

Set 2 — Applications + Function Behavior

  1. Differentiate \(y=\dfrac{3x+2}{x-4}\).
  2. Differentiate \(y=\sin(3x)e^{-x}\).
  3. If \(h(x)=x^3-3x^2-24x+5\), find \(h''(x)\) and the inflection point.
  4. Production cost is \(C(q)=q^3-9q^2+24q+40\), \(q\ge0\). Find where cost is increasing/decreasing.
  5. Evaluate \(\int \dfrac{2x}{x^2+1}\,dx\).
  6. Evaluate \(\int_1^3 (x^2-2x+4)\,dx\).
  7. Evaluate \(\int (2x-1)^5\,dx\).
  8. A rectangle has perimeter \(100\) m. Find dimensions maximizing area.
  9. Find \(\lim_{x\to\infty}\dfrac{3x^2+1}{x^2-5}\).
  10. Determine whether \(f(x)=\begin{cases}x+1,&x<2\\3,&x\ge2\end{cases}\) is continuous at \(x=2\).

Set 3 — Vectors & Geometry Bridge

  1. For \(\mathbf{a}=(2,-1,3)\), find \(|\mathbf{a}|\) and \(\hat{\mathbf{a}}\).
  2. For \(\mathbf{a}=(1,2,3)\), \(\mathbf{b}=(2,-1,4)\), find \(\mathbf{a}\cdot\mathbf{b}\).
  3. Find the angle between \(\mathbf{a}=(1,0,1)\) and \(\mathbf{b}=(1,1,0)\).
  4. Find \(\mathbf{a}\times\mathbf{b}\) for \(\mathbf{a}=(1,2,0)\), \(\mathbf{b}=(0,1,3)\).
  5. Find a normal vector to the plane through \(A(1,0,0), B(0,1,0), C(0,0,1)\).
  6. Write vector equation of the line through \((1,2,0)\) in direction \((2,-1,3)\).
  7. Find equation of plane with normal \((2,-1,4)\) through \((1,0,2)\).
  8. Find distance between points \(P(1,-2,3)\) and \(Q(4,2,-1)\).
  9. Find projection of \(\mathbf{u}=(3,1,2)\) onto \(\mathbf{v}=(1,0,1)\).
  10. Check if vectors \((1,2,3)\), \((2,4,6)\), \((0,1,1)\) are coplanar (use scalar triple product idea).

Set 4 — Matrices & Linear Systems Bridge

  1. State shape of \(A=\begin{pmatrix}1&2&3\\4&5&6\end{pmatrix}\).
  2. Compute \(A+B\) where \(A=\begin{pmatrix}1&0\\2&3\end{pmatrix}, B=\begin{pmatrix}4&-1\\0&2\end{pmatrix}\).
  3. Compute \(3\begin{pmatrix}2&-1\\0&4\end{pmatrix}\).
  4. Compute \(\begin{pmatrix}1&2\\3&4\end{pmatrix}\begin{pmatrix}5&6\\7&8\end{pmatrix}\).
  5. Find \(\det\begin{pmatrix}3&1\\2&4\end{pmatrix}\).
  6. Find inverse of \(\begin{pmatrix}2&1\\5&3\end{pmatrix}\).
  7. Write augmented matrix for \(\begin{cases}2x+3y=7\\x-y=1\end{cases}\).
  8. Solve \(\begin{cases}x+2y=5\\3x-y=4\end{cases}\).
  9. For \(A=\begin{pmatrix}k&2\\1&k\end{pmatrix}\), find values of \(k\) where \(A\) is not invertible.
  10. Explain briefly why \(AB\neq BA\) in general (one sentence).

Set 5 — Mixed MATH141 Readiness Challenge

  1. Differentiate \(y=\dfrac{\ln x}{x}\).
  2. Differentiate implicitly: \(x^2+y^2=25\).
  3. Evaluate \(\int_0^1 x\sqrt{x^2+1}\,dx\).
  4. Find area enclosed by \(y=4x-x^2\) and \(y=x^2\) on \([0,2]\).
  5. Find stationary points of \(f(x)=2x^3-15x^2+36x\).
  6. For \(g(x)=x^3+3x^2-24x+1\), find inflection point.
  7. Compute \(\mathbf{a}\times\mathbf{b}\) for \(\mathbf{a}=(2,1,-1)\), \(\mathbf{b}=(1,0,3)\).
  8. Find \(\det\begin{pmatrix}1&2&0\\3&1&1\\0&2&2\end{pmatrix}\).
  9. Solve system \(\begin{cases}2x+y=7\\x-2y=-1\end{cases}\).
  10. State one algebra weakness that can break both derivative and matrix questions, and how to avoid it.
