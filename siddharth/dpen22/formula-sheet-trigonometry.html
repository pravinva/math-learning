Trigonometry Formula Sheet (Intuitive)

For DPEN22: formulas + derivations + canonical right triangles so you can derive values instead of memorizing blindly.

1) Build From One Right Triangle (SOH CAH TOA)

General Right Triangle

\[ \sin\theta=\frac{\text{opp}}{\text{hyp}},\quad \cos\theta=\frac{\text{adj}}{\text{hyp}},\quad \tan\theta=\frac{\text{opp}}{\text{adj}} \]

Why these are enough

  • Given angle + one side, you can find missing sides.
  • Given sides, you can find unknown angles.
  • Everything else (identities, equations, graphs) builds on these ratios and the unit circle.
Memory trick: SOH = sine, CAH = cosine, TOA = tangent.

2) Canonical Right Triangles (Exact Values Without Rote Memorization)

45°-45°-90° Triangle

\[ \sin45^\circ=\cos45^\circ=\frac{1}{\sqrt{2}}=\frac{\sqrt{2}}{2},\quad \tan45^\circ=1 \]
Plain text fallback: sin45 = cos45 = 1/sqrt(2) = sqrt(2)/2, tan45 = 1

30°-60°-90° Triangle

\[ \sin30^\circ=\frac{1}{2},\ \cos30^\circ=\frac{\sqrt{3}}{2},\ \tan30^\circ=\frac{1}{\sqrt{3}}=\frac{\sqrt{3}}{3} \]
Plain text fallback: sin30 = 1/2, cos30 = sqrt(3)/2, tan30 = 1/sqrt(3) = sqrt(3)/3
\[ \sin60^\circ=\frac{\sqrt{3}}{2},\ \cos60^\circ=\frac{1}{2},\ \tan60^\circ=\sqrt{3} \]
Plain text fallback: sin60 = sqrt(3)/2, cos60 = 1/2, tan60 = sqrt(3)
Exam habit: derive 30/45/60 from triangles first, then use unit-circle symmetry for 120°, 135°, 150°, etc.

3) Essential Trig Identities

\[ \sin^2\theta+\cos^2\theta=1,\quad 1+\tan^2\theta=\sec^2\theta,\quad 1+\cot^2\theta=\csc^2\theta \]
\[ \tan\theta=\frac{\sin\theta}{\cos\theta},\quad \cot\theta=\frac{\cos\theta}{\sin\theta} \]

Sum / Difference (very high value)

\[ \sin(A\pm B)=\sin A\cos B\pm\cos A\sin B \]
\[ \cos(A\pm B)=\cos A\cos B\mp\sin A\sin B \]
\[ \tan(A\pm B)=\frac{\tan A\pm\tan B}{1\mp\tan A\tan B} \]

\[ c^2=a^2+b^2-2ab\cos C \quad\text{(Cosine rule)} \]
