DPEN22 Applied Reverse-Identity Training

Rewritten to use only the core identities: \(\sin^2x+\cos^2x=1\), sum formulas, and double-angle formulas (plus direct derived forms like \(1+\tan^2x=\sec^2x\)).
How to use each question:
1) Spot the pattern (Pythagorean, sum/difference, or double-angle).
2) Rewrite first, then solve/simplify.
3) Check interval/domain at the end.
Identity Toolbox (only these are used)
Core:
\(\sin^2x+\cos^2x=1\)
\(1+\tan^2x=\sec^2x,\quad 1+\cot^2x=\csc^2x\)
Sum of angles:
\(\sin(A\pm B)=\sin A\cos B\pm\cos A\sin B\)
\(\cos(A\pm B)=\cos A\cos B\mp\sin A\sin B\)
Double angle:
\(\sin2A=2\sin A\cos A\)
\(\cos2A=\cos^2A-\sin^2A=1-2\sin^2A=2\cos^2A-1\)

Set 1 (Guided, 8 Problems)

  1. Simplify \(\cos^2x-\sin^2x\).
    Observe: direct \(\cos2x\) pattern.
  2. Simplify \(2\sin x\cos x\).
    Observe: direct \(\sin2x\) pattern.
  3. Simplify \(1-2\sin^2x\).
    Observe: reverse double-angle cosine.
  4. Simplify \(2\cos^2x-1\).
    Observe: same reverse \(\cos2x\) form.
  5. Simplify \(\sin(4x)\cos x+\cos(4x)\sin x\).
    Observe: \(\sin(A+B)\).
  6. Simplify \(\cos(3x)\cos(2x)-\sin(3x)\sin(2x)\).
    Observe: \(\cos(A+B)\).
  7. Solve on \(0\le x\le2\pi\): \(\cos^2x-\sin^2x=\dfrac12\).
  8. Solve on \(0\le x\le2\pi\): \(2\sin x\cos x=\dfrac{\sqrt3}{2}\).

Set 2 (Practice, 8 Problems)

  1. Simplify \(\dfrac{1-\cos2x}{2}\).
  2. Simplify \(\dfrac{1+\cos2x}{2}\).
  3. Simplify \((\cos^2x-\sin^2x)^2+(2\sin x\cos x)^2\).
  4. Simplify \(\sin A\cos B-\cos A\sin B\).
  5. Simplify \(\cos A\cos B+\sin A\sin B\).
  6. Solve on \(0\le x\le2\pi\): \(1-2\sin^2x=-\dfrac12\).
  7. Solve on \(0\le x\le2\pi\): \(2\cos^2x-1=0\).
  8. If \(g(x)=3(1-2\sin^2x)+2(2\sin x\cos x)\), write \(g(x)\) as \(a\cos2x+b\sin2x\).

Set 3 (Applied, 8 Problems)

  1. A torque model is \(\tau(\theta)=8(\cos^2\theta-\sin^2\theta)\). Rewrite using one trig function.
  2. Using Q1, find all \(\theta\in[0,2\pi]\) where \(\tau(\theta)=0\).
  3. A lift height model is \(h(t)=5+2[\sin(3t)\cos t+\cos(3t)\sin t]\). Rewrite with one trig function.
  4. For Q3, solve \(h(t)=5\) on \(0\le t\le2\pi\).
  5. A signal \(v(t)=4(1-2\sin^2t)\). Rewrite with one trig function and solve \(v(t)=0\), \(0\le t\le2\pi\).
  6. Simplify \((1+\tan^2x)-\sec^2x\).
  7. Simplify \(\sin(5x)\cos x-\cos(5x)\sin x\).
  8. Solve on \(0\le x\le2\pi\): \(\sin(5x)\cos x-\cos(5x)\sin x=\dfrac12\).

Worked Solutions — Set 1

  1. \(\cos^2x-\sin^2x=\cos2x\).
  2. \(2\sin x\cos x=\sin2x\).
  3. \(1-2\sin^2x=\cos2x\).
  4. \(2\cos^2x-1=\cos2x\).
  5. \(\sin(4x+x)=\sin5x\).
  6. \(\cos(3x+2x)=\cos5x\).
  7. \(\cos2x=\dfrac12\Rightarrow 2x=\dfrac{\pi}{3},\dfrac{5\pi}{3},\dfrac{7\pi}{3},\dfrac{11\pi}{3}\).
    \(\Rightarrow x=\dfrac{\pi}{6},\dfrac{5\pi}{6},\dfrac{7\pi}{6},\dfrac{11\pi}{6}\).
  8. \(\sin2x=\dfrac{\sqrt3}{2}\Rightarrow 2x=\dfrac{\pi}{3},\dfrac{2\pi}{3},\dfrac{7\pi}{3},\dfrac{8\pi}{3}\).
    \(\Rightarrow x=\dfrac{\pi}{6},\dfrac{\pi}{3},\dfrac{7\pi}{6},\dfrac{4\pi}{3}\).

Worked Solutions — Set 2

  1. \(\dfrac{1-\cos2x}{2}=\sin^2x\).
  2. \(\dfrac{1+\cos2x}{2}=\cos^2x\).
  3. \((\cos2x)^2+(\sin2x)^2=1\).
  4. \(\sin(A-B)\).
  5. \(\cos(A-B)\).
  6. \(\cos2x=-\dfrac12\Rightarrow 2x=\dfrac{2\pi}{3},\dfrac{4\pi}{3},\dfrac{8\pi}{3},\dfrac{10\pi}{3}\).
    \(\Rightarrow x=\dfrac{\pi}{3},\dfrac{2\pi}{3},\dfrac{4\pi}{3},\dfrac{5\pi}{3}\).
  7. \(\cos2x=0\Rightarrow 2x=\dfrac{\pi}{2},\dfrac{3\pi}{2},\dfrac{5\pi}{2},\dfrac{7\pi}{2}\).
    \(\Rightarrow x=\dfrac{\pi}{4},\dfrac{3\pi}{4},\dfrac{5\pi}{4},\dfrac{7\pi}{4}\).
  8. \(g(x)=3\cos2x+2\sin2x\), so \(a=3,\ b=2\).

Worked Solutions — Set 3

  1. \(\tau(\theta)=8\cos2\theta\).
  2. \(\cos2\theta=0\Rightarrow \theta=\dfrac{\pi}{4},\dfrac{3\pi}{4},\dfrac{5\pi}{4},\dfrac{7\pi}{4}\).
  3. \(h(t)=5+2\sin(3t+t)=5+2\sin4t\).
  4. \(h(t)=5\Rightarrow \sin4t=0\Rightarrow 4t=n\pi\Rightarrow t=\dfrac{n\pi}{4}\).
    On \([0,2\pi]\): \(n=0,1,\dots,8\).
  5. \(v(t)=4\cos2t\). For \(v(t)=0\): \(\cos2t=0\Rightarrow t=\dfrac{\pi}{4},\dfrac{3\pi}{4},\dfrac{5\pi}{4},\dfrac{7\pi}{4}\).
  6. \((1+\tan^2x)-\sec^2x=0\).
  7. \(\sin(5x-x)=\sin4x\).
  8. \(\sin4x=\dfrac12\Rightarrow 4x=\dfrac{\pi}{6},\dfrac{5\pi}{6},\dfrac{13\pi}{6},\dfrac{17\pi}{6}\).
    \(\Rightarrow x=\dfrac{\pi}{24},\dfrac{5\pi}{24},\dfrac{13\pi}{24},\dfrac{17\pi}{24},\dfrac{25\pi}{24},\dfrac{29\pi}{24},\dfrac{37\pi}{24},\dfrac{41\pi}{24}\).
