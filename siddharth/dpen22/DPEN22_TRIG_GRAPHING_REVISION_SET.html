How to Choose the 4 Step-Points (Sin, Cos, Tan)

Students often ask: “Why these points?” The short answer is: we choose x-values where the trig value is exact and the graph changes role (crossing, maximum, minimum, or center).

Sine: the 4 quarter-step points (plus start)

For y=Asin⁡(B(x−h))+Dy=A\sin(B(x-h))+Dy=Asin(B(x−h))+D:

  • One cycle length is T=2π∣B∣T=\frac{2\pi}{|B|}T=∣B∣2π​, so step size is T/4T/4T/4.
  • Start at x0=hx_0=hx0​=h, then use 4 equal steps: x1=h+T4,x2=h+T2,x3=h+3T4,x4=h+T. x_1=h+\frac{T}{4},\quad x_2=h+\frac{T}{2},\quad x_3=h+\frac{3T}{4},\quad x_4=h+T. x1​=h+4T​,x2​=h+2T​,x3​=h+43T​,x4​=h+T.
  • At those argument angles 0,π2,π,3π2,2π0,\frac{\pi}{2},\pi,\frac{3\pi}{2},2\pi0,2π​,π,23π​,2π, sine values are exactly 0,1,0,−1,00,1,0,-1,00,1,0,−1,0.
  • What each of the 5 y-values means (plain language):
    • Point 0 (start): y-value is DDD. This is the midline level, so you begin on the center line.
    • Point 1 (after 1 step): y-value is D+AD+AD+A. You move up by amplitude AAA to the maximum (if A>0A>0A>0).
    • Point 2 (after 2 steps): y-value goes back to DDD. This is the second midline crossing.
    • Point 3 (after 3 steps): y-value is D−AD-AD−A. You move down by amplitude AAA to the minimum.
    • Point 4 (after 4 steps): y-value returns to DDD. One full cycle is complete.
  • Concrete example: for y=3sin⁡(x−π/2)+1y=3\sin(x-\pi/2)+1y=3sin(x−π/2)+1, D=1D=1D=1, A=3A=3A=3, so the y-sequence is 1, 4, 1, −2, 1. 1,\ 4,\ 1,\ -2,\ 1. 1, 4, 1, −2, 1. This is exactly “midline, maximum, midline, minimum, midline”.
Why these points? They capture the full sine behavior in one cycle: midline crossing up, maximum, midline crossing down, minimum, and return to midline.

Core Method (Teach This First)

1) Scale the Domain Before Solving

When the equation has a transformed argument, solve for the whole argument first (call it θ\thetaθ). Adjust the domain boundaries to match that argument window.

  • If equation uses 2x2x2x, original 0≤x≤2π0\le x\le2\pi0≤x≤2π becomes 0≤2x≤4π0\le2x\le4\pi0≤2x≤4π (two full rotations).
  • If equation uses x/2x/2x/2, original 0≤x≤2π0\le x\le2\pi0≤x≤2π becomes 0≤x/2≤π0\le x/2\le\pi0≤x/2≤π (half-speed argument).
  • Negative windows (for example −2π≤x≤2π-2\pi\le x\le2\pi−2π≤x≤2π) include clockwise (negative) and counterclockwise (positive) angles.

2) Solve the Whole Argument θ\thetaθ, Not xxx, First

For sin⁡(Bx+C)=k\sin(Bx+C)=ksin(Bx+C)=k, set θ=Bx+C\theta=Bx+Cθ=Bx+C, solve trig values for θ\thetaθ inside the transformed domain, then back-solve to xxx.

3) Isolate the Trig Function First (Amplitude/Vertical Shift)

For general form Asin⁡(B(x−C))+D=kA\sin(B(x-C))+D=kAsin(B(x−C))+D=k, first isolate:

sin⁡(B(x−C))=k−DA \sin(B(x-C))=\frac{k-D}{A} sin(B(x−C))=Ak−D​

Only then transform domain boundaries for B(x−C)B(x-C)B(x−C).

4) Back-Solve at the End

After collecting all valid θ\thetaθ-solutions in the transformed window, undo the shifts and scale to find xxx.

Common error: dividing by BBB too early and missing valid angle repeats. Always solve in θ\thetaθ-space first.
