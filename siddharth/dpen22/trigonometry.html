Topic 1
Trig Ratios & Exact Values
SOH CAH TOA — and the exact values every student must know cold
Trigonometry starts in a right-angled triangle. The three ratios — sine, cosine, tangent — relate the angles to the sides. Exact values for 30°, 45°, and 60° appear in almost every trig question at this level. You need these in your memory, not derived each time.
DefinitionThe three trig ratios in a right triangle▶

For an angle \(\theta\) in a right-angled triangle, label the sides as opposite (O), adjacent (A), and hypotenuse (H):

\[ \sin\theta = \frac{\text{Opposite}}{\text{Hypotenuse}} \qquad \cos\theta = \frac{\text{Adjacent}}{\text{Hypotenuse}} \qquad \tan\theta = \frac{\text{Opposite}}{\text{Adjacent}} \]

Memory aid: SOH CAH TOA. The hypotenuse is always the longest side, opposite the right angle.

Three reciprocal ratios also exist:

\[ \csc\theta = \frac{1}{\sin\theta} \qquad \sec\theta = \frac{1}{\cos\theta} \qquad \cot\theta = \frac{1}{\tan\theta} \]

Exact values — memorise all of these:

Canonical exact-value triangles (drawn explicitly)
Use these two triangles to reconstruct the exact values in an exam: the 45-45-90 triangle with sides \(1,1,\sqrt{2}\), and the 30-60-90 triangle with sides \(1,\sqrt{3},2\).
From these: \(\sin 45°=\frac{1}{\sqrt{2}}\), \(\cos 45°=\frac{1}{\sqrt{2}}\), \(\sin 30°=\frac{1}{2}\), \(\cos 30°=\frac{\sqrt{3}}{2}\), \(\sin 60°=\frac{\sqrt{3}}{2}\), \(\cos 60°=\frac{1}{2}\).
Acute-angle cofunction identities (complements): for \(0°<\theta<90°\), \(\sin\theta=\cos(90°-\theta)\), \(\cos\theta=\sin(90°-\theta)\), and \(\tan\theta=\cot(90°-\theta)\). Same idea for reciprocals: \(\sec\theta=\csc(90°-\theta)\), \(\csc\theta=\sec(90°-\theta)\), \(\cot\theta=\tan(90°-\theta)\).
Quick use: \(\sin 20°=\cos 70°\), \(\cos 35°=\sin 55°\), \(\tan 15°=\cot 75°\). Think: complementary angles in a right triangle swap opposite and adjacent sides.
Negative-angle identities (odd/even): \(\sin(-\theta)=-\sin\theta\), \(\cos(-\theta)=\cos\theta\), \(\tan(-\theta)=-\tan\theta\). So sine and tangent are odd, cosine is even.
Quick use: \(\sin(-30°)=-\frac12\), \(\cos(-60°)=\frac12\), \(\tan(-45°)=-1\). On the unit circle, \(-\theta\) reflects \(\theta\) across the x-axis: x-value stays the same, y-value changes sign.
Instant derive method (from canonical triangles): first place side lengths in order against angle sizes: smallest angle opposite smallest side, largest angle opposite largest side (in a right triangle, the hypotenuse is opposite \(90°\) and is always the largest side). So for 45-45-90, equal acute angles mean equal legs: \(1,1,\sqrt2\), giving \(\sin45°=\cos45°=\frac{1}{\sqrt2}\). For 30-60-90, side order is \(1,\sqrt3,2\) opposite \(30°,60°,90°\), giving \(\sin30°=\frac12,\ \cos30°=\frac{\sqrt3}{2},\ \sin60°=\frac{\sqrt3}{2},\ \cos60°=\frac12\). Add edge values \(\sin0°=0,\ \cos0°=1,\ \sin90°=1,\ \cos90°=0\). This reconstructs the full exact-value table without memorising a pattern.

NoteCommon mistakes with trig ratios▶
Watch out: the hypotenuse is always opposite the right angle — it is always the longest side. Do not confuse "opposite" and "adjacent" — they change depending on which angle you are working with.
Watch out: \(\sin^{-1}(x)\) means arcsin(x), not \(\frac{1}{\sin x}\). These are completely different things. When finding an angle from a ratio, use the inverse function on your calculator.
The exact values for 30° and 60° come from an equilateral triangle with side 2 (split in half). The exact values for 45° come from a right isoceles triangle with legs of length 1. Drawing these triangles is the best way to reconstruct exact values in an exam if you forget them.

Topic 2
The Unit Circle & Radians
Extending trig beyond right triangles — to all angles
Right-triangle trig only works for angles between 0° and 90°. To handle angles like 120°, 270°, or even negative angles, we need the unit circle. It extends sine and cosine to any angle — and explains why trig functions have negative values in certain quadrants.
DefinitionRadians and the unit circle▶

Radians: one radian is the angle subtended at the centre of a circle by an arc equal in length to the radius.

\[ \pi \text{ rad} = 180° \qquad 1\text{ rad} = \frac{180°}{\pi} \approx 57.3° \]

Converting: degrees → radians: multiply by \(\dfrac{\pi}{180}\). Radians → degrees: multiply by \(\dfrac{180}{\pi}\).

The unit circle has radius 1, centred at the origin. For any angle \(\theta\) measured anticlockwise from the positive x-axis, the point \(P\) on the circle has coordinates:

\[ P = (\cos\theta,\; \sin\theta) \]

This defines \(\cos\theta\) and \(\sin\theta\) for any angle — not just those in a right triangle.

NoteRadian traps and CAST mistakes▶
Watch out: when your calculator is in degree mode and the question is in radians (or vice versa), you get wrong answers. Always check your calculator mode first.
Watch out: the reference angle is always measured to the nearest part of the x-axis — not the y-axis. In Q2, reference = 180° − θ. In Q3, reference = θ − 180°. In Q4, reference = 360° − θ.
Arc length and sector area formulas (in radians only): \(s = r\theta\) and \(A = \frac{1}{2}r^2\theta\). These only work in radians — if given degrees, convert first.

PatternVisual explanation: why positive and negative coterminal angles exist▶

Think of angle as a turn from the positive x-axis to the terminal arm.

1
Positive angle means anticlockwise turn (for example \(+\frac{5\pi}{6}\)).
2
If you add one full turn \(2\pi\), you land on the same terminal arm: \(\frac{5\pi}{6}+2\pi=\frac{17\pi}{6}\).
3
Negative angle means clockwise turn. From the same arm, subtract one full turn: \(\frac{5\pi}{6}-2\pi=-\frac{7\pi}{6}\).
So coterminal angles are not random numbers — they are different descriptions of the same final direction. Positive/negative comes from turning direction.
Coterminal-angle diagram (same terminal arm)
Blue arc: \(+\frac{5\pi}{6}\). Orange arc: \(+\frac{17\pi}{6}\) (one extra full turn). Red clockwise arc: \(-\frac{7\pi}{6}\). All end on the same ray.
Arc length and sector area: where formulas come from
For \(\theta=1\) radian, arc length equals radius, so for \(\theta\) radians, arc scales linearly: \(s=r\theta\). Sector area is the same fraction of circle area as angle is of \(2\pi\): \(A=\frac{\theta}{2\pi}\pi r^2=\frac12 r^2\theta\).
Clarified diagram: what “arc” means (boundary on the circle)
Blue/orange/red are turning paths on the circumference from the same initial side (positive x-axis) to the same terminal side (150°). On the right, \(s\) is the curved boundary of the sector.
Definition reminder: an arc is the curved part of the circle boundary between two rays. It is not a straight radius line.

Graphs of Trig Functions
Amplitude, period, phase shift — reading and writing transformed trig graphs
Trig graphs repeat — they are periodic. Understanding the four parameters (a, b, c, d) in \(y = a\sin(bx+c)+d\) lets you sketch any transformed trig graph and write the equation of a graph from its shape. This is a core exam topic.
DefinitionThe general form and its four parameters▶
\[ y = a\sin(bx + c) + d \]
ParameterNameEffectFormula
\(a\)AmplitudeVertical stretch. If \(a<0\), reflected in x-axis.Amplitude = \(|a|\)
\(b\)Angular frequencyHorizontal compression. Larger b → shorter period.Period = \(\dfrac{2\pi}{|b|}\)
\(c\)Phase shiftHorizontal shift. Moves graph left (if +) or right (if −).Shift = \(-\dfrac{c}{b}\)
\(d\)Vertical shiftMoves midline up (if +) or down (if −).Midline at \(y = d\)

Range: from \(d - |a|\) to \(d + |a|\).

PatternThorough graph method: sketch, read, and reverse-engineer equations▶

Use this method every time:

1
Choose function family first: does shape look like sine/cosine wave, tangent branches with asymptotes, or reciprocal branches (sec/csc/cot)?
2
Find midline and amplitude: \(d = \frac{\max+\min}{2}\), \(|a| = \frac{\max-\min}{2}\). For tan, amplitude is not used.
3
Find period: sine/cos/sec/csc use \(T=\frac{2\pi}{|b|}\), tan/cot use \(T=\frac{\pi}{|b|}\).
4
Find horizontal shift carefully: rewrite as \(y=a\sin(b(x-h))+d\) or \(y=a\cos(b(x-h))+d\), where \(h=-c/b\).
5
Generate key x-values: one cycle is split into quarters. For sine/cos, use \(h,\ h+\frac{T}{4},\ h+\frac{T}{2},\ h+\frac{3T}{4},\ h+T\).
6
For tan/cot: locate asymptotes first, then centerline crossing between them, then direction (increasing or decreasing).
7
For sec/csc: sketch parent cos/sin first, then place reciprocal branches above maxima and below minima.

NoteHigh-yield graph traps and checking rules▶
Trap 1: \(c\) is not the shift. Shift is \(-c/b\). Always factor \(b\) first before reading shift.
Trap 2: tan/cot period is \(\pi/|b|\), not \(2\pi/|b|\).
Trap 3: sec/csc are undefined where cos/sin are zero. Mark asymptotes before drawing branches.
Fast self-check: verify one cycle by substituting 3 strategic x-values: start point, quarter-period point, and endpoint. If any miss expected max/min/midline behavior, re-check \(h\) and \(b\).

Topic 4
Solving Trig Equations
Finding all solutions — not just the first one the calculator gives
The calculator's \(\sin^{-1}\) button gives only one answer. But trig equations almost always have multiple solutions within a given range because the functions repeat. The systematic method — CAST diagram plus symmetry — ensures you find every solution.
DefinitionPrincipal value and all solutions▶

The principal value is the one angle in the standard range that your calculator returns:

\[ \sin^{-1}: \left[-\frac{\pi}{2}, \frac{\pi}{2}\right] \qquad \cos^{-1}: [0, \pi] \qquad \tan^{-1}: \left(-\frac{\pi}{2}, \frac{\pi}{2}\right) \]

To find all solutions in \([0°, 360°]\) or \([0, 2\pi]\):

1
Find the reference angle: the positive acute angle whose trig value has the same magnitude.
2
Use CAST to determine which quadrants give the correct sign.
3
Write the angle in each valid quadrant using the reference angle.
4
Check all answers are within the required range.

General solution (all possible solutions, not just one cycle):

\[ \sin\theta = k: \quad \theta = \sin^{-1}k + 360°n \quad\text{or}\quad \theta = 180° - \sin^{-1}k + 360°n \]
\[ \cos\theta = k: \quad \theta = \pm\cos^{-1}k + 360°n \]
\[ \tan\theta = k: \quad \theta = \tan^{-1}k + 180°n \]
PatternFull solving workflow: isolate, transform, range-check, and conclude▶
1
Normalize first: move everything to one side or isolate one trig function (for example \(\sin\theta=k\)).
2
Classify equation type: linear trig, quadratic in \(\sin\theta/\cos\theta\), multiple-angle (\(2\theta,3\theta\)), or mixed-form requiring identities.
3
Handle transformed angle carefully: if \(u=2\theta\), solve in the doubled range first, then map back.
4
Use CAST/sign logic: reference angle + allowed quadrants gives all solutions in interval.
5
Reject impossible values: if \(|\sin\theta|>1\) or \(|\cos\theta|>1\), discard branch immediately.
6
Never divide by variable trig factor unless guaranteed nonzero; factor and solve each factor instead.

PatternDomain adjustment every time: \(2x\), \(3x\), \(x/2\), and shifted arguments▶

Rule: solve for the whole argument first. If \(\theta=\text{(inside trig bracket)}\), transform the original \(x\)-domain into a \(\theta\)-domain before finding any angles.

Inside argumentOriginal domain \(x\in[a,b]\)Working domain for argumentMeaning
\(2x\)\([a,b]\)\([2a,2b]\)double sweep (more solutions)
\(3x\)\([a,b]\)\([3a,3b]\)triple sweep
\(x/2\)\([a,b]\)\([a/2,b/2]\)half sweep (fewer solutions)
\(Bx+C\)\([a,b]\)\([Ba+C,\ Bb+C]\)apply same operations to boundaries
Negative ranges stay negative: example \(-2\pi\le x\le2\pi\Rightarrow -4\pi\le2x\le4\pi\). Do not drop the clockwise (negative) part.

Five-line routine:

1
Isolate trig function (\(\sin,\cos,\tan\) alone on one side).
2
Set \(\theta=\) whole inside argument.
3
Transform domain boundaries into the \(\theta\)-window.
4
Find all \(\theta\)-solutions in that transformed window.
5
Back-substitute \(\theta=\) argument and solve for \(x\).

Topic 5
Trig Identities
Fundamental relationships that are always true — and how to prove new ones
Identities are equations that are true for every value of the angle. They are not equations to solve — they are relationships you use to simplify or transform expressions. The Pythagorean identity is the most important and appears everywhere.
DefinitionThe core identities▶

Pythagorean identities (derived from \(x^2+y^2=1\) on the unit circle):

\[ \sin^2\theta + \cos^2\theta = 1 \]
\[ 1 + \tan^2\theta = \sec^2\theta \qquad (\text{divide first by }\cos^2\theta) \]
\[ \cot^2\theta + 1 = \csc^2\theta \qquad (\text{divide first by }\sin^2\theta) \]

Quotient identity:

\[ \tan\theta = \frac{\sin\theta}{\cos\theta} \qquad \cot\theta = \frac{\cos\theta}{\sin\theta} \]

Rearrangements of the Pythagorean identity — memorise all three forms:

\[ \sin^2\theta = 1 - \cos^2\theta \qquad \cos^2\theta = 1 - \sin^2\theta \]
PatternThorough identity method: simplify and prove with control▶
1

2

3
Prioritize algebra before trig tricks: factorization, common denominator, cancellation (with domain awareness), then identities.
4
Use identity families deliberately: Pythagorean, quotient, reciprocal, double-angle/half-angle, conjugate rationalization.
5

6
Finish by matching exact target form (not just an equivalent-looking expression).

Topic 6
Sine & Cosine Rules
Solving non-right triangles — the complete toolkit
Right-triangle trig only works when there is a right angle. The sine rule and cosine rule handle any triangle. Knowing which rule to use — and when the ambiguous case applies — is what separates good students from great ones.
DefinitionBoth rules and the area formula▶

Sine Rule — for any triangle with sides a, b, c opposite angles A, B, C:

\[ \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} \]

Cosine Rule:

\[ a^2 = b^2 + c^2 - 2bc\cos A \]

Rearranged to find an angle:

\[ \cos A = \frac{b^2 + c^2 - a^2}{2bc} \]

Area of a triangle:

\[ \text{Area} = \frac{1}{2}ab\sin C \]
GivenUse
Two angles + one side (AAS or ASA)Sine Rule
Two sides + angle opposite one of them (SSA)Sine Rule (check ambiguous case)
Two sides + included angle (SAS)Cosine Rule
Three sides (SSS)Cosine Rule
PatternThe ambiguous case (SSA)▶

When given two sides and an angle NOT between them (SSA), there may be two valid triangles. When \(\sin B = k\) and you are finding angle B:

1
Find \(B_1 = \sin^{-1}(k)\) (acute angle).
2
Check if \(B_2 = 180° - B_1\) is valid: it is valid if \(A + B_2 < 180°\).
3
If valid, both triangles exist and you must give both solutions.
The ambiguous case only arises with the Sine Rule when you are finding an angle, not a side. The cosine rule, when finding an angle, gives a unique answer (cosine is 1-to-1 on [0°,180°]).
ExampleFour fully worked examples▶
Example 1 — Sine Rule: find side a given A=40°, B=75°, b=12 cm
1

ANSWER Area ≈ 12.9 cm²
NoteRule selection and the most common exam traps▶
Watch out: always label the triangle consistently. Side a is opposite angle A, side b opposite angle B. If you mix these up, the rule gives wrong answers.
Watch out: in the cosine rule, \(a^2 = b^2 + c^2 - 2bc\cos A\). Students often forget to halve the last term or apply the wrong sign. The formula for angle is always \(\cos A = (b^2+c^2-a^2)/(2bc)\).
If you get \(\cos A > 1\) or \(\cos A < -1\) when using the cosine rule, you have made an arithmetic error — no real angle exists outside this range.

Topic 7
Compound & Double Angle Formulas
Expanding sin(A±B), cos(A±B), and their double-angle special cases
The compound angle formulas let you find exact values of angles like 75° or 15° that are not in the standard exact value table. They also appear constantly in proofs and in solving equations. The double angle formulas are just special cases when A = B — they are worth knowing separately because they appear so often.
DefinitionAll the compound and double angle formulas▶

Compound angle formulas:

\[ \sin(A+B) = \sin A\cos B + \cos A\sin B \]
\[ \sin(A-B) = \sin A\cos B - \cos A\sin B \]
\[ \cos(A+B) = \cos A\cos B - \sin A\sin B \]
\[ \cos(A-B) = \cos A\cos B + \sin A\sin B \]
\[ \tan(A\pm B) = \frac{\tan A \pm \tan B}{1 \mp \tan A\tan B} \]

Double angle formulas (set B = A):

\[ \sin 2A = 2\sin A\cos A \]
\[ \cos 2A = \cos^2 A - \sin^2 A = 2\cos^2 A - 1 = 1 - 2\sin^2 A \]
\[ \tan 2A = \frac{2\tan A}{1 - \tan^2 A} \]
The three forms of cos 2A are all equal — choose whichever is most convenient. If the target has only cos terms, use \(2\cos^2A-1\). Only sin? Use \(1-2\sin^2A\). Both? Use \(\cos^2A-\sin^2A\).

Inverse Trig Functions
arcsin, arccos, arctan — and why domain restrictions matter
Sine, cosine, and tangent are not one-to-one functions — the same output can come from many different inputs. To define an inverse, we must restrict the domain. Getting the domain and range of each inverse function correct is essential — it determines which angle your calculator returns and why.
DefinitionWhy these domain and range choices are fixed▶

Core inverse-rule: if \(f^{-1}\) exists, then

\[ \text{domain}(f^{-1})=\text{range}(f), \qquad \text{range}(f^{-1})=\text{domain}(f) \]

For trig, we first choose a one-to-one branch (restricted angle interval), then invert on that branch.

InverseWhy input domain is thisWhy output range is this
\(\arcsin x\)\(\sin\theta\in[-1,1]\), so input must be \([-1,1]\).Use one-to-one sine branch \(\theta\in[-\pi/2,\pi/2]\), so outputs are in that interval.
\(\arccos x\)\(\cos\theta\in[-1,1]\), so input must be \([-1,1]\).Use one-to-one cosine branch \(\theta\in[0,\pi]\), so outputs are in that interval.
\(\arctan x\)\(\tan\theta\) can be any real number, so input is all reals.Use one-to-one tangent branch \(\theta\in(-\pi/2,\pi/2)\), endpoints excluded where tan is undefined.
Principal-value idea: the inverse function returns one standard angle only (its branch angle), not every coterminal angle that gives the same trig value.
Fast checks: \(\arcsin(1/2)=\pi/6\) (not \(5\pi/6\)); \(\arccos(1/2)=\pi/3\); \(\arctan(1)=\pi/4\).
PatternVisual branch explanation (circular form)▶

What must be true for an inverse: each output must come from exactly one input (one-to-one).

1
On \([0,\pi]\), sine is not one-to-one because \(\sin\theta=\sin(\pi-\theta)\).
2
Example pair: \(\sin(\pi/6)=1/2\) and \(\sin(5\pi/6)=1/2\), same y-value from two different angles.
3
So we choose \([-\pi/2,\pi/2]\), where sine increases strictly from \(-1\) to \(1\).
4
Endpoint check: \(\sin(\pm\pi/2)=\pm1\), so arcsin range includes endpoints; tan is undefined at \(\pm\pi/2\), so arctan range excludes them.
Invalid branch view: \([0,\pi]\) (fails one-to-one)
Two different angles hit the same y-value, so an inverse would return two answers for one input.
Valid branch view: \([-\pi/2,\pi/2]\) (works for arcsin)
Single sweep from bottom to top: sine increases from \(-1\) to \(1\), so inverse is single-valued.
Direction and endpoints for principal values
Positive principal angle = anticlockwise, negative principal angle = clockwise from +x-axis.

Important identities:

\[ \sin(\arcsin x) = x \qquad \arcsin(\sin\theta) = \theta \text{ only if } \theta\in\left[-\frac{\pi}{2},\frac{\pi}{2}\right] \]
