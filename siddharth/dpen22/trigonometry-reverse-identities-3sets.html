DPEN22 Trigonometry Reverse-Identity Practice

3 sets of 8 questions designed to train reverse recognition: identify when to condense to double-angle, sum/difference, product-to-sum, and Pythagorean transformations.
Focus: go both directions (expand and condense), especially patterns like \( \cos^2\theta-\sin^2\theta\to\cos2\theta \), \(2\sin A\cos A\to\sin2A\), and sum/difference angle compression.

Set A (8 Questions) — Double-Angle Recognition

  1. Write as a single trig function: \(\cos^2\theta-\sin^2\theta\).
  2. Write as a single trig function: \(2\sin x\cos x\).
  3. Condense: \(1-2\sin^2 t\).
  4. Condense: \(2\cos^2\alpha-1\).
  5. Rewrite in terms of \(2x\): \(\sin^2 x\).
  6. Rewrite in terms of \(2x\): \(\cos^2 x\).
  7. Simplify fully: \(\dfrac{1-\cos2u}{2}\).
  8. Simplify fully: \(\dfrac{1+\cos2u}{2}-\dfrac{1-\cos2u}{2}\).

Set B (8 Questions) — Angle Addition/Subtraction Reverse

  1. Condense: \(\sin A\cos B+\cos A\sin B\).
  2. Condense: \(\sin A\cos B-\cos A\sin B\).
  3. Condense: \(\cos A\cos B-\sin A\sin B\).
  4. Condense: \(\cos A\cos B+\sin A\sin B\).
  5. Express as a product: \(\cos(5x)+\cos(3x)\).
  6. Express as a product: \(\cos(5x)-\cos(3x)\).
  7. Express as a sum: \(2\sin(4x)\cos x\).
  8. Express as a sum: \(2\cos(3x)\cos(2x)\).

Set C (8 Questions) — Mixed Reverse Identity Selection

  1. Simplify: \((\cos^2\theta-\sin^2\theta)(2\sin\theta\cos\theta)\).
  2. Condense to a single cosine: \(\cos^2x-\sin^2x\) then evaluate at \(x=15^\circ\).
  3. Simplify: \(\sin(3x)\cos x+\cos(3x)\sin x\).
  4. Simplify: \(\cos(4x)\cos x-\sin(4x)\sin x\).
  5. Rewrite in terms of \(\cos(2x)\): \(4\cos^2x-2\).
  6. Simplify: \(\dfrac{1-\cos(2x)}{\sin x}\) (where defined).
  7. Condense: \(\sin(2x)\cos(3x)+\cos(2x)\sin(3x)\).
  8. Condense: \(\cos(2x)\cos(3x)+\sin(2x)\sin(3x)\).

Worked Answers — Set A

  1. \(\cos2\theta\)
  2. \(\sin2x\)
  3. \(\cos2t\)
  4. \(\cos2\alpha\)
  5. \(\sin^2x=\dfrac{1-\cos2x}{2}\)
  6. \(\cos^2x=\dfrac{1+\cos2x}{2}\)
  7. \(\sin^2u\)
  8. \(\cos2u\)

Worked Answers — Set B

  1. \(\sin(A+B)\)
  2. \(\sin(A-B)\)
  3. \(\cos(A+B)\)
  4. \(\cos(A-B)\)
  5. \(2\cos(4x)\cos x\)
  6. \(-2\sin(4x)\sin x\)
  7. \(\sin(5x)+\sin(3x)\)
  8. \(\cos x+\cos(5x)\)

Worked Answers — Set C

  1. \(\cos2\theta\sin2\theta=\dfrac12\sin4\theta\)
  2. \(\cos(2x)\), so at \(x=15^\circ\): \(\cos30^\circ=\dfrac{\sqrt3}{2}\)
  3. \(\sin(4x)\)
  4. \(\cos(5x)\)
  5. \(4\cos^2x-2=2(2\cos^2x-1)=2\cos2x\)
  6. \(\dfrac{2\sin^2x}{\sin x}=2\sin x\)
  7. \(\sin(5x)\)
  8. \(\cos(x)\)
