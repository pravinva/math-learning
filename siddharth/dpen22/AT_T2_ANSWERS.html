  1. Given \(A=\frac{\pi}{3}\), \(b=6\), \(c=8\). Use cosine rule: \[ a^2=b^2+c^2-2bc\cos A =36+64-2(6)(8)\cos\frac{\pi}{3} =100-48=52. \] \[ a=\sqrt{52}=2\sqrt{13}\approx 7.21. \]
  2. Given \(b=4\), \(c=8\), \(B=\frac{\pi}{6}\). Use sine rule: \[ \frac{b}{\sin B}=\frac{c}{\sin C} \Rightarrow \frac{4}{1/2}=\frac{8}{\sin C} \Rightarrow 8=\frac{8}{\sin C} \Rightarrow \sin C=1. \] So \(C=\frac{\pi}{2}\). Then \[ A=\pi-B-C =\pi-\frac{\pi}{6}-\frac{\pi}{2} =\frac{\pi}{3}. \]
  3. Let base be \(O=(0,0)\), tip be \(T=(x,y)\), and shadow end be \(S=(-12,0)\).
    Sun elevation \(32^\circ\) means ray \(ST\) makes angle \(32^\circ\) with horizontal: \[ y=\tan32^\circ\,(x+12). \] Pole length is \(10\): \[ x^2+y^2=100. \] Substitute: \[ x^2+\tan^2(32^\circ)(x+12)^2=100. \] Solving gives \(x\approx 3.18\) (the “leaning toward sun” root), \(y\approx 9.49\).
    Pole angle with horizontal: \[ \theta=\tan^{-1}\!\left(\frac{y}{x}\right)\approx\tan^{-1}(2.98)\approx 71.5^\circ. \]
  4. \(y=\frac13\cos\!\left(\frac{x}{2}\right)\).
    Amplitude \(=\frac13\).
    Period \(=\frac{2\pi}{1/2}=4\pi\).
  5. \(y=4\tan(2x)\).
    Tangent period \(=\frac{\pi}{2}\).
    Vertical stretch factor \(=4\). (No amplitude for tangent.)
  6. \(\sec x=-\frac32\) and \(\frac{\pi}{2}\le x\le\pi\) (Quadrant II).
    \[ \cos x=\frac{1}{\sec x}=-\frac23. \] Then \[ \sin x=\sqrt{1-\cos^2x} =\sqrt{1-\frac49} =\frac{\sqrt5}{3} \] (positive in QII). So \[ \cot x=\frac{\cos x}{\sin x} =\frac{-2/3}{\sqrt5/3} =-\frac{2}{\sqrt5} =-\frac{2\sqrt5}{5}. \]
  7. \[ \frac{\sin x\sec x}{\tan x+\cot x} =\frac{\tan x}{\frac{\sin x}{\cos x}+\frac{\cos x}{\sin x}} =\frac{\tan x}{\frac{\sin^2x+\cos^2x}{\sin x\cos x}} =\tan x\cdot \sin x\cos x =\sin^2x. \]
  8. Check: \[ \cos\!\left(\frac{3\pi}{2}-x\right) =\cos\frac{3\pi}{2}\cos x+\sin\frac{3\pi}{2}\sin x =0-\sin x =-\sin x. \] Statement says it equals \(\sin x\), so it is false.
  9. \[ \sin(x+y)+\sin(x-y)=2\sin x\cos y \] is a standard sum-to-product identity, so this is true.
  10. Show: \[ (\sec x-\tan x)^2=\frac{1-\sin x}{1+\sin x}. \] Start from LHS: \[ \sec x-\tan x =\frac{1}{\cos x}-\frac{\sin x}{\cos x} =\frac{1-\sin x}{\cos x}. \] Square: \[ (\sec x-\tan x)^2 =\frac{(1-\sin x)^2}{\cos^2x} =\frac{(1-\sin x)^2}{1-\sin^2x} =\frac{(1-\sin x)^2}{(1-\sin x)(1+\sin x)} =\frac{1-\sin x}{1+\sin x}. \] Done.
