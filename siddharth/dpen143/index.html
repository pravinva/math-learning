How to Study DPEN143 (Fast-Paced Survival Plan) NEW
DPEN143 feels fast because topics switch quickly. Study it as recurring problem patterns, not isolated chapters.
🧭 Pattern-first approach ⚙️ Formula-to-question mapping 📝 Exam-usable structure
Use a 3-layer structure for every topic
  1. Core idea in one sentence — what physical principle is controlling the situation.
  2. 3-5 key formulas + units — keep only formulas that repeatedly appear in exam questions.
  3. 2 canonical problem types — one direct calculation and one interpretation/setup-style question.
Master sheet of patterns (not chapter summaries)
1) Field/Force Pattern (Coulomb, Electric Field, Potential)
  • Core idea in one sentence: fields and potentials from multiple charges combine by superposition, with direction/sign determined by geometry.
  • Key formulas + units:
  • \(F = k\frac{|q_1 q_2|}{r^2}\) (N), \(\;k=8.99\times10^9\;\text{N m}^2\!/\text{C}^2\)
  • \(\vec E = k\frac{q}{r^2}\hat r\) (N/C), and \(\vec E_{net}=\sum \vec E_i\)
  • \(V = k\frac{q}{r}\) (V), and \(V_{net}=\sum V_i\)
  • \(\Delta U = q\Delta V\) (J), \(\;W=q\Delta V\)
  • Canonical problems: (i) compute net \(\vec E\) or \(V\) at a point from 2-3 charges; (ii) explain which components cancel by symmetry and which add.
2) Circuit Pattern (Ohm, Kirchhoff, RC/RL Transients)
  • Core idea in one sentence: choose circuit model first (series/parallel, KCL/KVL, or transient law), then solve systematically.
  • Key formulas + units:
  • \(V=IR\) (V, A, \(\Omega\)); \(\;P=VI=I^2R=V^2/R\) (W)
  • KCL: \(\sum I_{in}=\sum I_{out}\); KVL: \(\sum \Delta V_{loop}=0\)
  • RC: \(v_C(t)=V_f+(V_0-V_f)e^{-t/RC}\), \(\tau_{RC}=RC\) (s)
  • RL: \(i_L(t)=I_f+(I_0-I_f)e^{-tR/L}\), \(\tau_{RL}=L/R\) (s)
  • \(E_C=\tfrac12CV^2\) (J), \(\;E_L=\tfrac12LI^2\) (J)
  • Canonical problems: (i) find unknown currents/voltages in multi-loop network using KCL/KVL; (ii) transient value at specified time (\(t=\tau,\;2\tau,\;3\tau\)).
3) Wave/Optics Pattern (Interference, Diffraction, Snell/TIR)
  • Core idea in one sentence: wave behavior is controlled by phase/path difference and interface geometry.
  • Key formulas + units:
  • \(v=f\lambda\) (\(\text{m/s}=\text{Hz}\cdot\text{m}\)); \(\omega=2\pi f\) (rad/s)
  • Snell: \(n_1\sin\theta_1=n_2\sin\theta_2\) (angles in deg or rad)
  • Critical angle: \(\sin\theta_c=n_2/n_1\) for \(n_1>n_2\)
  • Double-slit maxima: \(d\sin\theta=m\lambda\); minima: \(d\sin\theta=(m+\tfrac12)\lambda\)
  • Single-slit minima: \(a\sin\theta=m\lambda\), \(m=1,2,\dots\)
  • Canonical problems: (i) compute fringe angle/spacing or diffraction minima location; (ii) determine whether refraction or TIR occurs and justify using \(\theta_c\).
4) Modern Physics Pattern (Photoelectric, de Broglie, Uncertainty)
  • Core idea in one sentence: microscopic behavior is quantized, so energy, momentum, and wavelength are linked by Planck-scale relations.
  • Key formulas + units:
  • \(E=hf=\frac{hc}{\lambda}\) (J), \(\;h=6.626\times10^{-34}\,\text{J s}\)
  • Photoelectric: \(K_{max}=hf-\phi\) (J or eV), \(\;eV_s=K_{max}\)
  • de Broglie: \(\lambda=\frac{h}{p}\) (m), with \(p=mv\) (kg m/s)
  • Uncertainty: \(\Delta x\,\Delta p \ge \hbar/2\), \(\Delta E\,\Delta t \ge \hbar/2\)
  • Canonical problems: (i) find threshold frequency, stopping potential, or emitted kinetic energy; (ii) compute de Broglie wavelength and discuss uncertainty implication.
How to run each study block
  • 10 min: write the one-sentence core idea + formula shortlist.
  • 25 min: solve 2 canonical questions without notes.
  • 10 min: error log (units, sign, model choice, algebra).
