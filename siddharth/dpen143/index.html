How to Study DPEN143 (Fast-Paced Survival Plan) NEW
DPEN143 feels fast because topics switch quickly. Study it as recurring problem patterns, not isolated chapters.
🧭 Pattern-first approach ⚙️ Formula-to-question mapping 📝 Exam-usable structure
Use a 3-layer structure for every topic
  1. Core idea in one sentence — what physical principle is controlling the situation.
  2. 3-5 key formulas + units — keep only formulas that repeatedly appear in exam questions.
  3. 2 canonical problem types — one direct calculation and one interpretation/setup-style question.
Master sheet of patterns (not chapter summaries)
1) Field/Force Pattern (Coulomb, Electric Field, Potential)
  • Core idea in one sentence: fields and potentials from multiple charges combine by superposition, with direction/sign determined by geometry.
  • Key formulas + units:
  • \(F = k\frac{|q_1 q_2|}{r^2}\) (N), \(\;k=8.99\times10^9\;\text{N m}^2\!/\text{C}^2\)
  • \(\vec E = k\frac{q}{r^2}\hat r\) (N/C), and \(\vec E_{net}=\sum \vec E_i\)
  • \(V = k\frac{q}{r}\) (V), and \(V_{net}=\sum V_i\)
  • \(\Delta U = q\Delta V\) (J), \(\;W=q\Delta V\)
  • Canonical problems: (i) compute net \(\vec E\) or \(V\) at a point from 2-3 charges; (ii) explain which components cancel by symmetry and which add.
2) Circuit Pattern (Ohm, Kirchhoff, RC/RL Transients)
  • Core idea in one sentence: choose circuit model first (series/parallel, KCL/KVL, or transient law), then solve systematically.
  • Key formulas + units:
  • \(V=IR\) (V, A, \(\Omega\)); \(\;P=VI=I^2R=V^2/R\) (W)
  • KCL: \(\sum I_{in}=\sum I_{out}\); KVL: \(\sum \Delta V_{loop}=0\)
  • RC: \(v_C(t)=V_f+(V_0-V_f)e^{-t/RC}\), \(\tau_{RC}=RC\) (s)
  • RL: \(i_L(t)=I_f+(I_0-I_f)e^{-tR/L}\), \(\tau_{RL}=L/R\) (s)
  • \(E_C=\tfrac12CV^2\) (J), \(\;E_L=\tfrac12LI^2\) (J)
  • Canonical problems: (i) find unknown currents/voltages in multi-loop network using KCL/KVL; (ii) transient value at specified time (\(t=\tau,\;2\tau,\;3\tau\)).
3) Wave/Optics Pattern (Interference, Diffraction, Snell/TIR)
  • Core idea in one sentence: wave behavior is controlled by phase/path difference and interface geometry.
  • Key formulas + units:
  • \(v=f\lambda\) (\(\text{m/s}=\text{Hz}\cdot\text{m}\)); \(\omega=2\pi f\) (rad/s)
  • Snell: \(n_1\sin\theta_1=n_2\sin\theta_2\) (angles in deg or rad)
  • Critical angle: \(\sin\theta_c=n_2/n_1\) for \(n_1>n_2\)
  • Double-slit maxima: \(d\sin\theta=m\lambda\); minima: \(d\sin\theta=(m+\tfrac12)\lambda\)
  • Single-slit minima: \(a\sin\theta=m\lambda\), \(m=1,2,\dots\)
  • Canonical problems: (i) compute fringe angle/spacing or diffraction minima location; (ii) determine whether refraction or TIR occurs and justify using \(\theta_c\).
4) Modern Physics Pattern (Photoelectric, de Broglie, Uncertainty)
  • Core idea in one sentence: microscopic behavior is quantized, so energy, momentum, and wavelength are linked by Planck-scale relations.
  • Key formulas + units:
  • \(E=hf=\frac{hc}{\lambda}\) (J), \(\;h=6.626\times10^{-34}\,\text{J s}\)
  • Photoelectric: \(K_{max}=hf-\phi\) (J or eV), \(\;eV_s=K_{max}\)
  • de Broglie: \(\lambda=\frac{h}{p}\) (m), with \(p=mv\) (kg m/s)
  • Uncertainty: \(\Delta x\,\Delta p \ge \hbar/2\), \(\Delta E\,\Delta t \ge \hbar/2\)
  • Canonical problems: (i) find threshold frequency, stopping potential, or emitted kinetic energy; (ii) compute de Broglie wavelength and discuss uncertainty implication.
How to run each study block
  • 10 min: write the one-sentence core idea + formula shortlist.
  • 25 min: solve 2 canonical questions without notes.
  • 10 min: error log (units, sign, model choice, algebra).

DPEN143 Topic-by-Topic Implementation NEW
This is the direct implementation: each topic below has its one-line core idea, exam-priority formulas with units, and two canonical question types.
🧱 3-layer per topic 📐 Formula + unit focus 🧪 Canonical question mapping
Week 1 — Electrostatics
  • Core idea: Fields from charges add by vector superposition with symmetry-based cancellation.
  • Formulas: \(F=k|q_1q_2|/r^2\) (N), \(\vec E=kq\hat r/r^2\) (N/C), \(V=kq/r\) (V), \(\Delta U=q\Delta V\) (J).
  • Canonical problems: (i) compute net \(E\) or \(V\) at a point; (ii) explain symmetry cancellation and surviving direction.
Week 2 — Capacitance, Current, Resistance
  • Core idea: Charge storage and current flow are linked by constitutive laws and energy balance.
  • Formulas: \(C=Q/V\) (F), \(C=\epsilon A/d\) (F), \(V=IR\) (V), \(P=VI\) (W), \(E_C=\tfrac12CV^2\) (J).
  • Canonical problems: (i) equivalent \(R\)/\(C\) and power/energy; (ii) choose series vs parallel reduction strategy.
Week 3 — Kirchhoff + Magnetism Intro
  • Core idea: Circuit conservation laws and magnetic-field source laws are both model-selection problems.
  • Formulas: KCL \(\sum I=0\), KVL \(\sum \Delta V=0\), \(F=qvB\sin\theta\) (N), \(B=\mu_0I/(2\pi r)\) (T).
  • Canonical problems: (i) solve unknown branch currents via KCL/KVL; (ii) determine magnetic force magnitude/direction.
Week 4 — Faraday + EM Waves
  • Core idea: Time-varying flux induces emf; wave speed/frequency/wavelength tie EM propagation together.
  • Formulas: \(\mathcal{E}=-d\Phi_B/dt\) (V), \(\Phi_B=BA\cos\theta\) (Wb), \(c=f\lambda\), \(E/B=c\).
  • Canonical problems: (i) induced emf from changing \(B\), \(A\), or angle; (ii) determine missing \(f,\lambda\), or \(c\)-based ratio.
Weeks 5 & 9 — Module Reviews
  • Core idea: Mixed questions test method choice more than raw calculation.
  • Formulas: all prior high-frequency laws; prioritize unit checks and sign conventions.
  • Canonical problems: (i) classify method before solving; (ii) repair a worked solution with a deliberate setup/sign error.
Week 6 — Special Relativity
  • Core idea: Space/time measurements depend on frame through Lorentz factor.
  • Formulas: \(\gamma=1/\sqrt{1-v^2/c^2}\), \(\Delta t=\gamma \Delta t_0\), \(L=L_0/\gamma\), \(E=\gamma mc^2\).
  • Canonical problems: (i) time dilation/length contraction numerics; (ii) identify proper time/length and frame assignment.
Week 7 — Photoelectric, X-ray, Hydrogen, Laser
  • Core idea: Quantum energy exchange is discrete, not continuous.
  • Formulas: \(E=hf=hc/\lambda\), \(K_{max}=hf-\phi\), \(eV_s=K_{max}\), \(\Delta E=13.6(1/n_f^2-1/n_i^2)\) eV.
  • Canonical problems: (i) threshold frequency/stopping potential; (ii) emission-line energy/wavelength reasoning.
Week 8 — Matter Waves, Uncertainty, Nuclear
  • Core idea: Matter has wave behavior and measurement limits are intrinsic.
  • Formulas: \(\lambda=h/p\), \(p=mv\), \(\Delta x\Delta p\ge\hbar/2\), \(N=N_0e^{-\lambda t}\), \(A=\lambda N\).
  • Canonical problems: (i) de Broglie wavelength and uncertainty bounds; (ii) decay activity/half-life calculations.
Week 10 — Wave Motion, Beats, Doppler
  • Core idea: Relative motion and close frequencies change observed wave behavior.
  • Formulas: \(v=f\lambda\), \(f_b=|f_1-f_2|\), \(f'=f\frac{v\pm v_o}{v\mp v_s}\).
  • Canonical problems: (i) beat frequency/source identification; (ii) Doppler observed frequency for moving source/observer.
Week 11 — Snell, TIR, Dispersion, Polarization
  • Core idea: Refraction behavior is interface-geometry constrained by refractive index.
  • Formulas: \(n_1\sin\theta_1=n_2\sin\theta_2\), \(\sin\theta_c=n_2/n_1\), \(v=c/n\), Malus: \(I=I_0\cos^2\theta\).
  • Canonical problems: (i) refracted angle vs TIR decision; (ii) transmitted intensity through polarizers.
Week 12 — Mirrors, Lenses, Optical Instruments
  • Core idea: Image formation is a sign-convention and geometry setup problem.
  • Formulas: \(1/f=1/d_o+1/d_i\), \(m=-d_i/d_o=h_i/h_o\), lens power \(P=1/f\) (D).
  • Canonical problems: (i) image position/type/magnification; (ii) instrument interpretation (virtual vs real image behavior).
Week 13 — Interference & Diffraction
  • Core idea: fringe/minima conditions come from phase/path-difference geometry.
  • Formulas: double-slit max: \(d\sin\theta=m\lambda\), min: \(d\sin\theta=(m+\tfrac12)\lambda\), single-slit min: \(a\sin\theta=m\lambda\), small-angle \(y\approx L\theta\).
  • Canonical problems: (i) fringe spacing or minima location; (ii) infer slit spacing/width from measured screen pattern.
