Week 5A · Topic 1
Composite Functions
Functions of functions — combining functions by placing one inside another.
Composite functions let you build complex transformations from simple ones. Think of it like a machine inside another machine: the output of the first function becomes the input of the second. The notation \((f \circ g)(x) = f(g(x))\) means "f of g of x".
Definition Composite Functions ▶

Functions can be added, subtracted, multiplied, and divided.

Further, we can find the composition of two functions by placing one inside the other. For two functions \(f\) and \(g\), this is denoted by:

\[(f \circ g)(x) = f(g(x))\]
Key idea: Start from the inside! Evaluate \(g(x)\) first, then use that result as the input for \(f\).

Important: \((f \circ g)(x)\) is generally NOT equal to \((g \circ f)(x)\). The order matters!

Pattern How to evaluate composite functions ▶

Method 1: Evaluate at a specific value

1
Evaluate the inner function at the given value
2
Use that result as input for the outer function
3
Simplify

Method 2: Find the general formula

1
Replace every \(x\) in the outer function with the entire inner function
2
Expand and simplify completely
Example Worked Examples ▶
Example 1 — Given \(f(x) = x^2 - 2\) and \(g(x) = 2 - x\), find \(f(g(x))\)
1
Start with \(f(x) = x^2 - 2\)
2
Replace every \(x\) with \((2-x)\): \(f(g(x)) = (2-x)^2 - 2\)
3
Expand: \((2-x)^2 = 4 - 4x + x^2\)
4
Simplify: \(4 - 4x + x^2 - 2 = x^2 - 4x + 2\)
ANSWER \(f(g(x)) = x^2 - 4x + 2\)
Example 2 — Given \(f(x) = x^2 - 2\) and \(g(x) = 2 - x\), find \(g(f(x))\)
1
Start with \(g(x) = 2 - x\)
2
Replace every \(x\) with \((x^2-2)\): \(g(f(x)) = 2 - (x^2-2)\)
3
Simplify: \(2 - x^2 + 2 = 4 - x^2\)
ANSWER \(g(f(x)) = 4 - x^2\)
Notice: \(f(g(x)) \neq g(f(x))\). The order of composition matters!
Example 3 — Given \(f(x) = x^2 + x\) and \(g(t) = \frac{1}{t}\), find \(f(g(2))\)
1
First evaluate \(g(2) = \frac{1}{2}\)
2
Now evaluate \(f(\frac{1}{2}) = (\frac{1}{2})^2 + \frac{1}{2}\)
3
Simplify: \(\frac{1}{4} + \frac{1}{2} = \frac{1}{4} + \frac{2}{4} = \frac{3}{4}\)
ANSWER \(\frac{3}{4}\)
Note Common Mistakes ▶
Mistake 1: Confusing \(f(g(x))\) with \(f(x) \cdot g(x)\). Composition is NOT multiplication!
Mistake 2: Forgetting to use parentheses when substituting. Always write \(f(g(x))\) as \(f(\text{entire expression for } g(x))\).
Mistake 3: Thinking order doesn't matter. Remember: \((f \circ g)(x) \neq (g \circ f)(x)\) in general.

Week 5A · Topic 2
Quadratic Functions
Understanding parabolas, their properties, and the quadratic formula.
Quadratic functions model countless real-world phenomena: projectile motion, profit functions, suspension bridge cables, satellite dishes, and more. The general form is \(f(x) = ax^2 + bx + c\) where \(a \neq 0\), and its graph is always a parabola.
Definition The Quadratic Function ▶

The general form of a quadratic function is:

\[f(x) = ax^2 + bx + c, \quad a \neq 0\]

Where:

  • \(a\) determines the concavity (if \(a > 0\), parabola opens upward; if \(a < 0\), downward)
  • \(b\) affects the position of the vertex
  • \(c\) is the y-intercept

The graph of a quadratic function is called a parabola.

Real-world applications: Projectile paths, profit/revenue functions, bridge cables, satellite dish shapes, optimization problems.
Pattern The Quadratic Formula ▶

To find where a parabola crosses the x-axis, we solve \(ax^2 + bx + c = 0\) using the quadratic formula:

\[x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\]

The solutions are called the roots or x-intercepts of the parabola.

1
Identify coefficients \(a\), \(b\), and \(c\)
2
Calculate the discriminant: \(\Delta = b^2 - 4ac\)
3
Substitute into the formula and simplify
Example Identifying Quadratic Functions ▶
Example 1 — Identify the coefficients in \(f(x) = 3x^2 - 5x + 2\)
1
\(a = 3\) (coefficient of \(x^2\))
2
\(b = -5\) (coefficient of \(x\))
3
\(c = 2\) (constant term)
4
Since \(a = 3 > 0\), the parabola opens upward
ANSWER \(a = 3, b = -5, c = 2\); opens upward
Example 2 — Find the y-intercept of \(y = 2x^2 + 4x - 3\)
1
The y-intercept occurs when \(x = 0\)
2
\(y = 2(0)^2 + 4(0) - 3 = -3\)
ANSWER y-intercept: \((0, -3)\)
Quick tip: The y-intercept is always \(c\) in the form \(ax^2 + bx + c\).
Note Key Properties of Parabolas ▶
  • Concavity: If \(a > 0\), parabola opens UP (minimum point); if \(a < 0\), opens DOWN (maximum point)
  • Vertex: The highest or lowest point on the parabola
  • Axis of symmetry: Vertical line through the vertex: \(x = -\frac{b}{2a}\)
  • Y-intercept: Always at \((0, c)\)
  • X-intercepts: Found using the quadratic formula (may be 0, 1, or 2 intercepts)

Week 5A · Topic 3
The Discriminant
Determine the nature and number of roots without solving the equation.
The discriminant \(\Delta = b^2 - 4ac\) is a powerful tool that tells you everything about the roots before you even solve the equation. It reveals whether roots are real or complex, rational or irrational, and how many there are!
Definition What is the Discriminant? ▶

In the quadratic formula \(x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\), the quantity under the square root is called the discriminant:

\[\Delta = b^2 - 4ac\]

The Greek letter \(\Delta\) (delta) represents the discriminant.

Why it matters: The discriminant determines whether the roots exist (are real), how many roots there are, and whether they're rational or irrational — all without solving!
Pattern Interpreting the Discriminant ▶

For the quadratic equation \(ax^2 + bx + c = 0\):

✓ CASE 1: \(\Delta > 0\)

Two distinct real roots
The parabola crosses the x-axis at two points.
Special case: If \(\Delta\) is a perfect square, the roots are rational.

✓ CASE 2: \(\Delta = 0\)

One real root (repeated/equal roots)
The parabola touches the x-axis at exactly one point (the vertex).

✗ CASE 3: \(\Delta < 0\)

No real roots
The parabola does not touch or cross the x-axis.
(The roots are complex/imaginary numbers)

Memory aid: Positive discriminant = 2 roots; Zero discriminant = 1 root; Negative discriminant = 0 real roots
Example Worked Examples ▶
Example 1 — Find the nature of roots for \(4x^2 + 11x - 3 = 0\)
1
Identify: \(a = 4\), \(b = 11\), \(c = -3\)
2
Calculate: \(\Delta = (11)^2 - 4(4)(-3) = 121 + 48 = 169\)
3
Since \(\Delta = 169 > 0\), there are two real roots
4
Since \(\sqrt{169} = 13\) is rational, the roots are rational
ANSWER Two distinct real rational roots
Example 2 — Find the nature of roots for \(x^2 - 4x + 4 = 0\)
1
Identify: \(a = 1\), \(b = -4\), \(c = 4\)
2
Calculate: \(\Delta = (-4)^2 - 4(1)(4) = 16 - 16 = 0\)
3
Since \(\Delta = 0\), there is one real root (equal roots)
ANSWER One real root (parabola touches x-axis at vertex)
Example 3 — Find the nature of roots for \(4x^2 + 3x + 1 = 0\)
1
Identify: \(a = 4\), \(b = 3\), \(c = 1\)
2
Calculate: \(\Delta = (3)^2 - 4(4)(1) = 9 - 16 = -7\)
3
Since \(\Delta = -7 < 0\), there are no real roots
ANSWER No real roots (parabola doesn't touch x-axis)
Note Advanced: Finding Values of Parameters ▶

You can use the discriminant to find values of unknown parameters that produce specific types of roots.

Example — For what values of \(k\) does \(2x^2 + 4x + 3k = 0\) have one solution?
1
For one solution, \(\Delta = 0\)
2
\(a = 2\), \(b = 4\), \(c = 3k\)
3
\((4)^2 - 4(2)(3k) = 0\)
4
\(16 - 24k = 0 \Rightarrow k = \frac{16}{24} = \frac{2}{3}\)
ANSWER \(k = \frac{2}{3}\)
Remember: When solving for parameters, you often need to solve inequalities (\(\Delta > 0\), \(\Delta < 0\)) or equations (\(\Delta = 0\)).

Week 5B · Topics 1-2
Graphing & Sketching Parabolas
Learn to find intercepts, vertices, axes of symmetry, and sketch accurate parabolas.
Understanding how to graph parabolas is essential for analyzing quadratic functions. You'll learn to identify key features: concavity (up/down), intercepts, vertex, and axis of symmetry. These skills apply to physics (projectile motion), economics (profit optimization), and engineering!
Definition Points of Intersection ▶

Y-intercept: The graph crosses the y-axis when \(x = 0\)

\[y = ax^2 + bx + c \Rightarrow \text{if } x = 0, \text{ then } y = c\]

X-intercepts: The graph crosses the x-axis when \(y = 0\)

\[y = ax^2 + bx + c \Rightarrow ax^2 + bx + c = 0\]

Solve using factoring or the quadratic formula. A parabola may have:

  • Two x-intercepts when \(\Delta > 0\) (cuts x-axis twice)
  • One x-intercept when \(\Delta = 0\) (touches x-axis once)
  • No x-intercepts when \(\Delta < 0\) (doesn't touch x-axis)
Pattern Finding the Vertex and Axis of Symmetry ▶

Every parabola has a vertical axis of symmetry passing through its vertex.

Method 1: Using x-intercepts (when parabola crosses x-axis)

1
Find the midpoint of the two x-intercepts
2
This x-value is the axis of symmetry
3
Substitute into the function to find the y-coordinate of the vertex

Method 2: Using the formula (always works)

\[\text{Axis of symmetry: } x = -\frac{b}{2a}\]
\[\text{Vertex: } \left(-\frac{b}{2a}, f\left(-\frac{b}{2a}\right)\right)\]
Pro tip: Use Method 2 when the parabola doesn't cross the x-axis or when you want to find the vertex quickly!
Example Sketching Parabolas — Step by Step ▶
Example — Sketch \(y = x^2 - x - 2\)
1
Concavity: \(a = 1 > 0\), so parabola opens upward
2
Y-intercept: \(y = 0^2 - 0 - 2 = -2\) → \((0, -2)\)
3
X-intercepts: \(x^2 - x - 2 = 0 \Rightarrow (x-2)(x+1) = 0\) → \(x = 2\) or \(x = -1\)
4
Vertex: Midpoint of \(x = -1\) and \(x = 2\) is \(x = 0.5\)
5
At \(x = 0.5\): \(y = (0.5)^2 - 0.5 - 2 = -2.25\) → Vertex: \((0.5, -2.25)\)
KEY FEATURES Vertex: \((0.5, -2.25)\); X-int: \((-1,0), (2,0)\); Y-int: \((0,-2)\)
Note Steps to Sketch Any Parabola ▶

Complete checklist for sketching \(y = ax^2 + bx + c\):

  1. Determine concavity: \(a > 0\) (opens up) or \(a < 0\) (opens down)
  2. Find y-intercept: set \(x = 0\), get \(y = c\)
  3. Find x-intercepts: set \(y = 0\), solve for \(x\) (may be 0, 1, or 2 solutions)
  4. Find vertex using \(x = -\frac{b}{2a}\), then substitute to get y-coordinate
  5. Plot all points and draw a smooth U-shaped curve

Domain & Range of Parabolas
Determine the input (domain) and output (range) values for quadratic functions.
Understanding domain and range is crucial for analyzing functions. Domain = all possible x-values (inputs); Range = all possible y-values (outputs). For parabolas, domain is always all real numbers, but range depends on whether the parabola opens up or down!
Definition Domain and Range ▶

Domain: The set of all x-values for which \(f(x)\) is defined (inputs)

Range: The set of all y-values obtained from the domain (outputs)

For all quadratic functions \(y = ax^2 + bx + c\):
Domain is ALWAYS \((-\infty, \infty)\) — all real numbers!

Finding the Range:

  • Find the y-coordinate of the vertex (the maximum or minimum value)
  • If \(a > 0\): parabola opens up → range is \([y_{\text{vertex}}, \infty)\)
  • If \(a < 0\): parabola opens down → range is \((-\infty, y_{\text{vertex}}]\)
Pattern Steps to Find Domain & Range Algebraically ▶
1
Domain: Write \((-\infty, \infty)\) (always for quadratics!)
2
Find x-coordinate of vertex: \(x = -\frac{b}{2a}\)
3
Substitute to find y-coordinate of vertex
4
If \(a > 0\): range is \([y_{\text{vertex}}, \infty)\)
5
If \(a < 0\): range is \((-\infty, y_{\text{vertex}}]\)
Example Worked Examples ▶
Example 1 — Find domain and range of \(y = x^2 - 3\)
1
Domain: \((-\infty, \infty)\)
2
\(a = 1 > 0\), so parabola opens upward (has minimum)
3
Vertex: \(x = 0\), \(y = -3\)
4
Minimum value is -3, so range: \(y \geq -3\)
ANSWER Domain: \((-\infty, \infty)\); Range: \([-3, \infty)\)
Example 2 — Find domain and range of \(f(x) = -x^2 + 1\)
1
Domain: \((-\infty, \infty)\)
2
\(a = -1 < 0\), so parabola opens downward (has maximum)
3
Vertex: \(x = 0\), \(y = 1\)
4
Maximum value is 1, so range: \(y \leq 1\)
ANSWER Domain: \((-\infty, \infty)\); Range: \((-\infty, 1]\)
Example 3 — Find domain and range of \(y = x^2 - 3x + 4\)
1
Domain: \((-\infty, \infty)\)
2
\(a = 1 > 0\) (opens up, has minimum)
3
Vertex x-coordinate: \(x = -\frac{-3}{2} = \frac{3}{2}\)
4
\(y = (\frac{3}{2})^2 - 3(\frac{3}{2}) + 4 = \frac{9}{4} - \frac{9}{2} + 4 = \frac{7}{4}\)
ANSWER Domain: \((-\infty, \infty)\); Range: \([\frac{7}{4}, \infty)\)
Note Common Mistakes ▶
Mistake 1: Forgetting to use interval notation correctly. Use square brackets \([\text{ or }]\) when including the endpoint, round brackets \((\text{ or })\) when excluding.
Mistake 2: Confusing domain and range direction. Domain is always horizontal (x-axis), range is vertical (y-axis).
Mistake 3: Not checking concavity. If \(a < 0\), range has upper bound, not lower bound!

Absolute Value Functions
Graph and analyze absolute value functions and their transformations.
Absolute value functions create V-shaped graphs. The absolute value \(|x|\) makes all negative values positive, creating a reflection in the x-axis. These functions appear in distance problems, error analysis, and optimization!
Definition Absolute Value Function ▶

Definition of absolute value:

\[|x| = \begin{cases} x, & \text{if } x \geq 0 \\ -x, & \text{if } x < 0 \end{cases}\]

The basic absolute value function is \(y = |x|\), which forms a V-shape with vertex at the origin.

Key property: Taking the absolute value of a negative number makes it positive. This reflects the negative portion of the graph across the x-axis.

General form: \(y = |ax + b| + c\)

  • \(b\) causes horizontal shift: \(y = |x + b|\) shifts left/right
  • \(c\) causes vertical shift: \(y = |x| + c\) shifts up/down
  • \(a\) affects steepness: larger \(|a|\) means steeper V-shape
Pattern Graphing Absolute Value Functions ▶

Method: Find intercepts and use symmetry

1
Find y-intercept: set \(x = 0\)
2
Find x-intercept(s): set \(y = 0\) and solve \(|ax + b| + c = 0\)
3
The x-intercept is the vertex (lowest/highest point)
4
Find one more point on each side for accuracy
5
Draw V-shape symmetric about the vertex
Shortcut for \(y = |ax + b| + c\): The vertex is where the expression inside absolute value equals zero: \(ax + b = 0\)
Example Worked Examples ▶
Example 1 — Graph \(y = |x| - 5\)
1
Vertex form with \(c = -5\): shifted down 5 units
2
Vertex at \((0, -5)\)
3

4
Y-intercept: \(y = -5\)
KEY FEATURES Vertex: \((0,-5)\); X-int: \((-5,0), (5,0)\); Y-int: \((0,-5)\)
Example 2 — Graph \(y = |x + 2| - 3\)
1
Shifted left 2, down 3
2
Vertex where \(x + 2 = 0\): \(x = -2\), \(y = -3\) → \((-2, -3)\)
3
Y-intercept: \(y = |2| - 3 = -1\)
4
X-intercepts: \(|x+2| = 3 \Rightarrow x + 2 = \pm 3\) → \(x = 1\) or \(x = -5\)
KEY FEATURES Vertex: \((-2,-3)\); X-int: \((1,0), (-5,0)\); Y-int: \((0,-1)\)
Example 3 — Graph \(y = |2x - 3|\)
1
Vertex where \(2x - 3 = 0\): \(x = \frac{3}{2}\), \(y = 0\)
2
Y-intercept: \(y = |-3| = 3\)
3

KEY FEATURES Vertex: \((\frac{3}{2}, 0)\); X-int: \((\frac{3}{2}, 0)\); Y-int: \((0,3)\)
Note Domain & Range of Absolute Value Functions ▶

For all absolute value functions \(y = |ax + b| + c\):

Domain: Always \((-\infty, \infty)\) — no restrictions on x

Range: Depends on the minimum value (the y-coordinate of the vertex)

  • Since \(|ax + b| \geq 0\) always, the minimum value of \(|ax + b| + c\) is \(c\)
  • Range: \([c, \infty)\)
Quick rule: For \(y = |ax + b| + c\), range is always \([c, \infty)\)
