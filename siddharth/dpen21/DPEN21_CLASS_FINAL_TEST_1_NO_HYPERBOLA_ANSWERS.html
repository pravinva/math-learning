DPEN21 CLASS FINAL TEST 1 NO HYPERBOLA — Worked Solutions

Full worked solutions (separate answer key).
  1. Slope $m=\frac{7-(-1)}{6-2}=2$. Point-slope: $y+1=2(x-2)\Rightarrow y=2x-5$.
  2. From $x+y=5$, $y=5-x$. Substitute: $2x-3(5-x)=7\Rightarrow5x=22\Rightarrow x=\frac{22}{5}$, $y=\frac{3}{5}$.
  3. $y=(x-1)^2-4$ gives vertex $(1,-4)$. For x-intercepts: $(x-1)^2=4\Rightarrow x=-1,3$. y-intercept: $x=0\Rightarrow y=-3$.
  4. Complete square: $(x^2-6x)+(y^2+8y)=11\Rightarrow(x-3)^2+(y+4)^2=36$. Centre $(3,-4)$, radius $6$.
  5. $9-x^2\ge0\Rightarrow -3\le x\le3$. Since square root, $0\le y\le3$.
  6. $f(g(x))=(3x-2)^2+1=9x^2-12x+5$. Also $f(1)=2$, so $g(f(1))=g(2)=4$.
  7. $\log_5(x-1)=2\Rightarrow x-1=5^2=25\Rightarrow x=26$.
  8. $\ln(2x+1)=3\Rightarrow2x+1=e^3\Rightarrow x=\frac{e^3-1}{2}$.
  9. Use log laws: $\log40-\log5=\log8$, and $2\log x=\log x^2$. So expression $=\log(8x^2)$.
  10. $27=3^3$, so $3^{2x-1}=3^3\Rightarrow2x-1=3\Rightarrow x=2$.
  11. $e^{x+1}=20\Rightarrow x+1=\ln20\Rightarrow x=\ln20-1\approx1.996$.
  12. $y=e^{x-2}-1$: horizontal asymptote $y=-1$, domain all real, range $(-1,\infty)$. x-intercept from $0=e^{x-2}-1\Rightarrow x=2$. y-intercept $x=0$: $e^{-2}-1$.
