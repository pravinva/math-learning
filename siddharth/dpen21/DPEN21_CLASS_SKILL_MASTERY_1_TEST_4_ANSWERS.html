DPEN21 CLASS SKILL MASTERY 1 TEST 4 — Worked Solutions

Full worked solutions (separate answer key).
  1. $(a+b)^2-ab=(3-1)^2-(3)(-1)=4+3=7$.
  2. $(4x-3)(x-2)=4x^2-8x-3x+6=4x^2-11x+6$.
  3. Using row for power 6: $(2x+1)^6=64x^6+192x^5+240x^4+160x^3+60x^2+12x+1$.
  4. $9x^2-24x+16=(3x-4)^2$.
  5. $8x^3+1=(2x)^3+1^3=(2x+1)(4x^2-2x+1)$.
  6. $\frac{45x^3}{15x^{-1}}=3x^{3-(-1)}=3x^4$.
  7. $\frac{3m^2}{14n}\div\frac{9m}{7n^2}=\frac{3m^2}{14n}\cdot\frac{7n^2}{9m}=\frac{mn}{6}$.
  8. $\frac{2}{x}+\frac{5}{3x}=\frac{6}{3x}+\frac{5}{3x}=\frac{11}{3x}$.
  9. $3\sqrt{20}=6\sqrt5$, $\sqrt{45}=3\sqrt5$. So $6\sqrt5-3\sqrt5+\sqrt5=4\sqrt5$.
  10. $\frac{6}{\sqrt7+\sqrt2}\cdot\frac{\sqrt7-\sqrt2}{\sqrt7-\sqrt2}=\frac{6(\sqrt7-\sqrt2)}{5}$.
  11. Inside bracket: $\frac{x^{-2}y^3}{xy^{-1}}=x^{-3}y^4$. Raise to $-1$: $x^3y^{-4}=\frac{x^3}{y^4}$.
  12. $\sqrt[3]{a^2b^5}=(a^2b^5)^{1/3}=a^{2/3}b^{5/3}$.
