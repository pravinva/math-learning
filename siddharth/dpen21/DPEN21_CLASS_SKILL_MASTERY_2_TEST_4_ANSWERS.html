DPEN21 CLASS SKILL MASTERY 2 TEST 4 — Worked Solutions

Full worked solutions (separate answer key).
  1. $2(5x-1)-3(x+4)=7\Rightarrow10x-2-3x-12=7\Rightarrow7x=21\Rightarrow x=3$.
  2. Cross multiply: $4(2x-1)=3(x+5)\Rightarrow8x-4=3x+15\Rightarrow5x=19\Rightarrow x=\frac{19}{5}$.
  3. $x^2+12x+20=0=(x+10)(x+2)$, so $x=-10,-2$.
  4. Let $u=2x-1$. Then $u^2-6u+5=0=(u-1)(u-5)$. So $u=1,5$. Hence $2x-1=1$ or $5$, giving $x=1,3$.
  5. $|x-7|+2=6\Rightarrow|x-7|=4\Rightarrow x-7=\pm4$, so $x=11$ or $3$.
  6. $\frac{x-1}{2}<3\Rightarrow x-1<6\Rightarrow x<7$. Interval $(-\infty,7)$.
  7. $m=\frac{1-(-2)}{7-0}=\frac37$.
  8. Given slope $-\frac13$, perpendicular slope is $3$. Through $(6,2)$: $y-2=3(x-6)\Rightarrow y=3x-16$.
  9. From $3x+y=7$, $y=7-3x$. Substitute into $5x-2y=9$: $5x-2(7-3x)=9\Rightarrow11x=23\Rightarrow x=\frac{23}{11}$, $y=\frac{8}{11}$.
  10. $f(2)=2^2-2(2)=0$. Then $g(f(2))=g(0)=3$.
  11. $f(-x)=(-x)^4+2=x^4+2=f(x)$, so even.
  12. $x^2-9\ne0\Rightarrow x\ne\pm3$. Domain $(-\infty,-3)\cup(-3,3)\cup(3,\infty)$.
