2) Indices Cheat Sheet (2C + 2D)

\[ a^m a^n = a^{m+n},\quad \frac{a^m}{a^n}=a^{m-n},\quad (a^m)^n=a^{mn} \] \[ (ab)^n=a^n b^n,\quad \left(\frac{a}{b}\right)^n=\frac{a^n}{b^n},\quad a^0=1\ (a\neq0) \] \[ a^{-n}=\frac{1}{a^n},\quad a^{m/n}=\sqrt[n]{a^m}=(\sqrt[n]{a})^m \]
Common error: \((a+b)^n \ne a^n+b^n\) in general.
Technique: simplify one layer at a time: brackets → powers → multiply/divide → final positive-index form.

3) Equations Technique Ladder (3A + 3B)

Linear / Fraction Equations

  • Clear fractions using LCD (or valid cross-multiplication when one fraction per side).
  • Collect variable terms on one side, constants on the other.
  • For variable in denominator, state restrictions first (denominator \(\ne 0\)).

Quadratics

  • Factorisation: fastest when obvious factors exist.
  • Completing square: good for exact + decimal form.
  • Quadratic formula: always works: \[ x=\frac{-b\pm\sqrt{b^2-4ac}}{2a} \]
  • Reducible form: if powers are \(x^4,x^2,\text{constant}\), set \(u=x^2\).
Exam trick: after solving, substitute quickly to verify and reject extraneous roots.

4) Formula Rearrangement Cheats (3C)

Core process

  • Target variable on one side only.
  • Use inverse operations on both sides.
  • If power is even, remember \(\pm\) when taking roots in equation-solving contexts.
Example forms: \[ \frac{8}{x}=y-4 \Rightarrow x=\frac{8}{y-4} \] \[ \frac{1}{R_t}=\frac{1}{R_1}+\frac{1}{R_2} \Rightarrow R_t=\frac{R_1R_2}{R_1+R_2} \]

5) Absolute Value + Inequalities (3C + 3D)

Absolute value equations

  • \(|A|=k\Rightarrow A=\pm k\) (with \(k\ge0\)).
  • When right side has variable, check domain/sign and verify solutions.

Inequalities

  • Add/subtract same value: sign unchanged.
  • Multiply/divide by negative: flip sign.
  • Use interval notation in final answer.
\[ |x-a|r \Rightarrow xa+r \]
