Time: 40 minutes | 12 MCQs | 2 marks each
Worked answers (separate): open answer file.

Questions

  1. [Indices] Simplify \((3a^4)\div a^{-2}\).
  2. [Indices] Write with positive indices: \(\left(\frac{x^{-2}y^3}{x\,y^{-1}}\right)\).
  3. [Indices] Simplify \(x^{5/2}\div x^{1/2}\).
  4. [Linear equation] Solve \(\frac{5y-1}{6}+\frac{y+2}{3}=4\).
  5. [Quadratic formula] Solve \(x^2-7x+10=0\).
  6. [Completing square] Solve \(x^2+6x+2=0\) by completing square.
  7. [Power-4 substitution] Solve \(x^4-13x^2+36=0\) using \(u=x^2\).
  8. [Powers equation] Solve \(5^{x-2}=25\).
  9. [Change subject] Rearrange \(\frac{3}{x}-\frac{1}{y}=2\) for \(x\).
  10. [Inequality] Solve \(-7\le 4x+1<9\).
  11. [Absolute value inequality] Solve \(|x+5|\ge 4\).
  12. [Absolute value equation] Solve \(|3x-2|=10\).
