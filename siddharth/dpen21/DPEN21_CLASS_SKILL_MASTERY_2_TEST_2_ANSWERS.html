DPEN21 CLASS SKILL MASTERY 2 TEST 2 — Worked Solutions

Full worked solutions (separate answer key).
  1. $4(3x-2)=5x+7\Rightarrow12x-8=5x+7\Rightarrow7x=15\Rightarrow x=\frac{15}{7}$.
  2. $\frac53x-\frac23=4\Rightarrow\frac53x=\frac{14}{3}\Rightarrow x=\frac{14}{5}$.
  3. $2x^2-9x-18=0=(2x+3)(x-6)$, so $x=-\frac32$ or $6$.
  4. Let $u=x-1$. Then $u^2-7u+12=0=(u-3)(u-4)$. So $u=3,4$, hence $x=4,5$.
  5. Need RHS $\ge0$: $3x-1\ge0\Rightarrow x\ge\frac13$. Cases: $x+5=3x-1\Rightarrow x=3$; and $-(x+5)=3x-1\Rightarrow x=-1$ (reject, not in domain). So $x=3$.
  6. $5-2x\ge9\Rightarrow-2x\ge4\Rightarrow x\le-2$. Interval $(-\infty,-2]$.
  7. $m=\frac{9-1}{1-(-3)}=\frac84=2$.
  8. Parallel to $y=2x-3$ means slope $2$. Through $(-1,5)$: $y-5=2(x+1)\Rightarrow y=2x+7$.
  9. From $x-2y=5$, $x=5+2y$. Substitute into $3x+4y=7$: $15+6y+4y=7\Rightarrow y=-\frac45$, $x=\frac{17}{5}$.
  10. $h(2a)=2(2a)^2-3(2a)=8a^2-6a$.
  11. $f(-x)=(-x)^3-(-x)=-x^3+x=-(x^3-x)=-f(x)$, so odd.
  12. $2x-5\ge0\Rightarrow x\ge\frac52$. Domain $[\frac52,\infty)$.
