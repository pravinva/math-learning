DPEN21 Class Skill Mastery Test 1D

Built from Week class-question styles.
12 questions. Suggested time: 50 minutes.
  1. If $a=3$, $b=-1$, evaluate $(a+b)^2-ab$.
  2. Expand and simplify $(4x-3)(x-2)$.
  3. Use Pascal's triangle to expand $(2x+1)^6$.
  4. Factorise completely: $9x^2-24x+16$.
  5. Factorise completely: $8x^3+1$.
  6. Simplify: $\dfrac{45x^3}{15x^{-1}}$.
  7. Simplify: $\left(\dfrac{3m^2}{14n}\right)\div\left(\dfrac{9m}{7n^2}\right)$.
  8. Simplify: $\dfrac{2}{x}+\dfrac{5}{3x}$.
  9. Simplify: $3\sqrt{20}-\sqrt{45}+\sqrt5$.
  10. Rationalise the denominator: $\dfrac{6}{\sqrt7+\sqrt2}$.
  11. Write with positive indices: $\left(\dfrac{x^{-2}y^3}{x y^{-1}}\right)^{-1}$.
  12. Write in index form: $\sqrt[3]{a^2b^5}$.
