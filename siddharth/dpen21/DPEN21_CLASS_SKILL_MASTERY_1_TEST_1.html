DPEN21 Class Skill Mastery Test 1A

Built from Week class-question styles.
12 questions. Suggested time: 50 minutes.
  1. If $a=4$, $b=-3$, $c=\tfrac14$, evaluate $\dfrac{a-b}{c}+ab$.
  2. Expand and simplify $(3x-1)(x+5)$.
  3. Use Pascal's triangle to expand $(2x-1)^5$.
  4. Factorise completely: $6x^2-15x$.
  5. Factorise by grouping: $3ax+6ay+bx+2by$.
  6. Simplify: $\dfrac{12x^3y^2}{18x^2y}$.
  7. Simplify: $\left(\dfrac{3x^2}{5y}\right)\times\left(\dfrac{10y^2}{9x}\right)$.
  8. Simplify: $\dfrac{2}{x}+\dfrac{3}{2x}$.
  9. Simplify: $\sqrt{75}-\sqrt{12}+\sqrt{27}$.
  10. Rationalise the denominator: $\dfrac{5}{2-\sqrt{3}}$.
  11. Write with positive indices: $\dfrac{x^{-3}y^2}{x^{-1}y^{-4}}$.
  12. Write without fractional indices: $x^{3/2}y^{1/2}$.
