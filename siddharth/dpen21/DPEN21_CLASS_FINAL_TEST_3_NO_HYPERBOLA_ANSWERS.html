DPEN21 CLASS FINAL TEST 3 NO HYPERBOLA — Worked Solutions

Full worked solutions (separate answer key).
  1. Intercept form: $\frac{x}{3}+\frac{y}{-6}=1\Rightarrow2x-y=6$.
  2. Set $2x+1=-3x+11\Rightarrow5x=10\Rightarrow x=2$, then $y=5$.
  3. For $y=-2x^2+8x-3$, axis $x=-\frac{b}{2a}=-\frac{8}{-4}=2$. Vertex value: $y(2)=5$, so vertex $(2,5)$.
  4. Already in centre-radius form: centre $(-1,4)$, radius $4$.
  5. $y=\frac1{x-3}$: domain $x\ne3$, range $y\ne0$.
  6. If $f(x)=x^2-4$ with $x\ge0$, then $f^{-1}(5)$ solves $x^2-4=5\Rightarrow x=3$ (principal branch).
  7. $\log_3(2x-1)=2\Rightarrow2x-1=9\Rightarrow x=5$.
  8. $\ln x+\ln(x-3)=2\Rightarrow\ln[x(x-3)]=2\Rightarrow x^2-3x=e^2$. Solve $x=\frac{3\pm\sqrt{9+4e^2}}{2}$; domain $x>3$, keep $x=\frac{3+\sqrt{9+4e^2}}{2}$.
  9. $\log_a(a^4)-\frac12\log_a(a^6)=4-\frac12(6)=1$.
  10. $10^{x+1}=10^{-2}\Rightarrow x+1=-2\Rightarrow x=-3$.
  11. $e^{2x}=15\Rightarrow2x=\ln15\Rightarrow x=\frac12\ln15\approx1.354$.
  12. $y=3-2e^{-x}$: as $x\to\infty$, $e^{-x}\to0$, asymptote $y=3$. y-int at $x=0$: $y=1$. x-int: $0=3-2e^{-x}\Rightarrow e^{-x}=\frac32\Rightarrow x=-\ln(3/2)$.
