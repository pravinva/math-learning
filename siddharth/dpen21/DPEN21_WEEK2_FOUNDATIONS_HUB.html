Section 2 — Lesson Blocks

Lesson A (Week 2A): Algebraic Fractions

  • Simplify/multiply/divide/add fractions using factor-first approach.
  • Cancellation only across common factors, never across terms joined by +/−.

Lesson B (Week 2B): Surds + Rationalising

  • Simplify using perfect-square factors; combine like surds only.
  • Rationalise single-surd and binomial denominators with conjugates.

Lesson C (Week 2C): Positive Index Rules

  • Product, quotient, power, product-to-power, quotient-to-power laws.
  • Write final answers cleanly with simplified exponents.

Lesson D (Week 2D): Zero/Negative/Fractional Indices

  • \(a^0=1,\ a^{-n}=1/a^n,\ a^{m/n}=\sqrt[n]{a^m}\).
  • Convert to positive indices and simplify fully.

Lesson E: Challenge Set Patterns

  • Error-spotting, advanced rationalising, mixed-index pressure questions.
  • Application-style patterns: dB intensity and half-life style decay forms.
