Section B — Long Answer

  1. \(m=\frac{4-(-2)}{-1-5}=\frac{6}{-6}=-1\). Using \((5,-2)\): \(-2=-1(5)+b\Rightarrow b=3\). So \(y=-x+3\).
  2. \(d=\sqrt{(4-(-1))^2+(14-2)^2}=\sqrt{25+144}=\sqrt{169}=13\).
  3. \(3x-y+5=0\Rightarrow y=3x+5\), slope \(3\); parallel line also has slope \(3\). Through \((2,-1)\): \(y+1=3(x-2)\Rightarrow 3x-y-7=0\).
  4. Substitute \(y=x^2\) into \(y=2x+3\): \(x^2=2x+3\Rightarrow x^2-2x-3=0\Rightarrow (x-3)(x+1)=0\). So \(x=3\ (y=9)\) or \(x=-1\ (y=1)\). Points \((3,9)\) and \((-1,1)\).
  5. \(g(0)=0^2+4=4\Rightarrow f(4)=\sqrt4=2\). \((g\circ f)(x)=g(\sqrt{x})=(\sqrt{x})^2+4=x+4\), with domain \(x\ge0\).
  6. \(f(-x)=(-x)^2-|-x|=x^2-|x|=f(x)\). Even.
  7. \(x_v=\frac{12}{2(3)}=2,\ y_v=3(4)-24+7=-5\Rightarrow (2,-5)\). Since \(a=3>0\), it is a minimum.
  8. Two distinct real roots \(\Rightarrow \Delta=(-k)^2-4(1)(4)>0\Rightarrow k^2>16\Rightarrow k<-4\) or \(k>4\).
  9. \(x^2+y^2-4x+6y+9=0\Rightarrow (x-2)^2+(y+3)^2=4\). So centre \((2,-3)\), radius \(2\). x-axis intercepts: set \(y=0\Rightarrow (x-2)^2+9=4\), no real solutions (no x-intercepts). y-axis intercept: set \(x=0\Rightarrow 4+(y+3)^2=4\Rightarrow y=-3\), so \((0,-3)\). Sketch below.
  10. \(y=-2|x-1|+5\): vertex \((1,5)\). x-intercepts from \(-2|x-1|+5=0\Rightarrow |x-1|=\frac52\Rightarrow x=-\frac32,\ \frac72\). y-intercept at \(x=0\): \(y=3\). Sketch below.
  11. \(\log_2 8 + \log_2 2 - \log_2 4 = 3 + 1 - 2 = 2\). (Equivalently \(\log_2\frac{8\cdot2}{4}=\log_2 4=2\).)
  12. \(200(0.5)^{t}=25\Rightarrow (0.5)^{t}=\tfrac18=(0.5)^3\Rightarrow t=3\) days.
  13. \(\ln[x(x-3)]=\ln 4\Rightarrow x^2-3x=4\Rightarrow x^2-3x-4=0\Rightarrow (x-4)(x+1)=0\). Domain requires \(x>3\), so \(x=4\) (reject \(x=-1\)).
  14. \(y=-x^2+2x+3\): vertex \((1,4)\), x-intercepts \(-1,3\), y-intercept \((0,3)\), opens downward. Sketch below.
  15. \(y=2e^{-x}\): y-intercept \((0,2)\); horizontal asymptote \(y=0\); decreasing. Sketch below.
  16. \(y=\ln x + 2\): x-intercept where \(\ln x=-2\Rightarrow x=e^{-2}\approx0.135\); vertical asymptote where \(x=0\). Sketch below.
