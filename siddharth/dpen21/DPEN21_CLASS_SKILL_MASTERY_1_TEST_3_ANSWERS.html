DPEN21 CLASS SKILL MASTERY 1 TEST 3 — Worked Solutions

Full worked solutions (separate answer key).
  1. $3(-2)^2-4(-2)+1=12+8+1=21$.
  2. $(x-4)^2-(x+1)(x-1)=(x^2-8x+16)-(x^2-1)=-8x+17$.
  3. $(3x+2)^4=(3x)^4+4(3x)^3(2)+6(3x)^2(2)^2+4(3x)(2)^3+2^4=81x^4+216x^3+216x^2+96x+16$.
  4. $12x^2-27x+15=3(4x^2-9x+5)=3(4x-5)(x-1)$.
  5. $x^2+3x+2x+6=x(x+3)+2(x+3)=(x+2)(x+3)$.
  6. $\frac{20x^2y^3}{25xy^2}=\frac45xy$.
  7. $\frac{5a}{6b}\cdot\frac{9b^2}{10a^2}=\frac{45ab^2}{60a^2b}=\frac{3b}{4a}$.
  8. $\frac{x}{x+2}+\frac{2}{x+2}=\frac{x+2}{x+2}=1$ (for $x\ne -2$).
  9. $\sqrt{48}=4\sqrt3$, $\sqrt{27}=3\sqrt3$. So $4\sqrt3-2\sqrt3+3\sqrt3=5\sqrt3$.
  10. $\frac{4}{\sqrt5-1}\cdot\frac{\sqrt5+1}{\sqrt5+1}=\frac{4(\sqrt5+1)}{4}=\sqrt5+1$.
  11. $\frac{p^{-4}q^{-1}}{p^{-2}q^{-5}}=p^{-2}q^4=\frac{q^4}{p^2}$.
  12. $(16x)^{1/4}=16^{1/4}x^{1/4}=2\sqrt[4]{x}$.
