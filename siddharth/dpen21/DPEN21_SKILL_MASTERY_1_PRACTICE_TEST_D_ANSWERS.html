  1. \(6x^2-4x+12+2x^2-10=8x^2-4x+2\).
  2. \(8a^2-5(a^2-4ab+4b^2)=3a^2+20ab-20b^2\).
  3. Set \(a=1,b=1\): \((3+1)^4=256\).
  4. \((x-2)^5=x^5-10x^4+40x^3-80x^2+80x-32\).

  12. \(\frac{4}{3+\sqrt{7}}\cdot\frac{3-\sqrt{7}}{3-\sqrt{7}}=\frac{4(3-\sqrt{7})}{9-7}=2(3-\sqrt{7})=6-2\sqrt{7}\).
