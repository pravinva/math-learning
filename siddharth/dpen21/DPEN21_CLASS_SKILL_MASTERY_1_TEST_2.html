DPEN21 Class Skill Mastery Test 1B

Built from Week class-question styles.
12 questions. Suggested time: 50 minutes.
  1. If $m=6$ and $n=-2$, evaluate $5m-3n+mn$.
  2. Expand and simplify $(2x+7)(2x-3)$.
  3. Use Pascal's triangle to expand $(x+3)^4$.
  4. Factorise completely: $x^2-16$.
  5. Factorise completely: $x^3-27$.
  6. Simplify: $\dfrac{14a^2b}{21ab^3}$.
  7. Simplify: $\left(\dfrac{8x}{15y}\right)\div\left(\dfrac{4}{9y}\right)$.
  8. Simplify: $\dfrac{3}{x-1}-\dfrac{1}{x+1}$.
  9. Simplify: $2\sqrt{18}+\sqrt{8}-\sqrt{50}$.
  10. Rationalise the denominator: $\dfrac{7}{3+2\sqrt2}$.
  11. Write with positive indices: $(a^{-2}b^3)^2\div(a^{-1}b)$.
  12. Write in radical form: $\dfrac{1}{x^{2/3}}$.
