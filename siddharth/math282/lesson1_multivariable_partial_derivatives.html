Lesson 1 - Multivariable Functions & Partial Derivatives

This lesson builds the core language for advanced engineering maths: how a quantity changes when it depends on more than one variable. You will use these ideas in heat transfer, fluid flow, optimization, and machine learning models.

1) Motivation

Engineering story: Imagine temperature in a plate given by \(T(x,y)\). Moving in the \(x\)-direction changes temperature one way, moving in the \(y\)-direction changes it another way. Partial derivatives isolate each directional effect.

2) Method Scaffold (How to Solve)

Scaffold A Find \(\partial f/\partial x\):

  1. Treat every variable except \(x\) as a constant.
  2. Differentiate using normal one-variable rules.
  3. Simplify and keep variables visible (do not over-cancel).

Scaffold B Chain rule for \(z=f(x,y)\), \(x=x(t)\), \(y=y(t)\):

\[ \frac{dz}{dt}=\frac{\partial f}{\partial x}\frac{dx}{dt}+\frac{\partial f}{\partial y}\frac{dy}{dt} \]

  1. Find \(f_x\) and \(f_y\).
  2. Find \(x'(t), y'(t)\).
  3. Substitute and then evaluate at requested \(t\).

3) Core Explanations

  • Gradient: \(\nabla f=(f_x,f_y)\), points in direction of steepest increase.
  • Tangent plane: for \(z=f(x,y)\) at \((a,b)\): \[ z \approx f(a,b)+f_x(a,b)(x-a)+f_y(a,b)(y-b) \]
  • Second partials: \(f_{xx},f_{yy},f_{xy}\) help classify surfaces and curvature.

4) Solved Examples (Step by Step)

Example 1 — Basic partials

Given \(f(x,y)=x^2y+3xy^2-4y\), find \(f_x, f_y\).

  1. \(f_x=2xy+3y^2\) (treat \(y\) constant).
  2. \(f_y=x^2+6xy-4\) (treat \(x\) constant).

Example 2 — Chain rule

If \(z=x^2y+y^3,\ x=t^2,\ y=2t-1\), find \(dz/dt\).

  1. \(f_x=2xy,\ f_y=x^2+3y^2\).
  2. \(x'=2t,\ y'=2\).
  3. \(\dfrac{dz}{dt}=f_xx'+f_yy'=2xy(2t)+(x^2+3y^2)2\).
  4. Substitute \(x=t^2,\ y=2t-1\) for final expression.

Example 3 — Tangent plane

Find tangent plane to \(z=x^2+xy+y^2\) at \((1,2)\).

  1. \(f(1,2)=1+2+4=7\).
  2. \(f_x=2x+y \Rightarrow f_x(1,2)=4\).
  3. \(f_y=x+2y \Rightarrow f_y(1,2)=5\).
  4. \(z=7+4(x-1)+5(y-2)\).

Example 4 — Implicit partial derivative

\(x^2+xy+y^2=7\). Find \(\dfrac{dy}{dx}\).

  1. Differentiate both sides w.r.t. \(x\): \(2x+(x\dfrac{dy}{dx}+y)+2y\dfrac{dy}{dx}=0\).
  2. Group \(\dfrac{dy}{dx}\): \((x+2y)\dfrac{dy}{dx}=-(2x+y)\).
  3. \(\dfrac{dy}{dx}=-\dfrac{2x+y}{x+2y}\).

5) Practice Set

  1. For \(f(x,y)=x^3y^2+2xy\), find \(f_x\) and \(f_y\).
  2. For \(f(x,y)=\ln(x^2+y^2)\), find \(f_x,f_y\).
  3. Find \(\nabla f\) for \(f(x,y)=x^2-3xy+4y^2\) at \((1,-1)\).
  4. Find tangent plane to \(z=x^2-y^2+2x\) at \((1,2)\).
  5. Use chain rule for \(z=x^2+xy\), \(x=e^t,\ y=t^2\).
  6. Find \(\dfrac{dy}{dx}\) if \(x^2+y^2=25\).
  7. Find \(f_{xx},f_{yy},f_{xy}\) for \(f=x^2y+y^3\).
  8. Linearize \(f(x,y)=\sqrt{x+y}\) near \((3,1)\).
  9. For \(T(x,y)=100-2x^2-y^2\), where is \(T\) increasing fastest at \((2,1)\)?
  10. Find directional derivative of \(f=x^2+y^2\) at \((1,2)\) in direction \((3,4)\).

6) Mini Test (No notes, 25 min)

  1. Compute \(f_x,f_y\) for \(f=x^2e^y+y\cos x\).
  2. Find \(\dfrac{dz}{dt}\) for \(z=x^2y\), \(x=t+1,\ y=t^2\).
  3. Find tangent plane to \(z=xy+x+y\) at \((1,1)\).
  4. Given \(x^2+xy+y^2=9\), find \(\dfrac{dy}{dx}\).
  5. Compute \(\nabla f(1,2)\) for \(f=x^2+3y^2\).
  6. Classify critical point at origin for \(f=x^2+y^2\).
Mini Test Answers

  3. \(z=3+2(x-1)+2(y-1)\)
  4. \(\dfrac{dy}{dx}=-(2x+y)/(x+2y)\)
  5. \((2,12)\)
  6. Local/global minimum
