Lesson 5 - Probability and Random Variables

This lesson covers the transition from probability rules to random variable models used in reliability, quality control, and process analysis.

1) Motivation

Real engineering systems have uncertainty. Good decisions come from modeling uncertainty with distributions and summary statistics, not just single values.

2) Method Scaffold

Probability workflow
  1. Define events clearly.
  2. Use complement/intersection/union rules.
  3. If conditional appears, switch to \(P(A|B)=P(A\cap B)/P(B)\).
  4. Check plausibility: result must be between 0 and 1.
Random variable workflow
  1. Identify distribution family (Binomial, Poisson, Normal, etc.).
  2. Extract parameters (\(n,p\), \(\lambda\), \(\mu,\sigma\)).
  3. Compute required probability or expectation.
  4. For large-sample mean, apply CLT and standardize with \(Z\).

3) Solved Examples

Example 1 — Conditional probability

\(P(A)=0.6,\ P(B)=0.5,\ P(A\cap B)=0.3\). Find \(P(A|B)\).

  1. \(P(A|B)=0.3/0.5=0.6\).

Example 2 — Binomial

\(X\sim\text{Bin}(10,0.2)\). Find \(P(X=2)\).

  1. \(P(X=2)=\binom{10}{2}(0.2)^2(0.8)^8\).
  2. Numeric value \(\approx 0.302\).

Example 3 — Poisson

Arrivals follow Poisson with \(\lambda=3\) per hour. Find \(P(X\le1)\).

  1. \(P(X\le1)=P(0)+P(1)=e^{-3}(1+3)\).
  2. \(\approx 0.199\).

Example 4 — CLT for sample mean

If \(X\) has \(\mu=50,\sigma=12\), \(n=36\), find \(P(\bar X>53)\).

  1. \(\bar X \approx N(50, (12/\sqrt{36})^2)=N(50,2^2)\).
  2. \(Z=(53-50)/2=1.5\).
  3. \(P(\bar X>53)=P(Z>1.5)\approx0.0668\).

4) Practice Set

  1. If \(P(A)=0.4,\ P(B)=0.7,\ P(A\cap B)=0.3\), find \(P(A\cup B)\).
  2. Find \(P(A^c)\) if \(P(A)=0.92\).
  3. For fair die, find probability of even number.
  4. \(X\sim\text{Bin}(8,0.25)\). Find \(E[X]\), \(\text{Var}(X)\).
  5. \(X\sim\text{Poisson}(4)\). Find \(P(X=0)\).
  6. \(X\sim N(100,15^2)\). Find \(P(X<115)\).
  7. Find z-score for \(x=72\), \(\mu=65,\sigma=5\).
  8. State one difference between PMF and PDF.
  9. For \(X\sim\text{Exp}(\lambda)\), write \(E[X]\).
  10. For sample mean with \(n=49\), \(\sigma=14\), find standard error.

5) Mini Test (25 min)

  1. Given \(P(A)=0.5,P(B)=0.6,P(A\cap B)=0.2\), find \(P(A|B)\).
  2. \(X\sim\text{Bin}(12,0.3)\), find \(P(X=3)\).
  3. \(X\sim\text{Poisson}(2)\), find \(P(X\ge1)\).
  4. \(X\sim N(10,2^2)\), find \(P(X>12)\).
  5. For \(\mu=80,\sigma=10,n=25\), find \(P(\bar X<84)\).
  6. Write one use case for normal approximation in engineering.
Mini Test Answers
  1. \(0.2/0.6=1/3\)
  2. \(\binom{12}{3}(0.3)^3(0.7)^9\)
  3. \(1-e^{-2}\)
  4. \(P(Z>1)\approx0.1587\)
  5. \(\bar X\sim N(80,2^2)\Rightarrow P(Z<2)\approx0.9772\)
  6. Quality-control acceptance windows, process capability checks, etc.
