MATH282 · Week 6-7
Fourier Series, Beta Function, and Error Function
Detailed tutorial: derive coefficients, use parity shortcuts, and work with non-elementary integral tools.
Why this matters: Fourier series decomposes periodic signals into frequency components; Beta/Gamma/erf appear when classic antiderivatives fail.
PatternBuild it: coefficient mechanics and symmetry logic▶
\[ f(x)\sim \frac{a_0}{2}+\sum_{n=1}^{\infty}\left(a_n\cos\frac{n\pi x}{L}+b_n\sin\frac{n\pi x}{L}\right) \]

Parity shortcut saves time: even \(f\Rightarrow b_n=0\); odd \(f\Rightarrow a_n=0\) (including \(a_0\)).

\[ B(p,q)=\int_0^1 t^{p-1}(1-t)^{q-1}dt,\quad B(p,q)=\frac{\Gamma(p)\Gamma(q)}{\Gamma(p+q)} \]
\[ \operatorname{erf}(x)=\frac{2}{\sqrt{\pi}}\int_0^x e^{-u^2}\,du \]
GuidedGuided routine for Fourier problems▶
  1. Set interval and period carefully (\(L\) matters).
  2. Test parity first and eliminate unnecessary coefficients.
  3. Compute coefficients with clean integration by parts steps.
  4. Write first few non-zero terms, then general form.
  5. For jump points, remember convergence is midpoint of left/right limits.
ExampleWorked examples▶
Example 1 — odd function \(f(x)=x\) on \((-\pi,\pi)\)

\(a_0=a_n=0\), and \(b_n=2(-1)^{n+1}/n\).

\[ x\sim 2\sum_{n=1}^{\infty}(-1)^{n+1}\frac{\sin(nx)}{n} \]
Example 2 — Beta integral mapping

\(\int_0^1 \sqrt{t}(1-t)^2 dt = B(3/2,3)=16/105\).

Example 3 — erf notation

\(\int_0^a e^{-u^2}du = (\sqrt{\pi}/2)\operatorname{erf}(a)\).

NoteCommon mistakes and recovery checks▶
Recovery 1
Wrong \(L\) means wrong frequency basis. Always write interval as \((-L,L)\) before coefficients.
Recovery 2
Do parity check before integrating — often halves workload.
Recovery 3
\(\int e^{-x^2}dx\) does not simplify elementarily; use erf.

Practice + Mini Mastery Test

1
For \(f(x)=x^2\), which Fourier coefficients vanish?
▶
Answer
All \(b_n\) vanish (even function).
2
Compute \(B(1/2,1/2)\).
▶
Answer
\(\pi\).
3
Write \(\int_0^2 e^{-u^2}du\) using erf.
▶
Answer
\((\sqrt{\pi}/2)\operatorname{erf}(2)\).
4
For jump discontinuity, Fourier converges to what value?
▶
Answer
Midpoint: \((f(x^-)+f(x^+))/2\).
5
State one engineering use of Fourier series.
▶
Answer
Signal decomposition/filtering, vibration mode analysis, heat PDE solutions.
6
For odd \(f\), what are \(a_0\) and \(a_n\)?
▶
Answer
Both are zero.
