MATH282 · Week 8-10
Probability, Random Variables, and Distributions
Detailed tutorial with event algebra, Binomial/Poisson/Normal models, and CLT-based sample mean reasoning.
Why this matters: Engineering systems are noisy and uncertain; probability and random-variable models quantify risk and expected behavior instead of guessing.
PatternBuild it: from events to models▶

Start with event relations, then elevate to random variables and distributions.

\[ P(A\cup B)=P(A)+P(B)-P(A\cap B),\quad P(A|B)=\frac{P(A\cap B)}{P(B)} \]

Core models:

  • \(X\sim \text{Bin}(n,p)\): fixed trials, success/failure.
  • \(X\sim \text{Pois}(\lambda)\): count in interval at constant rate.
  • \(X\sim N(\mu,\sigma^2)\): continuous bell model; CLT anchor.
GuidedGuided routine for selecting distributions▶
  1. Is there fixed number of trials? If yes, Binomial candidate.
  2. Is it count-per-time/space with steady average rate? Poisson candidate.
  3. Is variable continuous and approximately symmetric? Normal candidate.
  4. For sample means, use CLT if \(n\) is large enough.
ExampleWorked examples▶
Example 1 — conditional probability

\(P(A)=0.6,\ P(B)=0.5,\ P(A\cap B)=0.3\Rightarrow P(A|B)=0.6\).

Example 2 — Binomial exact

\(X\sim Bin(10,0.2)\): \(P(X=2)=\binom{10}{2}(0.2)^2(0.8)^8\approx0.302\).

Example 3 — Poisson cumulative

\(X\sim Pois(3)\): \(P(X\le1)=e^{-3}(1+3)\approx0.199\).

Example 4 — CLT mean probability

\(\mu=50,\sigma=12,n=36\): \(P(\bar X>53)=P(Z>1.5)\approx0.0668\).

NoteCommon mistakes and recovery checks▶
Recovery 1
If you wrote Poisson but have fixed \(n\) trials, reconsider Binomial first.
Recovery 2
CLT standard error is \(\sigma/\sqrt{n}\), not \(\sigma/n\).
Recovery 3
Every final probability must lie between 0 and 1. If not, re-check arithmetic or model.
