Lesson 2 Tutorial · Weeks 2-3

Double Integrals, Order Switching, Polar Coordinates

Full tutorial workflow: region sketching, integration order decisions, Cartesian-to-polar conversion, Jacobian meaning, and area/volume applications.

1) Why this topic exists

Single integrals add contributions along a line. Many engineering quantities are distributed over areas (plates, membranes, pressure fields), so we need area integration:

\[ \iint_R f(x,y)\,dA \]

Intuition: cut region \(R\) into tiny cells; evaluate \(f\) at each cell; multiply by tiny area; add all cells.

2) Core build-up (no jumps)

2.1 Iterated integral form

\[ \iint_R f(x,y)\,dA = \int_a^b\int_{g_1(x)}^{g_2(x)} f(x,y)\,dy\,dx \] or \[ = \int_c^d\int_{h_1(y)}^{h_2(y)} f(x,y)\,dx\,dy. \]

2.2 Why order switching matters

Some integrands become much easier in one order. If one order gives ugly inner bounds or impossible antiderivatives, switch.

2.3 Polar conversion

\[ x=r\cos\theta,\quad y=r\sin\theta,\quad dA=r\,dr\,d\theta \]

The extra \(r\) (Jacobian) is not optional — it adjusts for stretched area elements in polar coordinates.

3) Method routines (scaffold)

Routine A — Cartesian setup
  1. Draw region and label boundaries clearly.
  2. Pick vertical or horizontal strips.
  3. Write lower/upper inner bounds first, then outer bounds.
Routine B — switch order
  1. Rewrite region as inequalities in opposite strip orientation.
  2. Check new bounds cover region exactly once.
Routine C — polar setup
  1. Convert curves to \(r,\theta\).
  2. Convert integrand to \(r,\theta\).
  3. Multiply by Jacobian \(r\).

4) Guided examples (fully worked)

Example 1: direct rectangular region

\(\int_0^2\int_1^3 (x+2y)\,dy\,dx\)

  1. Inner integral: \([xy+y^2]_1^3=2x+8\).
  2. Outer integral: \(\int_0^2(2x+8)\,dx=[x^2+8x]_0^2=20\).
Answer: 20.

Example 2: reverse order to simplify

\(\int_0^1\int_x^1 f(x,y)\,dy\,dx\)

  1. Region: \(0\le x\le1,\ x\le y\le1\).
  2. Equivalent description: \(0\le y\le1,\ 0\le x\le y\).
  3. Reversed integral: \(\int_0^1\int_0^y f(x,y)\,dx\,dy\).

Example 3: area by double integral

Find area between \(y=2x\) and \(y=x^2\) from \(x=0\) to \(x=2\).

  1. Area \(=\int_0^2\int_{x^2}^{2x}1\,dy\,dx\).
  2. Inner gives \(2x-x^2\).
  3. Area \(=\int_0^2(2x-x^2)\,dx=4/3\).

Example 4: polar disk integral

\(\iint_{x^2+y^2\le4}(x^2+y^2)\,dA\)

  1. Region is disk: \(0\le r\le2,\ 0\le\theta\le2\pi\).
  2. Integrand becomes \(r^2\), and \(dA\to r\,dr\,d\theta\).
  3. \(\int_0^{2\pi}\int_0^2 r^3\,dr\,d\theta = \int_0^{2\pi}4\,d\theta=8\pi\).

Example 5: annulus in polar

\(\iint_R 1\,dA\) where \(R:\ 1\le x^2+y^2\le9\).

  1. Polar bounds: \(1\le r\le3,\ 0\le\theta\le2\pi\).
  2. Area \(=\int_0^{2\pi}\int_1^3 r\,dr\,d\theta = \int_0^{2\pi}4\,d\theta=8\pi\).

5) Common mistakes and recovery checks

  • Mistake: forgetting Jacobian \(r\). Recovery: write "replace \(dA\)" explicitly every time.
  • Mistake: wrong swapped bounds. Recovery: test with one point inside region.
  • Mistake: integrating wrong variable first. Recovery: circle differential in inner integral.
