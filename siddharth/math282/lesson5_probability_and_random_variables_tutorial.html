Lesson 5 Tutorial · Weeks 8-10

Probability, Random Variables, and Core Distributions

From event rules to Binomial/Poisson/Normal modeling, with interpretation and CLT sample-mean workflow.

1) Why this topic exists

Engineering systems have randomness (noise, defects, arrivals, failures). Probability gives the language; random variables give models; statistics turns data into decisions.

Mindset: write assumptions first (independence? constant rate? finite trials?), then select model.

2) Build-up concepts

2.1 Event probability basics

\[ P(A\cup B)=P(A)+P(B)-P(A\cap B),\quad P(A|B)=\frac{P(A\cap B)}{P(B)} \]

2.2 Random variables

  • Discrete: PMF \(P(X=x)\), e.g. Binomial/Poisson.
  • Continuous: PDF \(f(x)\), probabilities via area/integral.

2.3 Key models

  • \(X\sim \text{Bin}(n,p)\): fixed number of independent yes/no trials.
  • \(X\sim \text{Poisson}(\lambda)\): event count per interval at constant rate.
  • \(X\sim N(\mu,\sigma^2)\): symmetric continuous model and CLT anchor.

3) Method scaffold

Routine A — probability question
  1. Define events in words and symbols.
  2. Choose union/intersection/conditional formula.
  3. Substitute, compute, sanity-check \(0\le P\le1\).
Routine B — distribution question
  1. Identify family and parameters.
  2. Write exact probability expression.
  3. Evaluate and interpret.
Routine C — sample mean via CLT
  1. \(\bar X\approx N(\mu,\sigma^2/n)\) for large \(n\).
  2. Standardize: \(Z=(\bar X-\mu)/(\sigma/\sqrt{n})\).

4) Guided worked examples

Example 1: conditional probability

\(P(A)=0.6,\ P(B)=0.5,\ P(A\cap B)=0.3\). Find \(P(A|B)\).

  1. \(P(A|B)=0.3/0.5=0.6\).

Example 2: Binomial exact probability

\(X\sim\text{Bin}(10,0.2)\). Find \(P(X=2)\).

  1. \(P(X=2)=\binom{10}{2}(0.2)^2(0.8)^8\).
  2. Value \(\approx 0.302\).

Example 3: Poisson cumulative

\(X\sim\text{Poisson}(3)\). Find \(P(X\le1)\).

  1. \(P(0)+P(1)=e^{-3}+3e^{-3}=4e^{-3}\approx0.199\).

Example 4: Normal probability

\(X\sim N(100,15^2)\). Find \(P(X<115)\).

  1. \(Z=(115-100)/15=1\).
  2. \(P(X<115)=P(Z<1)\approx0.8413\).

Example 5: CLT sample mean

\(\mu=50,\sigma=12,n=36\). Find \(P(\bar X>53)\).

  1. SE \(=12/\sqrt{36}=2\).
  2. \(Z=(53-50)/2=1.5\).
  3. \(P(\bar X>53)=P(Z>1.5)\approx0.0668\).

5) Common mistakes and fixes

  • Mistake: using Poisson where trials are fixed (should be Binomial). Fix: ask "fixed \(n\)?" first.
  • Mistake: forgetting variance formulas. Fix: keep formula strip: Binomial \(np(1-p)\), Poisson \(\lambda\).
  • Mistake: CLT with wrong standard error. Fix: always divide \(\sigma\) by \(\sqrt{n}\), not \(n\).
