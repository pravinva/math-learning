Lesson 6 - Inference, Regression, ANOVA, Nonparametric Methods

Final-week toolkit for data-driven decisions: estimate, test, model relationships, compare groups, and use robust alternatives.

1) Motivation

In engineering, we rarely know population truth exactly. Inference converts sample evidence into confidence statements and decisions.

2) Method Scaffold

Hypothesis test workflow
  1. State \(H_0\) and \(H_1\).
  2. Select test statistic (z, t, F, chi-square, etc.).
  3. Compute statistic and p-value.
  4. Compare with \(\alpha\), conclude in context.
Regression workflow
  1. Plot data and inspect linearity/outliers.
  2. Fit \( \hat y=a+bx \).
  3. Interpret slope and \(R^2\).
  4. Check residual pattern before trusting predictions.

3) Solved Examples

Example 1 — Confidence interval for mean (known \(\sigma\))

\(\bar x=52,\ \sigma=10,\ n=64\). 95% CI for \(\mu\).

  1. SE \(=10/\sqrt{64}=1.25\).
  2. Use \(z_{0.975}=1.96\).
  3. \(52\pm1.96(1.25)=52\pm2.45\Rightarrow(49.55,54.45)\).

Example 2 — One-sample t-test

\(\bar x=101,\ s=6,\ n=25\), test \(H_0:\mu=100\) vs \(H_1:\mu\ne100\).

  1. \(t=(101-100)/(6/\sqrt{25})=1/1.2=0.833\).
  2. df \(=24\). p-value \(>0.4\), fail to reject \(H_0\).

Example 3 — Simple regression

Suppose fitted model is \(\hat y=4+1.8x,\ R^2=0.81\).

  1. Slope: +1 unit \(x\) predicts +1.8 units \(y\).
  2. \(R^2=0.81\): 81% variation in \(y\) explained by linear model.
  3. For \(x=5\), prediction \(\hat y=13\).

Example 4 — One-way ANOVA idea

Three process means compared. ANOVA \(F=5.9,\ p=0.008\).

  1. Since \(p<0.05\), reject equal-means hypothesis.
  2. At least one process mean differs; follow with post-hoc comparisons.

  4. State when to use paired t-test instead of two-sample t-test.

  6. Given \(r=0.9\), explain relationship strength/direction.
  7. For line \(\hat y=2+0.5x\), predict at \(x=20\).

5) Mini Test (30 min)

  1. Compute 95% CI: \(\bar x=70,s=8,n=25\) (use \(t_{0.975,24}\approx2.064\)).
  2. Two-sample means: \(\bar x_1=30,\bar x_2=27\). Write hypothesis for “group 1 larger”.
  3. Given p-value 0.12 at \(\alpha=0.05\), decision?
  4. Regression line \(\hat y=10-0.4x\): interpret slope.
  5. ANOVA p-value 0.001: what next step?
  6. When data are strongly non-normal with outliers, suggest a robust test family.

  1. SE \(=1.6\), margin \(=2.064(1.6)=3.3024\), CI \((66.70,73.30)\)
