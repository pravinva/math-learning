MATH282 · Week 1-2
Multivariable Functions and Partial Derivatives
Full tutorial: from intuitive surface thinking to gradient, directional derivatives, tangent planes, chain rule, and local optimization tests.
Why this matters: Most engineering models depend on more than one input variable. Partial derivatives tell you how the system responds if you nudge one input while holding the others fixed.
Pattern Build it: what changes when there are two variables? ▶

For a surface \(z=f(x,y)\), each point in the \(xy\)-plane has a height. Instead of one slope, we now have at least two natural slopes: one moving parallel to \(x\), one parallel to \(y\).

\[ f_x=\frac{\partial f}{\partial x}\quad(\text{hold }y\text{ fixed}),\qquad f_y=\frac{\partial f}{\partial y}\quad(\text{hold }x\text{ fixed}) \]

The gradient packs both into one vector:

\[ \nabla f=(f_x,f_y) \]

This is not random notation: \(\nabla f\) points in steepest increase direction and its magnitude is steepest rate of increase.

Guided Guided lesson (complete build routine) ▶
  1. Start with region/surface picture: what are independent variables?
  2. Compute \(f_x,f_y\) cleanly (freeze the other variable each time).
  3. If asked for rate in a direction \(\mathbf{u}\), normalize \(\mathbf{u}\).
  4. Use directional derivative:
    \[ D_{\mathbf{u}}f=\nabla f\cdot \mathbf{u} \]
  5. For local linear model near \((a,b)\), use tangent plane/linearization:
    \[ L(x,y)=f(a,b)+f_x(a,b)(x-a)+f_y(a,b)(y-b) \]
Definition Formal statements used in this week ▶
\[ \frac{dz}{dt}=f_x\frac{dx}{dt}+f_y\frac{dy}{dt} \quad\text{(chain rule for }z=f(x,y)\text{)} \]
\[ D_{\mathbf{u}}f(a,b)=\nabla f(a,b)\cdot\mathbf{u},\quad \|\mathbf{u}\|=1 \]
\[ \text{Tangent plane at }(a,b):\ z=f(a,b)+f_x(a,b)(x-a)+f_y(a,b)(y-b) \]
Example Worked examples (step-by-step, no jumps) ▶
Example 1 — compute partial derivatives

Given \(f(x,y)=x^2y+3xy^2-4y\), find \(f_x,f_y\).

Hold \(y\) constant: \(f_x=2xy+3y^2\).

Hold \(x\) constant: \(f_y=x^2+6xy-4\).

Answer: \(\;f_x=2xy+3y^2,\;f_y=x^2+6xy-4\).
Example 2 — tangent plane

Find tangent plane to \(z=x^2+xy+y^2\) at \((1,2)\).

\(f(1,2)=7\), \(f_x=2x+y\Rightarrow f_x(1,2)=4\), \(f_y=x+2y\Rightarrow f_y(1,2)=5\).

\[ z=7+4(x-1)+5(y-2) \]
Example 3 — chain rule in motion model

\(z=x^2y+y^3,\ x=t^2,\ y=2t-1\). Find \(dz/dt\).

\(f_x=2xy,\ f_y=x^2+3y^2,\ x'=2t,\ y'=2\).

\[ \frac{dz}{dt}=f_xx'+f_yy'=(2xy)(2t)+(x^2+3y^2)\cdot2 \]
Note Common mistakes and recovery checks ▶
Recovery check 1
If you are finding \(f_x\), any stray \(dy/dx\) means you accidentally treated \(y\) as dependent. Restart with "freeze \(y\)".
Recovery check 2
Directional derivative requires a unit direction vector. Normalize first.
Recovery check 3
Tangent plane uses values at the base point only; do not leave raw \(f_x(x,y)\) in final plane equation.

Practice + Mini Mastery Test

1
Find \(f_x,f_y\) for \(f=x^3y^2+2xy\).
▶
Answer
\(f_x=3x^2y^2+2y,\quad f_y=2x^3y+2x\).
2
Find \(\nabla f(1,-1)\) for \(f=x^2-3xy+4y^2\).
▶
Answer
\(f_x=2x-3y,\ f_y=-3x+8y\Rightarrow \nabla f(1,-1)=(5,-11)\).
3
Find \(dy/dx\) from \(x^2+xy+y^2=9\).
▶
Answer
\(\dfrac{dy}{dx}=-(2x+y)/(x+2y)\).
4
Find tangent plane to \(z=xy+x+y\) at \((1,1)\).
▶
Answer
\(f(1,1)=3,\ f_x=y+1=2,\ f_y=x+1=2\Rightarrow z=3+2(x-1)+2(y-1)\).
5
If \(z=x^2y,\ x=t+1,\ y=t^2\), find \(dz/dt\).
▶
Answer
\(dz/dt=2xyx'+x^2y' = 2(t+1)t^2 + (t+1)^2(2t)\).
6
Explain gradient in one sentence.
▶
Answer
The gradient points in the direction of fastest local increase and its length is that maximum rate.
