Lesson 2 - Double Integrals, Order Switching, Polar Coordinates

This lesson turns area-sums into exact values and gives you a reliable setup routine for engineering volume, mass, and average-value problems.

1) Motivation

In mechanics and heat-flow models, quantities vary over regions, not just lines. Double integrals add all local contributions over a 2D domain.

2) Method Scaffold

Scaffold A — Cartesian double integral
  1. Sketch region \(R\).
  2. Choose order: \(dy\,dx\) or \(dx\,dy\) based on simpler bounds.
  3. Write inner bounds as functions of outer variable.
  4. Integrate inner first, then outer.
Scaffold B — Polar change

\[ x=r\cos\theta,\quad y=r\sin\theta,\quad dA=r\,dr\,d\theta \]

  1. Convert region to \((r,\theta)\) cleanly.
  2. Convert integrand \(f(x,y)\to f(r\cos\theta,r\sin\theta)\).
  3. Do not forget Jacobian \(r\).

3) Solved Examples

Example 1 — Basic rectangle

\(\int_0^2\int_1^3 (x+2y)\,dy\,dx\)

  1. Inner: \(\int_1^3 (x+2y)\,dy = [xy+y^2]_1^3 = 2x+8\).
  2. Outer: \(\int_0^2 (2x+8)\,dx = [x^2+8x]_0^2=20\).

Example 2 — Switch order

\(\int_0^1\int_x^1 f(x,y)\,dy\,dx\)

  1. Region: \(0\le x\le1,\ x\le y\le1\).
  2. Reverse: \(0\le y\le1,\ 0\le x\le y\).
  3. Equivalent integral: \(\int_0^1\int_0^y f(x,y)\,dx\,dy\).

Example 3 — Area with double integral

Find area bounded by \(y=x^2\) and \(y=2x\).

  1. Intersections: \(x^2=2x \Rightarrow x=0,2\).
  2. Area \(=\int_0^2 (2x-x^2)\,dx = \left[x^2-\frac{x^3}{3}\right]_0^2=\frac{4}{3}\).

Example 4 — Polar integral

Evaluate \(\iint_R (x^2+y^2)\,dA\), \(R:\ x^2+y^2\le 4\).

  1. In polar: integrand \(=r^2\), region \(0\le r\le2,\ 0\le\theta\le2\pi\).
  2. \(\int_0^{2\pi}\int_0^2 r^2(r)\,dr\,d\theta = \int_0^{2\pi}\left[\frac{r^4}{4}\right]_0^2 d\theta\).
  3. \(= \int_0^{2\pi}4\,d\theta = 8\pi\).

4) Practice Set

  1. Evaluate \(\int_0^1\int_0^2 (3x-y)\,dy\,dx\).
  2. Switch order: \(\int_0^2\int_{x/2}^1 f(x,y)\,dy\,dx\).
  3. Find area of triangle with vertices \((0,0),(2,0),(2,3)\) using double integral.
  4. Evaluate \(\iint_R 1\,dA\), \(R:\ x^2+y^2\le9\).
  5. Evaluate \(\iint_R x\,dA\), \(R:\ 0\le y\le x\le1\).
  6. Evaluate in polar: \(\iint_R (x^2+y^2)\,dA\), \(R:\ 1\le x^2+y^2\le4\).
  7. Set up (do not solve) \(\iint_R e^{x^2+y^2}\,dA\) over first-quadrant quarter-circle radius 3.
  8. Compute average value of \(f=x+y\) over rectangle \(0\le x\le2,\ 0\le y\le1\).
  9. Find mass for density \(\rho=x+y\) on \(0\le x\le1,\ 0\le y\le1\).
  10. Evaluate \(\int_0^1\int_0^{1-x} (x+y)\,dy\,dx\).

5) Mini Test (30 min)

  1. Evaluate \(\int_0^2\int_0^x (x+2y)\,dy\,dx\).
  2. Reverse order: \(\int_0^1\int_{y}^{1} f(x,y)\,dx\,dy\).
  3. Evaluate \(\iint_{x^2+y^2\le1} (x^2+y^2)\,dA\).
  4. Set up polar bounds for region between circles \(r=1\) and \(r=3\), upper half-plane.
  5. Find area enclosed by \(y=x\) and \(y=x^2\).
  6. Compute \(\int_0^1\int_0^{2-y} 1\,dx\,dy\).
Mini Test Answers
  1. \(\frac{20}{3}\)
  2. \(\int_0^1\int_0^x f(x,y)\,dy\,dx\)
  3. \(\pi/2\)
  4. \(0\le\theta\le\pi,\ 1\le r\le3\)
  5. \(\int_0^1 (x-x^2)\,dx=\frac16\)
  6. \(\frac32\)
