Lesson 4 Tutorial · Weeks 6-7

Fourier Series, Beta Function, Error Function

Complete walkthrough of periodic decomposition plus special-function tools for non-elementary integrals.

1) Why this topic exists

Fourier series models periodic signals as sums of sines and cosines. Special functions (Beta, Gamma, erf) appear when integrals cannot be written in elementary antiderivative form.

Engineering link: heat conduction, vibration, signal processing, diffusion, and probability all use these representations.

2) Build-up concepts

2.1 Fourier structure on \((-L,L)\)

\[ f(x)\sim\frac{a_0}{2}+\sum_{n=1}^\infty\left(a_n\cos\frac{n\pi x}{L}+b_n\sin\frac{n\pi x}{L}\right) \]

with \[ a_n=\frac{1}{L}\int_{-L}^L f(x)\cos\frac{n\pi x}{L}\,dx,\quad b_n=\frac{1}{L}\int_{-L}^L f(x)\sin\frac{n\pi x}{L}\,dx. \]

2.2 Parity shortcut

  • \(f\) even \(\Rightarrow b_n=0\)
  • \(f\) odd \(\Rightarrow a_0=a_n=0\)

2.3 Special functions

\[ B(p,q)=\int_0^1 t^{p-1}(1-t)^{q-1}dt,\quad \Gamma(p)=\int_0^\infty t^{p-1}e^{-t}dt,\quad B(p,q)=\frac{\Gamma(p)\Gamma(q)}{\Gamma(p+q)} \]

\[ \operatorname{erf}(x)=\frac{2}{\sqrt{\pi}}\int_0^x e^{-u^2}du. \]

3) Method scaffold

Routine A — Fourier coefficients
  1. Check parity first.
  2. Compute only non-zero family (\(a_n\) or \(b_n\)).
  3. Use integration by parts as needed.
Routine B — Beta/Gamma integral recognition
  1. Try mapping integral to \(t^{p-1}(1-t)^{q-1}\).
  2. Read \(p,q\), then use Gamma identity.
Routine C — error function usage
  1. If integral has \(e^{-u^2}\), write in erf form.
  2. Do not try symbolic elementary antiderivative.

4) Guided worked examples

Example 1: classic odd function Fourier

\(f(x)=x\) on \((-\pi,\pi)\)

  1. Odd function \(\Rightarrow a_0=a_n=0\).
  2. \(b_n=\frac{1}{\pi}\int_{-\pi}^{\pi}x\sin(nx)\,dx=\frac{2}{\pi}\int_0^\pi x\sin(nx)\,dx\).
  3. By parts: \(b_n=2(-1)^{n+1}/n\).
  4. \(x\sim2\sum_{n=1}^\infty (-1)^{n+1}\frac{\sin nx}{n}\).

Example 2: even function shortcut

For \(f(x)=x^2\) on \((-\pi,\pi)\), only cosine terms survive; sine terms vanish by odd integrand symmetry in \(b_n\).

Example 3: Beta integral

\(\int_0^1 \sqrt{t}(1-t)^2 dt = B(3/2,3)\).

  1. \(= \Gamma(3/2)\Gamma(3)/\Gamma(9/2)\).
  2. \(=16/105\).

Example 4: erf conversion

\(\int_0^a e^{-u^2}du = (\sqrt{\pi}/2)\operatorname{erf}(a)\).

5) Common mistakes and fixes

  • Mistake: computing both \(a_n\) and \(b_n\) without parity check. Fix: test \(f(-x)\) first.
  • Mistake: wrong interval scaling by \(\pi/L\). Fix: write master formula before substitution.
  • Mistake: trying to integrate \(e^{-x^2}\) elementarily. Fix: switch to erf notation.
