Lesson 3 - Laplace Transforms and ODEs

Laplace transforms turn differential equations into algebraic equations. This is a core engineering tool for control systems and transient analysis.

1) Motivation

Solving ODEs directly is often messy, especially with forcing terms and piecewise inputs. Laplace provides a structured, repeatable workflow.

2) Method Scaffold

Scaffold — Solve IVP with Laplace
  1. Take Laplace transform of each term.
  2. Insert initial conditions immediately.
  3. Solve for \(Y(s)\).
  4. Use partial fractions/shift rules to invert.

Main identities: \(\mathcal{L}\{y'\}=sY-y(0)\), \(\mathcal{L}\{y''\}=s^2Y-sy(0)-y'(0)\).

3) Solved Examples

Example 1 — Basic transform

Find \(\mathcal{L}\{t e^{2t}\}\).

  1. \(\mathcal{L}\{te^{at}\}=\frac{1}{(s-a)^2}\).
  2. Hence \(\mathcal{L}\{te^{2t}\}=\frac{1}{(s-2)^2}\).

Example 2 — IVP

\(y'+3y=6,\ y(0)=1\).

  1. \(sY-1+3Y=\frac{6}{s}\).
  2. \((s+3)Y=\frac{6}{s}+1=\frac{s+6}{s}\).
  3. \(Y=\frac{s+6}{s(s+3)}=\frac{2}{s}-\frac{1}{s+3}\).
  4. \(y(t)=2-e^{-3t}\).

Example 3 — Heaviside input

\(f(t)=u(t-2)(t-2)\). Find \(F(s)\).

  1. Use shift: \(\mathcal{L}\{u(t-a)g(t-a)\}=e^{-as}G(s)\).
  2. \(g(t)=t\Rightarrow G(s)=1/s^2\).
  3. \(F(s)=e^{-2s}/s^2\).

Example 4 — Delta impulse

\(\mathcal{L}\{\delta(t-3)\}=e^{-3s}\). For \(2y'+y=\delta(t-3), y(0)=0\), solve in \(s\)-domain:

  1. \((2s+1)Y=e^{-3s}\).
  2. \(Y=\dfrac{e^{-3s}}{2s+1}\).
  3. \(y(t)=u(t-3)\frac{1}{2}e^{-(t-3)/2}\).

4) Practice Set

  1. Find \(\mathcal{L}\{t^2\}\).
  2. Find \(\mathcal{L}\{\sin 5t\}\).
  3. Find \(\mathcal{L}^{-1}\left\{\dfrac{3}{s(s+2)}\right\}\).
  4. Solve \(y'+2y=4,\ y(0)=0\).
  5. Solve \(y''+y=0,\ y(0)=1,\ y'(0)=0\).
  6. Find \(\mathcal{L}\{u(t-1)\}\).
  7. Find \(\mathcal{L}^{-1}\left\{\dfrac{e^{-2s}}{s}\right\}\).
  8. Solve \(y''+3y'+2y=0,\ y(0)=2,\ y'(0)=0\).
  9. Find \(\mathcal{L}\{t\cos 2t\}\) (use derivative in \(s\)-domain idea).
  10. Write forcing \(f(t)=0\ (t<3),\ f(t)=5\ (t\ge3)\) using Heaviside.

5) Mini Test (30 min)

  1. Find \(\mathcal{L}\{e^{-4t}\sin 2t\}\).
  2. Find \(\mathcal{L}^{-1}\left\{\dfrac{1}{(s+1)^2}\right\}\).
  3. Solve \(y'+y=t,\ y(0)=0\) using Laplace.
  4. Find \(\mathcal{L}\{u(t-3)(t-3)^2\}\).
  5. Find \(\mathcal{L}\{\delta(t-5)\}\).
  6. Solve \(y''+4y=8,\ y(0)=0,\ y'(0)=0\).
Mini Test Answers
  1. \(\dfrac{2}{(s+4)^2+4}\)
  2. \(te^{-t}\)
  3. \(y=t-1+e^{-t}\)
  4. \(e^{-3s}\dfrac{2!}{s^3}=2e^{-3s}/s^3\)
  5. \(e^{-5s}\)
  6. \(y=2(1-\cos 2t)\)
