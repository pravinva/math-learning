Week 2 Deep Dive: Mechanical Properties of Metals

Outline mapping: Ch6 §6.1-6.12.

Full worked scaffold (what to do for any stress-strain question)

  1. Define test context: tensile/compressive, monotonic/cyclic, room temperature/elevated temperature.
  2. Read curve landmarks: linear region, yield point (or 0.2% offset), ultimate strength, necking, fracture strain.
  3. Extract equations: engineering stress/strain first; convert to true values if large plastic strain is relevant.
  4. Compute energy metrics: resilience (elastic) and toughness (full area).
  5. Close with design implication: safety factor and likely failure mode (yield, fracture, fatigue, buckling).

Step 1 - Convert words to measurable targets

Design statementProperty targetReasoning
No permanent set under service loadWorking stress less than yield / safety factorPlastic deformation starts near yield threshold
Low elastic deflectionHigher modulus EElastic strain = stress/E
Absorb occasional impactHigher toughnessLarger area under full stress-strain curve
Spring-like recoveryHigher resilience in elastic rangeRecoverable strain energy before yield

Step 2 - Core equations with interpretation

Engineering stress: sigma = F / A0
Engineering strain: epsilon = (L - L0) / L0
True stress: sigma_t = sigma * (1 + epsilon)
True strain: epsilon_t = ln(1 + epsilon)
Resilience (linear elastic): Ur = sigma_y^2 / (2E)

True values become important when area reduction and large plastic strain occur (for example beyond uniform elongation).

Step 3 - Full numerical example

Given: A0=120 mm^2, load F=43.2 kN, elongation from 50 mm to 58 mm, E=210 GPa, yield stress sigma_y=450 MPa.

  1. Engineering stress: sigma = 43.2e3 / 120e-6 = 360 MPa.
  2. Engineering strain: epsilon = (58-50)/50 = 0.16.
  3. True stress: sigma_t = 360*(1+0.16)=417.6 MPa.
  4. True strain: epsilon_t = ln(1.16)=0.148.
  5. Resilience: Ur = (450e6)^2 / (2*210e9) = 4.82e5 J/m^3.

Conclusion: for the same test point, true stress is higher due to area reduction; resilience quantifies recoverable energy if loading remains elastic.

Step 4 - Decision logic with safety factor

If maximum service stress is 180 MPa and yield is 450 MPa, yield safety factor is:

N_y = sigma_y / sigma_work = 450/180 = 2.5

Now check if 2.5 is adequate for variability, temperature, and consequence of failure. If high risk, increase margin or redesign cross-section.

Common mistake: picking a material only by ultimate tensile strength and ignoring yield, toughness, and expected loading mode.

Quiz (with full worked solutions)

  1. A steel sample has A0=100 mm^2 and F=32 kN. Compute engineering stress and explain if yield is likely when sigma_y=280 MPa.
  2. Given epsilon=0.22 at a point where engineering stress is 410 MPa, compute true stress and true strain.
  3. Two candidate alloys have same yield but one has much higher toughness. Which is preferred for impact-prone service and why?
Solution 1

sigma = 32e3 / 100e-6 = 320 MPa. Since 320 MPa > 280 MPa, yielding is likely (assuming uniaxial tension and no unusual constraint effects).

Solution 2

sigma_t = 410*(1+0.22)=500.2 MPa. epsilon_t = ln(1.22)=0.199. Both converted values indicate stronger effective stress state under large strain.

Solution 3

Choose higher-toughness alloy. In impact-prone loading, energy absorption before fracture is critical; equal yield alone does not guarantee resistance to sudden fracture.
