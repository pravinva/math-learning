Pascal Quick Memorisation Box (only up to \(n=8\))▼
Use limit: prepare Pascal rows only up to power \(8\).
\[ \begin{aligned} n=0 &: 1\\ n=1 &: 1,\ 1\\ n=2 &: 1,\ 2,\ 1\\ n=3 &: 1,\ 3,\ 3,\ 1\\ n=4 &: 1,\ 4,\ 6,\ 4,\ 1\\ n=5 &: 1,\ 5,\ 10,\ 10,\ 5,\ 1\\ n=6 &: 1,\ 6,\ 15,\ 20,\ 15,\ 6,\ 1\\ n=7 &: 1,\ 7,\ 21,\ 35,\ 35,\ 21,\ 7,\ 1\\ n=8 &: 1,\ 8,\ 28,\ 56,\ 70,\ 56,\ 28,\ 8,\ 1 \end{aligned} \]
  • Sum of coefficients of \((a+b)^n\): \(2^n\).
  • Alternating sum of coefficients: \(0\) for \(n\ge1\).
  • For \((u-v)^n\), signs alternate \(+,-,+,-,\dots\).

Lesson A — Algebraic Fractions (Week 2A)▶
Core routine: factor numerator/denominator, cancel only common factors, not terms.
Multiply: factor all parts first, then cross-cancel before multiplying.
Divide: multiply by reciprocal of second fraction, then simplify.
Add/Subtract: use LCD, rewrite each fraction, combine numerators, simplify final result.
Do not do: \(\dfrac{a+m}{b+m}\neq\dfrac{a}{b}\), and \(\dfrac{y+a}{yb}\neq\dfrac{1+a}{b}\).
Lesson B — Surds + Rationalising (Week 2B)▶
Surd basics: simplify using largest perfect square factor first (e.g. \(\sqrt{48}=4\sqrt3\)).
Like-surds only: you can combine \(a\sqrt n+b\sqrt n\), but not \(\sqrt x+\sqrt y\) when radicands differ.
Multiply/divide: \(\sqrt x\sqrt y=\sqrt{xy}\), and \(\dfrac{\sqrt x}{\sqrt y}=\sqrt{\dfrac{x}{y}}\) where valid.
Rationalise single surd denominator: multiply top and bottom by that surd.
Rationalise binomial denominator: multiply by conjugate \((a+b)\leftrightarrow(a-b)\).
Lesson C — Positive Index Rules (Week 2C)▶
Rule 1: \(a^m\cdot a^n=a^{m+n}\).
Rule 2: \(\dfrac{a^m}{a^n}=a^{m-n}\) (same base, \(a\neq0\)).
Rule 3: \((ab)^n=a^n b^n\).
Rule 4: \(\left(\dfrac{a}{b}\right)^n=\dfrac{a^n}{b^n}\) (\(b\neq0\)).
Rule 5: \((a^m)^n=a^{mn}\).
Lesson D — Indices Extension (Week 2D)▶
Zero rule: \(a^0=1\) for \(a\neq0\); \(0^0\) undefined.
Negative rule: \(a^{-n}=\dfrac1{a^n}\), and \(\left(\dfrac{a}{b}\right)^{-n}=\left(\dfrac{b}{a}\right)^n\).
Fractional rule: \(a^{m/n}=\sqrt[n]{a^m}=(\sqrt[n]{a})^m\).
Combined laws: \(a^m a^n=a^{m+n}\), \(\dfrac{a^m}{a^n}=a^{m-n}\), \((a^m)^n=a^{mn}\).
Lesson E — Challenge Set Patterns (Week 2)▶
Surd error-checking (valid vs invalid simplifications), rationalising binomial denominators, and mixed-index expressions under time pressure.
Engineering-style word setups (sound intensity \(I=I_0\,10^{L/10}\), decay \(A=A_0\,2^{-t/T}\)) translate directly to index simplification and growth/decay interpretation questions.
