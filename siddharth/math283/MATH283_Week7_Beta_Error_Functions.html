Lay of the land — this week's toolkit
  • Beta function: $B(m,n)=\displaystyle\int_0^1 t^{m-1}(1-t)^{n-1}\,dt$, defined for $m,n>0$.
  • Beta-Gamma relationship: $B(m,n)=\dfrac{\Gamma(m)\Gamma(n)}{\Gamma(m+n)}$ — this is the standard way Beta integrals are actually evaluated, rather than the defining integral directly.
  • Symmetry: $B(m,n)=B(n,m)$.
  • Trigonometric form: $B(m,n)=2\displaystyle\int_0^{\pi/2}\sin^{2m-1}\theta\cos^{2n-1}\theta\,d\theta$ (via the substitution $t=\sin^2\theta$).
  • Error function: $\text{erf}(x)=\dfrac{2}{\sqrt\pi}\displaystyle\int_0^x e^{-t^2}\,dt$; $\text{erf}(0)=0$, $\text{erf}(\infty)=1$; odd function: $\text{erf}(-x)=-\text{erf}(x)$.
  • Complementary error function: $\text{erfc}(x)=1-\text{erf}(x)=\dfrac{2}{\sqrt\pi}\displaystyle\int_x^\infty e^{-t^2}\,dt$.

0 The Beta Function

The Beta function is a two-parameter integral that, remarkably, can always be expressed exactly in terms of the (single-parameter) Gamma function — making the Beta-Gamma relationship the practical tool for evaluating it, rather than grinding through the defining integral by hand.

Where it comes from. $B(m,n)=\int_0^1t^{m-1}(1-t)^{n-1}dt$ generalises binomial-coefficient-like integrals; the trigonometric substitution $t=\sin^2\theta$ converts it into an integral of $\sin$ and $\cos$ powers over $[0,\pi/2]$, connecting it back to the trig-integral techniques from MATH142.

Key idea

$$B(m,n)=\int_0^1 t^{m-1}(1-t)^{n-1}\,dt = \frac{\Gamma(m)\Gamma(n)}{\Gamma(m+n)} = B(n,m)$$

Worked example

Example 1. Evaluate $B(3,4)$ using the Beta-Gamma relationship.

$B(3,4)=\dfrac{\Gamma(3)\Gamma(4)}{\Gamma(7)}=\dfrac{2!\cdot3!}{6!}=\dfrac{2\times6}{720}=\dfrac{12}{720}=\dfrac{1}{60}$.

Example 2. Evaluate $\displaystyle\int_0^{\pi/2}\sin^3\theta\cos^5\theta\,d\theta$ using the trigonometric Beta form.

Match $2m-1=3\implies m=2$; $2n-1=5\implies n=3$. Integral $=\dfrac12 B(2,3)=\dfrac12\cdot\dfrac{\Gamma(2)\Gamma(3)}{\Gamma(5)}=\dfrac12\cdot\dfrac{1\times2}{24}=\dfrac{1}{24}$.

Practice 1

Evaluate $B(2,5)$ using the Beta-Gamma relationship.

Solution

$B(2,5)=\dfrac{\Gamma(2)\Gamma(5)}{\Gamma(7)}=\dfrac{1!\cdot4!}{6!}=\dfrac{24}{720}=\dfrac{1}{30}$.

Practice 2

Evaluate $B(1/2,1)$ using the Beta-Gamma relationship and $\Gamma(1/2)=\sqrt\pi$.

Solution

$B(1/2,1)=\dfrac{\Gamma(1/2)\Gamma(1)}{\Gamma(3/2)}=\dfrac{\sqrt\pi\cdot1}{\tfrac12\sqrt\pi}=2$.

Practice 3

Evaluate $\displaystyle\int_0^1 t^2(1-t)^3\,dt$ by recognising it as a Beta function.

Solution

Matches $B(m,n)$ with $m-1=2\implies m=3$; $n-1=3\implies n=4$. So this is $B(3,4)=\dfrac{1}{60}$ (from the worked example).

Practice 4

Evaluate $\displaystyle\int_0^{\pi/2}\sqrt{\sin\theta}\,d\theta$ using the trigonometric Beta form.

Solution

$\sqrt{\sin\theta}=\sin^{1/2}\theta\cos^0\theta$. Match $2m-1=1/2\implies m=3/4$; $2n-1=0\implies n=1/2$.

Integral $=\dfrac12 B(3/4,1/2)=\dfrac12\cdot\dfrac{\Gamma(3/4)\Gamma(1/2)}{\Gamma(5/4)}$ (left in terms of Gamma values, since $\Gamma(3/4)$ and $\Gamma(5/4)$ are not elementary).

Practice 5

Evaluate $\displaystyle\int_0^1 \sqrt{t}(1-t)^2\,dt$ using the Beta function.

Solution

$\sqrt t=t^{1/2}$, so $m-1=1/2\implies m=3/2$; $n-1=2\implies n=3$.

$B(3/2,3)=\dfrac{\Gamma(3/2)\Gamma(3)}{\Gamma(9/2)}$. $\Gamma(3/2)=\tfrac12\sqrt\pi$, $\Gamma(3)=2$, $\Gamma(9/2)=\tfrac{105}{16}\sqrt\pi$ (from Week 3).

$=\dfrac{(\tfrac12\sqrt\pi)(2)}{\tfrac{105}{16}\sqrt\pi}=\dfrac{\sqrt\pi}{\tfrac{105}{16}\sqrt\pi}=\dfrac{16}{105}$.

Practice 6

Show that $B(m,n)=B(n,m)$ using the defining integral and the substitution $u=1-t$.

Solution

$B(m,n)=\displaystyle\int_0^1 t^{m-1}(1-t)^{n-1}\,dt$. Let $u=1-t,du=-dt$; when $t=0,u=1$; $t=1,u=0$.

$=\displaystyle\int_1^0(1-u)^{m-1}u^{n-1}(-du)=\int_0^1 u^{n-1}(1-u)^{m-1}\,du = B(n,m)$.

Answer: confirmed, $B(m,n)=B(n,m)$ — the substitution simply swaps the roles of the two exponents, matching the definition of $B(n,m)$.

Practice 7

Evaluate $\displaystyle\int_0^{\pi/2}\cos^7\theta\,d\theta$ using the trigonometric Beta form (with $\sin^0\theta$).

Solution

Match $2m-1=0\implies m=1/2$; $2n-1=7\implies n=4$.

Integral $=\dfrac12B(1/2,4)=\dfrac12\cdot\dfrac{\Gamma(1/2)\Gamma(4)}{\Gamma(9/2)}=\dfrac12\cdot\dfrac{\sqrt\pi\cdot6}{\tfrac{105}{16}\sqrt\pi}=\dfrac12\cdot\dfrac{6\times16}{105}=\dfrac{48}{105}=\dfrac{16}{35}$.

Practice 8

Use the relationship $\Gamma(m+n)B(m,n)=\Gamma(m)\Gamma(n)$ to find $\Gamma(6)$ given that $B(2,4)=\dfrac{1}{20}$ and $\Gamma(2)=1$.

Solution

$\Gamma(2+4)B(2,4)=\Gamma(2)\Gamma(4) \implies \Gamma(6)\cdot\dfrac{1}{20}=1\times6 \implies \Gamma(6)=120$.

Check: $\Gamma(6)=5!=120$. ✓ Matches, confirming the relationship works both as a definition and as a computational shortcut when one of the three quantities is already known.

Practice 9

Evaluate $\displaystyle\int_0^4 x^2(4-x)^3\,dx$ (a Beta-type integral scaled to a general interval $[0,4]$ rather than $[0,1]$, using the substitution $x=4t$).

Solution

Let $x=4t,dx=4\,dt$; when $x=0,t=0$; $x=4,t=1$.

$\displaystyle\int_0^1(4t)^2(4-4t)^3(4\,dt)=\int_0^1 16t^2\cdot64(1-t)^3\cdot4\,dt=4096\int_0^1t^2(1-t)^3\,dt=4096\,B(3,4)$.

$B(3,4)=\dfrac{1}{60}$ (from earlier): $4096\times\dfrac{1}{60}=\dfrac{4096}{60}=\dfrac{1024}{15}$.

Answer: $\dfrac{1024}{15}\approx68.27$.

Practice 10

Explain why the Beta-Gamma relationship $B(m,n)=\Gamma(m)\Gamma(n)/\Gamma(m+n)$ is far more practically useful than trying to evaluate $\int_0^1t^{m-1}(1-t)^{n-1}dt$ directly for non-integer $m,n$.

Solution

For general (especially non-integer) values of $m$ and $n$, the integral $\int_0^1t^{m-1}(1-t)^{n-1}dt$ has no elementary antiderivative — there's no direct way to "just integrate" it using ordinary techniques (substitution, parts, etc.), the same way $\int e^{-t^2}dt$ resists elementary methods. The Beta-Gamma relationship instead converts the problem into evaluating (or looking up/tabulating) the Gamma function at three specific points, which is a much more tractable and standard task — Gamma function values are well-tabulated and satisfy simple relationships (the recursion $\Gamma(n+1)=n\Gamma(n)$, and $\Gamma(1/2)=\sqrt\pi$) that make hand-computation feasible even without a full derivation from scratch each time.

1 The Error Function

The integral $\int e^{-t^2}dt$ has no elementary antiderivative (it's the same integral underlying the normal distribution, met again via polar coordinates back in MATH142 Week 12 and again this week's Beta function's trigonometric cousin). The error function packages this integral, scaled and normalised, into a standard named function.

Key properties. $\text{erf}(0)=0$; $\text{erf}(x)\to1$ as $x\to\infty$ (the total area under the full Gaussian curve, once normalised, is exactly $1$); $\text{erf}$ is an odd function, $\text{erf}(-x)=-\text{erf}(x)$, reflecting the symmetry of $e^{-t^2}$.

Key idea

$$\text{erf}(x)=\frac{2}{\sqrt\pi}\int_0^x e^{-t^2}\,dt, \qquad \text{erfc}(x)=1-\text{erf}(x)$$

Worked example

Example 1. Express $\displaystyle\int_0^2 e^{-t^2}\,dt$ in terms of $\text{erf}$.

By definition, $\text{erf}(2)=\dfrac{2}{\sqrt\pi}\displaystyle\int_0^2e^{-t^2}dt \implies \displaystyle\int_0^2e^{-t^2}dt=\dfrac{\sqrt\pi}{2}\text{erf}(2)$.

Example 2. Express $\displaystyle\int_1^3 e^{-t^2}\,dt$ in terms of $\text{erf}$.

$\displaystyle\int_1^3=\int_0^3-\int_0^1=\dfrac{\sqrt\pi}{2}\big[\text{erf}(3)-\text{erf}(1)\big]$.

Practice 11

Express $\displaystyle\int_0^5 e^{-t^2}\,dt$ in terms of $\text{erf}$.

Solution

Answer: $\dfrac{\sqrt\pi}{2}\text{erf}(5)$.

Practice 12

Express $\displaystyle\int_{-2}^2 e^{-t^2}\,dt$ in terms of $\text{erf}$, using the fact that $e^{-t^2}$ is an even function.

Solution

By symmetry: $\displaystyle\int_{-2}^2 e^{-t^2}dt=2\int_0^2 e^{-t^2}dt=2\cdot\dfrac{\sqrt\pi}{2}\text{erf}(2)=\sqrt\pi\,\text{erf}(2)$.

Practice 13

Express $\displaystyle\int_{-1}^{3} e^{-t^2}\,dt$ in terms of $\text{erf}$, using the oddness of $\text{erf}$ where needed.

Solution

$\displaystyle\int_{-1}^3=\int_{-1}^0+\int_0^3$. First: $\int_{-1}^0e^{-t^2}dt=\int_0^1e^{-t^2}dt$ (even integrand, symmetric bounds) $=\dfrac{\sqrt\pi}{2}\text{erf}(1)$.

Second: $\dfrac{\sqrt\pi}{2}\text{erf}(3)$.

Answer: $\dfrac{\sqrt\pi}{2}\big[\text{erf}(1)+\text{erf}(3)\big]$.

Practice 14

Using the substitution $u=t/\sqrt2$, express $\displaystyle\int_0^x e^{-u^2/2}\,du$ in terms of $\text{erf}$ (this exact form appears constantly in the normal distribution, Week 10).

Solution

Let $u=\sqrt2\,w$ (so that $u^2/2=w^2$), $du=\sqrt2\,dw$; when $u=0,w=0$; $u=x,w=x/\sqrt2$.

$\displaystyle\int_0^x e^{-u^2/2}\,du=\sqrt2\int_0^{x/\sqrt2}e^{-w^2}\,dw=\sqrt2\cdot\dfrac{\sqrt\pi}{2}\text{erf}\!\left(\dfrac{x}{\sqrt2}\right)$.

Answer: $\displaystyle\int_0^xe^{-u^2/2}\,du=\sqrt{\dfrac\pi2}\,\text{erf}\!\left(\dfrac{x}{\sqrt2}\right)$.

Practice 15

Explain why $\text{erf}(x)\to1$ as $x\to\infty$, connecting this to the classic Gaussian integral result $\displaystyle\int_0^\infty e^{-t^2}\,dt=\dfrac{\sqrt\pi}{2}$.

Solution

By definition, $\text{erf}(\infty)=\dfrac2{\sqrt\pi}\displaystyle\int_0^\infty e^{-t^2}\,dt$. Using the known Gaussian integral result $\int_0^\infty e^{-t^2}dt=\dfrac{\sqrt\pi}{2}$ (derivable via polar coordinates, as in MATH142 Week 12/the Week 3 normalisation constant this course met): $\text{erf}(\infty)=\dfrac2{\sqrt\pi}\cdot\dfrac{\sqrt\pi}{2}=1$.

This is precisely why the $\dfrac{2}{\sqrt\pi}$ normalisation factor was chosen in the definition of $\text{erf}$ — so that it approaches exactly $1$ (rather than some other constant) as $x\to\infty$, making it directly usable as a probability-like quantity.

Practice 16

Find $\text{erfc}(0)$ and explain its value using the definitions of $\text{erf}$ and $\text{erfc}$.

Solution

$\text{erfc}(0)=1-\text{erf}(0)=1-0=1$ (since $\text{erf}(0)=\dfrac2{\sqrt\pi}\int_0^0e^{-t^2}dt=0$, an integral over zero width).

Practice 17

A heat-conduction problem gives the temperature at position $x$ and time $t$ as $T(x,t)=T_0\,\text{erfc}\!\left(\dfrac{x}{2\sqrt{\alpha t}}\right)$. Find $\displaystyle\lim_{t\to\infty}T(x,t)$ for fixed $x>0$, using the behaviour of erfc as its argument approaches 0.

Solution

As $t\to\infty$ (with $x$ fixed), $\dfrac{x}{2\sqrt{\alpha t}}\to0$.

$\text{erfc}(0)=1$ (from the previous question), so $\displaystyle\lim_{t\to\infty}T(x,t)=T_0\times1=T_0$.

Answer: $T_0$ — physically, this says that after a very long time, every point eventually reaches the boundary/reference temperature $T_0$, a sensible result for a heat-diffusion process.

Practice 18

Express $\displaystyle\int_2^\infty e^{-t^2}\,dt$ in terms of $\text{erfc}$.

Solution

By definition, $\text{erfc}(2)=\dfrac2{\sqrt\pi}\displaystyle\int_2^\infty e^{-t^2}\,dt \implies \displaystyle\int_2^\infty e^{-t^2}\,dt=\dfrac{\sqrt\pi}{2}\text{erfc}(2)$.

Practice 19

Show that $\text{erf}(-x)=-\text{erf}(x)$ directly from the defining integral, using the substitution $u=-t$.

Solution

$\text{erf}(-x)=\dfrac2{\sqrt\pi}\displaystyle\int_0^{-x}e^{-t^2}\,dt$. Let $u=-t,du=-dt$; when $t=0,u=0$; $t=-x,u=x$.

$=\dfrac2{\sqrt\pi}\displaystyle\int_0^{x}e^{-u^2}(-du)\cdot(-1)$... more carefully: $\int_0^{-x}e^{-t^2}dt=\int_0^xe^{-u^2}(-du)$ with limits flipping sign appropriately, giving $-\int_0^xe^{-u^2}du$.

$\text{erf}(-x)=\dfrac2{\sqrt\pi}\left(-\displaystyle\int_0^xe^{-u^2}\,du\right)=-\text{erf}(x)$.

Answer: confirmed — $\text{erf}$ is odd, directly reflecting the even symmetry of the Gaussian integrand $e^{-t^2}$ itself.

Practice 20

Given that $\text{erf}(1)\approx0.8427$, find $\text{erfc}(1)$ and $\text{erf}(-1)$, and briefly state what each represents in terms of area under the Gaussian curve.

Solution

$\text{erfc}(1)=1-0.8427=0.1573$. $\text{erf}(-1)=-0.8427$ (using the odd-function property).

Interpretation: $\text{erf}(1)\approx0.8427$ represents (a normalised version of) the area under $e^{-t^2}$ between $t=0$ and $t=1$; $\text{erfc}(1)\approx0.1573$ represents the remaining area from $t=1$ out to $t=\infty$ (the two together account for the entire area from $0$ to $\infty$, normalised to sum to $1$).
