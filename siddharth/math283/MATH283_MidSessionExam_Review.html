Lay of the land — this week's toolkit
  • Multivariable functions: domain constraints (square roots, denominators, logs); level curves $f(x,y)=k$; limits must agree along every path (two-path test to disprove; polar coordinates to prove).
  • Partial derivatives: differentiate treating other variables as constants; $f_{xy}=f_{yx}$ (Clairaut); chain rule $\dfrac{dz}{dt}=z_x\dfrac{dx}{dt}+z_y\dfrac{dy}{dt}$; implicit diff $\dfrac{dy}{dx}=-F_x/F_y$; differentials $dz=f_xdx+f_ydy$ for error estimation.
  • Double integrals: iterated integrals, reverse order when the given order is unsolvable; area $\iint1\,dA$, volume $\iint f\,dA$; polar form $dA=r\,dr\,d\theta$.
  • Laplace transforms: $\mathcal L\{y'\}=sY-y(0)$; solve ODEs by transforming, solving algebraically, inverting; Heaviside $u(t-a)$, delta $\delta(t-a)$; shift theorems ($e^{at}f(t)\to F(s-a)$; $f(t-a)u(t-a)\to e^{-as}F(s)$); derivative-of-transform for $t^nf(t)$; convolution $F(s)G(s)\to(f*g)(t)$; periodic-function formula.
  • Fourier series: $a_n,b_n$ formulas; even/odd shortcuts; convergence to the average at jumps; half-range cosine/sine series; general period via $L=T/2$.
  • Special functions: $\Gamma(n+1)=n\Gamma(n)$, $\Gamma(1/2)=\sqrt\pi$; $B(m,n)=\Gamma(m)\Gamma(n)/\Gamma(m+n)$; $\text{erf}(x)=\tfrac{2}{\sqrt\pi}\int_0^xe^{-t^2}dt$.

0 Recap: Multivariable Functions, Partial Derivatives & Double Integrals

The exam-critical skill across Weeks 1–3 is correct setup: identifying domain restrictions, picking the right path(s) to test a limit, choosing which variable to hold constant, and correctly identifying integration bounds (Cartesian or polar) for a given region.

Key idea

$$\frac{dz}{dt}=z_x\frac{dx}{dt}+z_y\frac{dy}{dt}, \qquad dA_{\text{polar}}=r\,dr\,d\theta$$

Worked example

Exam-style. Find the domain of $f(x,y)=\ln(x^2+y^2-1)$, then find $f_x$ and $f_y$, then find the volume under $z=f(x,y)+2$ is not directly needed — instead, evaluate $\displaystyle\iint_R (x^2+y^2)\,dA$ where $R$ is the annulus $1\le x^2+y^2\le4$.

Domain: $x^2+y^2>1$ (outside the unit circle). $f_x=\dfrac{2x}{x^2+y^2-1}$, $f_y=\dfrac{2y}{x^2+y^2-1}$.

Polar: $\displaystyle\int_0^{2\pi}\int_1^2 r^2\cdot r\,dr\,d\theta=\int_0^{2\pi}\left[\dfrac{r^4}{4}\right]_1^2d\theta=\int_0^{2\pi}\dfrac{15}{4}\,d\theta=\dfrac{15\pi}{2}$.

Practice 1

Find the domain of $f(x,y)=\sqrt{y-x}+\ln(x)$.

Solution

Need $y-x\ge0$ (i.e. $y\ge x$) AND $x>0$.

Answer: $\{(x,y): y\ge x,\ x>0\}$.

Practice 2

Show that $\displaystyle\lim_{(x,y)\to(0,0)}\frac{x^2y}{x^4+y^2}$ does not exist, using the path $y=x^2$.

Solution

Along $y=0$: limit is $0$. Along $y=x^2$: $\dfrac{x^2\cdot x^2}{x^4+x^4}=\dfrac12$.

Answer: different limits, so the limit does not exist.

Practice 3

If $z=x^2y^3$, $x=e^t,y=\cos t$, find $\dfrac{dz}{dt}$ at $t=0$ using the chain rule.

Solution

$z_x=2xy^3,z_y=3x^2y^2$; $dx/dt=e^t,dy/dt=-\sin t$.

At $t=0$: $x=1,y=1$. $\dfrac{dz}{dt}=2(1)(1)(1)+3(1)(1)(0)=2$.

Practice 4

Evaluate $\displaystyle\int_0^1\int_{\sqrt x}^1 e^{y^3}\,dy\,dx$ by reversing the order of integration.

Solution

Region: $0\le x\le1,\sqrt x\le y\le1 \to 0\le y\le1,0\le x\le y^2$.

$\displaystyle\int_0^1\int_0^{y^2}e^{y^3}\,dx\,dy=\int_0^1y^2e^{y^3}\,dy=\dfrac13(e-1)$.

Practice 5

The volume of a cylinder is $V=\pi r^2h$. If $r=4$ cm and $h=10$ cm are each measured with possible error $\pm0.1$ cm, estimate the maximum error in $V$.

Solution

$V_r=2\pi rh=80\pi$; $V_h=\pi r^2=16\pi$.

$dV=80\pi(0.1)+16\pi(0.1)=8\pi+1.6\pi=9.6\pi\approx30.16\ \text{cm}^3$.

1 Recap: Laplace Transforms & Solving ODEs

Exam questions typically test the full pipeline: transform, solve algebraically for $Y(s)$, then invert (via partial fractions, shift theorems, or convolution). Watch for resonance (repeated factors matching the forcing frequency) and delayed/switched forcing (Heaviside functions), which both require extra care at the inversion step.

Key idea

$$\mathcal L\{y''\}=s^2Y-sy(0)-y'(0), \qquad \mathcal L\{e^{at}f(t)\}=F(s-a), \qquad \mathcal L\{f(t-a)u(t-a)\}=e^{-as}F(s)$$

Worked example

Exam-style. Solve $y''+2y'+y=e^{-t}$, $y(0)=0,y'(0)=0$ (repeated root, resonant-type forcing).

$s^2Y+2sY+Y=\dfrac{1}{s+1} \implies Y(s+1)^2=\dfrac{1}{s+1} \implies Y=\dfrac{1}{(s+1)^3}$.

Matching $\mathcal L\{t^ne^{at}\}=n!/(s-a)^{n+1}$ with $n=2,a=-1$: $Y=\dfrac12\cdot\dfrac{2}{(s+1)^3}$, so $y(t)=\dfrac{t^2e^{-t}}{2}$.

Practice 6

Solve $y'-5y=0$, $y(0)=2$.

Solution

$sY-2-5Y=0\implies Y=\dfrac{2}{s-5}$.

Answer: $y(t)=2e^{5t}$.

Practice 7

Find $\mathcal L\{t^2e^{3t}\}$ using the derivative-of-transform formula.

Solution

$\mathcal L\{t^2e^{3t}\}=\dfrac{2!}{(s-3)^3}=\dfrac{2}{(s-3)^3}$.

Practice 8

Solve $y''+4y=8u(t-1)$, $y(0)=0,y'(0)=0$.

Solution

$s^2Y+4Y=\dfrac{8e^{-s}}{s} \implies Y=\dfrac{8e^{-s}}{s(s^2+4)}$.

$\dfrac{8}{s(s^2+4)}=\dfrac2s-\dfrac{2s}{s^2+4}$ (partial fractions), inverse-transforming to $2-2\cos2t$.

Answer: $y(t)=\big[2-2\cos(2(t-1))\big]u(t-1)$ (second shift theorem).

Practice 9

Use convolution to find $\mathcal L^{-1}\left\{\dfrac{1}{s(s-2)}\right\}$ with $f(t)=1,g(t)=e^{2t}$.

Solution

$(f*g)(t)=\displaystyle\int_0^t e^{2(t-\tau)}\,d\tau=e^{2t}\int_0^te^{-2\tau}d\tau=e^{2t}\left[\dfrac{1-e^{-2t}}{2}\right]=\dfrac{e^{2t}-1}{2}$.

Practice 10

Find $\mathcal L\{f(t)\}$ for a square wave with $f(t)=4$ on $[0,1)$, $f(t)=0$ on $[1,2)$, period $T=2$.

Solution

$\displaystyle\int_0^2e^{-st}f(t)\,dt=4\int_0^1e^{-st}dt=\dfrac{4(1-e^{-s})}{s}$.

Answer: $\mathcal L\{f(t)\}=\dfrac{1}{1-e^{-2s}}\cdot\dfrac{4(1-e^{-s})}{s}=\dfrac{4}{s(1+e^{-s})}$.

2 Recap: Fourier Series & Special Functions

Fourier questions typically hinge on spotting even/odd symmetry before computing (saves half the work), and correctly identifying $L$ for the given period. Beta/Gamma questions almost always reduce to correctly matching the integral's form to $B(m,n)$ or $\Gamma(n)$ and then using the recursion or the Beta-Gamma relationship.

Key idea

$$a_n=\frac1L\int_{-L}^Lf(t)\cos\frac{n\pi t}{L}dt, \qquad B(m,n)=\frac{\Gamma(m)\Gamma(n)}{\Gamma(m+n)}$$

Worked example

Exam-style. Find $a_0$ and $b_n$ for $f(t)=t$ on $(0,2)$ extended as an odd function (half-range sine series, $L=2$), then evaluate $\Gamma(11/2)$.

Odd extension $\implies a_n=0$ for all $n$ including $a_0$.

$b_n=\dfrac22\displaystyle\int_0^2t\sin\!\left(\dfrac{n\pi t}{2}\right)dt$; by parts, this evaluates (following the Week 6 pattern) to $b_n=\dfrac{4(-1)^{n+1}}{n\pi}$.

$\Gamma(11/2)=\dfrac92\Gamma(9/2)=\dfrac92\cdot\dfrac{105}{16}\sqrt\pi=\dfrac{945}{32}\sqrt\pi$.

Practice 11

Explain (without computing) why $f(t)=\cos(3t)+5$ has $b_n=0$ for all $n$ in its Fourier series.

Solution

Both $\cos(3t)$ and the constant $5$ are even functions, and the sum of even functions is even — so $f(t)$ itself is even, guaranteeing all $b_n=0$.

Practice 12

Find what the Fourier series of a square wave converges to at a jump where $f(t^-)=3$ and $f(t^+)=-1$.

Solution

Average: $\dfrac{3+(-1)}{2}=1$.

Practice 13

Evaluate $B(4,2)$ using the Beta-Gamma relationship.

Solution

$B(4,2)=\dfrac{\Gamma(4)\Gamma(2)}{\Gamma(6)}=\dfrac{6\times1}{120}=\dfrac{1}{20}$.

Practice 14

Express $\displaystyle\int_0^{\pi/2}\sin^5\theta\cos^3\theta\,d\theta$ as a Beta function and evaluate.

Solution

$2m-1=5\implies m=3$; $2n-1=3\implies n=2$. Integral $=\tfrac12B(3,2)=\tfrac12\cdot\dfrac{\Gamma(3)\Gamma(2)}{\Gamma(5)}=\tfrac12\cdot\dfrac{2\times1}{24}=\dfrac{1}{24}$.

Practice 15

Express $\displaystyle\int_0^3 e^{-t^2}\,dt$ in terms of erf.

Solution

Answer: $\dfrac{\sqrt\pi}{2}\text{erf}(3)$.

3 Mixed Exam-Style Problems (Weeks 1–7)

Real exam questions often combine two or more of this phase's techniques in one multi-part problem. Read each fully before starting — later parts often reuse an earlier result.

Key idea

No single formula — the skill is recognising which tool each part needs, and carrying results between parts.

Worked example

Exam-style, multi-part. (a) Solve $y''+y=0$, $y(0)=1,y'(0)=0$. (b) Confirm your solution is an even function, and find its Fourier cosine series on $(-\pi,\pi)$ (a trick question — think about what's actually being asked).

(a) $s^2Y-s+Y=0\implies Y=\dfrac{s}{s^2+1}\implies y(t)=\cos t$.

(b) $\cos t$ is already even, and is already a single pure cosine term — its own Fourier series (on any symmetric interval containing a full period) is trivially itself: $a_1=1$, every other $a_n=0$, and $b_n=0$ throughout. No further decomposition is needed since the function already matches one basis element exactly.

Practice 16

(a) Find the volume under $z=x^2+y^2$ over the disk $x^2+y^2\le1$ using polar coordinates. (b) Find $\Gamma(5/2)$ and note that its value doesn't directly relate to part (a), but both required recognising a substitution pattern.

Solution

(a) $V=\displaystyle\int_0^{2\pi}\int_0^1 r^2\cdot r\,dr\,d\theta=\int_0^{2\pi}\dfrac14\,d\theta=\dfrac{\pi}{2}$.

(b) $\Gamma(5/2)=\dfrac32\Gamma(3/2)=\dfrac32\cdot\dfrac12\sqrt\pi=\dfrac34\sqrt\pi$.

Practice 17

A tank's salt content satisfies $y'+\tfrac{1}{10}y=5$, $y(0)=0$ (constant-volume mixing, MATH142-style). (a) Solve using the Laplace transform. (b) Find the steady-state value as $t\to\infty$, and confirm it matches $5/(1/10)=50$.

Solution

(a) $sY+\tfrac{1}{10}Y=\dfrac5s \implies Y\left(s+\tfrac{1}{10}\right)=\dfrac5s \implies Y=\dfrac{5}{s(s+0.1)}$.

Partial fractions: $\dfrac{50}{s}-\dfrac{50}{s+0.1}$, giving $y(t)=50-50e^{-0.1t}=50(1-e^{-0.1t})$.

(b) As $t\to\infty$, $e^{-0.1t}\to0$, so $y\to50$ — matches the steady-state ratio $5/0.1=50$.

Practice 18

(a) Find $\mathcal L\{t\cos(2t)\}$ using the derivative-of-transform formula. (b) Use this result to solve $y''+4y=\cos(2t)$, $y(0)=0,y'(0)=0$ (a resonance case).

Solution

(a) $F(s)=\dfrac{s}{s^2+4}$; $-\dfrac{d}{ds}F(s)=-\dfrac{(s^2+4)-s(2s)}{(s^2+4)^2}=\dfrac{s^2-4}{(s^2+4)^2}$.

(b) $s^2Y+4Y=\dfrac{s}{s^2+4} \implies Y=\dfrac{s}{(s^2+4)^2}$.

This matches $\mathcal L\{t\sin2t\}=\dfrac{4s}{(s^2+4)^2}$, so $Y=\dfrac14\cdot\dfrac{4s}{(s^2+4)^2}$: $y(t)=\dfrac{t\sin(2t)}{4}$ — the growing amplitude confirms resonance, as the forcing frequency (2) matches the system's natural frequency.

Practice 19

(a) Find the half-range sine series coefficient $b_1$ for $f(t)=1$ on $(0,\pi)$. (b) Using the Beta function, evaluate $\displaystyle\int_0^{\pi/2}\sin\theta\,d\theta$ (as a sanity check, since this should just equal 1 directly).

Solution

(a) $b_1=\dfrac2\pi\displaystyle\int_0^\pi\sin t\,dt=\dfrac2\pi\big[-\cos t\big]_0^\pi=\dfrac2\pi(1-(-1))=\dfrac4\pi$.

(b) Using the Beta form: $2m-1=1\implies m=1$; $2n-1=0\implies n=1/2$. Integral $=\tfrac12B(1,1/2)=\tfrac12\cdot\dfrac{\Gamma(1)\Gamma(1/2)}{\Gamma(3/2)}=\tfrac12\cdot\dfrac{1\times\sqrt\pi}{\tfrac12\sqrt\pi}=\tfrac12\times2=1$. Matches the direct evaluation $\int_0^{\pi/2}\sin\theta\,d\theta=[-\cos\theta]_0^{\pi/2}=1$. ✓

Mastery Challenge (Multi-part)

Purpose: exam-level synthesis across techniques with fully worked steps.

Practice 39

Exam-style mixed problem: (a) evaluate $\iint_R (x-y)\,dA$ where $R$ is between $y=x^2$ and $y=2x$; (b) solve $y'+2y=u(t-1)$ with $y(0)=0$.

Solution

(a) $x\in[0,2]$, so integral is $\int_0^2\int_{x^2}^{2x}(x-y)dy\,dx=\int_0^2(2x^2-x^3-\tfrac12x^4)dx=\frac{4}{15}$. (b) Laplace gives $Y=\frac{e^{-s}}{s(s+2)}$. Since $\mathcal L^{-1}\{1/[s(s+2)]\}=\frac12(1-e^{-2t})$, the answer is $y(t)=\frac12u(t-1)\big(1-e^{-2(t-1)}\big)$.

Practice 40

Find Fourier series for odd extension of $f(x)=x(\pi-x)$ on $(0,\pi)$ (setup + first nonzero coefficient).

Solution

Odd extension implies sine series only: $b_n=\frac{2}{\pi}\int_0^\pi x(\pi-x)\sin(nx)dx$. Integrate by parts twice; coefficients simplify to $b_n=\frac{8}{\pi n^3}$ for odd $n$, $0$ for even $n$. First nonzero term is $\frac{8}{\pi}\sin x$.

Practice 41

Use Beta/Gamma to evaluate $\int_0^1 \sqrt{x}(1-x)^{3/2}dx$.

Solution

Integral is $B(3/2,5/2)=\dfrac{\Gamma(3/2)\Gamma(5/2)}{\Gamma(4)}=\dfrac{(\sqrt\pi/2)(3\sqrt\pi/4)}{6}=\frac{\pi}{16}$.

Practice 42

Inverse Laplace challenge: $F(s)=\dfrac{e^{-2s}(3s+1)}{s^2+4s+13}$. Find $f(t)$.

Solution

Complete square: $s^2+4s+13=(s+2)^2+3^2$, and $3s+1=3(s+2)-5$. Without shift: $3e^{-2t}\cos3t-\frac{5}{3}e^{-2t}\sin3t$. Apply $e^{-2s}$ shift: $f(t)=u(t-2)\left[3e^{-2(t-2)}\cos3(t-2)-\frac{5}{3}e^{-2(t-2)}\sin3(t-2)\right]$.
