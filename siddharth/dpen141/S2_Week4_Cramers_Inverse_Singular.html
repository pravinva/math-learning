Cramer's Rule, Inverse & Singular Matrices

Two more ways to solve (or fail to solve) a linear system: Cramer's Rule, which solves directly via determinants, and the inverse matrix method, which solves any number of systems with the same coefficient matrix almost instantly once A⁻¹ is known. Both methods break down in exactly the same circumstance — a singular coefficient matrix — which is explored in its own right at the end. Solutions stay hidden until you click.

0 Cramer's Rule

Cramer's Rule solves a linear system directly using determinants, with no row reduction at all. It's not always the fastest method for large systems, but for $2\times2$ and $3\times3$ systems it's often quicker than elimination, and it reveals immediately (via the denominator) exactly when a unique solution fails to exist.

Key formula

For $A\mathbf{x}=\mathbf{b}$ with $A$ an $n\times n$ matrix, let $A_i$ be $A$ with column $i$ replaced by $\mathbf{b}$. Then, provided $\det A\ne0$: $$x_i = \frac{\det A_i}{\det A}.$$

If $\det A=0$, Cramer's Rule cannot be used at all — the system either has no solution or infinitely many.

Worked example

Use Cramer's Rule to solve $4x_1-x_2-x_3=1,\ 2x_1+2x_2+3x_3=10,\ 5x_1-2x_2-2x_3=-1$.

$A=\begin{pmatrix}4&-1&-1\\2&2&3\\5&-2&-2\end{pmatrix}$. Expanding along row 1: $\det A = 4\det\begin{pmatrix}2&3\\-2&-2\end{pmatrix}-(-1)\det\begin{pmatrix}2&3\\5&-2\end{pmatrix}+(-1)\det\begin{pmatrix}2&2\\5&-2\end{pmatrix}$ $= 4(-4+6)+1(-4-15)-1(-4-10) = 4(2)+1(-19)-1(-14) = 8-19+14=3$.

$A_1$ (replace column 1 with $\mathbf{b}=(1,10,-1)$): $\det\begin{pmatrix}1&-1&-1\\10&2&3\\-1&-2&-2\end{pmatrix}$. Expanding along row 1: $1\det\begin{pmatrix}2&3\\-2&-2\end{pmatrix}+1\det\begin{pmatrix}10&3\\-1&-2\end{pmatrix}-1\det\begin{pmatrix}10&2\\-1&-2\end{pmatrix} = 1(2)+1(-20+3)-1(-20+2)=2-17+18=3$.

$x_1=\dfrac{\det A_1}{\det A}=\dfrac{3}{3}=1$. (Finding $x_2,x_3$ follows the same pattern, replacing columns 2 and 3 in turn.)

Practice 1

Use Cramer's Rule to solve $2x+3y=7,\ x-y=1$.

Solution

$A=\begin{pmatrix}2&3\\1&-1\end{pmatrix}$, $\det A=-2-3=-5$. $A_1=\begin{pmatrix}7&3\\1&-1\end{pmatrix}$, $\det A_1=-7-3=-10$. $A_2=\begin{pmatrix}2&7\\1&1\end{pmatrix}$, $\det A_2=2-7=-5$. $x=\dfrac{-10}{-5}=2$, $\ y=\dfrac{-5}{-5}=1$.

Practice 2

Use Cramer's Rule to solve $5x-4y=2,\ 6x-5y=1$.

Solution

$\det A = 5(-5)-(-4)(6)=-25+24=-1$. $\det A_1=\det\begin{pmatrix}2&-4\\1&-5\end{pmatrix}=-10+4=-6$. $\det A_2=\det\begin{pmatrix}5&2\\6&1\end{pmatrix}=5-12=-7$. $x=\dfrac{-6}{-1}=6$, $\ y=\dfrac{-7}{-1}=7$.

Practice 3

Explain what it means, in terms of the system's solution, if Cramer's Rule gives $\det A = 0$ but $\det A_1\ne0$.

Solution

The system has no solution (it's inconsistent). $\det A=0$ means the coefficient rows are dependent (the lines/planes are parallel), but $\det A_1\ne0$ shows the right-hand side breaks that same proportionality — so no point satisfies every equation simultaneously.

Practice 4

Attempt to use Cramer's Rule for the system $4x_1-x_2-x_3=1,\ 2x_1+2x_2+3x_3=10,\ 5x_1-2x_2-2x_3=-1$ to find $x_2$, given $\det A=3$ from the worked example, and $A_2=\begin{pmatrix}4&1&-1\\2&10&3\\5&-1&-2\end{pmatrix}$.

Solution

Expand $\det A_2$ along row 1: $4\det\begin{pmatrix}10&3\\-1&-2\end{pmatrix}-1\det\begin{pmatrix}2&3\\5&-2\end{pmatrix}+(-1)\det\begin{pmatrix}2&10\\5&-1\end{pmatrix} = 4(-20+3)-1(-4-15)-1(-2-50)=4(-17)+19+52=-68+19+52=3$. So $x_2=\dfrac{3}{3}=1$.

Practice 5

A $3\times3$ system has $\det A=0$. Explain, without computing anything further, why Cramer's Rule cannot be used to find a value for $x_1$.

Solution

Cramer's Rule requires dividing by $\det A$; division by zero is undefined, so the formula simply breaks down. This isn't a computational inconvenience — it's the formula's own signal that the system doesn't have a unique solution at all.

1 The Inverse of a Matrix

The inverse $A^{-1}$ plays the same role for matrices that $\dfrac1a$ plays for numbers: $AA^{-1}=A^{-1}A=I$. For a $2\times2$ matrix there's a direct formula; for larger matrices, the inverse is usually found via row-reducing the augmented matrix $[A\,|\,I]$ to $[I\,|\,A^{-1}]$.

Key formula — the $2\times2$ inverse

$$A=\begin{pmatrix}a&b\\c&d\end{pmatrix} \ \Rightarrow\ A^{-1}=\frac{1}{\det A}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}, \qquad \det A = ad-bc\ne0.$$

Once you have $A^{-1}$, the system $A\mathbf{x}=\mathbf{b}$ solves instantly as $\mathbf{x}=A^{-1}\mathbf{b}$.

Worked example

Find the inverse of $\begin{pmatrix}1&4\\-1&-3\end{pmatrix}$.

$\det A = 1(-3)-4(-1) = -3+4=1$.

$$A^{-1} = \frac{1}{1}\begin{pmatrix}-3&-4\\1&1\end{pmatrix} = \begin{pmatrix}-3&-4\\1&1\end{pmatrix}.$$

Practice 6

Find the inverse of $\begin{pmatrix}2&1\\5&3\end{pmatrix}$.

Solution

$\det=6-5=1$. $A^{-1}=\begin{pmatrix}3&-1\\-5&2\end{pmatrix}$.

Practice 7

Find the inverse of $\begin{pmatrix}3&-2\\1&4\end{pmatrix}$, and verify $AA^{-1}=I$.

Solution

$\det=12-(-2)=14$. $A^{-1}=\dfrac{1}{14}\begin{pmatrix}4&2\\-1&3\end{pmatrix}$. Check: $AA^{-1}=\dfrac{1}{14}\begin{pmatrix}3&-2\\1&4\end{pmatrix}\begin{pmatrix}4&2\\-1&3\end{pmatrix}=\dfrac{1}{14}\begin{pmatrix}12+2&6-6\\4-4&2+12\end{pmatrix}=\dfrac{1}{14}\begin{pmatrix}14&0\\0&14\end{pmatrix}=I$. ✓

Practice 8

Use the inverse matrix method to solve $3x-2y=7,\ x+4y=-6$.

Solution

$A=\begin{pmatrix}3&-2\\1&4\end{pmatrix}$, using the inverse from Practice 6: $A^{-1}=\dfrac{1}{14}\begin{pmatrix}4&2\\-1&3\end{pmatrix}$. $\mathbf{x}=A^{-1}\mathbf{b}=\dfrac{1}{14}\begin{pmatrix}4&2\\-1&3\end{pmatrix}\begin{pmatrix}7\\-6\end{pmatrix}=\dfrac{1}{14}\begin{pmatrix}28-12\\-7-18\end{pmatrix}=\dfrac{1}{14}\begin{pmatrix}16\\-25\end{pmatrix}$. So $x=\dfrac{16}{14}=\dfrac87$, $\ y=-\dfrac{25}{14}$.

Practice 9

Find the inverse of the system's coefficient matrix $A=\begin{pmatrix}1&2&2\\1&1&1\\3&-3&-1\end{pmatrix}$ using the augmented matrix method, reducing $[A\,|\,I]$ to $[I\,|\,A^{-1}]$. (You may present the row operations and the final result.)

Solution

Start: $\left(\begin{array}{ccc|ccc}1&2&2&1&0&0\\1&1&1&0&1&0\\3&-3&-1&0&0&1\end{array}\right)$. $R_2\to R_2-R_1$: $(0,-1,-1,-1,1,0)$. $R_3\to R_3-3R_1$: $(0,-9,-7,-3,0,1)$. $R_2\to-R_2$: $(0,1,1,1,-1,0)$. $R_3\to R_3+9R_2$: $(0,0,2,6,-9,1)$. $R_3\to\tfrac12R_3$: $(0,0,1,3,-4.5,0.5)$. Back-substituting to clear above each pivot gives, after full reduction, $A^{-1}=\begin{pmatrix}2&4&0\\-4&-7&1\\6&9&-1\end{pmatrix}$ (matching what direct verification $AA^{-1}=I$ confirms).

Practice 10

For the system $A\mathbf{x}=\mathbf{b}$ with $A=\begin{pmatrix}1&2&2\\1&1&1\\3&-3&-1\end{pmatrix}$ and $\mathbf{b}=\begin{pmatrix}2\\0\\0\end{pmatrix}$, use the inverse from Practice 9 to solve for $\mathbf{x}$.

Solution

$\mathbf{x}=A^{-1}\mathbf{b}=\begin{pmatrix}2&4&0\\-4&-7&1\\6&9&-1\end{pmatrix}\begin{pmatrix}2\\0\\0\end{pmatrix}=\begin{pmatrix}4\\-8\\12\end{pmatrix}$. So $x=4,\ y=-8,\ z=12$.

2 Singular Matrices

A matrix with $\det A=0$ is singular: it has no inverse at all, and the row-reduction attempt to find one will always produce a row of zeros somewhere, making the identity matrix impossible to reach on the left-hand block.

Key idea

$A$ is singular $\iff$ $\det A=0$ $\iff$ $A$ has rank less than $n$ $\iff$ the rows (or columns) of $A$ are linearly dependent $\iff$ $A\mathbf{x}=\mathbf{0}$ has a nonzero solution. All of these are different windows onto the exact same underlying fact.

Worked example

Show that $A=\begin{pmatrix}1&2&3\\2&4&6\\1&0&2\end{pmatrix}$ is singular, and identify the dependency among its rows.

Notice immediately that row 2 is exactly $2\times$ row 1: $(2,4,6)=2(1,2,3)$.

Whenever one row is a scalar multiple of another, the determinant is automatically $0$: $\det A=0$, confirming $A$ is singular.

This is the cleanest way to spot singularity: before ever expanding a determinant, scan for a row (or column) that's a multiple of another, or a sum of others — if you find one, you already know the determinant is $0$ without computing anything.

Practice 11

Determine whether $\begin{pmatrix}2&4\\3&6\end{pmatrix}$ is singular, and if so, state the relationship between its rows.

Solution

$\det=12-12=0$: singular. Row 2 is $\tfrac32\times$ row 1.

Practice 12

Find the value(s) of $k$ for which $\begin{pmatrix}1&2&3\\2&k&6\\1&2&k\end{pmatrix}$ is singular.

Solution

Expand along row 1: $1\det\begin{pmatrix}k&6\\2&k\end{pmatrix}-2\det\begin{pmatrix}2&6\\1&k\end{pmatrix}+3\det\begin{pmatrix}2&k\\1&2\end{pmatrix} = (k^2-12)-2(2k-6)+3(4-k) = k^2-12-4k+12+12-3k=k^2-7k+12$. Set to $0$: $(k-3)(k-4)=0 \Rightarrow k=3$ or $k=4$.

Practice 13

Explain why every matrix with a row of all zeros is automatically singular, without computing a determinant.

Solution

Expanding the determinant along that all-zero row: every term in the cofactor expansion is multiplied by $0$, so the whole determinant is $0$ regardless of any other entries.

Practice 14

A $3\times3$ system $A\mathbf{x}=\mathbf{b}$ has $A$ singular. Explain (in terms of rank) the two possible outcomes for the system's solution set, and what distinguishes them.

Solution

Since $A$ is singular, its rank is less than $3$. If the augmented matrix $[A\,|\,\mathbf{b}]$ has the same rank as $A$, the system is consistent with infinitely many solutions. If the augmented matrix has a higher rank than $A$ (a row like $0=c$ for nonzero $c$ appears after reduction), the system is inconsistent and has no solution.

Practice 15

Find the value of $\lambda$ for which $\begin{pmatrix}\lambda&1\\4&\lambda\end{pmatrix}$ is singular, and hence explain why the matrix has no inverse at that value.

Solution

$\det=\lambda^2-4=0 \Rightarrow \lambda=\pm2$. At either value the inverse formula $A^{-1}=\dfrac{1}{\det A}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}$ would require dividing by $0$, which is undefined — so no inverse exists.
