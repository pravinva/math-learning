0 Composition of Functions

Composing two functions means feeding the output of one straight into the other: $(f\circ g)(x)=f(g(x))$ first applies $g$, then applies $f$ to whatever $g$ produced. Order matters enormously — $f\circ g$ and $g\circ f$ are generally completely different functions.

Key idea

$(f\circ g)(x)=f(g(x))$. The domain of $f\circ g$ is every $x$ in the domain of $g$ for which $g(x)$ also lies in the domain of $f$ — composing can shrink the domain even further than either function alone.

Worked example

Let $f(x)=e^{2x}$ and $g(x)=\ln(2x-4)$. Find $(f\circ g)(x)$ in simplified form, with its domain.

$(f\circ g)(x) = f(g(x)) = e^{2\ln(2x-4)} = e^{\ln(2x-4)^2} = (2x-4)^2$.

Domain: need $g(x)$ defined first, i.e. $2x-4>0 \Rightarrow x>2$. There's no further restriction from $f$ (its domain is all reals), so $\text{dom}(f\circ g) = (2,\infty)$.

Practice 1

Let $f(x)=x^2+1$ and $g(x)=x-3$. Find $(f\circ g)(x)$ and $(g\circ f)(x)$, and confirm they differ.

Solution

$(f\circ g)(x)=f(x-3)=(x-3)^2+1$. $(g\circ f)(x)=g(x^2+1)=x^2+1-3=x^2-2$. Clearly different.

Practice 2

Let $f(x)=\sqrt{x}$ and $g(x)=x^2-9$. Find $(f\circ g)(x)$ and its domain.

Solution

$(f\circ g)(x)=\sqrt{x^2-9}$. Need $x^2-9\ge0 \Rightarrow |x|\ge3$: domain $(-\infty,-3]\cup[3,\infty)$.

Practice 3

Let $f(x)=\dfrac{1}{x-1}$ and $g(x)=2x+3$. Find $(f\circ g)(x)$, and state any excluded value(s) of $x$.

Solution

$(f\circ g)(x)=\dfrac{1}{(2x+3)-1}=\dfrac{1}{2x+2}$. Excluded: $2x+2=0 \Rightarrow x=-1$.

Practice 4

If $h(x)=(3x-1)^4$, express $h$ as a composition $f\circ g$ of two simpler functions.

Solution

Let $g(x)=3x-1$ (inner) and $f(x)=x^4$ (outer). Then $(f\circ g)(x)=(3x-1)^4=h(x)$.

Practice 5

Let $f(x)=\ln x$ and $g(x)=e^{3x}$. Find $(f\circ g)(x)$ and $(g\circ f)(x)$, and simplify both fully.

Solution

$(f\circ g)(x)=\ln(e^{3x})=3x$. $(g\circ f)(x)=e^{3\ln x}=e^{\ln x^3}=x^3$ (for $x>0$).

Practice 6

A manufacturing cost function is $C(u)=50+2u$ (dollars, for $u$ units), and the number of units produced depends on time as $u(t)=10t^2$. Find $(C\circ u)(t)$ and interpret what it represents.

Solution

$(C\circ u)(t)=C(10t^2)=50+2(10t^2)=50+20t^2$. This gives the total cost as a direct function of time $t$.

1 One-to-One Functions

A function is one-to-one (injective) if it never sends two different inputs to the same output. This property is exactly what's needed for an inverse function to exist — if two inputs shared an output, the "inverse" wouldn't know which one to send that output back to.

Key idea — the horizontal line test

Algebraically: $f$ is one-to-one if $f(a)=f(b)$ always forces $a=b$. Graphically: $f$ is one-to-one exactly when no horizontal line crosses its graph more than once.

Worked example

Determine whether $f(x)=x^2-4x+3$ (with domain all reals) is one-to-one.

Try $f(a)=f(b)$: $a^2-4a+3=b^2-4b+3 \Rightarrow a^2-b^2=4a-4b \Rightarrow (a-b)(a+b)=4(a-b)$.

If $a\ne b$, divide both sides by $(a-b)$: $a+b=4$. This has solutions with $a\ne b$ — for example $a=1,b=3$: check $f(1)=1-4+3=0$ and $f(3)=9-12+3=0$. Equal outputs, different inputs.

Not one-to-one (as expected — it's a parabola, which fails the horizontal line test on either side of its vertex).

Practice 7

Determine whether $f(x)=3x-7$ is one-to-one, using the algebraic definition.

Solution

$f(a)=f(b) \Rightarrow 3a-7=3b-7 \Rightarrow a=b$. One-to-one.

Practice 8

Determine whether $f(x)=x^3$ is one-to-one over all reals.

Solution

$f(a)=f(b) \Rightarrow a^3=b^3 \Rightarrow a=b$ (cube root is unique over the reals). One-to-one.

Practice 9

Determine whether $f(x)=|x|$ is one-to-one over all reals, and if not, suggest a restricted domain on which it is.

Solution

Not one-to-one: $f(2)=f(-2)=2$. Restricting to $x\ge0$ (or $x\le0$) makes it one-to-one, since $|x|=x$ (or $-x$) on that restricted domain, which is strictly monotonic.

Practice 10

Determine whether $f(x)=\dfrac{1}{x+2}$ is one-to-one on its natural domain.

Solution

$f(a)=f(b) \Rightarrow \dfrac{1}{a+2}=\dfrac{1}{b+2} \Rightarrow a+2=b+2 \Rightarrow a=b$. One-to-one.

Practice 11

For $f(x)=1-x^2$, find the largest domain of the form $[0,\infty)$ or $(-\infty,0]$ on which $f$ is one-to-one, and justify your choice using the algebraic test.

Solution

On $x\ge0$: if $f(a)=f(b)$ with $a,b\ge0$, then $1-a^2=1-b^2 \Rightarrow a^2=b^2 \Rightarrow a=\pm b$; since both are non-negative, $a=b$. So restricting to $[0,\infty)$ makes $f$ one-to-one (equally, $(-\infty,0]$ would work by the same argument).

2 Inverse Functions

If $f$ is one-to-one, its inverse $f^{-1}$ undoes it exactly: $f^{-1}(f(x))=x$ and $f(f^{-1}(x))=x$. Finding $f^{-1}$ algebraically is a mechanical process, and the domain/range always swap between a function and its inverse.

Key method and facts

To find $f^{-1}$: write $y=f(x)$, swap $x$ and $y$, then solve for $y$. $\text{dom}(f^{-1})=\text{range}(f)$ and $\text{range}(f^{-1})=\text{dom}(f)$. Graphically, $y=f^{-1}(x)$ is the reflection of $y=f(x)$ in the line $y=x$.

Worked example

Let $f(x)=\dfrac{1}{x^2-1}$ (restricted to $x>1$ to ensure one-to-one). Find the domain and range of $f$, and find $f^{-1}(x)$.

Domain (given): $x>1$. As $x\to1^+$, $f(x)\to+\infty$; as $x\to\infty$, $f(x)\to0^+$. Range: $(0,\infty)$... more precisely $f$ is decreasing from $+\infty$ toward (but never reaching) $0$: range $(0,\infty)$.

Swap and solve: $x=\dfrac{1}{y^2-1} \Rightarrow y^2-1=\dfrac{1}{x} \Rightarrow y^2=1+\dfrac{1}{x} \Rightarrow y=\sqrt{1+\dfrac1x}$ (taking the positive root, since $x>1$ was required for $f$).

$$f^{-1}(x) = \sqrt{1+\frac1x}, \qquad \text{domain } x>0.$$

Practice 12

Find the inverse of $f(x)=5x-2$.

Solution

$x=5y-2 \Rightarrow y=\dfrac{x+2}{5}$. $f^{-1}(x)=\dfrac{x+2}{5}$.

Practice 13

Find the inverse of $f(x)=\sqrt{x-3}$ (with domain $x\ge3$), stating the domain of $f^{-1}$.

Solution

$x=\sqrt{y-3} \Rightarrow x^2=y-3 \Rightarrow y=x^2+3$. Domain of $f^{-1}$ is the range of $f$, which is $x\ge0$ (since a square root is never negative). $f^{-1}(x)=x^2+3,\ x\ge0$.

Practice 14

Find the inverse of $f(x)=\dfrac{2x+1}{x-3}$.

Solution

$x=\dfrac{2y+1}{y-3} \Rightarrow x(y-3)=2y+1 \Rightarrow xy-3x=2y+1 \Rightarrow xy-2y=3x+1 \Rightarrow y(x-2)=3x+1 \Rightarrow y=\dfrac{3x+1}{x-2}$.

Practice 15

Find the inverse of $f(x)=e^{2x}-1$.

Solution

$x=e^{2y}-1 \Rightarrow x+1=e^{2y} \Rightarrow 2y=\ln(x+1) \Rightarrow y=\dfrac12\ln(x+1)$. Domain of $f^{-1}$: need $x+1>0 \Rightarrow x>-1$.

Practice 16

For $f(x)=\dfrac{1}{x^2-1}$ (domain), sketch $y=f(x)$, find its domain and range, and (if $f$ is one-to-one, find $f^{-1}$; if not, restrict the domain first).

Solution

Natural domain excludes $x=\pm1$: $x\in(-\infty,-1)\cup(-1,1)\cup(1,\infty)$. Range: since $x^2-1$ ranges over $(-1,\infty)\setminus\{0\}$ ... more directly, $f$ is not one-to-one on its natural domain (e.g. $f(2)=f(-2)=\tfrac13$), so restrict to $x>1$: there $f$ is a strictly decreasing bijection from $(1,\infty)$ onto $(0,\infty)$, matching the worked example, giving $f^{-1}(x)=\sqrt{1+\tfrac1x}$.

Practice 17

If $f(2)=7$ and $f$ is one-to-one, state $f^{-1}(7)$, and explain the reasoning in one sentence.

Solution

$f^{-1}(7)=2$, since the inverse undoes $f$ exactly: whatever input $f$ sent to $7$, $f^{-1}$ must send $7$ back to.
