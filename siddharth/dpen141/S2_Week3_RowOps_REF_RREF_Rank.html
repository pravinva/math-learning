0 Augmented Matrices and Elementary Row Operations

A system of linear equations can be packed into an augmented matrix: the coefficients on the left, a vertical bar, and the constants on the right. Solving the system then becomes a purely mechanical process — applying a small set of legal moves (row operations) until the answer is obvious, with no risk of accidentally changing what the system means.

The three elementary row operations

1) Swap two rows. 2) Multiply a row by a nonzero scalar. 3) Add a multiple of one row to another. Each of these leaves the solution set unchanged — they just rewrite the same system in a more convenient form.

Worked example

Write the augmented matrix for the system $3x+2y+2z=-1,\ 2x+2y+2z=2,\ -4x+4y+3z=0$, and perform $R_1\to R_1-R_2$ to simplify.

$$\left(\begin{array}{ccc|c}3&2&2&-1\\2&2&2&2\\-4&4&3&0\end{array}\right)$$

$R_1\to R_1-R_2$: new row 1 is $(3-2,\ 2-2,\ 2-2,\ -1-2)=(1,0,0,-3)$, immediately revealing $x=-3$.

Practice 1

Write the augmented matrix for the system $2x-y+3z=5,\ x+4y-z=0,\ 3x-2y+2z=7$.

Solution

$$\left(\begin{array}{ccc|c}2&-1&3&5\\1&4&-1&0\\3&-2&2&7\end{array}\right)$$

Practice 2

For the augmented matrix $\left(\begin{array}{cc|c}2&4&6\\1&-1&1\end{array}\right)$, perform $R_1\to\tfrac12R_1$ and state the resulting matrix.

Solution

$$\left(\begin{array}{cc|c}1&2&3\\1&-1&1\end{array}\right)$$

Practice 3

For $\left(\begin{array}{cc|c}1&2&3\\1&-1&1\end{array}\right)$, perform $R_2\to R_2-R_1$ and hence solve the resulting system.

Solution

$R_2\to R_2-R_1$: new row $2$ is $(1-1,-1-2,1-3)=(0,-3,-2)$: $$\left(\begin{array}{cc|c}1&2&3\\0&-3&-2\end{array}\right)$$ This gives $-3y=-2\Rightarrow y=\tfrac23$, then $x+2(\tfrac23)=3\Rightarrow x=3-\tfrac43=\tfrac53$.

Practice 4

Explain why swapping two rows of an augmented matrix never changes the solution of the corresponding system.

Solution

The rows of an augmented matrix are just the equations of the system, listed in some order. Writing the same equations in a different order doesn't change what values of the unknowns satisfy all of them simultaneously — a system is a set of conditions, not an ordered list.

Practice 5

For $\left(\begin{array}{ccc|c}1&2&1&4\\2&5&3&10\\1&0&2&2\end{array}\right)$, perform $R_2\to R_2-2R_1$ and $R_3\to R_3-R_1$ in one step, and state the resulting matrix.

Solution

New $R_2 = (2-2,5-4,3-2,10-8)=(0,1,1,2)$. New $R_3=(1-1,0-2,2-1,2-4)=(0,-2,1,-2)$. $$\left(\begin{array}{ccc|c}1&2&1&4\\0&1&1&2\\0&-2&1&-2\end{array}\right)$$

1 Gaussian Elimination: Row Echelon Form

The systematic strategy: use row operations to create zeros below the diagonal, working column by column, left to right. Once every row starts (from the left) with more leading zeros than the row above it, the matrix is in row echelon form (REF), and the system can be solved directly by back-substitution.

Key idea

A matrix is in row echelon form if: 1) any all-zero rows are at the bottom, 2) each row's first nonzero entry (its pivot) is strictly to the right of the pivot in the row above. REF is not unique — different valid sequences of row operations can produce different, equally correct, echelon forms.

Worked example

Reduce $\begin{pmatrix}2&1&3\\0&-2&7\\3&4&5\end{pmatrix}$ to row echelon form without introducing fractions.

Target: eliminate the $3$ in row 3, column 1, using row 1. To avoid fractions, use $2R_3-3R_1$ instead of $R_3-\tfrac32R_1$: new $R_3 = 2(3,4,5)-3(2,1,3) = (6-6,\,8-3,\,10-9)=(0,5,1)$.

$$\begin{pmatrix}2&1&3\\0&-2&7\\0&5&1\end{pmatrix}$$

Eliminate the $5$ in row 3 using row 2: use $2R_3+5R_2$: new $R_3=2(0,5,1)+5(0,-2,7)=(0,10-10,2+35)=(0,0,37)$.

$$\begin{pmatrix}2&1&3\\0&-2&7\\0&0&37\end{pmatrix}$$ — row echelon form.

Practice 6

Reduce $\begin{pmatrix}1&2&3\\2&5&7\\1&0&5\end{pmatrix}$ to row echelon form.

Solution

$R_2\to R_2-2R_1$: $(0,1,1)$. $R_3\to R_3-R_1$: $(0,-2,2)$. Now $R_3\to R_3+2R_2$: $(0,0,4)$. Result: $\begin{pmatrix}1&2&3\\0&1&1\\0&0&4\end{pmatrix}$.

Practice 7

Solve the system whose augmented matrix, after row reduction, is $\left(\begin{array}{ccc|c}1&-1&2&5\\0&1&-1&-2\\0&0&3&6\end{array}\right)$ using back-substitution.

Solution

Bottom row: $3z=6\Rightarrow z=2$. Middle row: $y-z=-2\Rightarrow y=z-2=0$. Top row: $x-y+2z=5\Rightarrow x=5+y-2z=5+0-4=1$. So $x=1,y=0,z=2$.

Practice 8

Reduce $\begin{pmatrix}1&-1&-1\\-2&1&1\\6&-1&-2\end{pmatrix}$ to row echelon form.

Solution

$R_2\to R_2+2R_1$: $(0,-1,-1)$. $R_3\to R_3-6R_1$: $(0,5,4)$. Now $R_3\to R_3+5R_2$: $(0,0,-1)$. Result: $\begin{pmatrix}1&-1&-1\\0&-1&-1\\0&0&-1\end{pmatrix}$.

Practice 9

Reduce the augmented matrix $\left(\begin{array}{ccc|c}1&-1&-1&-11\\-2&1&1&15\\6&-1&-2&-36\end{array}\right)$ to row echelon form and solve the system by back-substitution.

Solution

$R_2\to R_2+2R_1$: $(0,-1,-1,-7)$. $R_3\to R_3-6R_1$: $(0,5,4,30)$. $R_3\to R_3+5R_2$: $(0,0,-1,-5)$. REF: $\left(\begin{array}{ccc|c}1&-1&-1&-11\\0&-1&-1&-7\\0&0&-1&-5\end{array}\right)$. Back-substitute: $-z=-5\Rightarrow z=5$; $-y-z=-7\Rightarrow y=7-z=2$; $x-y-z=-11\Rightarrow x=-11+y+z=-11+2+5=-4$. So $x=-4,y=2,z=5$.

Practice 10

Reduce $\begin{pmatrix}4&0&-3&4\\-2&0&7&1\\-2&0&7&4\end{pmatrix}$ to row echelon form (as far as possible), and comment on what you notice about column 2.

Solution

Column 2 is entirely zero, so no pivot can ever appear there — it stays zero throughout. $R_2\to 2R_2+R_1$: $(0,0,11,6)$. $R_3\to R_3-R_2^{\text{orig}}$, i.e. using the original row 2: $(0,0,0,3)$. REF: $\begin{pmatrix}4&0&-3&4\\0&0&11&6\\0&0&0&3\end{pmatrix}$. This has a pivot missing in column 2, which will matter for rank (Section 2).

2 Reduced Row Echelon Form and Rank

Pushing row echelon form one step further — making every pivot exactly $1$ and clearing out everything above each pivot too, not just below — gives reduced row echelon form (RREF). Unlike REF, RREF is unique for a given matrix, which makes it the natural place to read off the rank: the number of pivots (nonzero rows) once fully reduced.

Key idea

The rank of a matrix is the number of pivot columns in its (reduced) row echelon form — equivalently, the number of linearly independent rows. For an $n\times n$ system, rank $=n$ means a unique solution exists; rank $

Worked example

Continue reducing $\begin{pmatrix}2&1&3\\0&-2&7\\0&0&37\end{pmatrix}$ (from Section 1's example) to reduced row echelon form.

Scale each row so its pivot is $1$: $R_1\to\tfrac12R_1$: $(1,0.5,1.5)$. $R_2\to-\tfrac12R_2$: $(0,1,-3.5)$. $R_3\to\tfrac{1}{37}R_3$: $(0,0,1)$.

Clear above each pivot, starting from the bottom: $R_2\to R_2+3.5R_3$: $(0,1,0)$. $R_1\to R_1-1.5R_3$: $(1,0.5,0)$. Then $R_1\to R_1-0.5R_2$: $(1,0,0)$.

$$\begin{pmatrix}1&0&0\\0&1&0\\0&0&1\end{pmatrix} = I.$$ Rank $=3$ (full rank), since every column has a pivot.

Practice 11

State the rank of $\begin{pmatrix}1&0&0\\0&1&0\\0&0&0\end{pmatrix}$, already in RREF.

Solution

Two nonzero rows (two pivots): rank $=2$.

Practice 12

Find the reduced row echelon form of $\begin{pmatrix}1&2\\2&4\end{pmatrix}$, and state its rank.

Solution

$R_2\to R_2-2R_1$: $(0,0)$. RREF: $\begin{pmatrix}1&2\\0&0\end{pmatrix}$. Rank $=1$ (only one pivot).

Practice 13

Find the reduced row echelon form for $A=\begin{pmatrix}1&-1&-1&-11\\-2&1&1&15\\6&-1&-2&-36\end{pmatrix}$ (continue from the REF found in Practice 9), and state the rank.

Solution

From Practice 9's REF $\begin{pmatrix}1&-1&-1&-11\\0&-1&-1&-7\\0&0&-1&-5\end{pmatrix}$: scale rows 2,3 by $-1$: $\begin{pmatrix}1&-1&-1&-11\\0&1&1&7\\0&0&1&5\end{pmatrix}$. Clear above pivots: $R_2\to R_2-R_3$: $(0,1,0,2)$; $R_1\to R_1+R_3$: $(1,-1,0,-6)$; then $R_1\to R_1+R_2$: $(1,0,0,-4)$. RREF: $\begin{pmatrix}1&0&0&-4\\0&1&0&2\\0&0&1&5\end{pmatrix}$. Rank of the coefficient part is $3$ (full rank), matching the unique solution found earlier.

Practice 14

A $3\times3$ augmented matrix reduces to $\left(\begin{array}{ccc|c}1&0&2&3\\0&1&-1&2\\0&0&0&5\end{array}\right)$. What does the bottom row tell you about the system, and what is the rank of the coefficient matrix (left of the bar)?

Solution

The bottom row reads $0x+0y+0z=5$, an impossible statement — the system is inconsistent (no solution). The coefficient matrix (ignoring the augmented column) has rank $2$, since only two nonzero rows remain there.

Practice 15

Determine the values of $a$ for which $\begin{pmatrix}a&0&b\\a&a&6\\0&a&3\end{pmatrix}$ has full rank $3$ (you may leave your answer in terms of conditions on $a$, treating $b$ as arbitrary).

Solution

Full rank requires a nonzero determinant. Expanding along column 1: $a\det\begin{pmatrix}a&6\\a&3\end{pmatrix}-a\det\begin{pmatrix}0&b\\a&3\end{pmatrix}+0 = a(3a-6a)-a(0-ab)=a(-3a)-a(-ab)=-3a^2+a^2b=a^2(b-3)$. Full rank needs this nonzero: $a\ne0$ and $b\ne3$.
