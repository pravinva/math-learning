1 Trig functions

$\int\sin x\,dx=-\cos x$ and $\int\cos x\,dx=\sin x$ are immediate from the derivatives you already know. The trouble starts with powers of trig functions, $\sin^2x$, $\sin^3x\cos^2x$, and so on, none of which is the derivative of anything obvious. Two separate tricks handle essentially every case: rewrite an even power using a double-angle identity to remove the power altogether, or peel one factor off an odd power and substitute.

Key idea

Even powers ($\sin^2x$, $\cos^2x$): use $\cos^2x=\dfrac{1+\cos2x}2$ and $\sin^2x=\dfrac{1-\cos2x}2$ to rewrite the power as a sum of easier terms.

Odd powers ($\sin^3x$, $\sin^2x\cos^3x$, etc.): peel off a single copy of whichever function has odd power (using $\sin^2x+\cos^2x=1$ on the rest), then substitute $u=\cos x$ or $u=\sin x$ for what remains.

Why the odd-power trick works

Substitution needs $du$ to appear alongside $u$. If you write $\sin^3x = \sin^2x\cdot\sin x = (1-\cos^2x)\sin x$, the leftover $\sin x\,dx$ is exactly $-d(\cos x)$, so setting $u=\cos x$ turns the whole integral into a polynomial in $u$, something you already know how to integrate. This only works because one factor was left over as a single, unpaired copy, which is precisely what having an odd total power guarantees; an even power leaves nothing spare to convert into $du$.

Worked example

Find $\int\cos^2x\,dx$ and $\int\sin^3x\,dx$.

Even power: $\displaystyle\int\cos^2x\,dx = \int\frac{1+\cos2x}2\,dx = \frac{x}2+\frac{\sin2x}4+C$.

Odd power: $\sin^3x = (1-\cos^2x)\sin x$. Let $u=\cos x$, $du=-\sin x\,dx$:

$$\int(1-\cos^2x)\sin x\,dx = -\int(1-u^2)\,du = -u+\frac{u^3}3+C = -\cos x+\frac{\cos^3x}3+C.$$

Practice 1

Find $\int\sin(3x)\,dx$.

Solution

$\displaystyle\int\sin(3x)\,dx = -\frac{\cos(3x)}3+C$ (chain rule in reverse: the inner derivative $3$ divides out).

Practice 2

Find $\int\cos(5x)\,dx$.

Solution

$\displaystyle\int\cos(5x)\,dx = \frac{\sin(5x)}5+C$.

Practice 3

Find $\int\sin^2x\,dx$.

Solution

$\sin^2x = \dfrac{1-\cos2x}2$, so $\displaystyle\int\sin^2x\,dx = \frac{x}2-\frac{\sin2x}4+C$.

Practice 4

Find $\int\tan^2x\,dx$.

Solution

Use $\tan^2x = \sec^2x-1$: $\displaystyle\int(\sec^2x-1)\,dx = \tan x - x+C$.

Practice 5

Find $\int\sec^2(2x)\,dx$.

Solution

$\displaystyle\int\sec^2(2x)\,dx = \frac{\tan(2x)}2+C$ (same chain-rule-in-reverse idea as Practice 1).

Practice 6

Find $\int\sin x\cos x\,dx$.

Solution

Let $u=\sin x$, $du=\cos x\,dx$: $\displaystyle\int u\,du = \frac{u^2}2+C = \frac{\sin^2x}2+C$.

(Equally valid: use $\sin x\cos x=\dfrac{\sin2x}2$, giving $-\dfrac{\cos2x}4+C$, a different-looking but equivalent answer, since the two forms differ only by the constant $\frac14$, absorbed into $C$.)

Practice 7

Find $\int\sin^2x\cos x\,dx$.

Solution

The $\cos x$ is already the odd, unpaired factor here. Let $u=\sin x$, $du=\cos x\,dx$: $\displaystyle\int u^2\,du = \frac{u^3}3+C = \frac{\sin^3x}3+C$.

Practice 8

Find $\int\sin^2x\cos^3x\,dx$.

Solution

$\cos^3x$ has the odd power, so peel one $\cos x$ off: $\cos^3x = (1-\sin^2x)\cos x$. Let $u=\sin x$, $du=\cos x\,dx$:

$$\int\sin^2x(1-\sin^2x)\cos x\,dx = \int u^2(1-u^2)\,du = \int(u^2-u^4)\,du = \frac{u^3}3-\frac{u^5}5+C = \frac{\sin^3x}3-\frac{\sin^5x}5+C.$$

2 Inverse trig functions

Differentiating $\arcsin x$ and $\arctan x$ produces two of the strangest-looking derivatives in calculus, $\dfrac{d}{dx}\arcsin x = \dfrac1{\sqrt{1-x^2}}$ and $\dfrac{d}{dx}\arctan x = \dfrac1{1+x^2}$, and precisely because they look so distinctive, spotting their pattern inside an integral is a genuinely useful skill, not just a formula to memorise.

Key idea

$$\int\frac{dx}{\sqrt{a^2-x^2}} = \arcsin\left(\frac{x}a\right)+C, \qquad \int\frac{dx}{a^2+x^2} = \frac1a\arctan\left(\frac{x}a\right)+C.$$

A denominator like $x^2+bx+c$ or $c-bx-x^2$ that doesn't obviously match either pattern usually does after completing the square, turning it into $(x-h)^2+k$ or $k-(x-h)^2$, which is exactly the $a^2+x^2$ or $a^2-x^2$ shape shifted sideways.

Worked example

Find $\displaystyle\int\frac{dx}{\sqrt{9-x^2}}$ and $\displaystyle\int\frac{dx}{x^2+2x+5}$.

First one matches the pattern directly with $a=3$: $\displaystyle\int\frac{dx}{\sqrt{9-x^2}} = \arcsin\left(\frac{x}3\right)+C$.

Second one: complete the square, $x^2+2x+5 = (x+1)^2+4$. Now it matches the $a^2+u^2$ pattern with $u=x+1$, $a=2$:

$$\int\frac{dx}{(x+1)^2+4} = \frac12\arctan\left(\frac{x+1}2\right)+C.$$

Integrating the inverse trig functions themselves

$\int\arctan x\,dx$ looks like it needs a new trick, but it doesn't: apply integration by parts (Section 6) with $u=\arctan x$, $dv=dx$, so $du=\dfrac{dx}{1+x^2}$, $v=x$:

$$\int\arctan x\,dx = x\arctan x - \int\frac{x}{1+x^2}\,dx = x\arctan x - \frac12\ln(1+x^2)+C,$$

using the $f'/f$ log pattern from Section 3 on the remaining integral. This is a recurring theme: a function that is hard to integrate directly is often easy to integrate once paired, via parts, with its own well-known derivative.

Practice 9

Find $\displaystyle\int\frac{dx}{4+x^2}$.

Solution

Matches $\dfrac1{a^2+x^2}$ with $a=2$: $\displaystyle\int\frac{dx}{4+x^2} = \frac12\arctan\left(\frac{x}2\right)+C$.

Practice 10

Find $\displaystyle\int\frac{dx}{\sqrt{1-4x^2}}$.

Solution

Write $1-4x^2 = 1-(2x)^2$. Let $u=2x$, $du=2\,dx$: $\displaystyle\int\frac{dx}{\sqrt{1-(2x)^2}} = \frac12\int\frac{du}{\sqrt{1-u^2}} = \frac12\arcsin(2x)+C$.

Practice 11

Find $\displaystyle\int\frac{dx}{\sqrt{16-x^2}}$.

Solution

$\displaystyle\int\frac{dx}{\sqrt{16-x^2}} = \arcsin\left(\frac{x}4\right)+C$.

Practice 12

Find $\displaystyle\int\frac{dx}{9+4x^2}$.

Solution

Factor out the $4$: $9+4x^2 = 4\left(x^2+\dfrac94\right)$, so this is $\dfrac14\displaystyle\int\dfrac{dx}{x^2+(3/2)^2}$.

$$\int\frac{dx}{9+4x^2} = \frac14\cdot\frac1{3/2}\arctan\left(\frac{x}{3/2}\right)+C = \frac16\arctan\left(\frac{2x}3\right)+C.$$

Practice 13

Find $\int\arcsin x\,dx$.

Solution

By parts: $u=\arcsin x$, $dv=dx$, so $du=\dfrac{dx}{\sqrt{1-x^2}}$, $v=x$.

$$\int\arcsin x\,dx = x\arcsin x - \int\frac{x\,dx}{\sqrt{1-x^2}}.$$

For the remaining integral, let $w=1-x^2$, $dw=-2x\,dx$: $\displaystyle\int\frac{x\,dx}{\sqrt{1-x^2}} = -\frac12\int w^{-1/2}\,dw = -\sqrt{w} = -\sqrt{1-x^2}$.

$$\int\arcsin x\,dx = x\arcsin x + \sqrt{1-x^2}+C.$$

Practice 14

Find $\displaystyle\int\frac{dx}{\sqrt{3-2x-x^2}}$.

Solution

Complete the square: $3-2x-x^2 = -(x^2+2x)+3 = -(x^2+2x+1)+1+3 = 4-(x+1)^2$.

This matches $a^2-u^2$ with $u=x+1$, $a=2$: $\displaystyle\int\frac{dx}{\sqrt{4-(x+1)^2}} = \arcsin\left(\frac{x+1}2\right)+C$.

Practice 15

Find $\displaystyle\int\frac{dx}{x^2+2x+5}$ (this is the second worked example, done here as practice, reproduce it yourself before checking).

Solution

$x^2+2x+5 = (x+1)^2+4$, so $\displaystyle\int\frac{dx}{(x+1)^2+4} = \frac12\arctan\left(\frac{x+1}2\right)+C$.

Practice 16

Find $\displaystyle\int_0^{1/2}\frac{dx}{\sqrt{1-x^2}}$.

Solution

Antiderivative is $\arcsin x$. Evaluating: $\arcsin\left(\dfrac12\right)-\arcsin(0) = \dfrac{\pi}6-0 = \dfrac{\pi}6$.

3 Log functions

The single most common source of a natural logarithm appearing in an integral is the pattern $\dfrac{f'(x)}{f(x)}$: whenever the numerator is (a multiple of) the derivative of the denominator, the antiderivative is $\ln|f(x)|$, directly from the chain rule run in reverse.

Key idea

$$\int\frac{f'(x)}{f(x)}\,dx = \ln|f(x)|+C.$$

Trained eyes learn to spot this pattern even when it isn't written in an obviously matching form, for example $\tan x = \dfrac{\sin x}{\cos x}$ has numerator $\sin x = -(-\sin x)$, exactly $-1$ times the derivative of $\cos x$, so $\int\tan x\,dx$ is an $f'/f$ integral in disguise.

Worked example

Find $\int\ln x\,dx$ and $\int\tan x\,dx$.

$\int\ln x\,dx$ needs integration by parts (Section 6): $u=\ln x$, $dv=dx$, so $du=\dfrac{dx}x$, $v=x$.

$$\int\ln x\,dx = x\ln x - \int x\cdot\frac1x\,dx = x\ln x - \int1\,dx = x\ln x - x+C.$$

$\int\tan x\,dx = \displaystyle\int\frac{\sin x}{\cos x}\,dx$. The numerator is $-1$ times the derivative of $\cos x$, so this is an $f'/f$ pattern with $f=\cos x$:

$$\int\tan x\,dx = -\int\frac{-\sin x}{\cos x}\,dx = -\ln|\cos x|+C.$$

Practice 17

Find $\displaystyle\int\frac{2x}{x^2+1}\,dx$.

Solution

The numerator $2x$ is exactly the derivative of the denominator $x^2+1$: $\displaystyle\int\frac{2x}{x^2+1}\,dx = \ln(x^2+1)+C$ (no absolute value bars needed, since $x^2+1>0$ always).

Practice 18

Find $\displaystyle\int\frac{3x^2}{x^3+1}\,dx$.

Solution

$3x^2$ is exactly the derivative of $x^3+1$: $\displaystyle\int\frac{3x^2}{x^3+1}\,dx = \ln|x^3+1|+C$.

Practice 19

Find $\int\cot x\,dx$.

Solution

$\cot x = \dfrac{\cos x}{\sin x}$, and the numerator is exactly the derivative of $\sin x$: $\displaystyle\int\cot x\,dx = \ln|\sin x|+C$.

Practice 20

Find $\int x\ln x\,dx$.

Solution

By parts: $u=\ln x$, $dv=x\,dx$, so $du=\dfrac{dx}x$, $v=\dfrac{x^2}2$.

$$\int x\ln x\,dx = \frac{x^2}2\ln x - \int\frac{x^2}2\cdot\frac1x\,dx = \frac{x^2}2\ln x - \int\frac{x}2\,dx = \frac{x^2}2\ln x - \frac{x^2}4+C.$$

Practice 21

Find $\displaystyle\int\frac{dx}{x\ln x}$.

Solution

Let $w=\ln x$, $dw=\dfrac{dx}x$: $\displaystyle\int\frac{dx}{x\ln x} = \int\frac{dw}w = \ln|w|+C = \ln|\ln x|+C$. A "log of a log," which looks unusual the first time but follows from an entirely ordinary substitution.

Practice 22

Find $\displaystyle\int\frac{\ln x}{x}\,dx$.

Solution

Let $w=\ln x$, $dw=\dfrac{dx}x$: $\displaystyle\int w\,dw = \frac{w^2}2+C = \frac{(\ln x)^2}2+C$.

Practice 23

Find $\displaystyle\int\frac{e^x}{e^x+1}\,dx$.

Solution

The numerator $e^x$ is exactly the derivative of the denominator $e^x+1$: $\displaystyle\int\frac{e^x}{e^x+1}\,dx = \ln(e^x+1)+C$ (again no absolute value needed, $e^x+1>0$ always).

Practice 24

Find $\displaystyle\int_1^e \frac{\ln x}x\,dx$.

Solution

From Practice 22, antiderivative is $\dfrac{(\ln x)^2}2$. Evaluating: $\dfrac{(\ln e)^2}2-\dfrac{(\ln1)^2}2 = \dfrac{1}2-0=\dfrac12$.

4 The t formula

Some integrals involving $\sin x$ and $\cos x$ together, like $\displaystyle\int\dfrac{dx}{3+5\cos x}$, don't match any pattern from Section 1: there is no clean odd power to peel off, and no double-angle identity simplifies things. The Weierstrass substitution (or "$t$-formula") is a single, universal trick that converts any rational combination of $\sin x$ and $\cos x$ into an ordinary rational function of a new variable $t$, which partial fractions (Section 5) can then usually finish off.

Key idea

Let $t=\tan\left(\dfrac{x}2\right)$. Then

$$\sin x = \frac{2t}{1+t^2}, \qquad \cos x = \frac{1-t^2}{1+t^2}, \qquad dx = \frac{2}{1+t^2}\,dt.$$

Where these formulas come from

Since $x=2\arctan t$, differentiating directly gives $dx = \dfrac{2}{1+t^2}\,dt$, confirming the last formula immediately. For $\sin x$ and $\cos x$, use the double-angle formulas with half-angle $x/2$: $\sin x = 2\sin(x/2)\cos(x/2)$ and $\cos x = \cos^2(x/2)-\sin^2(x/2)$. Divide top and bottom of each by $\cos^2(x/2)$ and use $t=\tan(x/2)$: for instance $\sin x = \dfrac{2\sin(x/2)\cos(x/2)}{\cos^2(x/2)+\sin^2(x/2)} = \dfrac{2\tan(x/2)}{1+\tan^2(x/2)} = \dfrac{2t}{1+t^2}$ (dividing numerator and denominator by $\cos^2(x/2)$, and using $\sin^2+\cos^2=1$ in the denominator). The $\cos x$ formula follows the same way.

Worked example

Find $\displaystyle\int\frac{dx}{1+\cos x}$.

Substitute: $1+\cos x = 1+\dfrac{1-t^2}{1+t^2} = \dfrac{(1+t^2)+(1-t^2)}{1+t^2} = \dfrac2{1+t^2}$.

$$\int\frac{dx}{1+\cos x} = \int\frac{2/(1+t^2)}{2/(1+t^2)}\,dt = \int1\,dt = t+C = \tan\left(\frac{x}2\right)+C.$$

The $(1+t^2)$ factors cancelled completely here, an unusually clean case, but it illustrates the general strategy: substitute, simplify the resulting rational function of $t$, integrate that, then substitute $x$ back at the very end.

Practice 25

Find $\displaystyle\int\frac{dx}{\sin x}$ using the $t$ formula.

Solution

$\sin x = \dfrac{2t}{1+t^2}$, $dx=\dfrac2{1+t^2}\,dt$:

$$\int\frac{dx}{\sin x} = \int\frac{2/(1+t^2)}{2t/(1+t^2)}\,dt = \int\frac{dt}{t} = \ln|t|+C = \ln\left|\tan\frac{x}2\right|+C.$$

Practice 26

Find $\displaystyle\int\frac{dx}{1+\sin x+\cos x}$.

Solution

$1+\sin x+\cos x = \dfrac{(1+t^2)+2t+(1-t^2)}{1+t^2} = \dfrac{2+2t}{1+t^2} = \dfrac{2(1+t)}{1+t^2}$.

$$\int\frac{dx}{1+\sin x+\cos x} = \int\frac{2/(1+t^2)}{2(1+t)/(1+t^2)}\,dt = \int\frac{dt}{1+t} = \ln|1+t|+C = \ln\left|1+\tan\frac{x}2\right|+C.$$

Practice 27

Find $\displaystyle\int\frac{dx}{3+5\cos x}$.

Solution

$3+5\cos x = \dfrac{3(1+t^2)+5(1-t^2)}{1+t^2} = \dfrac{8-2t^2}{1+t^2}$.

$$\int\frac{dx}{3+5\cos x} = \int\frac{2/(1+t^2)}{(8-2t^2)/(1+t^2)}\,dt = \int\frac{2\,dt}{8-2t^2} = \int\frac{dt}{4-t^2}.$$

This now matches the inverse-trig-adjacent log pattern $\displaystyle\int\frac{dt}{a^2-t^2} = \frac1{2a}\ln\left|\frac{a+t}{a-t}\right|+C$ with $a=2$ (derivable via partial fractions, Section 5):

$$\int\frac{dx}{3+5\cos x} = \frac14\ln\left|\frac{2+\tan(x/2)}{2-\tan(x/2)}\right|+C.$$

Practice 28

Use the $t$ formula to re-derive $\displaystyle\int\sec x\,dx$, and confirm it matches the standard result $\ln|\sec x+\tan x|+C$ at $x=0$ and $x=\pi/2$ conceptually (i.e.\ explain why both should agree, without necessarily reducing algebraically).

Solution

$\sec x = \dfrac1{\cos x} = \dfrac{1+t^2}{1-t^2}$, and $dx=\dfrac2{1+t^2}dt$, so

$$\int\sec x\,dx = \int\frac{1+t^2}{1-t^2}\cdot\frac2{1+t^2}\,dt = \int\frac{2\,dt}{1-t^2} = \ln\left|\frac{1+t}{1-t}\right|+C$$

(again via the $\int dt/(a^2-t^2)$ pattern, here with $a=1$). Both this result and the standard form $\ln|\sec x+\tan x|$ are antiderivatives of the same function $\sec x$, so they must differ by at most a constant; they are simply two different-looking, algebraically equivalent expressions for the same family of antiderivatives, exactly the kind of thing that happens whenever a function is integrated by two genuinely different routes.

Practice 29

Write down $\sin x$, $\cos x$, and $dx$ in terms of $t=\tan(x/2)$ from memory, without looking back at Section 4's key idea, then check yourself.

Solution

$$\sin x = \frac{2t}{1+t^2}, \qquad \cos x = \frac{1-t^2}{1+t^2}, \qquad dx = \frac{2\,dt}{1+t^2}.$$

A useful way to keep these straight: the denominator is always $1+t^2$; the $\sin x$ numerator is "$2t$" (linear, odd); the $\cos x$ numerator is "$1-t^2$" (starts at $1$ when $t=0$, matching $\cos0=1$).

5 Partial fractions

A rational function like $\dfrac{3x+5}{(x+1)(x+2)}$ is not directly an $f'/f$ pattern (Section 3) and has no obvious antiderivative as it stands. But it can always be rewritten as a sum of simpler fractions, each with just a single linear (or irreducible quadratic) factor in the denominator, and each of those simpler pieces integrates immediately to a logarithm or arctangent.

Key idea: the decomposition rules

For a proper rational function (numerator degree less than denominator degree):

A distinct linear factor $(x-a)$ in the denominator contributes a term $\dfrac{A}{x-a}$.

A repeated linear factor $(x-a)^n$ contributes $\dfrac{A_1}{x-a}+\dfrac{A_2}{(x-a)^2}+\cdots+\dfrac{A_n}{(x-a)^n}$, one term for every power up to $n$.

An irreducible quadratic factor $(x^2+bx+c)$ contributes $\dfrac{Ax+B}{x^2+bx+c}$ (linear numerator, not just a constant).

Find the unknown constants by combining the right-hand side over a common denominator and matching it to the original numerator, either by comparing coefficients or by substituting convenient values of $x$.

Worked example

Find $\displaystyle\int\frac{3x+5}{(x+1)(x+2)}\,dx$.

Write $\dfrac{3x+5}{(x+1)(x+2)} = \dfrac{A}{x+1}+\dfrac{B}{x+2}$. Multiply both sides by $(x+1)(x+2)$: $3x+5 = A(x+2)+B(x+1)$.

Set $x=-1$: $3(-1)+5 = A(1) \Rightarrow A=2$. Set $x=-2$: $3(-2)+5=B(-1) \Rightarrow -1=-B \Rightarrow B=1$.

$$\int\left(\frac2{x+1}+\frac1{x+2}\right)dx = 2\ln|x+1|+\ln|x+2|+C.$$

Practice 30

Find $\displaystyle\int\frac{x}{(x-1)(x+3)}\,dx$.

Solution

$\dfrac{x}{(x-1)(x+3)} = \dfrac{A}{x-1}+\dfrac{B}{x+3}$, so $x=A(x+3)+B(x-1)$.

$x=1$: $1=4A \Rightarrow A=\dfrac14$. $\ x=-3$: $-3=-4B \Rightarrow B=\dfrac34$.

$$\int\left(\frac{1/4}{x-1}+\frac{3/4}{x+3}\right)dx = \frac14\ln|x-1|+\frac34\ln|x+3|+C.$$

Practice 31

Find $\displaystyle\int\frac{2x+1}{x^2-1}\,dx$.

Solution

$x^2-1=(x-1)(x+1)$. $\dfrac{2x+1}{(x-1)(x+1)} = \dfrac{A}{x-1}+\dfrac{B}{x+1}$, so $2x+1=A(x+1)+B(x-1)$.

$x=1$: $3=2A \Rightarrow A=\dfrac32$. $\ x=-1$: $-1=-2B \Rightarrow B=\dfrac12$.

$$\int\left(\frac{3/2}{x-1}+\frac{1/2}{x+1}\right)dx = \frac32\ln|x-1|+\frac12\ln|x+1|+C.$$

Practice 32

Find $\displaystyle\int\frac{dx}{x^2(x+1)}$.

Solution

The repeated factor $x^2$ needs two separate terms: $\dfrac1{x^2(x+1)} = \dfrac{A}x+\dfrac{B}{x^2}+\dfrac{C}{x+1}$.

Multiply through: $1 = Ax(x+1)+B(x+1)+Cx^2$. Set $x=0$: $1=B$. Set $x=-1$: $1=C$. Matching $x^2$ coefficients: $0=A+C \Rightarrow A=-1$.

$$\int\left(\frac{-1}x+\frac1{x^2}+\frac1{x+1}\right)dx = -\ln|x|-\frac1x+\ln|x+1|+C.$$

Practice 33

Find $\displaystyle\int\frac{x+2}{x^2+4}\,dx$.

Solution

$x^2+4$ is irreducible (no real roots), so this is already in the right form to split, but not to integrate directly: split the numerator to separate an $f'/f$ piece from an arctan piece.

$$\int\frac{x+2}{x^2+4}\,dx = \int\frac{x}{x^2+4}\,dx + \int\frac{2}{x^2+4}\,dx = \frac12\ln(x^2+4) + \arctan\left(\frac{x}2\right)+C,$$

using the $f'/f$ pattern on the first piece (numerator $x$ is half the derivative $2x$ of $x^2+4$) and the standard arctan formula on the second.

Practice 34

Find $\displaystyle\int\frac{5x-3}{(x-2)(x+1)}\,dx$.

Solution

$5x-3 = A(x+1)+B(x-2)$. $\ x=2$: $7=3A \Rightarrow A=\dfrac73$. $\ x=-1$: $-8=-3B \Rightarrow B=\dfrac83$.

$$\int\left(\frac{7/3}{x-2}+\frac{8/3}{x+1}\right)dx = \frac73\ln|x-2|+\frac83\ln|x+1|+C.$$

Practice 35

Find $\displaystyle\int\frac{x^2+1}{x(x-1)^2}\,dx$.

Solution

$\dfrac{x^2+1}{x(x-1)^2} = \dfrac{A}x+\dfrac{B}{x-1}+\dfrac{C}{(x-1)^2}$. Multiply through: $x^2+1 = A(x-1)^2+Bx(x-1)+Cx$.

$x=0$: $1=A$. $\ x=1$: $2=C$. Matching $x^2$ coefficients: $1=A+B \Rightarrow B=0$.

Interesting outcome: $B=0$, so the plain $\dfrac1{x-1}$ term vanishes entirely, leaving only the repeated-factor piece:

$$\int\left(\frac1x+\frac2{(x-1)^2}\right)dx = \ln|x| - \frac2{x-1}+C.$$

A coefficient coming out exactly $0$ is not an error, it is a legitimate outcome of the method, and worth double-checking rather than assuming a mistake was made.

Practice 36

Find $\displaystyle\int\frac{x+1}{x^2(x-2)}\,dx$.

Solution

$\dfrac{x+1}{x^2(x-2)} = \dfrac{A}x+\dfrac{B}{x^2}+\dfrac{C}{x-2}$. Multiply through: $x+1 = Ax(x-2)+B(x-2)+Cx^2$.

$x=0$: $1=-2B \Rightarrow B=-\dfrac12$. $\ x=2$: $3=4C \Rightarrow C=\dfrac34$. Matching $x^2$: $0=A+C \Rightarrow A=-\dfrac34$.

$$\int\left(\frac{-3/4}{x}+\frac{-1/2}{x^2}+\frac{3/4}{x-2}\right)dx = -\frac34\ln|x|+\frac1{2x}+\frac34\ln|x-2|+C.$$

6 Integration by parts

The product rule for derivatives, $(uv)'=u'v+uv'$, can be rearranged and integrated to give a technique for integrating products of functions, exactly the situation where substitution alone usually fails.

Key idea

$$\int u\,dv = uv - \int v\,du.$$

The skill is choosing which factor is $u$ (to be differentiated) and which is $dv$ (to be integrated). A common guide is the acronym LIATE, choose $u$ in this priority order: Logarithms, Inverse trig, Algebraic (polynomials), Trig, Exponentials, since these are roughly ordered from "gets simpler when differentiated" to "stays about the same complexity."

Why the choice of $u$ matters

Differentiating $\ln x$ gives the much simpler $1/x$; differentiating $e^x$ just gives $e^x$ back, no progress at all. So if a product involves both $\ln x$ and $e^x$, setting $u=\ln x$ makes real progress (its derivative is genuinely simpler), while setting $u=e^x$ would leave you no better off, and possibly worse, since you would then need to integrate $\ln x$, itself a nontrivial task. LIATE is really just a shortcut for "pick whichever factor gets simpler under differentiation."

Worked example

Find $\int xe^x\,dx$ and the cyclic integral $\int e^x\sin x\,dx$.

First: $u=x$ (algebraic, simplifies to $1$), $dv=e^x\,dx$, so $du=dx$, $v=e^x$: $\displaystyle\int xe^x\,dx = xe^x-\int e^x\,dx = xe^x-e^x+C$.

Second is trickier: neither $e^x$ nor $\sin x$ simplifies under differentiation. Apply parts twice and watch the original integral reappear. Let $I=\int e^x\sin x\,dx$. Take $u=\sin x$, $dv=e^x dx$: $I = e^x\sin x - \int e^x\cos x\,dx$.

Apply parts again to the remaining integral, again taking $u=\cos x$, $dv=e^xdx$: $\displaystyle\int e^x\cos x\,dx = e^x\cos x+\int e^x\sin x\,dx = e^x\cos x + I$.

Substitute back: $I = e^x\sin x - (e^x\cos x+I) = e^x\sin x - e^x\cos x - I$. So $2I = e^x(\sin x-\cos x)$, giving $I = \dfrac{e^x(\sin x-\cos x)}2+C$.

Practice 37

Find $\int x\sin x\,dx$.

Solution

$u=x$, $dv=\sin x\,dx$, so $du=dx$, $v=-\cos x$:

$$\int x\sin x\,dx = -x\cos x + \int\cos x\,dx = -x\cos x+\sin x+C.$$

Practice 38

Find $\int x\cos(2x)\,dx$.

Solution

$u=x$, $dv=\cos(2x)\,dx$, so $du=dx$, $v=\dfrac{\sin(2x)}2$:

$$\int x\cos(2x)\,dx = \frac{x\sin(2x)}2 - \int\frac{\sin(2x)}2\,dx = \frac{x\sin(2x)}2+\frac{\cos(2x)}4+C.$$

Practice 39

Find $\int x^2e^x\,dx$ (this needs integration by parts twice, once to reduce the power of $x$ from $2$ to $1$, and once more from $1$ to $0$).

Solution

First pass: $u=x^2$, $dv=e^x dx$: $\displaystyle\int x^2e^x\,dx = x^2e^x - \int2xe^x\,dx$.

Second pass on the remaining integral (this is exactly Practice 39's sibling, the worked example's $\int xe^x dx=xe^x-e^x$): $\displaystyle\int2xe^x\,dx = 2(xe^x-e^x)$.

$$\int x^2e^x\,dx = x^2e^x - 2xe^x+2e^x+C = (x^2-2x+2)e^x+C.$$

Practice 40

Find $\int x^2\ln x\,dx$.

Solution

$u=\ln x$ (logarithm, highest LIATE priority), $dv=x^2\,dx$, so $du=\dfrac{dx}x$, $v=\dfrac{x^3}3$:

$$\int x^2\ln x\,dx = \frac{x^3}3\ln x - \int\frac{x^3}3\cdot\frac1x\,dx = \frac{x^3}3\ln x - \int\frac{x^2}3\,dx = \frac{x^3}3\ln x - \frac{x^3}9+C.$$

Practice 41

Find $\int\ln(x+1)\,dx$.

Solution

$u=\ln(x+1)$, $dv=dx$, so $du=\dfrac{dx}{x+1}$, $v=x$:

$$\int\ln(x+1)\,dx = x\ln(x+1) - \int\frac{x}{x+1}\,dx.$$

For the remaining integral, write $\dfrac{x}{x+1} = 1-\dfrac1{x+1}$ (polynomial division, since the degree of the numerator matches the denominator):

$$\int\left(1-\frac1{x+1}\right)dx = x - \ln|x+1|.$$

$$\int\ln(x+1)\,dx = x\ln(x+1) - x+\ln|x+1|+C.$$

Practice 42

Find $\int x\arcsin x\,dx$.

Solution

$u=\arcsin x$ (inverse trig, ahead of algebraic in LIATE), $dv=x\,dx$, so $du=\dfrac{dx}{\sqrt{1-x^2}}$, $v=\dfrac{x^2}2$:

$$\int x\arcsin x\,dx = \frac{x^2}2\arcsin x - \int\frac{x^2}{2\sqrt{1-x^2}}\,dx.$$

For the remaining integral, write $x^2 = -(1-x^2)+1$: $\displaystyle\int\frac{x^2}{\sqrt{1-x^2}}\,dx = -\int\sqrt{1-x^2}\,dx + \int\frac{dx}{\sqrt{1-x^2}}$. Using the standard result $\int\sqrt{1-x^2}\,dx = \dfrac12\left(x\sqrt{1-x^2}+\arcsin x\right)$ and $\int\frac{dx}{\sqrt{1-x^2}}=\arcsin x$:

$$\int x\arcsin x\,dx = \frac{x^2}2\arcsin x + \frac{x\sqrt{1-x^2}}4 - \frac{\arcsin x}4+C.$$

7 Recurrence relations

Some families of integrals, $\int\sin^nx\,dx$, $\int x^ne^x\,dx$, for every whole number $n$, are too numerous to solve individually from scratch. A reduction formula (or recurrence relation) expresses the integral for a given $n$ in terms of the same type of integral for a smaller value of $n$, so that repeatedly applying it eventually bottoms out at a case you already know how to do directly.

Key idea

Writing $I_n = \int\sin^nx\,dx$, integration by parts (peeling off $\sin x$ as $dv$ and treating $\sin^{n-1}x$ as $u$) leads to the reduction formula

$$I_n = -\frac{\sin^{n-1}x\cos x}n + \frac{n-1}n I_{n-2}.$$

Similarly, for $I_n = \int x^ne^x\,dx$, parts with $u=x^n$, $dv=e^xdx$ gives $I_n = x^ne^x - nI_{n-1}$.

Deriving the $\sin^n x$ reduction formula

Write $\sin^nx = \sin^{n-1}x\cdot\sin x$, and apply parts with $u=\sin^{n-1}x$, $dv=\sin x\,dx$, so $du=(n-1)\sin^{n-2}x\cos x\,dx$, $v=-\cos x$:

$$I_n = -\sin^{n-1}x\cos x + (n-1)\int\sin^{n-2}x\cos^2x\,dx.$$

Replace $\cos^2x = 1-\sin^2x$: $\displaystyle\int\sin^{n-2}x\cos^2x\,dx = \int\sin^{n-2}x\,dx - \int\sin^nx\,dx = I_{n-2}-I_n$.

$$I_n = -\sin^{n-1}x\cos x+(n-1)(I_{n-2}-I_n) \quad\Longrightarrow\quad I_n+(n-1)I_n = -\sin^{n-1}x\cos x+(n-1)I_{n-2}.$$

$$nI_n = -\sin^{n-1}x\cos x+(n-1)I_{n-2} \quad\Longrightarrow\quad I_n = -\frac{\sin^{n-1}x\cos x}n+\frac{n-1}nI_{n-2}.$$

This is exactly the same self-referencing trick used for the cyclic integral $\int e^x\sin x\,dx$ in Section 6: the original integral reappears partway through, and solving for it algebraically finishes the job, rather than needing an entirely fresh technique.

Worked example

Use the reduction formula to find $\int\sin^4x\,dx$.

$I_4 = -\dfrac{\sin^3x\cos x}4+\dfrac34I_2$.

Need $I_2=\int\sin^2x\,dx$: from Section 1, $I_2 = \dfrac{x}2-\dfrac{\sin2x}4$ (or apply the reduction formula once more: $I_2=-\dfrac{\sin x\cos x}2+\dfrac12I_0$, with $I_0=\int1\,dx=x$).

$$I_4 = -\frac{\sin^3x\cos x}4+\frac34\left(\frac{x}2-\frac{\sin2x}4\right) = -\frac{\sin^3x\cos x}4+\frac{3x}8-\frac{3\sin2x}{16}+C.$$

Practice 43

Using $I_n = x^ne^x-nI_{n-1}$ (and $I_0=e^x$), find $I_1=\int xe^x\,dx$, $I_2=\int x^2e^x\,dx$, and check $I_2$ against your Practice 39 answer.

Solution

$I_1 = xe^x - 1\cdot I_0 = xe^x-e^x$.

$I_2 = x^2e^x - 2I_1 = x^2e^x-2(xe^x-e^x) = x^2e^x-2xe^x+2e^x = (x^2-2x+2)e^x$, matching Practice 39 exactly.

Practice 44

Continue the recurrence from Practice 43 to find $I_3=\int x^3e^x\,dx$.

Solution

$I_3 = x^3e^x - 3I_2 = x^3e^x - 3(x^2-2x+2)e^x = (x^3-3x^2+6x-6)e^x+C$.

Practice 45

Derive the analogous reduction formula for $J_n = \int\cos^nx\,dx$, by mirroring the Section 7 "why" derivation for $\sin^nx$ (peel off $\cos x$ as $dv$, use $\sin^2x=1-\cos^2x$).

Solution

By an identical argument with the roles of sine and cosine swapped:

$$J_n = \frac{\cos^{n-1}x\sin x}n + \frac{n-1}nJ_{n-2}.$$

The only sign difference from the $\sin^nx$ formula is that the leading term is $+$ rather than $-$, a consequence of $\dfrac{d}{dx}\sin x=\cos x$ having no minus sign, unlike $\dfrac{d}{dx}\cos x=-\sin x$.

Practice 46

Use the reduction formula from Practice 45 to find $\int\cos^3x\,dx$, and check it against direct u-substitution ($\cos^3x = (1-\sin^2x)\cos x$, substitute $u=\sin x$).

Solution

Via the reduction formula: $J_3 = \dfrac{\cos^2x\sin x}3+\dfrac23J_1$, and $J_1=\int\cos x\,dx=\sin x$.

$$J_3 = \frac{\cos^2x\sin x}3+\frac{2\sin x}3.$$

Via direct substitution: $u=\sin x$, $du=\cos x\,dx$: $\displaystyle\int(1-u^2)\,du = u-\frac{u^3}3 = \sin x - \frac{\sin^3x}3$.

These look different, but expand the reduction-formula answer using $\cos^2x=1-\sin^2x$: $\dfrac{(1-\sin^2x)\sin x}3+\dfrac{2\sin x}3 = \dfrac{\sin x-\sin^3x}3+\dfrac{2\sin x}3 = \sin x - \dfrac{\sin^3x}3$, exactly matching.

Practice 47

Without fully expanding, explain why the reduction formula $I_n=-\dfrac{\sin^{n-1}x\cos x}n+\dfrac{n-1}nI_{n-2}$ reduces $n$ by $2$ each time rather than by $1$, and what this means for how you would compute $I_5$ versus $I_4$.

Solution

The derivation used $\cos^2x=1-\sin^2x$ to convert $\int\sin^{n-2}x\cos^2x\,dx$ into $I_{n-2}-I_n$, a step that intrinsically drops the power by $2$, not $1$, since it replaces $\cos^2x$ (an even power) rather than removing a single $\cos x$. Consequently, computing $I_5$ requires $I_3$, which requires $I_1=\int\sin x\,dx=-\cos x$ (an odd chain, bottoming out at $n=1$), while computing $I_4$ requires $I_2$, which requires $I_0=\int1\,dx=x$ (an even chain, bottoming out at $n=0$). Odd and even starting values never mix, you always land on the "same parity" base case.

8 Applications

Every technique so far has been about finding an antiderivative. The reason any of this matters is that a definite integral, $\int_a^bf(x)\,dx$, computes real geometric and physical quantities: areas, volumes, averages, directly from the Fundamental Theorem of Calculus, $\int_a^bf(x)\,dx = F(b)-F(a)$, where $F$ is any antiderivative of $f$.

Key idea: common applications

Area under a curve: $\displaystyle\int_a^bf(x)\,dx$ (assuming $f\ge0$ on $[a,b]$).

Area between two curves $y=f(x)$ (on top) and $y=g(x)$ (below), from $x=a$ to $x=b$: $\displaystyle\int_a^b\big(f(x)-g(x)\big)\,dx$.

Volume of revolution about the $x$-axis (disk method), rotating $y=f(x)$ from $x=a$ to $x=b$: $\displaystyle V=\pi\int_a^b f(x)^2\,dx$.

Average value of $f$ on $[a,b]$: $\displaystyle\bar f = \frac1{b-a}\int_a^bf(x)\,dx$.

Why the disk method squares the function

Slice the solid of revolution into thin disks perpendicular to the $x$-axis, each of thickness $dx$. Rotating the point $(x,f(x))$ around the $x$-axis traces out a circle of radius $f(x)$, so each disk has cross-sectional area $\pi\big(f(x)\big)^2$ (the ordinary area-of-a-circle formula), not $\pi f(x)$. Adding up (integrating) the volumes $\pi f(x)^2\,dx$ of all these infinitesimally thin disks from $a$ to $b$ gives the total volume. The squaring is not an extra rule to memorise, it falls straight out of the fact that a circle's area involves the radius squared.

Worked example

Find the area under $y=x^2$ from $x=0$ to $x=2$, and the volume when that same region is rotated about the $x$-axis.

Area: $\displaystyle\int_0^2x^2\,dx = \left[\frac{x^3}3\right]_0^2 = \frac83-0 = \frac83$.

Volume: $\displaystyle\pi\int_0^2(x^2)^2\,dx = \pi\int_0^2x^4\,dx = \pi\left[\frac{x^5}5\right]_0^2 = \pi\cdot\frac{32}5 = \frac{32\pi}5$.

Practice 48

Find the area between $y=x$ and $y=x^2$ for $0\le x\le1$.

Solution

On $[0,1]$, $x\ge x^2$ (check at $x=0.5$: $0.5>0.25$), so $y=x$ is on top.

$$\int_0^1(x-x^2)\,dx = \left[\frac{x^2}2-\frac{x^3}3\right]_0^1 = \frac12-\frac13 = \frac16.$$

Practice 49

Find the area enclosed between $y=4-x^2$ and the $x$-axis.

Solution

First find where the curve meets the axis: $4-x^2=0 \Rightarrow x=\pm2$. The region runs from $x=-2$ to $x=2$.

$$\int_{-2}^2(4-x^2)\,dx = \left[4x-\frac{x^3}3\right]_{-2}^2 = \left(8-\frac83\right)-\left(-8+\frac83\right) = 16-\frac{16}3 = \frac{32}3.$$

Practice 50

Find the volume when $y=\sqrt x$, for $0\le x\le4$, is rotated about the $x$-axis.

Solution

$$V = \pi\int_0^4(\sqrt x)^2\,dx = \pi\int_0^4x\,dx = \pi\left[\frac{x^2}2\right]_0^4 = \pi\cdot8 = 8\pi.$$

Practice 51

Find the average value of $f(x)=\sin x$ on $[0,\pi]$.

Solution

$$\bar f = \frac1{\pi-0}\int_0^\pi\sin x\,dx = \frac1\pi\Big[-\cos x\Big]_0^\pi = \frac1\pi\big((-\cos\pi)-(-\cos0)\big) = \frac1\pi(1+1) = \frac2\pi.$$

Notice this is less than the function's peak value of $1$, which makes sense, $\sin x$ spends much of $[0,\pi]$ well below its maximum.

Practice 52

Find the area under $y=\dfrac1{1+x^2}$ from $x=0$ to $x=1$ (this uses Section 2's arctan formula).

Solution

$$\int_0^1\frac{dx}{1+x^2} = \Big[\arctan x\Big]_0^1 = \arctan1-\arctan0 = \frac{\pi}4-0 = \frac{\pi}4.$$

Practice 53

Find the volume when the region under $y=e^x$ from $x=0$ to $x=1$ is rotated about the $x$-axis.

Solution

$$V = \pi\int_0^1(e^x)^2\,dx = \pi\int_0^1e^{2x}\,dx = \pi\left[\frac{e^{2x}}2\right]_0^1 = \frac{\pi}2(e^2-1).$$

Practice 54

Find the area between the curves $y=3-x^2$ and $y=x^2-3$, between their two intersection points.

Solution

Intersections: $3-x^2=x^2-3 \Rightarrow 6=2x^2 \Rightarrow x=\pm\sqrt3$. On this interval $3-x^2\ge x^2-3$ (check $x=0$: $3\ge-3$), so $y=3-x^2$ is on top.

$$\int_{-\sqrt3}^{\sqrt3}\big((3-x^2)-(x^2-3)\big)\,dx = \int_{-\sqrt3}^{\sqrt3}(6-2x^2)\,dx = \left[6x-\frac{2x^3}3\right]_{-\sqrt3}^{\sqrt3}.$$

At $x=\sqrt3$: $6\sqrt3-\dfrac{2\cdot3\sqrt3}3 = 6\sqrt3-2\sqrt3=4\sqrt3$. At $x=-\sqrt3$: by symmetry (the integrand $6-2x^2$ is even, so this is an odd antiderivative), the value is $-4\sqrt3$. Total: $4\sqrt3-(-4\sqrt3)=8\sqrt3$.
