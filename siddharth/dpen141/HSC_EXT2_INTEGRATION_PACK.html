HSC Ext 2 Integration Pack

Past-paper style integration problem types only. 10 questions per subtopic with hidden answers.
Designed for MATH141 Calculus extension practice under `dpen141`.

How To Use

Method: Attempt each question first on paper, then open the hidden answer. Mark every error by category (algebra, method choice, sign, limits, constants).
\[ \text{Goal per session: }10\text{ questions} \rightarrow \text{review} \rightarrow \text{redo incorrect ones cold} \]

Subtopic 1: Core + Substitution (10)

  1. \(\int (4x^3-5x+2)\,dx\)
    Show answer
    \(x^4-\frac{5x^2}{2}+2x+C\).
  2. \(\int \frac{3x}{x^2+7}\,dx\)
    Show answer
    Let \(u=x^2+7\): \(\frac{3}{2}\ln(x^2+7)+C\).
  3. \(\int e^{5x}\,dx\)
    Show answer
    \(\frac{1}{5}e^{5x}+C\).
  4. \(\int \sin(3x)\,dx\)
    Show answer
    \(-\frac{1}{3}\cos(3x)+C\).
  5. \(\int \frac{2x+1}{x^2+x+4}\,dx\)
    Show answer
    \(\ln(x^2+x+4)+C\).
  6. \(\int x\sqrt{x^2+9}\,dx\)
    Show answer
    \(\frac{1}{3}(x^2+9)^{3/2}+C\).
  7. \(\int \frac{1}{x}\,dx\)
    Show answer
    \(\ln|x|+C\).
  8. \(\int \cos(2x-1)\,dx\)
    Show answer
    \(\frac{1}{2}\sin(2x-1)+C\).
  9. \(\int \frac{x^2}{x^3+1}\,dx\)
    Show answer
    \(\frac{1}{3}\ln(x^3+1)+C\).
  10. \(\int \frac{1}{\sqrt{1-4x^2}}\,dx\)
    Show answer
    \(\frac{1}{2}\arcsin(2x)+C\).

Subtopic 2: Integration by Parts (10)

  1. \(\int x e^x\,dx\)
    Show answer
    \(e^x(x-1)+C\).
  2. \(\int x\cos x\,dx\)
    Show answer
    \(x\sin x+\cos x+C\).
  3. \(\int \ln x\,dx\)
    Show answer
    \(x\ln x-x+C\).
  4. \(\int x^2 e^{2x}\,dx\)
    Show answer
    \(\frac{e^{2x}}{4}(2x^2-2x+1)+C\).
  5. \(\int x\ln x\,dx\)
    Show answer
    \(\frac{x^2}{2}\ln x-\frac{x^2}{4}+C\).
  6. \(\int e^x\sin x\,dx\)
    Show answer
    \(\frac{e^x}{2}(\sin x-\cos x)+C\).
  7. \(\int x^2\cos x\,dx\)
    Show answer
    \(x^2\sin x+2x\cos x-2\sin x+C\).
  8. \(\int x\arctan x\,dx\)
    Show answer
    \(\frac{x^2+1}{2}\arctan x-\frac{x}{2}+C\).
  9. \(\int \ln(1+x)\,dx\)
    Show answer
    \((x+1)\ln(1+x)-x+C\).
  10. \(\int x^3 e^x\,dx\)
    Show answer
    \(e^x(x^3-3x^2+6x-6)+C\).

Subtopic 3: Partial Fractions (10)

  1. \(\int \frac{1}{x^2-1}\,dx\)
    Show answer
    \(\frac12\ln\left|\frac{x-1}{x+1}\right|+C\).
  2. \(\int \frac{3x+5}{x^2+x-2}\,dx\)
    Show answer
    \(-\frac13\ln|x+2|+\frac{10}{3}\ln|x-1|+C\).
  3. \(\int \frac{x+1}{x(x+4)}\,dx\)
    Show answer
    \(\frac14\ln|x|+\frac34\ln|x+4|+C\).
  4. \(\int \frac{2x+1}{x^2+3x+2}\,dx\)
    Show answer
    \(-\ln|x+1|+3\ln|x+2|+C\).
  5. \(\int \frac{5x+7}{(x+1)(x+3)}\,dx\)
    Show answer
    \(\ln|x+1|+4\ln|x+3|+C\).
  6. \(\int \frac{1}{x(x-2)}\,dx\)
    Show answer
    \(-\frac12\ln|x|+\frac12\ln|x-2|+C\).
  7. \(\int \frac{x^2+1}{x(x^2+1)}\,dx\)
    Show answer
    \(\ln|x|+C\).
  8. \(\int \frac{1}{(x+1)^2}\,dx\)
    Show answer
    \(-\frac{1}{x+1}+C\).
  9. \(\int \frac{4x+1}{(2x-1)(x+2)}\,dx\)
    Show answer
    \(\frac{9}{5}\ln|2x-1|+\frac{1}{5}\ln|x+2|+C\).
  10. \(\int \frac{3x^2+2x+4}{x(x^2+1)}\,dx\)
    Show answer
    \(4\ln|x|+\frac{3}{2}\ln(x^2+1)-2\arctan x+C\).

Subtopic 4: Definite Integrals + Area (10)

  1. \(\int_0^2 (3x^2-1)\,dx\)
    Show answer
    \(6\).
  2. \(\int_0^\pi \sin x\,dx\)
    Show answer
    \(2\).
  3. Area between \(y=x\) and \(y=x^2\) on \([0,1]\)
    Show answer
    \(\frac16\).
  4. \(\int_1^e \frac{1}{x}\,dx\)
    Show answer
    \(1\).
  5. \(\int_0^{\pi/2}\cos(2x)\,dx\)
    Show answer
    \(0\).
  6. Area between \(y=2x\) and \(y=x^2\)
    Show answer
    Intersections \(0,2\), area \(\frac{4}{3}\).
  7. \(\int_{-1}^1 x^3\,dx\)
    Show answer
    \(0\).
  8. \(\int_0^1 \sqrt{x}\,dx\)
    Show answer
    \(\frac{2}{3}\).
  9. Area bounded by \(y=4-x^2\) and x-axis
    Show answer
    Bounds \([-2,2]\), area \(\frac{32}{3}\).
  10. \(\int_0^1 xe^{x^2}\,dx\)
    Show answer
    \(\frac{e-1}{2}\).

Subtopic 5: Improper + Mixed Extension (10)

  1. \(\int_1^\infty \frac{1}{x^2}\,dx\)
    Show answer
    Convergent, value \(1\).
  2. \(\int_1^\infty \frac{1}{x}\,dx\)
    Show answer
    Divergent.
  3. \(\int_0^1 \frac{1}{\sqrt{x}}\,dx\)
    Show answer
    Convergent, value \(2\).
  4. \(\int_0^\infty e^{-x}\,dx\)
    Show answer
    Convergent, value \(1\).
  5. \(\int_0^\infty xe^{-x}\,dx\)
    Show answer
    Convergent, value \(1\).
  6. \(\int_1^\infty \frac{\ln x}{x^2}\,dx\)
    Show answer
    Convergent, value \(1\).
  7. \(\int_0^\infty \frac{1}{1+x^2}\,dx\)
    Show answer
    \(\frac{\pi}{2}\).
  8. \(\int_0^1 \ln x\,dx\)
    Show answer
    \(-1\).
  9. \(\int_0^\infty \frac{1}{\sqrt{x}(1+x)}\,dx\)
    Show answer
    \(\pi\) via \(x=t^2\).
  10. \(\int_0^\infty \frac{x}{(1+x^2)^2}\,dx\)
    Show answer
    \(\frac12\).
