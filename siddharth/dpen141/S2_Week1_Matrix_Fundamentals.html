0 Sigma (Summation) Notation

$\sum$ is just shorthand for "add up a list of similar terms." Everything about it — the letter underneath, the number on top, the expression to the right — is telling you exactly which terms to add and in what order.

Key idea

$$\sum_{k=1}^{n} a_k = a_1+a_2+\cdots+a_n.$$

The index ($k$ here) is a dummy variable — it disappears once you evaluate the sum, so it doesn't matter whether you call it $k$, $i$, or $\alpha$. Sums are linear: $\sum(a_k+b_k)=\sum a_k+\sum b_k$ and $\sum c\,a_k = c\sum a_k$ for a constant $c$.

Worked example

Evaluate $\displaystyle\sum_{k=1}^{4}k\sin\frac{k\pi}{2}$.

List the terms: $k=1: 1\sin(\pi/2)=1(1)=1$. $\ k=2: 2\sin(\pi)=2(0)=0$. $\ k=3: 3\sin(3\pi/2)=3(-1)=-3$. $\ k=4: 4\sin(2\pi)=4(0)=0$.

Sum: $1+0-3+0=-2$.

Practice 1

Evaluate $\displaystyle\sum_{\alpha=1}^{2}(2\alpha+1)$, and separately $\displaystyle\sum_{\alpha=1}^{2}2\alpha+1$. State which one equals $7$.

Solution

First: terms are $(2(1)+1)+(2(2)+1)=3+5=8$. Second (the $+1$ is outside the sum): $\left(\sum_{\alpha=1}^2 2\alpha\right)+1 = (2+4)+1=7$. So the second expression equals $7$ — bracket placement changes the answer.

Practice 2

Evaluate $\displaystyle\sum_{k=0}^{4}1$.

Solution

This adds the constant $1$ once for each value of $k$ from $0$ to $4$, i.e. $5$ terms: sum $=5$.

Practice 3

Evaluate $\displaystyle\sum_{k=1}^{20}\left(\frac{1}{k}-\frac{1}{k+1}\right)$.

Solution

This telescopes: every interior term cancels, leaving only the first and last: $\dfrac{1}{1}-\dfrac{1}{21}=\dfrac{20}{21}$.

Practice 4

Evaluate $\displaystyle\sum_{n=0}^{n}\alpha r^{k} - r\sum_{k=0}^{n}\alpha r^{k}$ by factoring, then use the result to explain why $\displaystyle\sum_{k=0}^n \alpha r^k = \dfrac{\alpha - \alpha r^{n+1}}{1-r}$ for $r\neq1$.

Solution

Let $S=\sum_{k=0}^n\alpha r^k$. Then $S - rS = \alpha + \alpha r+\cdots+\alpha r^n - (\alpha r+\cdots+\alpha r^{n+1})$, and every middle term cancels, leaving $S(1-r)=\alpha-\alpha r^{n+1}$. Dividing by $(1-r)$ (valid since $r\ne1$) gives $S=\dfrac{\alpha-\alpha r^{n+1}}{1-r}$, the geometric series formula.

Practice 5

Write $\displaystyle\sum_{i=1}^{5}(3i-2)$ out as an explicit sum of terms, then evaluate it.

Solution

Terms ($i=1,\dots,5$): $1,4,7,10,13$. Sum $=1+4+7+10+13=35$.

Practice 6

A discrete signal has samples $x_k = (-1)^k k$ for $k=1,\dots,6$. Evaluate $\displaystyle\sum_{k=1}^{6}x_k$.

Solution

Terms: $-1,2,-3,4,-5,6$. Sum $=(-1+2)+(-3+4)+(-5+6)=1+1+1=3$.

1 Index Notation and the Kronecker Delta

Once matrices get large, writing out every entry by hand becomes impractical. Index notation refers to a matrix entry by its row and column number, $a_{ij}$ meaning "the entry in row $i$, column $j$." The Kronecker delta $\delta_{ij}$ is a compact way to write "$1$ if the indices match, $0$ otherwise" — exactly the pattern that defines the identity matrix.

Key idea

$$\delta_{ij} = \begin{cases}1 & i=j \\ 0 & i\ne j\end{cases}$$

For a matrix $A=(a_{ij})$, the entry $a_{ij}\delta_{ij}$ (no sum) is just $a_{ii}$ when $i=j$ and $0$ otherwise — the Kronecker delta acts as a selector, picking out diagonal entries and zeroing everything off-diagonal.

Worked example

Let $A = \begin{pmatrix}2&8&-1\\0&3&17\end{pmatrix}$ (so $a_{ij}$ is the entry in row $i$, column $j$, with $i=1,2$ and $j=1,2,3$). Evaluate $a_{13}\delta_{j1}$ when $j=1$, and explain why the same expression is $0$ when $j=2$.

$a_{13}=-1$ (row 1, column 3). When $j=1$: $\delta_{11}=1$, so $a_{13}\delta_{j1}=(-1)(1)=-1$.

When $j=2$: $\delta_{21}=0$ (indices don't match), so the whole product is $0$, regardless of what $a_{13}$ is.

Practice 7

For $A=\begin{pmatrix}2&8&-1\\0&3&17\end{pmatrix}$, state $a_{22}$ and $a_{11}$.

Solution

$a_{22}=3$ (row 2, column 2). $a_{11}=2$ (row 1, column 1).

Practice 8

Evaluate $\delta_{33}+\delta_{12}+\delta_{21}+\delta_{44}$.

Solution

$\delta_{33}=1$, $\delta_{12}=0$, $\delta_{21}=0$, $\delta_{44}=1$. Sum $=1+0+0+1=2$.

Practice 9

For $A=\begin{pmatrix}2&8&-1\\0&3&17\end{pmatrix}$ and the Kronecker delta $\delta_{ij}$, evaluate $\displaystyle\sum_{i=1}^{2}a_{i3}\delta_{i1}+1$.

Solution

Only the $i=1$ term survives (since $\delta_{i1}=0$ unless $i=1$): $a_{13}\delta_{11}=(-1)(1)=-1$. Adding the $+1$ outside the sum: $-1+1=0$.

Practice 10

Explain in words why $\displaystyle\sum_{j=1}^{n}\delta_{ij}\,x_j = x_i$ for any fixed $i$ and any list of numbers $x_1,\dots,x_n$.

Solution

The Kronecker delta $\delta_{ij}$ is zero for every $j$ except $j=i$, where it equals $1$. So every term in the sum vanishes except the one where $j=i$, which contributes exactly $x_i$. This identity is the index-notation way of saying "multiplying by the identity matrix picks out the original vector unchanged."

Practice 11

For a $3\times3$ matrix $B$ with entries $b_{ij}=i+j$, write out $B$ explicitly as a grid of numbers.

Solution

$b_{11}=2,\ b_{12}=3,\ b_{13}=4,\ b_{21}=3,\ b_{22}=4,\ b_{23}=5,\ b_{31}=4,\ b_{32}=5,\ b_{33}=6$. So $B=\begin{pmatrix}2&3&4\\3&4&5\\4&5&6\end{pmatrix}$.

Practice 12

For a $3\times3$ matrix $C$ with entries $c_{ij}=(-1)^{i+j}\delta_{ij}$, write $C$ explicitly. What well-known matrix is this?

Solution

$\delta_{ij}$ forces $i=j$ for any nonzero entry, and then $(-1)^{2i}=1$ always, so $c_{ii}=1$ for every $i$ and everything off-diagonal is $0$: $C=\begin{pmatrix}1&0&0\\0&1&0\\0&0&1\end{pmatrix}$, the $3\times3$ identity matrix.

2 Matrices: Definitions and Types

A matrix is simply a rectangular array of numbers, described by its order (rows × columns). Certain shapes and patterns come up so often they get their own names, and recognising them on sight saves time later.

Key definitions

An $m\times n$ matrix has $m$ rows and $n$ columns. A square matrix has $m=n$. A row vector is $1\times n$; a column vector is $m\times 1$. The zero matrix $O$ has every entry $0$. The identity matrix $I$ is square with $1$s on the diagonal and $0$s elsewhere ($I_{ij}=\delta_{ij}$). A diagonal matrix has nonzero entries only on the main diagonal. Two matrices are equal only if they have the same order and every corresponding entry matches.

Worked example

Solve for $a,b,c,d$ given $\begin{pmatrix}a-b & b+c\\3d+c & 2a-4d\end{pmatrix}=\begin{pmatrix}8&1\\7&6\end{pmatrix}$.

Matrix equality gives four simultaneous equations: $a-b=8$, $\ b+c=1$, $\ 3d+c=7$, $\ 2a-4d=6$.

From the first, $a=8+b$. Substituting into the fourth: $2(8+b)-4d=6 \Rightarrow 16+2b-4d=6 \Rightarrow 2b-4d=-10 \Rightarrow b-2d=-5$.

From the second, $c=1-b$. Substituting into the third: $3d+1-b=7 \Rightarrow 3d-b=6 \Rightarrow b=3d-6$.

Substitute into $b-2d=-5$: $(3d-6)-2d=-5 \Rightarrow d-6=-5 \Rightarrow d=1$. Then $b=3(1)-6=-3$, $\ a=8+(-3)=5$, $\ c=1-(-3)=4$.

So $a=5,\ b=-3,\ c=4,\ d=1$.

Practice 13

State the order of the matrix $\begin{pmatrix}1&-1&-1&-11\\-2&1&1&15\\6&-1&-2&-36\end{pmatrix}$.

Solution

3 rows, 4 columns: order $3\times4$.

Practice 14

Write down the $2\times2$ zero matrix and the $2\times2$ identity matrix.

Solution

$O=\begin{pmatrix}0&0\\0&0\end{pmatrix}$, $\quad I=\begin{pmatrix}1&0\\0&1\end{pmatrix}$.

Practice 15

For a general $4\times4$ matrix, how many entries lie strictly above the main diagonal?

Solution

A $4\times4$ matrix has $16$ entries total, $4$ on the diagonal, leaving $12$ off-diagonal, split evenly above and below: $6$ strictly above.

Practice 16

Solve for $x,y,z,w$ given $\begin{pmatrix}x+y&2w\\3z&x-y\end{pmatrix}=\begin{pmatrix}6&8\\9&2\end{pmatrix}$.

Solution

$x+y=6$, $\ 2w=8\Rightarrow w=4$, $\ 3z=9\Rightarrow z=3$, $\ x-y=2$. Adding the first and last: $2x=8\Rightarrow x=4$, so $y=2$.

Practice 17

Solve for $a,b,c,d$ given $\begin{pmatrix}2a+b&c-d\\a-3b&2c+d\end{pmatrix}=\begin{pmatrix}5&1\\-1&11\end{pmatrix}$.

Solution

From $2a+b=5$ and $a-3b=-1$: multiply the first by $3$ to get $6a+3b=15$, then add to the second to eliminate $b$: $7a=14 \Rightarrow a=2$, so $b=5-2a=1$. From $c-d=1$ and $2c+d=11$: adding eliminates $d$: $3c=12 \Rightarrow c=4$, then $d=c-1=3$. So $a=2,\ b=1,\ c=4,\ d=3$.

Practice 18

Two matrices $A=\begin{pmatrix}x&2\\3&y\end{pmatrix}$ and $B=\begin{pmatrix}5&2\\3&-1\end{pmatrix}$ are equal. Find $x$ and $y$, then state whether $A$ is diagonal, and why or why not.

Solution

Equality forces $x=5,\ y=-1$. $A=\begin{pmatrix}5&2\\3&-1\end{pmatrix}$ is not diagonal, since it has nonzero off-diagonal entries ($2$ and $3$).

3 Matrix Addition, Subtraction, and Scalar Multiplication

Adding matrices and multiplying by a scalar both work entrywise — there's no new machinery here beyond "do the same arithmetic operation to every matching pair (or every single entry)." The one rule that matters: you can only add or subtract matrices of the same order.

Key idea

$(A+B)_{ij}=A_{ij}+B_{ij}$ and $(cA)_{ij}=c\,A_{ij}$. These operations are commutative and associative, exactly like ordinary number arithmetic, entry by entry.

Worked example

For $A=\begin{pmatrix}2&-1\\3&0\end{pmatrix}$ and $B=\begin{pmatrix}4&5\\-2&1\end{pmatrix}$, find $3A-2B$.

$3A=\begin{pmatrix}6&-3\\9&0\end{pmatrix}$, $\quad 2B=\begin{pmatrix}8&10\\-4&2\end{pmatrix}$.

$3A-2B=\begin{pmatrix}6-8&-3-10\\9-(-4)&0-2\end{pmatrix}=\begin{pmatrix}-2&-13\\13&-2\end{pmatrix}$.

Practice 19

For $A=\begin{pmatrix}1&2\\3&4\end{pmatrix}$ and $B=\begin{pmatrix}5&-1\\0&2\end{pmatrix}$, find $A+B$ and $A-B$.

Solution

$A+B=\begin{pmatrix}6&1\\3&6\end{pmatrix}$, $\quad A-B=\begin{pmatrix}-4&3\\3&2\end{pmatrix}$.

Practice 20

For $A=\begin{pmatrix}2&0&-1\\1&3&4\end{pmatrix}$, find $-2A$.

Solution

$-2A=\begin{pmatrix}-4&0&2\\-2&-6&-8\end{pmatrix}$.

Practice 21

Solve for the matrix $X$ given $2X+\begin{pmatrix}1&2\\3&4\end{pmatrix}=\begin{pmatrix}7&8\\1&0\end{pmatrix}$.

Solution

$2X=\begin{pmatrix}7-1&8-2\\1-3&0-4\end{pmatrix}=\begin{pmatrix}6&6\\-2&-4\end{pmatrix}$, so $X=\begin{pmatrix}3&3\\-1&-2\end{pmatrix}$.

Practice 22

For $A=\begin{pmatrix}1&-2\\0&3\end{pmatrix}$, $B=\begin{pmatrix}2&1\\-1&4\end{pmatrix}$, $C=\begin{pmatrix}0&5\\2&-2\end{pmatrix}$, find $2A-3B+C$.

Solution

$2A=\begin{pmatrix}2&-4\\0&6\end{pmatrix}$, $3B=\begin{pmatrix}6&3\\-3&12\end{pmatrix}$. $2A-3B=\begin{pmatrix}-4&-7\\3&-6\end{pmatrix}$. Adding $C$: $\begin{pmatrix}-4&-2\\5&-8\end{pmatrix}$.

Practice 23

Explain why $A+B$ is undefined for $A=\begin{pmatrix}1&2&3\\4&5&6\end{pmatrix}$ and $B=\begin{pmatrix}1&0\\0&1\end{pmatrix}$.

Solution

$A$ is $2\times3$ and $B$ is $2\times2$ — different orders. Matrix addition requires entrywise matching, which is only possible when both matrices have exactly the same number of rows and the same number of columns.

Practice 24

A load-monitoring system records forces (in N) at three sensors over two time steps as matrices $F_1=\begin{pmatrix}12&8&-3\end{pmatrix}$ and $F_2=\begin{pmatrix}5&-2&7\end{pmatrix}$. Find the average force reading at each sensor, i.e. $\tfrac12(F_1+F_2)$.

Solution

$F_1+F_2=\begin{pmatrix}17&6&4\end{pmatrix}$, so $\tfrac12(F_1+F_2)=\begin{pmatrix}8.5&3&2\end{pmatrix}$ N.
