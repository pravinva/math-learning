0 Why this topic?

The DPEN22 Calc 1 revision set already covers antiderivatives (power, trig, exp/log), a reverse chain rule for linear inner functions, the Fundamental Theorem of Calculus in both parts, and the standard area/motion/average-value applications. That's a solid, complete foundation.

MATH141's own Integration unit, in its own words, adds "algebraic substitution," "integral tables," and "integrals by inspection" on top of that. In practice this means three things: substitution with a genuinely non-linear inner function (DPEN22's reverse chain rule only handled the linear case), recognising a small set of standard integral forms straight from a table (several of which are exactly the reverse of the inverse trig and inverse hyperbolic derivatives from the Differentiation Extensions lesson), and one especially useful pattern, $\int f'(x)/f(x)\,dx=\ln|f(x)|+C$, worth being able to spot on sight.

This is deliberately not the full integration toolkit. Partial fractions, integration by parts, the Weierstrass substitution, and reduction formulas are real techniques, but they belong to a fuller, dedicated calculus subject beyond MATH141's 4-hour Integration unit, not here.

1 Simplifying before integrating

Before reaching for substitution, it's always worth checking whether ordinary algebra, expanding brackets, splitting a fraction into separate terms, or dividing out, turns an intimidating-looking integral into a sum of pieces you can already integrate "by inspection," directly reversing a known derivative.

Key idea

Before choosing a technique, try: expanding any brackets or powers, splitting a single fraction with a sum on top into several simpler fractions, and dividing out when the numerator's degree is at least the denominator's. Only reach for substitution or a table once the expression is in its simplest algebraic form.

Worked example

Find $\displaystyle\int(x+1)^2\,dx$ and $\displaystyle\int\frac{3x^2-2}x\,dx$.

Expand first: $(x+1)^2 = x^2+2x+1$. $$\int(x^2+2x+1)\,dx = \frac{x^3}3+x^2+x+C.$$

Split the fraction first: $\dfrac{3x^2-2}x = 3x-\dfrac2x$. $$\int\left(3x-\frac2x\right)dx = \frac{3x^2}2-2\ln|x|+C.$$

Practice 1

Find $\displaystyle\int(2x-3)^2\,dx$.

Solution

Expand: $(2x-3)^2 = 4x^2-12x+9$. $$\int(4x^2-12x+9)\,dx = \frac{4x^3}3-6x^2+9x+C.$$

Practice 2

Find $\displaystyle\int\frac{x^3+2x}x\,dx$.

Solution

Divide out: $\dfrac{x^3+2x}x = x^2+2$. $$\int(x^2+2)\,dx = \frac{x^3}3+2x+C.$$

Practice 3

Find $\displaystyle\int\sqrt x\,(x+1)\,dx$.

Solution

Expand: $\sqrt x(x+1) = x^{3/2}+x^{1/2}$. $$\int\left(x^{3/2}+x^{1/2}\right)dx = \frac25x^{5/2}+\frac23x^{3/2}+C.$$

Practice 4

Find $\displaystyle\int(\sin x+\cos x)^2\,dx$.

Solution

Expand: $(\sin x+\cos x)^2 = \sin^2x+2\sin x\cos x+\cos^2x = 1+\sin(2x)$, using $\sin^2x+\cos^2x=1$ and $2\sin x\cos x=\sin(2x)$.

$$\int\big(1+\sin(2x)\big)\,dx = x-\frac{\cos(2x)}2+C.$$

Practice 5

Find $\displaystyle\int(e^x+e^{-x})^2\,dx$.

Solution

Expand: $(e^x+e^{-x})^2 = e^{2x}+2+e^{-2x}$. $$\int\left(e^{2x}+2+e^{-2x}\right)dx = \frac{e^{2x}}2+2x-\frac{e^{-2x}}2+C.$$

Practice 6

Find $\displaystyle\int\frac{x-1}{\sqrt x}\,dx$.

Solution

Split: $\dfrac{x-1}{\sqrt x} = x^{1/2}-x^{-1/2}$. $$\int\left(x^{1/2}-x^{-1/2}\right)dx = \frac23x^{3/2}-2x^{1/2}+C.$$

2 General algebraic substitution

DPEN22's reverse chain rule handled the case where the inner function is linear, $\sin(3x)$, $e^{2x}$, and so on, where the inner derivative is just a constant. Substitution as a genuine technique goes further: it works whenever the integrand contains some inner function $g(x)$ and (a multiple of) its own derivative $g'(x)$ sitting alongside it, no matter how complicated $g(x)$ itself is.

Key idea

For $\displaystyle\int f\big(g(x)\big)\,g'(x)\,dx$: let $u=g(x)$, so $du=g'(x)\,dx$. Rewrite the entire integral in terms of $u$ (no $x$ should remain), integrate as an ordinary function of $u$, then substitute $g(x)$ back in for $u$ at the very end.

Worked example

Find $\displaystyle\int2x\cos(x^2)\,dx$ and $\displaystyle\int x^2e^{x^3}\,dx$.

$\displaystyle\int2x\cos(x^2)\,dx$: let $u=x^2$, $du=2x\,dx$, exactly the factor sitting alongside $\cos(x^2)$. $$\int\cos u\,du = \sin u+C = \sin(x^2)+C.$$

$\displaystyle\int x^2e^{x^3}\,dx$: let $u=x^3$, $du=3x^2\,dx$, so $x^2\,dx = \dfrac13du$. $$\frac13\int e^u\,du = \frac13e^u+C = \frac13e^{x^3}+C.$$

Practice 7

Find $\displaystyle\int4x^3\cos(x^4)\,dx$.

Solution

$u=x^4$, $du=4x^3\,dx$: $$\int\cos u\,du = \sin u+C = \sin(x^4)+C.$$

Practice 8

Find $\displaystyle\int x\sqrt{x^2+1}\,dx$.

Solution

$u=x^2+1$, $du=2x\,dx$, so $x\,dx=\dfrac12du$: $$\frac12\int u^{1/2}\,du = \frac12\cdot\frac23u^{3/2}+C = \frac13(x^2+1)^{3/2}+C.$$

Practice 9

Find $\displaystyle\int\sin x\cos^3x\,dx$.

Solution

$u=\cos x$, $du=-\sin x\,dx$, so $\sin x\,dx=-du$: $$-\int u^3\,du = -\frac{u^4}4+C = -\frac{\cos^4x}4+C.$$

Practice 10

Find $\displaystyle\int(2x+3)(x^2+3x)^4\,dx$.

Solution

$u=x^2+3x$, $du=(2x+3)\,dx$, exactly the other factor present: $$\int u^4\,du = \frac{u^5}5+C = \frac{(x^2+3x)^5}5+C.$$

Practice 11

Find $\displaystyle\int\frac{x}{x^2+4}\,dx$.

Solution

$u=x^2+4$, $du=2x\,dx$, so $x\,dx=\dfrac12du$: $$\frac12\int\frac{du}u = \frac12\ln|u|+C = \frac12\ln(x^2+4)+C.$$

Practice 12

Find $\displaystyle\int\frac{\cos(\sqrt x)}{\sqrt x}\,dx$.

Solution

$u=\sqrt x$, $du=\dfrac1{2\sqrt x}\,dx$, so $\dfrac{dx}{\sqrt x} = 2\,du$: $$2\int\cos u\,du = 2\sin u+C = 2\sin(\sqrt x)+C.$$

3 Standard forms: inverse trig integrals

These two forms are exactly the reverse of the $\arcsin$ and $\arctan$ derivatives from the Differentiation Extensions lesson, worth recognising directly from an integral table rather than re-deriving each time.

Key idea

$$\int\frac{dx}{\sqrt{a^2-x^2}} = \arcsin\left(\frac{x}a\right)+C, \qquad \int\frac{dx}{a^2+x^2} = \frac1a\arctan\left(\frac{x}a\right)+C.$$

If the coefficient in front of $x^2$ inside the square root or denominator isn't $1$, factor it out (or substitute) until the expression matches one of these two shapes exactly.

Worked example

Find $\displaystyle\int\frac{dx}{\sqrt{9-x^2}}$ and $\displaystyle\int\frac{dx}{x^2+16}$.

Matches the first pattern with $a=3$: $$\int\frac{dx}{\sqrt{9-x^2}} = \arcsin\left(\frac{x}3\right)+C.$$

Matches the second pattern with $a=4$: $$\int\frac{dx}{x^2+16} = \frac14\arctan\left(\frac{x}4\right)+C.$$

Practice 13

Find $\displaystyle\int\frac{dx}{\sqrt{4-x^2}}$.

Solution

$a=2$: $$\int\frac{dx}{\sqrt{4-x^2}} = \arcsin\left(\frac{x}2\right)+C.$$

Practice 14

Find $\displaystyle\int\frac{dx}{x^2+25}$.

Solution

$a=5$: $$\int\frac{dx}{x^2+25} = \frac15\arctan\left(\frac{x}5\right)+C.$$

Practice 15

Find $\displaystyle\int\frac{dx}{\sqrt{1-9x^2}}$.

Solution

Write $1-9x^2=1-(3x)^2$. Let $u=3x$, $du=3\,dx$: $$\frac13\int\frac{du}{\sqrt{1-u^2}} = \frac13\arcsin(u)+C = \frac13\arcsin(3x)+C.$$

Practice 16

Find $\displaystyle\int\frac{dx}{4+x^2}$.

Solution

$a=2$: $$\int\frac{dx}{4+x^2} = \frac12\arctan\left(\frac{x}2\right)+C.$$

Practice 17

Find $\displaystyle\int\frac{3\,dx}{\sqrt{25-x^2}}$.

Solution

Pull the constant $3$ out front, $a=5$: $$3\int\frac{dx}{\sqrt{25-x^2}} = 3\arcsin\left(\frac{x}5\right)+C.$$

4 Standard forms: hyperbolic integrals

The same reversing idea applies to the hyperbolic derivatives from the Differentiation Extensions lesson: each derivative rule there gives a matching integral rule here directly.

Key idea

$$\int\sinh x\,dx = \cosh x+C, \qquad \int\cosh x\,dx=\sinh x+C, \qquad \int\operatorname{sech}^2x\,dx = \tanh x+C.$$

And two standard forms that reverse the inverse hyperbolic derivatives:

$$\int\frac{dx}{\sqrt{x^2+a^2}} = \operatorname{arcsinh}\left(\frac{x}a\right)+C, \qquad \int\frac{dx}{\sqrt{x^2-a^2}} = \operatorname{arccosh}\left(\frac{x}a\right)+C \ \ (x>a).$$

Worked example

Find $\displaystyle\int\cosh(2x)\,dx$ and $\displaystyle\int\frac{dx}{\sqrt{x^2+4}}$.

$\displaystyle\int\cosh(2x)\,dx$: let $u=2x$, $du=2\,dx$: $$\frac12\int\cosh u\,du = \frac12\sinh u+C = \frac12\sinh(2x)+C.$$

Matches the $\operatorname{arcsinh}$ pattern with $a=2$: $$\int\frac{dx}{\sqrt{x^2+4}} = \operatorname{arcsinh}\left(\frac{x}2\right)+C.$$

Practice 18

Find $\displaystyle\int\sinh(3x)\,dx$.

Solution

$$\int\sinh(3x)\,dx = \frac13\cosh(3x)+C.$$

Practice 19

Find $\displaystyle\int\operatorname{sech}^2\left(\frac{x}2\right)dx$.

Solution

$u=\dfrac{x}2$, $du=\dfrac12dx$: $$2\int\operatorname{sech}^2u\,du = 2\tanh u+C = 2\tanh\left(\frac{x}2\right)+C.$$

Practice 20

Find $\displaystyle\int\frac{dx}{\sqrt{x^2+9}}$.

Solution

$a=3$: $$\int\frac{dx}{\sqrt{x^2+9}} = \operatorname{arcsinh}\left(\frac{x}3\right)+C.$$

Practice 21

Find $\displaystyle\int\frac{dx}{\sqrt{x^2-16}}$ (for $x>4$).

Solution

$a=4$: $$\int\frac{dx}{\sqrt{x^2-16}} = \operatorname{arccosh}\left(\frac{x}4\right)+C.$$

Practice 22

Find $\displaystyle\int x\cosh(x^2)\,dx$.

Solution

$u=x^2$, $du=2x\,dx$: $$\frac12\int\cosh u\,du = \frac12\sinh u+C = \frac12\sinh(x^2)+C.$$

5 The f′/f log pattern

Whenever the numerator of a fraction is (a multiple of) the derivative of the denominator, the antiderivative is a natural logarithm, directly from the chain rule run backwards. Trained eyes learn to spot this even when it isn't written in an obviously matching form.

Key idea

$$\int\frac{f'(x)}{f(x)}\,dx = \ln|f(x)|+C.$$

Worked example

Find $\displaystyle\int\frac{2x}{x^2+1}\,dx$ and $\displaystyle\int\tan x\,dx$.

The numerator $2x$ is exactly the derivative of the denominator $x^2+1$: $$\int\frac{2x}{x^2+1}\,dx = \ln(x^2+1)+C$$ (no absolute value needed, since $x^2+1>0$ always).

$\tan x = \dfrac{\sin x}{\cos x}$, and the numerator $\sin x$ is $-1$ times the derivative of $\cos x$: $$\int\tan x\,dx = -\int\frac{-\sin x}{\cos x}\,dx = -\ln|\cos x|+C.$$

Practice 23

Find $\displaystyle\int\frac{3x^2}{x^3+5}\,dx$.

Solution

Numerator is exactly the derivative of the denominator: $$\int\frac{3x^2}{x^3+5}\,dx = \ln|x^3+5|+C.$$

Practice 24

Find $\int\cot x\,dx$.

Solution

$\cot x = \dfrac{\cos x}{\sin x}$, numerator is exactly the derivative of $\sin x$: $$\int\cot x\,dx = \ln|\sin x|+C.$$

Practice 25

Find $\displaystyle\int\frac{e^x}{e^x+2}\,dx$.

Solution

Numerator is exactly the derivative of the denominator: $$\int\frac{e^x}{e^x+2}\,dx = \ln(e^x+2)+C.$$

Practice 26

Find $\displaystyle\int\frac{2x}{x^2-9}\,dx$.

Solution

$$\int\frac{2x}{x^2-9}\,dx = \ln|x^2-9|+C.$$

Practice 27

Find $\displaystyle\int\frac{\cosh x}{\sinh x}\,dx$ (i.e. $\int\coth x\,dx$).

Solution

Numerator is exactly the derivative of the denominator: $$\int\coth x\,dx = \ln|\sinh x|+C.$$

Practice 28

Find $\displaystyle\int\frac{2x+3}{x^2+3x+1}\,dx$.

Solution

Numerator is exactly the derivative of the denominator: $$\int\frac{2x+3}{x^2+3x+1}\,dx = \ln|x^2+3x+1|+C.$$
