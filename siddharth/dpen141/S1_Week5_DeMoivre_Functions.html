0 De Moivre's Theorem

De Moivre's Theorem is the payoff for putting a complex number into polar form: raising it to a power, or taking an $n$th root, becomes almost effortless — multiply the argument (or divide it, for roots) and raise (or take the root of) the modulus.

Key statement

For any integer $n$: $$\big(r(\cos\theta+j\sin\theta)\big)^n = r^n(\cos n\theta+j\sin n\theta).$$

$n$th roots: the $n$ roots of $z=r(\cos\theta+j\sin\theta)$ are $$w_k = r^{1/n}\left(\cos\frac{\theta+2k\pi}{n}+j\sin\frac{\theta+2k\pi}{n}\right), \qquad k=0,1,\dots,n-1.$$ They're evenly spaced around a circle of radius $r^{1/n}$.

Worked example

Find the polar form of $z=-1+j\sqrt3$, and hence find $z^{10}$.

$r=\sqrt{1+3}=2$. Second quadrant, reference angle $\dfrac{\pi}{3}$: $\theta=\pi-\dfrac{\pi}{3}=\dfrac{2\pi}{3}$.

$z=2\left(\cos\dfrac{2\pi}{3}+j\sin\dfrac{2\pi}{3}\right)$.

$z^{10}=2^{10}\left(\cos\dfrac{20\pi}{3}+j\sin\dfrac{20\pi}{3}\right)$. Reduce the angle mod $2\pi$: $\dfrac{20\pi}{3}-6\pi=\dfrac{20\pi-18\pi}{3}=\dfrac{2\pi}{3}$.

$$z^{10} = 1024\left(\cos\frac{2\pi}{3}+j\sin\frac{2\pi}{3}\right) = 1024\left(-\frac12+j\frac{\sqrt3}{2}\right) = -512+512\sqrt3\,j.$$

Practice 1

Find $(1+j)^8$ using De Moivre's Theorem.

Solution

Polar form: $r=\sqrt2$, $\theta=\dfrac{\pi}{4}$. $(1+j)^8 = (\sqrt2)^8\left(\cos2\pi+j\sin2\pi\right)=16(1+0)=16$.

Practice 2

Find $\left(\sqrt3-j\right)^6$ using De Moivre's Theorem.

Solution

$r=\sqrt{3+1}=2$, $\theta=-\dfrac{\pi}{6}$ (Q4). $z^6=2^6\left(\cos(-\pi)+j\sin(-\pi)\right)=64(-1+0)=-64$.

Practice 3

Solve $z^4+1=j\sqrt3$ for all values of $z$ (leave your answer in polar form).

Solution

Rewrite: $z^4 = -1+j\sqrt3$. Polar form of RHS: $r=2$, $\theta=\dfrac{2\pi}{3}$ (Q2, as in the worked example). Fourth roots: $w_k = 2^{1/4}\left(\cos\dfrac{\tfrac{2\pi}{3}+2k\pi}{4}+j\sin\dfrac{\tfrac{2\pi}{3}+2k\pi}{4}\right)$ for $k=0,1,2,3$, giving angles $\dfrac{\pi}{6},\dfrac{\pi}{6}+\dfrac{\pi}{2},\dfrac{\pi}{6}+\pi,\dfrac{\pi}{6}+\dfrac{3\pi}{2}$, i.e. $\dfrac{\pi}{6},\dfrac{2\pi}{3},\dfrac{7\pi}{6},\dfrac{5\pi}{3}$, each with modulus $2^{1/4}$.

Practice 4

Find all cube roots of $z=8$ (i.e. solve $w^3=8$).

Solution

$8=8(\cos0+j\sin0)$. Roots: $w_k=8^{1/3}\left(\cos\dfrac{2k\pi}{3}+j\sin\dfrac{2k\pi}{3}\right)$, $k=0,1,2$: $w_0=2$, $w_1=2\left(\cos\tfrac{2\pi}{3}+j\sin\tfrac{2\pi}{3}\right)=-1+j\sqrt3$, $w_2=2\left(\cos\tfrac{4\pi}{3}+j\sin\tfrac{4\pi}{3}\right)=-1-j\sqrt3$.

Practice 5

Use De Moivre's Theorem with $n=3$ to derive an identity for $\cos3\theta$ in terms of $\cos\theta$.

Solution

$(\cos\theta+j\sin\theta)^3=\cos3\theta+j\sin3\theta$. Expand the left side using the binomial theorem: $\cos^3\theta+3\cos^2\theta(j\sin\theta)+3\cos\theta(j\sin\theta)^2+(j\sin\theta)^3 = \cos^3\theta+3j\cos^2\theta\sin\theta-3\cos\theta\sin^2\theta-j\sin^3\theta$. Equating real parts: $\cos3\theta=\cos^3\theta-3\cos\theta\sin^2\theta=\cos^3\theta-3\cos\theta(1-\cos^2\theta)=4\cos^3\theta-3\cos\theta$.

1 Introduction to Functions: Domain, Range, and Algebra of Functions

A function is a rule assigning exactly one output to each valid input. Its domain is every input the rule can legally accept; its range is every value the rule can actually produce. Combining functions (adding, multiplying, dividing) can only ever shrink the domain — never grow it — since every combined operation must itself be defined.

Key idea

For $(f+g)(x)$, $(fg)(x)$: domain is $\text{dom}(f)\cap\text{dom}(g)$. For $\left(\dfrac{f}{g}\right)(x)$: domain is $\text{dom}(f)\cap\text{dom}(g)$, further excluding any $x$ where $g(x)=0$.

Worked example

Let $f(x)=\sqrt{x^2-1}$ and $g(x)=\dfrac{1}{\sqrt{4-x^2}}$. Find $(fg)(x)$ and state the domain of each function, including $f$ and $g$.

$\text{dom}(f)$: need $x^2-1\ge0 \Rightarrow |x|\ge1 \Rightarrow x\in(-\infty,-1]\cup[1,\infty)$.

$\text{dom}(g)$: need $4-x^2>0$ (strict, since it's in a denominator under a root) $\Rightarrow |x|<2 \Rightarrow x\in(-2,2)$.

$(fg)(x) = \sqrt{x^2-1}\cdot\dfrac{1}{\sqrt{4-x^2}} = \sqrt{\dfrac{x^2-1}{4-x^2}}$.

$\text{dom}(fg) = \text{dom}(f)\cap\text{dom}(g) = (-2,-1]\cup[1,2)$.

Practice 6

Find the domain of $f(x)=\dfrac{1}{x-3}$.

Solution

All reals except where the denominator is zero: $x\ne3$, i.e. $x\in(-\infty,3)\cup(3,\infty)$.

Practice 7

Find the domain of $g(x)=\sqrt{5-x}$.

Solution

Need $5-x\ge0 \Rightarrow x\le5$, i.e. $x\in(-\infty,5]$.

Practice 8

For $f(x)=x^2-4$, find the range of $f$, given its domain is all real numbers.

Solution

Vertex at $(0,-4)$, opening upward, so the minimum value is $-4$ and $f$ increases without bound either side: range $=[-4,\infty)$.

Practice 9

Let $f(x)=\sqrt{x}$ and $g(x)=\dfrac{1}{x-2}$. Find $\left(\dfrac{f}{g}\right)(x)$ and its domain.

Solution

$\left(\dfrac{f}{g}\right)(x) = \sqrt{x}\cdot(x-2) = (x-2)\sqrt{x}$. Domain of $f$: $x\ge0$. Domain of $g$: $x\ne2$. Also need $g(x)\ne0$, which is automatic since $g$ is never zero. Domain of $f/g$: $x\ge0,\ x\ne2$, i.e. $[0,2)\cup(2,\infty)$.

Practice 11

A signal's power is $P(t)=\dfrac{1}{\sqrt{t^2-4}}$. Find the domain of $P$, and explain the physical significance of the excluded region.

Solution

Need $t^2-4>0$ (strict, as it's under a root in a denominator) $\Rightarrow |t|>2 \Rightarrow t\in(-\infty,-2)\cup(2,\infty)$. For $-2\le t\le2$, the formula is either undefined (at $t=\pm2$) or would require a square root of a negative number (strictly between), so the model simply doesn't apply in that time window — the signal's power isn't described by this formula there.

2 Piecewise Functions and Graphical Transformations

Many real functions aren't given by a single formula everywhere — they're defined piece by piece, switching rule at certain break points. Recognising how shifts, stretches, and reflections move a graph around lets you sketch a transformed function quickly from a known parent shape.

Key transformations of $y=f(x)$

$f(x)+a$: shift up $a$ (down if $a<0$). $\ f(x-a)$: shift right $a$ (left if $a<0$). $\ -f(x)$: reflect in the $x$-axis. $\ f(-x)$: reflect in the $y$-axis. $\ cf(x)$ ($c>1$): stretch vertically by factor $c$.

Worked example

Describe how the graph of $g(x)=-2f(x-3)+1$ relates to the graph of $y=f(x)$.

Read the transformations from the inside out: $f(x-3)$ shifts $f$ right by $3$.

$-2f(x-3)$ then reflects that graph in the $x$-axis and stretches it vertically by a factor of $2$.

$-2f(x-3)+1$ finally shifts the whole thing up by $1$. Order: shift right 3, reflect & stretch vertically by 2, shift up 1.

Practice 12

Given $f(x)=x^2$, describe the graph of $g(x)=(x+4)^2-2$ as a transformation of $f$.

Solution

Shift left $4$, then shift down $2$.

Practice 13

Given $f(x)=\sqrt{x}$, describe the graph of $h(x)=-\sqrt{x}+5$ as a transformation of $f$.

Solution

Reflect in the $x$-axis, then shift up $5$.

Practice 14

A function is defined piecewise as $f(x)=\begin{cases}x+2 & x<0\\x^2 & x\ge0\end{cases}$. Evaluate $f(-3)$, $f(0)$, and $f(2)$.

Solution

$f(-3)=-3+2=-1$ (using the first rule, since $-3<0$). $f(0)=0^2=0$ (using the second rule, since $0\ge0$). $f(2)=2^2=4$.

Practice 15

Sketch (describe key features of) $y=|x-2|$ as a transformation of $y=|x|$, and state its minimum value and where it occurs.

Solution

Shift right $2$. Minimum value $0$, occurring at $x=2$ (the vertex of the V-shape).

Practice 17

For parametric curve x=t^2-3t, y=t^3-2t, (a) find dy/dx, (b) find stationary points in y with respect to x, (c) tangent equations.

Solution

Compute dx/dt and dy/dt; then dy/dx=(dy/dt)/(dx/dt).

Stationary in y wrt x occurs when dy/dt=0 and dx/dt!=0.

Substitute t-values into (x,y) and slope for tangent equations.

Practice 18

Use substitution to evaluate integral from 0 to 4: x*sqrt(x^2+3) dx, then interpret geometric meaning of positive result.

Solution

Let u=x^2+3, du=2x dx.

Convert limits: x=0 -> u=3, x=4 -> u=19.

Integrate (1/2) integral u^(1/2) du and back-substitute/check units.

Practice 19

For f(x)=((x^2-5x+4)/(x-1)), (a) state domain, (b) simplify f(x) for x!= 1, (c) solve f(x)= 6.

Solution

(a) Domain excludes denominator zero: x != 1.

(b) Factor numerator where possible, then cancel only if a common factor with (x-1) exists; keep restriction x != 1.

(c) Set simplified expression equal to 6, solve resulting equation, then reject any root violating x != 1.

Practice 22

Differentiate y=(x^3+7x)^(4) and evaluate y' at x=1; then classify stationary points of y' on x>0.

Solution

Use chain rule: y'=4(x^3+7x)^(3)(3x^2+7).

Substitute x=1 directly.

For stationary analysis of y, set y'=0 and inspect multiplicity/sign changes.

Practice 24

Solve inequality ln(x-2) + ln(x+5) >= ln(45) and express solution set in interval notation.

Solution

Domain: x>2 and x>-5; hence x>2.

Combine logs: ln((x-2)(x+5)) >= ln(45) => (x-2)(x+5)>=45.

Solve quadratic inequality with sign chart, then intersect with domain.
