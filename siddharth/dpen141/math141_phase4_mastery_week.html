Cross-Phase Challenge
Combining Calculus & Linear Algebra
The kind of question the Nov mock exams actually test
47
A line has vector equation $\mathbf{r}=(1,2,0)+\lambda(1,-1,2)$. Find the value of $t$ (rename $\lambda=t$) where the line crosses the plane $x+y+z=6$, then find $dy/dx$ along the parametric curve $x=1+t,y=2-t$ at that value of $t$.
Cross-Phase Challenge
▶
Solution

Plane: $(1+t)+(2-t)+2t=6\implies3+2t=6\implies t=1.5$. For $x=1+t,y=2-t$: $dx/dt=1,dy/dt=-1\implies dy/dx=-1$ (constant, since it's a straight line)

48
Find $\det(A)$ for $A=\begin{pmatrix}2&1\\4&2\end{pmatrix}$, then explain what this implies both about $A^{-1}$ and about whether the rows of $A$, treated as 2D vectors, are parallel.
Cross-Phase Challenge
▶
Solution

$\det(A)=4-4=0$ — no inverse exists; and the rows $(2,1)$, $(4,2)$ are parallel ($(4,2)=2(2,1)$), which is exactly the 2D reason a $2\times2$ determinant vanishes

49
A parametric curve has $x=\cos t, y=\sin t$. Find the scalar triple product interpretation: is the vector $(\cos t,\sin t,0)$ ever coplanar with $(1,0,0)$ and $(0,1,0)$ for all $t$? Explain.
Cross-Phase Challenge
▶
Solution

Yes, always — $(\cos t,\sin t,0)$ has zero $z$-component for every $t$, so it always lies in the $xy$-plane spanned by $(1,0,0)$ and $(0,1,0)$; the scalar triple product of the three is $0$ for every $t$

50
Use Gaussian elimination to solve $\begin{cases}2x+y=8\\x-y=1\end{cases}$, then verify using the 2×2 inverse matrix method $\mathbf{x}=A^{-1}\mathbf{b}$.
Cross-Phase Challenge
▶
Solution

Elimination: $R_1+R_2\cdot? $ — add rows: $(2x+y)+(x-y)=8+1\implies3x=9\implies x=3,y=2$. Inverse check: $A=\begin{pmatrix}2&1\\1&-1\end{pmatrix}$, $\det=-3$, $A^{-1}=\dfrac{1}{-3}\begin{pmatrix}-1&-1\\-1&2\end{pmatrix}$; $A^{-1}\mathbf{b}=\dfrac{1}{-3}\begin{pmatrix}-9\\-6\end{pmatrix}=\begin{pmatrix}3\\2\end{pmatrix}$ — matches
