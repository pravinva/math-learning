Q2Inverses / Cramer's Rule

Solve $3x+2y=8,\ x-y=1$ using Cramer's Rule.

Solution

$\det(A)=\begin{vmatrix}3&2\\1&-1\end{vmatrix}=-3-2=-5$.

$\det(A_x)=\begin{vmatrix}8&2\\1&-1\end{vmatrix}=-8-2=-10 \Rightarrow x=\frac{-10}{-5}=2$.

$\det(A_y)=\begin{vmatrix}3&8\\1&1\end{vmatrix}=3-8=-5 \Rightarrow y=\frac{-5}{-5}=1$. Solution: $(2,1)$.

$A=\begin{pmatrix}7&0&0\\3&2&0\\1&-1&4\end{pmatrix}$ is lower triangular. State its eigenvalues directly.

Solution

For a triangular matrix, the eigenvalues are exactly the diagonal entries: $\lambda=7,2,4$.

Q5Matrix Fundamentals

Construct the $2\times3$ matrix $B$ whose entries satisfy $b_{ij}=ij$. What do you notice about its two rows?

Solution

$$B=\begin{pmatrix}1&2&3\\2&4&6\end{pmatrix}$$ Row 2 is exactly twice row 1, an example of a rank-deficient matrix.

Q6Summation / Kronecker Delta

For $A=\begin{pmatrix}1&4\\3&2\end{pmatrix}$, find $\mathrm{tr}(A)$ using $\displaystyle\sum_i\sum_jA_{ij}\delta_{ij}$; confirm it matches the diagonal sum.

Solution

The delta ties $j$ to $i$, collapsing the double sum to $\displaystyle\sum_iA_{ii}$: $A_{11}+A_{22}=1+2=3$.

Direct check: $\mathrm{tr}(A)=1+2=3$, matching.

Q7Row Operations

A $3\times3$ matrix row-reduces to $\begin{pmatrix}2&1&3\\0&-1&4\\0&0&0\end{pmatrix}$. State the rank, the determinant, and whether $A\mathbf{x}=\mathbf{0}$ has nontrivial solutions.

Solution

Only $2$ nonzero rows survive: $\mathrm{rank}(A)=2<3$, so $\det(A)=0$, $A$ is singular, and $A\mathbf{x}=\mathbf{0}$ has infinitely many nontrivial solutions.

Q2Eigenvalues

A $4\times4$ matrix — without solving anything, state how many eigenvalues it has (counting multiplicity), and why.

Solution

Four. The characteristic polynomial $\det(A-\lambda I)$ has degree exactly equal to the matrix size, so by the fundamental theorem of algebra it has exactly $4$ roots, counting multiplicity (real or complex).

Q3Row Operations

Is $\begin{pmatrix}2&1&4\\0&0&3\\0&0&0\end{pmatrix}$ in row echelon form? Justify.

Solution

Yes. Pivots $2$ (col 1) and $3$ (col 3) move strictly rightward down the rows, everything below each pivot is zero, and the zero row sits at the bottom.

Q4Inverses / Cramer's Rule

A $2\times2$ system has $\det(A)=0$. Can Cramer's Rule tell you how many solutions it has? What should you do instead?

Solution

No. Every variable's formula divides by $\det(A)$, so with $\det(A)=0$ it's undefined. Row-reduce the augmented matrix instead to tell whether the system has no solution or infinitely many.

Q5Eigenvalues

Find the eigenvector for $\lambda=0$ of $A=\begin{pmatrix}4&2\\2&1\end{pmatrix}$ (from Set 8 Q6). What does this eigenvector represent, geometrically?

Solution

$A-0I=A$. Row 1: $4x+2y=0\Rightarrow y=-2x$. Eigenvector $\begin{pmatrix}1\\-2\end{pmatrix}$.

This vector spans the null space of $A$ — consistent with $A$ being singular ($\det(A)=0$), it's a direction $A$ collapses entirely to the origin.

Q6Row Operations

A $3\times3$ matrix $A$ has $\det(A)=7$. State what this tells you about $\mathrm{rank}(A)$, invertibility, and the solutions to $A\mathbf{x}=\mathbf{0}$.

Solution

By the equivalence theorem: $\mathrm{rank}(A)=3$ (full rank), $A$ is invertible, and $A\mathbf{x}=\mathbf{0}$ has only the trivial solution $\mathbf{x}=\mathbf{0}$.
