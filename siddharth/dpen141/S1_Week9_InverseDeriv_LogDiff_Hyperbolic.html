0 The Derivative of an Inverse Function

If you know how fast $f$ changes, you automatically know how fast $f^{-1}$ changes too — they're reciprocals of each other, evaluated at corresponding points. This single idea is what lets you differentiate $\ln x$ once you know how to differentiate $e^x$, and $\arcsin x$ once you know how to differentiate $\sin x$.

Key formula

$$\big(f^{-1}\big)'(x) = \frac{1}{f'\big(f^{-1}(x)\big)}.$$

Equivalently, if $y=f^{-1}(x)$ so that $x=f(y)$, differentiating implicitly gives $1=f'(y)\dfrac{dy}{dx}$, so $\dfrac{dy}{dx}=\dfrac{1}{f'(y)}$.

Worked example

Given $f(x)=x^3+2x$, find $(f^{-1})'(3)$, using the fact that $f(1)=3$.

$f'(x)=3x^2+2$.

Since $f(1)=3$, we have $f^{-1}(3)=1$.

$$(f^{-1})'(3) = \frac{1}{f'(f^{-1}(3))} = \frac{1}{f'(1)} = \frac{1}{3(1)+2} = \frac{1}{5}.$$

Practice 1

Given $f(x)=2x^3+1$ and $f(1)=3$, find $(f^{-1})'(3)$.

Solution

$f'(x)=6x^2$. $f'(1)=6$. $(f^{-1})'(3)=\dfrac16$.

Practice 2

Given $f(x)=x^5+x$ and $f(1)=2$, find $(f^{-1})'(2)$.

Solution

$f'(x)=5x^4+1$. $f'(1)=6$. $(f^{-1})'(2)=\dfrac16$.

Practice 3

Given $f(x)=\tan x$ (with $f'(x)=\sec^2x$) and $f\!\left(\dfrac{\pi}{4}\right)=1$, find $(f^{-1})'(1)$.

Solution

$f'\!\left(\tfrac{\pi}{4}\right)=\sec^2\!\left(\tfrac{\pi}{4}\right)=\left(\sqrt2\right)^2=2$. $(f^{-1})'(1)=\dfrac12$.

Practice 4

Use implicit differentiation on $x=\sin y$ (with $-\tfrac{\pi}{2}\le y\le\tfrac{\pi}{2}$) to derive $\dfrac{d}{dx}(\arcsin x)=\dfrac{1}{\sqrt{1-x^2}}$.

Solution

Differentiate $x=\sin y$ with respect to $x$: $1=\cos y\dfrac{dy}{dx} \Rightarrow \dfrac{dy}{dx}=\dfrac{1}{\cos y}$. Since $\cos^2y+\sin^2y=1$ and $\sin y=x$: $\cos y=\sqrt{1-x^2}$ (positive, since $y\in[-\tfrac{\pi}{2},\tfrac{\pi}{2}]$). So $\dfrac{dy}{dx}=\dfrac{1}{\sqrt{1-x^2}}$.

Practice 5

Given $g(x)=x^3+3x+2$ is one-to-one with $g(0)=2$, find the equation of the tangent line to $y=g^{-1}(x)$ at $x=2$.

Solution

$g'(x)=3x^2+3$, $g'(0)=3$. $(g^{-1})'(2)=\dfrac13$. Since $g(0)=2$, $g^{-1}(2)=0$. Tangent line: $y-0=\dfrac13(x-2) \Rightarrow y=\dfrac13x-\dfrac23$.

1 Logarithmic Differentiation

When a function has a variable in both the base and the exponent (like $x^x$), or is a complicated product/quotient of many pieces, taking the logarithm of both sides before differentiating turns multiplication into addition and powers into simple products — often the only practical way to differentiate the function at all.

Key method

1) Take $\ln$ of both sides of $y=f(x)$. 2) Use log laws to expand fully. 3) Differentiate implicitly with respect to $x$ (the left side always gives $\dfrac{1}{y}\dfrac{dy}{dx}$). 4) Multiply through by $y$ (substituting the original expression for $y$) to isolate $\dfrac{dy}{dx}$.

Worked example

Differentiate $y=x^x$ (for $x>0$).

Take logs: $\ln y = x\ln x$.

Differentiate both sides with respect to $x$ (product rule on the right): $\dfrac{1}{y}\dfrac{dy}{dx} = \ln x + x\cdot\dfrac1x = \ln x+1$.

$$\frac{dy}{dx} = y(\ln x+1) = x^x(\ln x+1).$$

Practice 6

Differentiate $y=x^{2x}$ using logarithmic differentiation.

Solution

$\ln y=2x\ln x$. $\dfrac1y\dfrac{dy}{dx}=2\ln x+2x\cdot\dfrac1x=2\ln x+2$. $\dfrac{dy}{dx}=x^{2x}(2\ln x+2)$.

Practice 7

Differentiate $y=(\sin x)^x$ using logarithmic differentiation.

Solution

$\ln y = x\ln(\sin x)$. $\dfrac1y\dfrac{dy}{dx}=\ln(\sin x)+x\cdot\dfrac{\cos x}{\sin x}=\ln(\sin x)+x\cot x$. $\dfrac{dy}{dx}=(\sin x)^x\big[\ln(\sin x)+x\cot x\big]$.

Practice 8

Differentiate $y=\dfrac{(x+1)^2\sqrt{x-3}}{(2x+5)^3}$ using logarithmic differentiation (leave your final answer with $y$ substituted back in).

Solution

$\ln y = 2\ln(x+1)+\tfrac12\ln(x-3)-3\ln(2x+5)$. $\dfrac1y\dfrac{dy}{dx}=\dfrac{2}{x+1}+\dfrac{1}{2(x-3)}-\dfrac{6}{2x+5}$. $\dfrac{dy}{dx}=y\left[\dfrac{2}{x+1}+\dfrac{1}{2(x-3)}-\dfrac{6}{2x+5}\right]$, with $y=\dfrac{(x+1)^2\sqrt{x-3}}{(2x+5)^3}$.

Practice 9

Differentiate $y=x^{\ln x}$ (for $x>0$) using logarithmic differentiation.

Solution

$\ln y = \ln x\cdot\ln x = (\ln x)^2$. $\dfrac1y\dfrac{dy}{dx}=2\ln x\cdot\dfrac1x$. $\dfrac{dy}{dx}=x^{\ln x}\cdot\dfrac{2\ln x}{x}$.

Practice 10

Differentiate $y=x^x\tanh\!\left(\dfrac1x\right)$, combining logarithmic differentiation with the product rule.

Solution

$\ln y = x\ln x + \ln\!\left[\tanh\!\left(\tfrac1x\right)\right]$. Differentiate: $\dfrac1y\dfrac{dy}{dx}=(\ln x+1) + \dfrac{\text{sech}^2(1/x)\cdot(-1/x^2)}{\tanh(1/x)}$. So $\dfrac{dy}{dx}=x^x\tanh\!\left(\tfrac1x\right)\left[\ln x+1-\dfrac{\text{sech}^2(1/x)}{x^2\tanh(1/x)}\right]$.

2 Hyperbolic Functions

Hyperbolic functions are built from $e^x$ in a way that deliberately echoes ordinary trigonometry — they even satisfy an almost-identical Pythagorean identity. They show up naturally in engineering wherever exponential growth and decay combine symmetrically, such as the shape of a hanging cable (a catenary).

Key definitions and identity

$$\sinh x = \frac{e^x-e^{-x}}{2}, \qquad \cosh x = \frac{e^x+e^{-x}}{2}, \qquad \tanh x = \frac{\sinh x}{\cosh x}.$$

Hyperbolic Pythagorean identity: $\cosh^2x-\sinh^2x=1$ (note the minus sign, unlike the ordinary trig identity).

Worked example

Given $\cosh x = \dfrac{13}{5}$ and $x>0$, find $\sinh x$ and $\tanh x$.

$\cosh^2x-\sinh^2x=1 \ \Rightarrow\ \sinh^2x = \cosh^2x-1 = \dfrac{169}{25}-1=\dfrac{144}{25}$.

Since $x>0$, $\sinh x>0$: $\sinh x = \dfrac{12}{5}$.

$\tanh x = \dfrac{\sinh x}{\cosh x} = \dfrac{12/5}{13/5} = \dfrac{12}{13}$.

Practice 11

Express $z=j\dfrac{\pi}{6}$ in the form $a+jb$ using $e^z=\cos b+j\sin b$ when $z$ is purely imaginary (Euler's formula), and separately evaluate $\cosh\!\left(j\dfrac{\pi}{6}\right)$ using the definition directly.

Solution

$e^{j\pi/6}=\cos\tfrac{\pi}{6}+j\sin\tfrac{\pi}{6}=\tfrac{\sqrt3}{2}+\tfrac12j$. Separately, $\cosh\!\left(j\tfrac{\pi}{6}\right)=\dfrac{e^{j\pi/6}+e^{-j\pi/6}}{2}=\dfrac{2\cos(\pi/6)}{2}=\cos\!\left(\tfrac{\pi}{6}\right)=\tfrac{\sqrt3}{2}$ (a real number, illustrating $\cosh(jx)=\cos x$).

Practice 12

Given $\sinh x = \dfrac{3}{4}$, find $\cosh x$ (for $x>0$).

Solution

$\cosh^2x=1+\sinh^2x=1+\dfrac9{16}=\dfrac{25}{16}$. $\cosh x=\dfrac54$ (always positive).

Practice 13

Prove the identity $\cosh(2x)=2\sinh^2x+1$ starting from the exponential definitions.

Solution

$2\sinh^2x+1 = 2\left(\dfrac{e^x-e^{-x}}{2}\right)^2+1 = 2\cdot\dfrac{e^{2x}-2+e^{-2x}}{4}+1=\dfrac{e^{2x}-2+e^{-2x}}{2}+1=\dfrac{e^{2x}+e^{-2x}}{2}-1+1=\dfrac{e^{2x}+e^{-2x}}{2}=\cosh(2x)$.

Practice 14

Solve $\cosh x = 2$ for $x$, expressing your answer using a natural logarithm.

Solution

$\dfrac{e^x+e^{-x}}{2}=2 \Rightarrow e^x+e^{-x}=4$. Multiply by $e^x$: $e^{2x}-4e^x+1=0$. Let $u=e^x$: $u^2-4u+1=0 \Rightarrow u=\dfrac{4\pm\sqrt{12}}{2}=2\pm\sqrt3$. Both roots are positive, so $x=\ln(2+\sqrt3)$ or $x=\ln(2-\sqrt3)$ (which is the negative of the first, since $(2+\sqrt3)(2-\sqrt3)=1$).

Practice 15

A hanging cable takes the shape $y=a\cosh\!\left(\dfrac{x}{a}\right)$ (a catenary) with $a=10$. Find the height $y$ at $x=10$, to 2 d.p. (use $\cosh(1)\approx1.5431$).

Solution

$y=10\cosh(1)\approx10(1.5431)=15.43$.

3 Derivatives of Hyperbolic Functions

Because hyperbolic functions are built directly from $e^x$, their derivatives follow immediately from differentiating the exponential definitions — and the resulting pattern mirrors ordinary trig derivatives almost exactly, again with the occasional sign flip to watch for.

Key derivatives

$$\frac{d}{dx}\sinh x=\cosh x, \qquad \frac{d}{dx}\cosh x=\sinh x, \qquad \frac{d}{dx}\tanh x=\text{sech}^2x.$$

Note there's no sign flip between $\sinh$ and $\cosh$ derivatives, unlike the ordinary $\sin,\cos$ pair.

Worked example

Differentiate $y=\cosh(3x^2-1)$.

Chain rule: outer function $\cosh u$ (derivative $\sinh u$), inner $u=3x^2-1$ (derivative $6x$).

$$\frac{dy}{dx} = \sinh(3x^2-1)\times6x = 6x\sinh(3x^2-1).$$

Practice 16

Differentiate $y=\sinh(5x)$.

Solution

$y'=5\cosh(5x)$.

Practice 17

Differentiate $y=\tanh(x^2)$.

Solution

$y'=\text{sech}^2(x^2)\times2x=2x\,\text{sech}^2(x^2)$.

Practice 18

Differentiate $y=x\cosh x$ using the product rule.

Solution

$y'=\cosh x + x\sinh x$.

Practice 19

Differentiate $y=\ln(\cosh x)$.

Solution

$y'=\dfrac{\sinh x}{\cosh x}=\tanh x$.

Practice 20

Verify that $y=\cosh x$ satisfies the differential equation $y''-y=0$.

Solution

$y'=\sinh x$, $y''=\cosh x=y$. So $y''-y=\cosh x-\cosh x=0$. ✓

Practice 24

Solve inequality ln(x-2) + ln(x+5) >= ln(45) and express solution set in interval notation.

Solution

Domain: x>2 and x>-5; hence x>2.

Combine logs: ln((x-2)(x+5)) >= ln(45) => (x-2)(x+5)>=45.

Solve quadratic inequality with sign chart, then intersect with domain.

Practice 25

For parametric curve x=t^2-3t, y=t^3-2t, (a) find dy/dx, (b) find stationary points in y with respect to x, (c) tangent equations.

Solution

Compute dx/dt and dy/dt; then dy/dx=(dy/dt)/(dx/dt).

Stationary in y wrt x occurs when dy/dt=0 and dx/dt!=0.

Substitute t-values into (x,y) and slope for tangent equations.

Practice 26

Use substitution to evaluate integral from 0 to 2: x*sqrt(x^2+4) dx, then interpret geometric meaning of positive result.

Solution

Let u=x^2+4, du=2x dx.

Convert limits: x=0 -> u=4, x=2 -> u=8.

Integrate (1/2) integral u^(1/2) du and back-substitute/check units.

Practice 27

For f(x)=((x^2-3x+5)/(x-1)), (a) state domain, (b) simplify f(x) for x!= 1, (c) solve f(x)= 8.

Solution

(a) Domain excludes denominator zero: x != 1.

(b) Factor numerator where possible, then cancel only if a common factor with (x-1) exists; keep restriction x != 1.

(c) Set simplified expression equal to 8, solve resulting equation, then reject any root violating x != 1.
