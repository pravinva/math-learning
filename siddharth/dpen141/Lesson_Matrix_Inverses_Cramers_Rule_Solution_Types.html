Matrix Inverses, Cramer's Rule & Classifying Solutions

0 Why these three tools?

The Row Operations lesson built the whole toolkit: the three legal row operations, echelon form, Gaussian elimination, rank, determinants, and the big equivalence theorem tying invertibility, rank, and the determinant together. That lesson mentioned, but never actually showed, a couple of things that are worth having on hand: a direct formula for a $2\times2$ inverse, the general method for finding an inverse of any size by row reduction, and Cramer's Rule, a determinant-only way to solve a small system without doing any elimination at all.

It also left one gap in how systems get classified. The equivalence theorem tells you when a square system has exactly one solution for every possible right-hand side. But plenty of systems you will actually be asked to solve are not that clean, some have no solution at all, and some have infinitely many. Section 4 fixes that: it shows exactly how to read the answer, unique, none, or infinite, straight off the echelon form.

Everything here still comes back to the same row operations and determinants from before. Nothing new to learn arithmetically, just three new ways to use what you already have.

1 The 2×2 inverse formula

For a $2\times2$ matrix, there is a direct shortcut for the inverse, no row reduction required. It uses exactly the determinant formula you already know.

Key idea

For $A=\begin{pmatrix}a&b\\c&d\end{pmatrix}$ with $\det(A)=ad-bc\neq0$:

$$A^{-1} = \frac{1}{\det(A)}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}.$$

In words: swap the two diagonal entries, negate the two off-diagonal entries, then divide everything by the determinant. If $\det(A)=0$, this formula would require dividing by zero, exactly matching the equivalence theorem: a $2\times2$ matrix with zero determinant has no inverse at all.

It is worth checking, once, that this formula actually works, rather than just trusting it.

Worked example

Find the inverse of $A=\begin{pmatrix}4&1\\2&3\end{pmatrix}$, and check $AA^{-1}=I$.

$\det(A) = 4(3)-1(2) = 10$.

$A^{-1} = \dfrac1{10}\begin{pmatrix}3&-1\\-2&4\end{pmatrix}$.

Check: $AA^{-1} = \dfrac1{10}\begin{pmatrix}4&1\\2&3\end{pmatrix}\begin{pmatrix}3&-1\\-2&4\end{pmatrix} = \dfrac1{10}\begin{pmatrix}12-2&-4+4\\6-6&-2+12\end{pmatrix} = \dfrac1{10}\begin{pmatrix}10&0\\0&10\end{pmatrix} = \begin{pmatrix}1&0\\0&1\end{pmatrix}$ ✓.

$\det(A) = 2(1)-1(1) = 1$.

$$A^{-1} = \frac11\begin{pmatrix}1&-1\\-1&2\end{pmatrix} = \begin{pmatrix}1&-1\\-1&2\end{pmatrix}.$$

Find the inverse of $A=\begin{pmatrix}3&2\\5&4\end{pmatrix}$.

Solution

$\det(A) = 3(4)-2(5) = 12-10=2$.

$$A^{-1} = \frac12\begin{pmatrix}4&-2\\-5&3\end{pmatrix} = \begin{pmatrix}2&-1\\-2.5&1.5\end{pmatrix}.$$

Practice 3

Find the inverse of $A=\begin{pmatrix}1&2\\3&4\end{pmatrix}$.

Solution

$\det(A) = 1(4)-2(3) = 4-6=-2$.

$$A^{-1} = \frac{1}{-2}\begin{pmatrix}4&-2\\-3&1\end{pmatrix} = \begin{pmatrix}-2&1\\1.5&-0.5\end{pmatrix}.$$

Practice 4

Does $A=\begin{pmatrix}2&4\\1&2\end{pmatrix}$ have an inverse? Explain using the formula.

Solution

$\det(A) = 2(2)-4(1) = 4-4=0$. The formula would require dividing by $0$, which is undefined, so $A$ has no inverse. (Row 2 is exactly half of row 1, so the rows are dependent, consistent with a zero determinant.)

Practice 5

Find the inverse of $A=\begin{pmatrix}5&3\\2&2\end{pmatrix}$.

Solution

$\det(A) = 5(2)-3(2) = 10-6=4$.

$$A^{-1} = \frac14\begin{pmatrix}2&-3\\-2&5\end{pmatrix} = \begin{pmatrix}0.5&-0.75\\-0.5&1.25\end{pmatrix}.$$

Practice 6

Using the inverse found in Practice 1 for $A=\begin{pmatrix}2&1\\1&1\end{pmatrix}$, verify directly that $AA^{-1}=I$.

Solution

$$AA^{-1} = \begin{pmatrix}2&1\\1&1\end{pmatrix}\begin{pmatrix}1&-1\\-1&2\end{pmatrix} = \begin{pmatrix}2-1&-2+2\\1-1&-1+2\end{pmatrix} = \begin{pmatrix}1&0\\0&1\end{pmatrix}$$ ✓, exactly $I$, confirming the formula gave a genuine inverse.

2 Finding an inverse by row reduction

The $2\times2$ formula is fast, but it does not generalise: there is no equally simple formula for a $4\times4$ inverse. What does generalise is row reduction. The idea is to augment $A$ with the identity matrix, exactly the way you augmented $A$ with a right-hand side vector $\mathbf{b}$ back in the Row Operations lesson, and row-reduce all the way to RREF. Whatever happens to the identity side along the way turns out to be exactly $A^{-1}$.

Key idea

To find $A^{-1}$ for an $n\times n$ matrix $A$: form the augmented matrix $(A \mid I)$, and row-reduce until the left-hand block becomes $I$. The right-hand block is then $A^{-1}$:

$$(A \mid I) \ \xrightarrow{\text{row reduce}}\ (I \mid A^{-1}).$$

If, along the way, a row on the left becomes entirely zero, $A$ has no inverse, exactly the same "zero row means singular" signal from Gaussian elimination.

Why this works

Row-reducing $(A\mid I)$ to $(I\mid A^{-1})$ is really solving $n$ systems at once. Reading the identity's columns one at a time, $e_1,\dots,e_n$, row-reducing $(A\mid e_i)$ finds the vector $\mathbf{x}_i$ solving $A\mathbf{x}_i = e_i$. Doing all $n$ columns simultaneously, and stacking the resulting $\mathbf{x}_i$ side by side, gives exactly the matrix $X$ satisfying $AX=I$, which is the definition of $A^{-1}$. Augmenting with the full identity, rather than one column at a time, just does all $n$ of these solves in a single pass of row operations.

Worked example

Find the inverse of $A=\begin{pmatrix}2&1\\1&1\end{pmatrix}$ (the same matrix as Practice 1) by row reduction, and check the two methods agree.

Augment: $\left(\begin{array}{cc|cc}2&1&1&0\\1&1&0&1\end{array}\right)$. Swap rows for a convenient leading $1$: $\left(\begin{array}{cc|cc}1&1&0&1\\2&1&1&0\end{array}\right)$.

$R_2 \to R_2 - 2R_1$: $(0,-1\mid1,-2)$.

$\left(\begin{array}{cc|cc}1&1&0&1\\0&-1&1&-2\end{array}\right)$. Scale $R_2\to-R_2$: $(0,1\mid-1,2)$.

$\left(\begin{array}{cc|cc}1&1&0&1\\0&1&-1&2\end{array}\right)$. Clear above: $R_1\to R_1-R_2$: $(1,0\mid1,-1)$.

$\left(\begin{array}{cc|cc}1&0&1&-1\\0&1&-1&2\end{array}\right)$, so $A^{-1} = \begin{pmatrix}1&-1\\-1&2\end{pmatrix}$, exactly matching Practice 1.

Worked example

Find the inverse of $A=\begin{pmatrix}1&2&1\\3&7&8\\1&2&3\end{pmatrix}$ (the coefficient matrix from the Row Operations lesson's running example).

Augment: $\left(\begin{array}{ccc|ccc}1&2&1&1&0&0\\3&7&8&0&1&0\\1&2&3&0&0&1\end{array}\right)$.

$R_2\to R_2-3R_1$: $(0,1,5\mid-3,1,0)$. $\ R_3\to R_3-R_1$: $(0,0,2\mid-1,0,1)$.

$\left(\begin{array}{ccc|ccc}1&2&1&1&0&0\\0&1&5&-3&1&0\\0&0&2&-1&0&1\end{array}\right)$. Scale $R_3\to\frac12R_3$: $(0,0,1\mid-0.5,0,0.5)$.

Clear above the row-3 pivot: $R_2\to R_2-5R_3$: $(0,1,0\mid-0.5,1,-2.5)$. $\ R_1\to R_1-R_3$: $(1,2,0\mid1.5,0,-0.5)$.

$\left(\begin{array}{ccc|ccc}1&2&0&1.5&0&-0.5\\0&1&0&-0.5&1&-2.5\\0&0&1&-0.5&0&0.5\end{array}\right)$. Clear above the row-2 pivot: $R_1\to R_1-2R_2$: $(1,0,0\mid2.5,-2,4.5)$.

$$A^{-1} = \begin{pmatrix}2.5&-2&4.5\\-0.5&1&-2.5\\-0.5&0&0.5\end{pmatrix} = \begin{pmatrix}\frac52&-2&\frac92\\-\frac12&1&-\frac52\\-\frac12&0&\frac12\end{pmatrix}.$$

Spot-check one entry: row 1 of $A$ dotted with column 1 of $A^{-1}$ should give $1$: $1\left(\frac52\right)+2\left(-\frac12\right)+1\left(-\frac12\right) = \frac52-1-\frac12 = 1$ ✓.

$\left(\begin{array}{cc|cc}1&0&0.5&0\\1&2&0&1\end{array}\right)$. $\ R_2\to R_2-R_1$: $(0,2\mid-0.5,1)$. Scale $R_2\to\frac12R_2$: $(0,1\mid-0.25,0.5)$.

Practice 9

Find the inverse of the upper triangular matrix $\begin{pmatrix}1&1&1\\0&1&1\\0&0&1\end{pmatrix}$ by row reduction.

Solution

Clear above the row-3 pivot first: $R_2\to R_2-R_3$: $(0,1,0\mid0,1,-1)$. $\ R_1\to R_1-R_3$: $(1,1,0\mid1,0,-1)$.

Then clear above the row-2 pivot: $R_1\to R_1-R_2$: $(1,0,0\mid1,-1,0)$.

$$A^{-1} = \begin{pmatrix}1&-1&0\\0&1&-1\\0&0&1\end{pmatrix}.$$ Pivots were already all $1$ with nothing below to clear, so only the "clear above" passes were needed.

Practice 10

Try to find the inverse of $\begin{pmatrix}2&4\\1&2\end{pmatrix}$ by row reduction. What happens?

Solution

$\left(\begin{array}{cc|cc}2&4&1&0\\1&2&0&1\end{array}\right)$. $\ R_2\to R_2-\frac12R_1$: $(0,0\mid-0.5,1)$.

The left-hand block now has an entirely zero row, so it can never be reduced to $I$. This confirms $A$ is not invertible (matching Practice 4, and $\det(A)=0$).

$\left(\begin{array}{cc|cc}3&1&1&0\\2&1&0&1\end{array}\right)$. Scale $R_1\to\frac13R_1$: $\left(1,\frac13\mid\frac13,0\right)$.

$R_2\to R_2-2R_1$: $\left(0,\frac13\mid-\frac23,1\right)$. Scale $R_2\to3R_2$: $(0,1\mid-2,3)$.

Practice 12

Without recomputing it, use a property of determinants to state $\det(A^{-1})$ for $A=\begin{pmatrix}1&2&1\\3&7&8\\1&2&3\end{pmatrix}$ (the Section 2 worked example, where $\det(A)=2$).

Solution

Since $AA^{-1}=I$, and $\det(I)=1$, taking determinants of both sides gives $\det(A)\det(A^{-1})=1$, so $\det(A^{-1}) = \dfrac{1}{\det(A)} = \dfrac12$. No need to expand the $3\times3$ inverse matrix at all.

3 Cramer's Rule

Row reduction is the general-purpose tool, but if you only need one variable out of a small system, and you already have a formula for the determinant, there is a shortcut that skips elimination entirely: Cramer's Rule.

Key idea

For a square system $A\mathbf{x}=\mathbf{b}$ with $\det(A)\neq0$, let $A_i$ be the matrix formed by replacing column $i$ of $A$ with $\mathbf{b}$. Then each variable is:

$$x_i = \frac{\det(A_i)}{\det(A)}.$$

If $\det(A)=0$, Cramer's Rule cannot be used at all (it would divide by zero); the system then has either no solution or infinitely many, and you need row reduction to tell which, exactly as Section 4 covers.

Worked example

Solve $2x+y=5,\ x+3y=10$ using Cramer's Rule.

$A=\begin{pmatrix}2&1\\1&3\end{pmatrix}$, $\ \det(A) = 2(3)-1(1) = 5$.

$A_x = \begin{pmatrix}5&1\\10&3\end{pmatrix}$ (column 1 replaced by $\mathbf{b}$), $\ \det(A_x) = 5(3)-1(10) = 5$. So $x = \dfrac55 = 1$.

$A_y = \begin{pmatrix}2&5\\1&10\end{pmatrix}$ (column 2 replaced by $\mathbf{b}$), $\ \det(A_y) = 2(10)-5(1) = 15$. So $y = \dfrac{15}{5} = 3$.

Check: $2(1)+3=5$ ✓, $1+3(3)=10$ ✓. Solution $(x,y)=(1,3)$.

Worked example

Solve $2x+y-z=3,\ x-y+2z=8,\ 3x+y+z=10$ using Cramer's Rule (this is Practice 27 from the Row Operations lesson, solved there by elimination; the answer should match).

$A=\begin{pmatrix}2&1&-1\\1&-1&2\\3&1&1\end{pmatrix}$. Expand along row 1: $\det(A) = 2\begin{vmatrix}-1&2\\1&1\end{vmatrix} - 1\begin{vmatrix}1&2\\3&1\end{vmatrix} + (-1)\begin{vmatrix}1&-1\\3&1\end{vmatrix} = 2(-3)-1(-5)-1(4) = -6+5-4=-5.$

$A_x=\begin{pmatrix}3&1&-1\\8&-1&2\\10&1&1\end{pmatrix}$: $\det(A_x) = 3(-3)-1(-12)-1(18) = -9+12-18=-15$. So $x=\dfrac{-15}{-5}=3$.

$A_y=\begin{pmatrix}2&3&-1\\1&8&2\\3&10&1\end{pmatrix}$: $\det(A_y) = 2(-12)-3(-5)-1(-14) = -24+15+14=5$. So $y=\dfrac{5}{-5}=-1$.

$A_z=\begin{pmatrix}2&1&3\\1&-1&8\\3&1&10\end{pmatrix}$: $\det(A_z) = 2(-18)-1(-14)+3(4) = -36+14+12=-10$. So $z=\dfrac{-10}{-5}=2$.

Solution $(x,y,z)=(3,-1,2)$, exactly matching the elimination answer from the earlier lesson.

Practice 13

Solve $x+y=3,\ 2x+5y=12$ using Cramer's Rule.

Solution

$\det(A)=\begin{vmatrix}1&1\\2&5\end{vmatrix}=5-2=3$.

$\det(A_x)=\begin{vmatrix}3&1\\12&5\end{vmatrix}=15-12=3 \Rightarrow x=1$.

$\det(A_y)=\begin{vmatrix}1&3\\2&12\end{vmatrix}=12-6=6 \Rightarrow y=2$.

Solution: $(x,y)=(1,2)$.

Practice 14

Solve $3x+2y=16,\ x+y=7$ using Cramer's Rule.

$\det(A)=\begin{vmatrix}3&2\\1&1\end{vmatrix}=3-2=1$.

$\det(A_x)=\begin{vmatrix}16&2\\7&1\end{vmatrix}=16-14=2 \Rightarrow x=2$.

$\det(A_y)=\begin{vmatrix}3&16\\1&7\end{vmatrix}=21-16=5 \Rightarrow y=5$.

Solution: $(x,y)=(2,5)$.

Practice 15

Solve $4x-y=9,\ 2x+y=9$ using Cramer's Rule.

Solution

$\det(A)=\begin{vmatrix}4&-1\\2&1\end{vmatrix}=4+2=6$.

$\det(A_x)=\begin{vmatrix}9&-1\\9&1\end{vmatrix}=9+9=18 \Rightarrow x=3$.

$\det(A_y)=\begin{vmatrix}4&9\\2&9\end{vmatrix}=36-18=18 \Rightarrow y=3$.

Solution: $(x,y)=(3,3)$.

Practice 16

Solve $x+y+z=6,\ 2x+y-z=1,\ x-y+2z=5$ using Cramer's Rule.

Solution

$\det(A) = \begin{vmatrix}1&1&1\\2&1&-1\\1&-1&2\end{vmatrix} = 1(1)-1(5)+1(-3) = -7$.

$\det(A_x) = \begin{vmatrix}6&1&1\\1&1&-1\\5&-1&2\end{vmatrix} = 6(1)-1(7)+1(-6)=-7 \Rightarrow x=1$.

$\det(A_y) = \begin{vmatrix}1&6&1\\2&1&-1\\1&5&2\end{vmatrix} = 1(7)-6(5)+1(9)=-14 \Rightarrow y=2$.

$\det(A_z) = \begin{vmatrix}1&1&6\\2&1&1\\1&-1&5\end{vmatrix} = 1(6)-1(9)+6(-3)=-21 \Rightarrow z=3$.

Solution: $(x,y,z)=(1,2,3)$.

Practice 17

A $2\times2$ system has $\det(A)=0$. Can Cramer's Rule be used to solve it? What should you do instead?

Solution

No. Every variable's formula $x_i=\det(A_i)/\det(A)$ divides by $\det(A)$, so with $\det(A)=0$ the formula is undefined. This does not mean the system has no solution, it means Cramer's Rule simply cannot decide the case; the system could have no solution or infinitely many. Row-reduce the augmented matrix instead (Section 4 shows exactly how to tell the two cases apart).

Practice 18

Solve $2x+3y=7,\ x-y=1$ using Cramer's Rule.

Solution

$\det(A)=\begin{vmatrix}2&3\\1&-1\end{vmatrix}=-2-3=-5$.

$\det(A_x)=\begin{vmatrix}7&3\\1&-1\end{vmatrix}=-7-3=-10 \Rightarrow x=\dfrac{-10}{-5}=2$.

$\det(A_y)=\begin{vmatrix}2&7\\1&1\end{vmatrix}=2-7=-5 \Rightarrow y=\dfrac{-5}{-5}=1$.

Solution: $(x,y)=(2,1)$.

4 Classifying solutions: one, none, or infinitely many

The equivalence theorem from the Row Operations lesson answers a clean question: for a square matrix with $\det(A)\neq0$, $A\mathbf{x}=\mathbf{b}$ has exactly one solution, for every $\mathbf{b}$. But not every system you meet is square, and not every square system is full rank. Row-reducing the augmented matrix $(A\mid\mathbf{b})$ always settles the question, whatever the shape of $A$, and it settles it by producing exactly one of three possible outcomes.

Key idea

Row-reduce the augmented matrix $(A\mid\mathbf{b})$ to echelon form, then look at what came out:

Unique solution. Every variable's column has a pivot, and no row reads $0=c$ for nonzero $c$. Back-substitute as usual.

No solution (inconsistent). Some row reads $0 = c$ for a nonzero constant $c$, an outright contradiction. No values of the variables can satisfy this, regardless of anything else in the matrix.

Infinitely many solutions. No contradiction row appears, but at least one variable's column never gets a pivot, a free variable. Set that variable equal to a parameter (commonly $t$), and express the pivot variables in terms of it.

Worked example: infinitely many solutions

Solve $x+2y-z=3,\ 2x+4y-2z=6,\ x-y+z=0$.

Augmented matrix: $\left(\begin{array}{ccc|c}1&2&-1&3\\2&4&-2&6\\1&-1&1&0\end{array}\right)$.

$R_2\to R_2-2R_1$: $(0,0,0\mid0)$, entirely zero, row 2 was just twice row 1. $\ R_3\to R_3-R_1$: $(0,-3,2\mid-3)$.

Swap the zero row to the bottom: $\left(\begin{array}{ccc|c}1&2&-1&3\\0&-3&2&-3\\0&0&0&0\end{array}\right)$. Pivots sit in columns 1 and 2 ($x$ and $y$); column 3 ($z$) has no pivot, so $z$ is free.

Let $z=t$. Row 2: $-3y+2t=-3 \Rightarrow y = 1+\dfrac23t$. Row 1: $x+2y-t=3 \Rightarrow x = 3-2y+t = 3-2\left(1+\dfrac23t\right)+t = 1-\dfrac13t$.

Solution: $(x,y,z) = \left(1-\dfrac13t,\ 1+\dfrac23t,\ t\right)$ for any real $t$, a whole line of solutions. Check $t=3$: $(0,3,3)$; eq1: $0+6-3=3$ ✓, eq3: $0-3+3=0$ ✓, eq2: $0+12-6=6$ ✓.

Worked example: no solution

Solve $x+y+z=4,\ 2x+2y+2z=9,\ x-y+z=1$.

Augmented matrix: $\left(\begin{array}{ccc|c}1&1&1&4\\2&2&2&9\\1&-1&1&1\end{array}\right)$.

$R_2\to R_2-2R_1$: $(0,0,0\mid9-8) = (0,0,0\mid1)$.

This row reads $0=1$, a flat contradiction, no values of $x,y,z$ can make that true. No further row operations can fix it (row 3 does not even need to be touched): the system is inconsistent, with no solution.

This makes sense on inspection too: doubling the first equation gives $2x+2y+2z=8$, but the second equation demands $2x+2y+2z=9$, an outright contradiction before any elimination is even needed.

Practice 19

Classify the system $x+y=2,\ 2x+2y=4$. If it has infinitely many solutions, parametrise them.

Solution

The second equation is exactly twice the first, so $R_2\to R_2-2R_1$ gives $(0,0\mid0)$, no contradiction, but $y$'s column has no pivot. Infinitely many solutions. Let $y=t$: $x=2-t$. Solution: $(x,y)=(2-t,\,t)$ for any real $t$.

Classify the system $x+y=2,\ 2x+2y=5$.

Practice 21

Practice 22

Classify and, if applicable, parametrise the solutions of $x+2y-z=1,\ 2x+4y-2z=2,\ x+y+z=3$.

$\left(\begin{array}{ccc|c}1&2&-1&1\\2&4&-2&2\\1&1&1&3\end{array}\right)$. $\ R_2\to R_2-2R_1$: $(0,0,0\mid0)$. $\ R_3\to R_3-R_1$: $(0,-1,2\mid2)$.

Practice 23

Classify the system $2x-y=4,\ -4x+2y=-9$.

Solution

The left-hand sides are proportional ($-4x+2y = -2(2x-y)$), but $-2(4)=-8\neq-9$: a contradiction. No solution.
