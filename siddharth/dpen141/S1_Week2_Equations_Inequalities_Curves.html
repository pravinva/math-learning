0 Solving Equations

"Solving an equation" always means the same thing: isolate the unknown by doing the same operation to both sides, in a sequence that peels away everything surrounding it. The specific tactics differ by equation type, but the goal never changes.

Key idea — matching the tactic to the equation type

Linear: collect terms, isolate. Quadratic: factorise if possible, otherwise use $x=\dfrac{-b\pm\sqrt{b^2-4ac}}{2a}$. Rational (fractions): multiply through by the LCD, solve, then check for extraneous solutions. Involving surds: isolate the surd, square both sides, then check for extraneous solutions (squaring can introduce false roots).

Worked example

Solve $\sqrt{2x+3} = x$.

Square both sides: $2x+3=x^2 \ \Rightarrow\ x^2-2x-3=0 \ \Rightarrow\ (x-3)(x+1)=0 \ \Rightarrow\ x=3$ or $x=-1$.

Check $x=3$: $\sqrt{2(3)+3}=\sqrt9=3$. ✓ Check $x=-1$: $\sqrt{2(-1)+3}=\sqrt1=1 \ne -1$. ✗

Only $x=3$ is valid; $x=-1$ is extraneous, introduced by squaring.

Practice 1

Solve $\dfrac{2x-1}{3}+\dfrac{x+2}{4}=5$.

Solution

Multiply through by $12$: $4(2x-1)+3(x+2)=60 \Rightarrow 8x-4+3x+6=60 \Rightarrow 11x+2=60 \Rightarrow x=\dfrac{58}{11}$.

Practice 2

Solve $3x^2-12x+5=0$ using the quadratic formula, giving exact answers.

Solution

$x=\dfrac{12\pm\sqrt{144-60}}{6}=\dfrac{12\pm\sqrt{84}}{6}=\dfrac{12\pm2\sqrt{21}}{6}=2\pm\dfrac{\sqrt{21}}{3}$.

Practice 3

Solve $\sqrt{3x+1}=x-1$, checking for extraneous solutions.

Solution

Square: $3x+1=x^2-2x+1 \Rightarrow x^2-5x=0 \Rightarrow x(x-5)=0 \Rightarrow x=0$ or $x=5$. Check $x=0$: $\sqrt1=1$ but $x-1=-1$, doesn't match — extraneous. Check $x=5$: $\sqrt{16}=4=5-1$. ✓ Only $x=5$ is valid.

Practice 4

Solve $x^4-13x^2+36=0$.

Solution

Let $u=x^2$: $u^2-13u+36=0 \Rightarrow (u-4)(u-9)=0 \Rightarrow u=4$ or $u=9$. So $x^2=4 \Rightarrow x=\pm2$, or $x^2=9 \Rightarrow x=\pm3$. Four solutions: $x=\pm2,\pm3$.

Practice 5

Find all values of $x$ that satisfy $\dfrac{3x}{x^2-4}=\dfrac{2}{x^2-3x+2}$.

Solution

Factorise: $x^2-4=(x-2)(x+2)$, $\ x^2-3x+2=(x-1)(x-2)$. LCD $(x-2)(x+2)(x-1)$. Multiply through: $3x(x-1)=2(x+2) \Rightarrow 3x^2-3x=2x+4 \Rightarrow 3x^2-5x-4=0$. Quadratic formula: $x=\dfrac{5\pm\sqrt{25+48}}{6}=\dfrac{5\pm\sqrt{73}}{6}$. Neither value equals $2, -2,$ or $1$, so both are valid.

Practice 6

A projectile's height (m) is $h(t)=-5t^2+20t+2$. Find the times at which $h(t)=17$.

Solution

$-5t^2+20t+2=17 \Rightarrow -5t^2+20t-15=0 \Rightarrow t^2-4t+3=0 \Rightarrow (t-1)(t-3)=0 \Rightarrow t=1$ or $t=3$ seconds.

Practice 7

Solve $2x^{2/3}-5x^{1/3}-3=0$.

Solution

Let $u=x^{1/3}$: $2u^2-5u-3=0 \Rightarrow (2u+1)(u-3)=0 \Rightarrow u=-\tfrac12$ or $u=3$. So $x^{1/3}=-\tfrac12 \Rightarrow x=-\tfrac18$, or $x^{1/3}=3\Rightarrow x=27$.

1 Solving Quadratic and Rational Inequalities

Inequalities can't be "cross-multiplied" the way equations can, because multiplying by a negative number flips the inequality — and with an algebraic expression, you often don't know its sign in advance. The safe, general method is always the same: move everything to one side, factorise, then use a sign diagram to track the sign of the whole expression across every interval marked out by its critical values.

Key method

1) Rearrange to get $0$ on one side. 2) Factorise fully (for rational inequalities, do not cross-multiply an unknown-sign denominator — instead combine into a single fraction first). 3) Find every critical value (root of numerator or denominator). 4) Mark these on a number line and test the sign of each factor in each region. 5) Read off the intervals matching the inequality, remembering that any value making a denominator zero is always excluded.

Worked example

Solve $\dfrac{x-1}{x+3}\ge 2$.

Move everything to one side (never cross-multiply here, since $x+3$ could be negative): $\dfrac{x-1}{x+3}-2\ge0 \ \Rightarrow\ \dfrac{x-1-2(x+3)}{x+3}\ge0 \ \Rightarrow\ \dfrac{-x-7}{x+3}\ge0$.

Critical values: numerator zero at $x=-7$; denominator zero (excluded) at $x=-3$.

Sign diagram across $(-\infty,-7),\ (-7,-3),\ (-3,\infty)$: testing $x=-8$: $\dfrac{1}{-5}<0$. Testing $x=-5$: $\dfrac{-2}{-2}>0$. Testing $x=0$: $\dfrac{-7}{3}<0$.

Need $\ge0$: this holds on $(-7,-3)$, and at $x=-7$ itself (numerator zero, included). So $x\in[-7,-3)$, excluding $-3$ since the denominator can't be zero.

Practice 8

Solve $x^2-2x-8\le0$.

Solution

Factorise: $(x-4)(x+2)\le0$. Parabola opens up, so $\le0$ between the roots: $x\in[-2,4]$.

Practice 9

Solve $2x^2+5x-3>0$.

Solution

Factorise: $(2x-1)(x+3)>0$. Roots at $x=\tfrac12,-3$; parabola opens up, so positive outside the roots: $x\in(-\infty,-3)\cup(\tfrac12,\infty)$.

Practice 10

Solve $\dfrac{x+2}{x-5}\le0$.

Solution

Critical values $x=-2$ (numerator zero, included) and $x=5$ (denominator zero, excluded). Sign test: negative between them (e.g. $x=0$: $\dfrac{2}{-5}<0$). So $x\in[-2,5)$.

Practice 11

Solve $3-2x-x^2\ge0$.

Solution

Rewrite as $-(x^2+2x-3)\ge0 \Rightarrow x^2+2x-3\le0 \Rightarrow (x+3)(x-1)\le0$. Between the roots: $x\in[-3,1]$.

Practice 12

Solve $\dfrac{1}{x-2}<\dfrac{1}{x+1}$.

Solution

Move to one side: $\dfrac{1}{x-2}-\dfrac{1}{x+1}<0 \Rightarrow \dfrac{(x+1)-(x-2)}{(x-2)(x+1)}<0 \Rightarrow \dfrac{3}{(x-2)(x+1)}<0$. Since $3>0$ always, need $(x-2)(x+1)<0$, i.e. $x\in(-1,2)$.

Practice 13

A component fails if its temperature $T$ (°C) satisfies $T^2-90T+2000>0$. Find the temperature range for safe operation (where the inequality is false, i.e. $T^2-90T+2000\le0$).

Solution

Factorise: $T^2-90T+2000=(T-40)(T-50)$. $\le0$ between the roots: safe range is $T\in[40,50]$°C.

Practice 14

Solve $\dfrac{(x-1)(x+4)}{x-3}\ge0$.

Solution

Critical values $-4,1$ (numerator, included), $3$ (denominator, excluded). Testing each region: $x=-5$: $\dfrac{(-)(-)}{(-)}<0$. $x=0$: $\dfrac{(-)(+)}{(-)}>0$. $x=2$: $\dfrac{(+)(+)}{(-)}<0$. $x=4$: $\dfrac{(+)(+)}{(+)}>0$. So $x\in[-4,1]\cup(3,\infty)$.

2 Systems of Linear Equations (by hand)

Before Linear Algebra gives you row operations and matrices for systems of any size, you should be completely comfortable solving small systems the direct way: substitution or elimination. These are exactly the same underlying moves that row operations later generalise.

Key idea

Substitution: solve one equation for one variable, substitute into the other(s). Elimination: scale equations so a variable's coefficients match (or are opposite), then add or subtract to cancel it.

Worked example

Solve the system $3x-2y=4$ and $5x+3y=1$.

Eliminate $y$: multiply the first equation by $3$ and the second by $2$: $9x-6y=12$ and $10x+6y=2$.

Add: $19x=14 \Rightarrow x=\dfrac{14}{19}$.

Substitute into $3x-2y=4$: $\dfrac{42}{19}-2y=4 \Rightarrow 2y=\dfrac{42}{19}-4=\dfrac{42-76}{19}=-\dfrac{34}{19} \Rightarrow y=-\dfrac{17}{19}$.

Practice 15

Solve the system $x+y=7$, $\ 2x-y=2$.

Solution

Add the equations: $3x=9\Rightarrow x=3$, then $y=7-3=4$.

Practice 16

Solve the system $2x+3y=12$, $\ 4x-y=5$.

Solution

From the second, $y=4x-5$. Substitute: $2x+3(4x-5)=12 \Rightarrow 14x-15=12 \Rightarrow x=\tfrac{27}{14}$. Then $y=4(\tfrac{27}{14})-5=\tfrac{108}{14}-\tfrac{70}{14}=\tfrac{38}{14}=\tfrac{19}{7}$.

Practice 17

Solve the system $x-2y=-1$, $\ 3x+4y=17$.

Solution

From the first, $x=2y-1$. Substitute: $3(2y-1)+4y=17 \Rightarrow 10y-3=17 \Rightarrow y=2$, then $x=2(2)-1=3$.

Practice 18

Two currents in a circuit satisfy $I_1+I_2=10$ and $2I_1-3I_2=5$ (amps). Find $I_1$ and $I_2$.

Solution

From the first, $I_1=10-I_2$. Substitute: $2(10-I_2)-3I_2=5 \Rightarrow 20-5I_2=5 \Rightarrow I_2=3$, so $I_1=7$ A.

Practice 19

Solve the system $\dfrac{x}{2}+\dfrac{y}{3}=4$, $\ x-y=1$.

Solution

Clear fractions in the first: $3x+2y=24$. From the second, $x=y+1$. Substitute: $3(y+1)+2y=24 \Rightarrow 5y+3=24 \Rightarrow y=\tfrac{21}{5}$, so $x=\tfrac{21}{5}+1=\tfrac{26}{5}$.

Practice 20

Determine whether the system $2x-y=3$, $\ 4x-2y=7$ has a unique solution, no solution, or infinitely many. Justify your answer.

Solution

Scaling the first equation by $2$ gives $4x-2y=6$, which contradicts $4x-2y=7$ (same left side, different right side) — the two lines are parallel and distinct. No solution.

3 Curve Sketching: Lines, Circles, and Parabolas

Recognising a curve's type from its equation, and knowing where its key features (intercepts, vertex, centre, radius) live, is what makes sketching fast rather than a point-by-point slog. Completing the square is the tool that turns a messy equation into a recognisable standard form.

Standard forms

Line: $y=mx+c$ (gradient $m$, $y$-intercept $c$). Circle: $(x-h)^2+(y-k)^2=r^2$ (centre $(h,k)$, radius $r$). Parabola: $y=a(x-h)^2+k$ (vertex $(h,k)$, opens up if $a>0$, down if $a<0$).

Worked example

Find the centre and radius of the circle $x^2+y^2-6x+4y-3=0$, and sketch its key features.

Group and complete the square: $(x^2-6x)+(y^2+4y)=3$.

$(x^2-6x+9)+(y^2+4y+4)=3+9+4 \ \Rightarrow\ (x-3)^2+(y+2)^2=16$.

Centre $(3,-2)$, radius $\sqrt{16}=4$.

Practice 21

Sketch (describe key features of) $y=\dfrac14x^2-4$: state the vertex and $x$-intercepts.

Solution

Vertex $(0,-4)$ (upward parabola, shifted down $4$). $x$-intercepts: $\tfrac14x^2=4\Rightarrow x^2=16\Rightarrow x=\pm4$.

Practice 22

Find the centre and radius of $(x-3)^2+(y-4)^2=1$, and state whether the point $(4,4)$ lies on, inside, or outside this circle.

Solution

Centre $(3,4)$, radius $1$. Distance from centre to $(4,4)$ is $\sqrt{(4-3)^2+(4-4)^2}=1$, exactly the radius, so $(4,4)$ lies on the circle.

Practice 23

Find the centre and radius of $x^2+y^2+6x-8y=0$.

Solution

Complete the square: $(x^2+6x+9)+(y^2-8y+16)=25 \Rightarrow (x+3)^2+(y-4)^2=25$. Centre $(-3,4)$, radius $5$.

Practice 24

Find the vertex of $y=2x^2-12x+7$ by completing the square.

Solution

$y=2(x^2-6x)+7=2(x^2-6x+9-9)+7=2(x-3)^2-18+7=2(x-3)^2-11$. Vertex $(3,-11)$.

Practice 25

A circular satellite dish has cross-section $x^2+y^2-4x-2y-4=0$ (metres). Find its centre and radius.

Solution

$(x^2-4x+4)+(y^2-2y+1)=4+4+1 \Rightarrow (x-2)^2+(y-1)^2=9$. Centre $(2,1)$, radius $3$ m.

Practice 26

A line has gradient $-2$ and passes through $(1,5)$. Find its equation, and state its $y$-intercept.

Solution

$y-5=-2(x-1) \Rightarrow y=-2x+2+5=-2x+7$. $y$-intercept: $c=7$.

Practice 27

Describe how the graph of $y=(x-2)^2+3$ relates to $y=x^2$, and state its vertex.

Solution

Shifted $2$ units right and $3$ units up from $y=x^2$. Vertex $(2,3)$.

4 Intersections of Curves

Two curves intersect exactly where both equations are satisfied simultaneously — so finding intersections is just solving a system, except now one or both equations are nonlinear. The strategy is the same as Section 2: substitute one equation into the other to reduce to a single-variable equation.

Worked example

Find the points where the line $y=x+1$ meets the circle $x^2+y^2=25$.

Substitute: $x^2+(x+1)^2=25 \ \Rightarrow\ x^2+x^2+2x+1=25 \ \Rightarrow\ 2x^2+2x-24=0 \ \Rightarrow\ x^2+x-12=0$.

Factorise: $(x+4)(x-3)=0 \Rightarrow x=-4$ or $x=3$.

At $x=-4$: $y=-3$, point $(-4,-3)$. At $x=3$: $y=4$, point $(3,4)$.

Practice 28

Find the points of intersection of $y=2x$ and $y=x^2$.

Solution

$x^2=2x \Rightarrow x^2-2x=0 \Rightarrow x(x-2)=0 \Rightarrow x=0$ or $x=2$. Points: $(0,0)$ and $(2,4)$.

Practice 29

Find where the line $y=x-1$ meets the parabola $y=x^2-3x+1$.

Solution

$x-1=x^2-3x+1 \Rightarrow x^2-4x+2=0 \Rightarrow x=\dfrac{4\pm\sqrt{16-8}}{2}=2\pm\sqrt2$. Corresponding $y=x-1=1\pm\sqrt2$.

Practice 30

Find the points of intersection of $x^2+y^2=13$ and $y=x-1$.

Solution

$x^2+(x-1)^2=13 \Rightarrow 2x^2-2x-12=0 \Rightarrow x^2-x-6=0 \Rightarrow (x-3)(x+2)=0 \Rightarrow x=3$ or $x=-2$. Points: $(3,2)$ and $(-2,-3)$.

Practice 31

Determine whether the line $y=x+5$ intersects the circle $x^2+y^2=4$, and explain what your working reveals if it doesn't.

Solution

$x^2+(x+5)^2=4 \Rightarrow 2x^2+10x+21=0$. Discriminant: $100-4(2)(21)=100-168=-68<0$. No real solutions: the line does not intersect the circle (it passes entirely outside it).

Practice 32

Find the points of intersection of $y=x^2-4$ and $y=8-x^2$.

Solution

$x^2-4=8-x^2 \Rightarrow 2x^2=12 \Rightarrow x^2=6 \Rightarrow x=\pm\sqrt6$. Corresponding $y=8-6=2$. Points: $(\sqrt6,2)$ and $(-\sqrt6,2)$.

Practice 33

A cable modelled by $y=\tfrac{1}{10}x^2$ must clear a horizontal support beam at $y=4$. Find the $x$-range over which the cable is above the beam.

Solution

Need $\tfrac{1}{10}x^2>4 \Rightarrow x^2>40 \Rightarrow |x|>\sqrt{40}=2\sqrt{10}$. The cable clears the beam for $x<-2\sqrt{10}$ or $x>2\sqrt{10}$.

Practice 37

For parametric curve x=t^2-3t, y=t^3-2t, (a) find dy/dx, (b) find stationary points in y with respect to x, (c) tangent equations.

Solution

Compute dx/dt and dy/dt; then dy/dx=(dy/dt)/(dx/dt).

Stationary in y wrt x occurs when dy/dt=0 and dx/dt!=0.

Substitute t-values into (x,y) and slope for tangent equations.

Combine logs: ln((x-2)(x+6)) >= ln(36) => (x-2)(x+6)>=36.

Solve quadratic inequality with sign chart, then intersect with domain.

Practice 61

For parametric curve x=t^2-3t, y=t^3-2t, (a) find dy/dx, (b) find stationary points in y with respect to x, (c) tangent equations.

Solution

Compute dx/dt and dy/dt; then dy/dx=(dy/dt)/(dx/dt).

Stationary in y wrt x occurs when dy/dt=0 and dx/dt!=0.

Substitute t-values into (x,y) and slope for tangent equations.

Practice 62

Use substitution to evaluate integral from 0 to 3: x*sqrt(x^2+6) dx, then interpret geometric meaning of positive result.

Solution

Let u=x^2+6, du=2x dx.

Convert limits: x=0 -> u=6, x=3 -> u=15.

Integrate (1/2) integral u^(1/2) du and back-substitute/check units.

Practice 63

For f(x)=((x^2-4x+7)/(x-1)), (a) state domain, (b) simplify f(x) for x!= 1, (c) solve f(x)= 6.

Solution

(a) Domain excludes denominator zero: x != 1.

(b) Factor numerator where possible, then cancel only if a common factor with (x-1) exists; keep restriction x != 1.

(c) Set simplified expression equal to 6, solve resulting equation, then reject any root violating x != 1.

Practice 64

Solve exactly: 2sin(2x)-1=0 for 0<=x<2pi, then verify each solution on unit-circle quadrants.

Solution

Rewrite as sin(2x)=1/2.

Find principal solutions for 2x in [0,4pi): 2x=pi/6,5pi/6,13pi/6,17pi/6.

Divide by 2: x=pi/12,5pi/12,13pi/12,17pi/12 and confirm all are in [0,2pi).

Practice 65

Given z=6+9i, (a) convert to polar form, (b) find z^3 using De Moivre, (c) give argument in principal range.

Solution

(a) r=sqrt(36+81) and theta=atan2(9,6).

(b) z^3=r^3(cos(3theta)+i sin(3theta)).

(c) Reduce angle modulo 2pi to principal argument range.

Practice 66

Differentiate y=(x^2+3x)^(4) and evaluate y' at x=1; then classify stationary points of y' on x>0.

Solution

Use chain rule: y'=4(x^2+3x)^(3)(2x^1+3).

Substitute x=1 directly.

For stationary analysis of y, set y'=0 and inspect multiplicity/sign changes.
