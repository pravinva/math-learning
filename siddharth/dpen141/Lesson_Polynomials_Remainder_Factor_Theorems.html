0 Why this matters for eigenvalues

In eigenvalue work, you repeatedly build and solve equations like $\det(A-\lambda I)=0$. That determinant becomes a polynomial in $\lambda$. Everything in this lesson trains the exact algebra needed to handle that step with confidence.

Connection map

Remainder theorem: fast polynomial substitution and consistency checks.

Factor theorem: root testing and extraction of linear factors.

Division workflow: reduce degree after finding one factor.

Parameter problems: solve for unknown coefficients from constraints, exactly like some matrix-derived polynomial tasks.

1 Remainder theorem
Key idea

If $P(x)$ is divided by $(x-a)$, the remainder is $P(a)$.

Equivalent form: $P(x)=(x-a)Q(x)+R$ with constant remainder $R$, so substituting $x=a$ gives $R=P(a)$.

Worked example A

Find the remainder when $P(x)=2x^3-5x^2+x+7$ is divided by $(x-2)$.

$R=P(2)=2(8)-5(4)+2+7=16-20+2+7=5$.

Remainder is $\boxed{5}$.

Worked example B

Given $P(x)=x^3+ax+b$ with remainder $10$ on division by $(x-2)$ and remainder $-2$ on division by $(x+1)$, find $a,b$.

$P(2)=8+2a+b=10\Rightarrow 2a+b=2$.

$P(-1)=-1-a+b=-2\Rightarrow -a+b=-1$.

Solve: $3a=3\Rightarrow a=1$, then $b=0$.

2 Factor theorem
Key idea

$(x-a)$ is a factor of $P(x)$ if and only if $P(a)=0$.

Worked example C

Check whether $(x-3)$ is a factor of $P(x)=x^3-6x^2+11x-6$.

$P(3)=27-54+33-6=0$, so $(x-3)$ is a factor.

Worked example D

Find $k$ if $(x+2)$ is a factor of $P(x)=2x^3+kx^2-5x+4$.

Need $P(-2)=0$.

$2(-8)+k(4)-5(-2)+4=0\Rightarrow -16+4k+10+4=0\Rightarrow 4k-2=0$.

Hence $k=\dfrac12$.

3 From one factor to full factorization

Typical cubic workflow: test candidate roots $\pm$ factors of constant term (for monic cubics), confirm one zero by substitution, divide out, then solve the remaining quadratic.

Worked example E

Factorize $P(x)=x^3-4x^2-x+4$ completely over $\mathbb R$.

Test $x=1$: $1-4-1+4=0$, so $(x-1)$ is a factor.

Divide to get $x^2-3x-4$.

$x^2-3x-4=(x-4)(x+1)$.

So $P(x)=(x-1)(x-4)(x+1)$.

Worked example F

Factorize $P(x)=2x^3-x^2-8x+4$ completely over $\mathbb R$.

Test $x=2$: $16-4-16+4=0$, so $(x-2)$ is a factor.

Division gives $2x^2+3x-2=(2x-1)(x+2)$.

Hence $P(x)=(x-2)(2x-1)(x+2)$.

4 Parameter and mixed conditions
Common university pattern

Many tasks give you a polynomial with unknown coefficients and multiple conditions: specific remainders, known factors, or known roots. Translate each condition into an equation using $P(a)$, then solve the resulting linear system.

Worked example G

Find $a,b$ if $P(x)=x^3+ax^2+bx-6$ is divisible by $(x-1)$ and leaves remainder $8$ on division by $(x+2)$.

Divisible by $(x-1)$ gives $P(1)=0\Rightarrow 1+a+b-6=0\Rightarrow a+b=5$.

$P(-2)=8\Rightarrow -8+4a-2b-6=8\Rightarrow 2a-b=11$.

Solve: $a=\dfrac{16}{3}$, $b=-\dfrac{1}{3}$.

Practice 1

Find the remainder when $P(x)=3x^3-2x^2+5x-4$ is divided by $(x-1)$.

Solution

$R=P(1)=3-2+5-4=2$.

Practice 2

Find the remainder when $P(x)=2x^4-x^3+7x-9$ is divided by $(x+2)$.

Solution

$R=P(-2)=2(16)-(-8)+7(-2)-9=32+8-14-9=17$.

Practice 3

Find $k$ if the remainder on dividing $P(x)=x^3+kx+1$ by $(x-2)$ is $7$.

Solution

$P(2)=8+2k+1=7\Rightarrow 2k=-2\Rightarrow k=-1$.

Practice 4

If $P(x)=x^3+ax+b$, and $P(1)=4$, $P(-1)=0$, find $a,b$.

Solution

$1+a+b=4\Rightarrow a+b=3$. Also $-1-a+b=0\Rightarrow -a+b=1$. Solve: $b=2$, $a=1$.

Practice 5

Determine whether $(x-3)$ is a factor of $P(x)=x^3-7x+6$.

Solution

$P(3)=27-21+6=12\neq0$, so it is not a factor.

Practice 6

Find $m$ if $(x+1)$ is a factor of $P(x)=2x^3+mx^2-5x+2$.

Solution

Need $P(-1)=0$: $-2+m+5+2=0\Rightarrow m+5=0\Rightarrow m=-5$.

Practice 7

Show that $(x-2)$ is a factor of $x^3-5x^2+2x+8$ and find the quotient.

Solution

$P(2)=8-20+4+8=0$, so yes. Division gives quotient $x^2-3x-4$.

Practice 8

Factorize completely: $x^3-6x^2+11x-6$.

Solution

Test $x=1$: zero, so factor $(x-1)$. Quotient $x^2-5x+6=(x-2)(x-3)$. Final: $(x-1)(x-2)(x-3)$.

Practice 9

Factorize completely: $2x^3+x^2-8x-4$.

Solution

Test $x=2$: $16+4-16-4=0$, so $(x-2)$ is factor. Quotient $2x^2+5x+2=(2x+1)(x+2)$. Final: $(x-2)(2x+1)(x+2)$.

Practice 10

If $P(x)=x^3+ax^2+bx+c$ and $P(0)=3$, $P(1)=0$, $P(-1)=4$, write the three equations in $a,b,c$.

Solution

$P(0)=c=3$. $P(1)=1+a+b+c=0\Rightarrow a+b=-4$. $P(-1)=-1+a-b+c=4\Rightarrow a-b=2$.

Practice 11

Find the remainder when $P(x)=x^5-3x^2+2$ is divided by $(x-1)$.

Solution

$P(1)=1-3+2=0$.

Practice 12

Find the remainder when $P(x)=x^4+4x^3+6x^2+4x+1$ is divided by $(x+1)$.

Solution

$P(-1)=1-4+6-4+1=0$.

Practice 13

If $(x-4)$ is a factor of $P(x)=x^3+kx^2-5x+8$, find $k$.

Solution

$P(4)=64+16k-20+8=0\Rightarrow 16k+52=0\Rightarrow k=-\dfrac{13}{4}$.

Practice 14

If $(x+3)$ is a factor of $P(x)=2x^3+ax^2+bx+9$, write the condition relating $a,b$.

Solution

$P(-3)=2(-27)+9a-3b+9=0\Rightarrow -45+9a-3b=0\Rightarrow 3a-b=15$.

Practice 15

Factorize $x^3+x^2-4x-4$ completely.

Solution

Group: $x^2(x+1)-4(x+1)=(x+1)(x^2-4)=(x+1)(x-2)(x+2)$.

Practice 16

Factorize $x^3-3x^2-4x+12$ completely.

Solution

Group: $x^2(x-3)-4(x-3)=(x-3)(x^2-4)=(x-3)(x-2)(x+2)$.

Practice 17

Given $P(x)=x^3+px^2+qx+r$, and $P(1)=0$, $P(2)=0$, $P(3)=0$, what is $P(x)$ up to a constant factor?

Solution

Roots are $1,2,3$, so $P(x)=k(x-1)(x-2)(x-3)$. Since leading coefficient is $1$, $k=1$.

Practice 18

Find the quotient and remainder when $P(x)=x^3-7x-6$ is divided by $(x-3)$.

Solution

Remainder $=P(3)=27-21-6=0$, so divisible. Quotient from division is $x^2+3x+2$.

Practice 19

Find all real roots of $P(x)=x^3-2x^2-5x+6$.

Solution

Test $x=1$: $1-2-5+6=0$. Divide by $(x-1)$ to get $x^2-x-6=(x-3)(x+2)$. Roots: $1,3,-2$.

Practice 20

Find all real roots of $P(x)=2x^3-3x^2-8x+12$.

Solution

Test $x=2$: $16-12-16+12=0$. Divide to get $2x^2+x-6=(2x-3)(x+2)$. Roots: $2,\frac{3}{2},-2$.

Practice 21

If $P(x)=x^3+ax^2+bx+c$ has factor $(x-2)$ and leaves remainder $5$ on division by $(x+1)$, write two equations.

Solution

$P(2)=0\Rightarrow 8+4a+2b+c=0$. Also $P(-1)=5\Rightarrow -1+a-b+c=5$, so $a-b+c=6$.

Practice 22

Find $a,b$ if $P(x)=x^3+ax^2+bx+2$ is divisible by $(x-1)$ and $(x+2)$.

Solution

$P(1)=0\Rightarrow a+b+3=0\Rightarrow a+b=-3$. $P(-2)=0\Rightarrow -8+4a-2b+2=0\Rightarrow 2a-b=3$. Solve: $a=0$, $b=-3$.

Practice 23

Given $P(x)=x^4-5x^3+6x^2+ax+b$, with $P(3)=0$ and $P(-1)=16$, find $a,b$.

Solution

$P(3)=81-135+54+3a+b=0\Rightarrow 3a+b=0$. $P(-1)=1+5+6-a+b=16\Rightarrow -a+b=4$. Solve: $a=-1$, $b=3$.

Practice 24

Bridge check: if $\det(A-\lambda I)=\lambda^3-6\lambda^2+11\lambda-6$, find the eigenvalues.

Solution

Factor as $(\lambda-1)(\lambda-2)(\lambda-3)=0$. Eigenvalues are $1,2,3$.
