1 Summation ($\Sigma$) notation

Summation notation is just shorthand for writing out a long addition. $\displaystyle\sum_{k=1}^{n}a_k$ means "add up $a_k$ for every whole-number value of $k$ from $1$ to $n$." The letter $k$ is called the index of summation, the number below $\Sigma$ is where it starts, and the number above is where it stops (inclusive, both ends included).

The single most important thing to internalise early: the index letter is a dummy variable. It exists only inside the sum, to control which terms get added, and has no meaning or value outside it. Swapping $k$ for $i$, $a$, or any other letter changes nothing about the final number.

Key idea

$$\sum_{k=1}^{n}a_k = a_1+a_2+a_3+\cdots+a_n$$

Two things worth being deliberate about: order of operations ($\Sigma$ binds only to what's written immediately after it, so $\sum_{k=1}^na_k+c$ means $\left(\sum_{k=1}^na_k\right)+c$, not $\sum_{k=1}^n(a_k+c)$, unless the $+c$ is explicitly inside brackets), and counting terms correctly (a sum from $k=m$ to $k=n$ has exactly $n-m+1$ terms, not $n-m$, an easy off-by-one mistake).

Worked example

Evaluate $\displaystyle\sum_{k=1}^{4}k^2$.

Write out every term, substituting $k=1,2,3,4$ in turn:

$$\sum_{k=1}^{4}k^2 = 1^2+2^2+3^2+4^2 = 1+4+9+16 = 30.$$

Practice 1

Evaluate $\displaystyle\sum_{a=1}^{3}(2a+1)$.

Solution

Here the $+1$ is inside the brackets, so it's added inside every term: $(2(1)+1)+(2(2)+1)+(2(3)+1) = 3+5+7 = 15$.

Practice 2

Evaluate $\displaystyle\sum_{a=1}^{3}2a+1$ (no brackets this time), and compare carefully with Practice 1.

Solution

Without brackets, $\Sigma$ only applies to the term written directly after it, here that's $2a$. The $+1$ sits outside the sum entirely:

$$\sum_{a=1}^{3}2a+1 = \left(\sum_{a=1}^{3}2a\right)+1 = (2+4+6)+1 = 12+1 = 13.$$

Notice this is a genuinely different number from Practice 1's $15$, purely because of where the brackets were (or weren't). Reading $\Sigma$-notation carefully, term by term, is not optional, misplacing a bracket silently changes the answer.

Practice 3

Evaluate $\displaystyle\sum_{i=1}^{3}2i+1$, and explain why this must give exactly the same value as Practice 2.

Solution

$$\sum_{i=1}^{3}2i+1 = \left(\sum_{i=1}^{3}2i\right)+1 = (2+4+6)+1 = 13.$$

Same answer as Practice 2, because the only thing that changed is the label on the index, $i$ instead of $a$. The index is a dummy variable: it never appears in the final numeric answer, so relabelling it can never change that answer. $\sum_{a=1}^3 2a+1$ and $\sum_{i=1}^3 2i+1$ are two different ways of writing the exact same instruction, "add $2$ times each whole number from $1$ to $3$, then add $1$."

Practice 4

Evaluate $\displaystyle\sum_{k=1}^{6}k\sin\left(\dfrac{k\pi}2\right)$.

Solution

$\sin\left(\dfrac{k\pi}2\right)$ cycles through $1,0,-1,0$ as $k=1,2,3,4$, then repeats. Write out each term:

$k=1$: $1\cdot\sin\frac{\pi}2 = 1(1)=1$. $\ k=2$: $2\sin\pi = 2(0)=0$. $\ k=3$: $3\sin\frac{3\pi}2=3(-1)=-3$. $\ k=4$: $4\sin(2\pi)=4(0)=0$. $\ k=5$: $5\sin\frac{5\pi}2=5(1)=5$. $\ k=6$: $6\sin(3\pi)=6(0)=0$.

$$\sum_{k=1}^{6}k\sin\left(\frac{k\pi}2\right) = 1+0-3+0+5+0 = 3.$$

Practice 5

Evaluate $\displaystyle\sum_{k=0}^{6}1$.

Solution

Every single term equals $1$ (the summand doesn't even depend on $k$), so this sum just counts how many values of $k$ there are, from $0$ up to and including $6$: that's $0,1,2,3,4,5,6$, seven values.

$$\sum_{k=0}^{6}1 = 7.$$

This is the classic off-by-one trap: it's tempting to say "$6$" since the upper limit is $6$, but the count is upper $-$ lower $+1 = 6-0+1=7$, always add the $1$.

Practice 6

Evaluate $\displaystyle\sum_{k=1}^{15}\left(\dfrac1k-\dfrac1{k+1}\right)$.

Solution

This telescopes: writing out a few terms, $\left(1-\dfrac12\right)+\left(\dfrac12-\dfrac13\right)+\left(\dfrac13-\dfrac14\right)+\cdots+\left(\dfrac1{15}-\dfrac1{16}\right)$, every interior piece cancels against its neighbour, leaving only the very first and very last pieces:

$$\sum_{k=1}^{15}\left(\frac1k-\frac1{k+1}\right) = 1-\frac1{16} = \frac{15}{16}.$$

Practice 7

Explain, in your own words, why the index of summation is called a "dummy" variable, and why this matters when you see the same sum written with two different index letters.

Solution

The index only exists to control the mechanical process of substitution and addition inside the sum, it takes on each whole-number value in turn, contributes a term, and then the sum is done. Once the addition is complete, the index has no leftover meaning, the final number depends only on the summand's formula and the start/end values, never on what the index was called along the way. This matters practically because it means $\sum_{k}$, $\sum_i$, and $\sum_a$ are completely interchangeable notations for the identical calculation (Practice 2 and 3 confirmed this numerically), so if two sums look different only because of their index letter, they are not actually different sums at all.

2 Properties of summation

Because $\Sigma$-notation is just repeated addition, it inherits ordinary algebra's rules for addition and constants directly: sums can be split apart, constants can be pulled out front, and a constant added the same number of times is just multiplication.

Key idea

Splitting: $\displaystyle\sum_{k=1}^{n}(a_k\pm b_k) = \sum_{k=1}^{n}a_k \pm \sum_{k=1}^{n}b_k$.

Constant factor: $\displaystyle\sum_{k=1}^{n}ca_k = c\sum_{k=1}^{n}a_k$.

Constant term: $\displaystyle\sum_{k=1}^{n}c = nc$ (adding the same constant $c$, once for each of the $n$ values of $k$, is just $n$ copies of $c$, i.e.\ $nc$).

Why the constant-term rule needs care

The formula $\sum_{k=1}^nc=nc$ assumes the sum runs from $k=1$, giving exactly $n$ terms. If a sum instead runs from $k=0$ to $n$, there are $n+1$ terms (Practice 5 hit exactly this), and the correct total is $(n+1)c$, not $nc$. There is no shortcut here beyond counting the terms correctly first, upper limit minus lower limit plus one, before multiplying by the constant.

Worked example

Given $\displaystyle\sum_{k=1}^{n}a_k=8$ and $\displaystyle\sum_{k=1}^{n}b_k=3$, find $\displaystyle\sum_{k=1}^{n}(4a_k-b_k)$.

Split using the key idea's rules: $\displaystyle\sum_{k=1}^n(4a_k-b_k) = 4\sum_{k=1}^na_k - \sum_{k=1}^nb_k$.

$$= 4(8)-3 = 29.$$

Notice this required no knowledge of what the individual $a_k$ or $b_k$ actually are, only their totals, exactly the kind of shortcut these properties are built for.

Practice 8

Evaluate $\displaystyle\sum_{k=3}^{7}5$ (a constant sum, but not starting at $k=1$, so count the terms carefully first).

Solution

Number of terms: $7-3+1=5$. Each term equals $5$.

$$\sum_{k=3}^{7}5 = 5\times5 = 25.$$

Practice 9

Evaluate $\displaystyle\sum_{i=1}^{10}7$.

Solution

$10$ terms, each equal to $7$: $\displaystyle\sum_{i=1}^{10}7 = 10\times7=70$.

Practice 10

Given $\displaystyle\sum_{k=1}^{n}a_k=12$ and $\displaystyle\sum_{k=1}^{n}b_k=5$, find $\displaystyle\sum_{k=1}^{n}(3a_k-2b_k)$.

Solution

$$\sum_{k=1}^{n}(3a_k-2b_k) = 3\sum_{k=1}^na_k - 2\sum_{k=1}^nb_k = 3(12)-2(5) = 36-10=26.$$

Practice 11

Explain the mistake in this reasoning: "$\displaystyle\sum_{k=0}^{4}1$ has upper limit $4$, so the answer is $4$." What's the correct value, and why?

Solution

The mistake is treating the upper limit alone as the term count, ignoring that $k=0$ is itself a valid starting value that contributes its own term. The values of $k$ actually used are $0,1,2,3,4$, five values in total, not four. So $\displaystyle\sum_{k=0}^{4}1 = 5\times1=5$, not $4$. The correct rule, upper limit minus lower limit plus one, correctly accounts for the fact that both endpoints are included.

Practice 12

Using the splitting and constant-factor properties, rewrite $\displaystyle\sum_{k=1}^{n}\big(5a_k-2b_k+3\big)$ as a combination of $\sum a_k$, $\sum b_k$, and $n$ alone (don't evaluate, just restructure it using the key idea's rules).

Solution

Split the sum into three separate pieces, then pull out constant factors:

$$\sum_{k=1}^{n}\big(5a_k-2b_k+3\big) = 5\sum_{k=1}^{n}a_k - 2\sum_{k=1}^{n}b_k + \sum_{k=1}^{n}3 = 5\sum_{k=1}^{n}a_k - 2\sum_{k=1}^{n}b_k + 3n.$$

This kind of restructuring is exactly what makes Section 3's formulas usable on more complicated summands, break the messy sum into simple pieces first, then handle each piece separately.

3 Standard summation formulas

Three specific sums come up so often that it's worth having closed-form formulas for them memorised, rather than adding up terms by hand every time. Combined with Section 2's splitting property, these three formulas let you evaluate the sum of any polynomial in $k$, no matter how complicated, without writing out a single term.

Key idea

$$\sum_{k=1}^{n}k = \frac{n(n+1)}2, \qquad \sum_{k=1}^{n}k^2 = \frac{n(n+1)(2n+1)}6, \qquad \sum_{k=1}^{n}k^3 = \left[\frac{n(n+1)}2\right]^2.$$

These only apply exactly as written when the sum starts at $k=1$. If a summand is a more complicated polynomial in $k$ (like $(k+2)^3$), expand it first into separate powers of $k$, then apply Section 2's splitting property before reaching for these formulas.

Worked example

Given $\displaystyle\sum_{k=1}^{n}k=\dfrac{n(n+1)}2$ and $\displaystyle\sum_{k=1}^{n}k^3=\left[\dfrac{n(n+1)}2\right]^2$, evaluate $\displaystyle\sum_{k=1}^{5}(3k^3-2k+4)$.

Split using Section 2's properties: $\displaystyle\sum_{k=1}^{5}(3k^3-2k+4) = 3\sum_{k=1}^{5}k^3 - 2\sum_{k=1}^{5}k + \sum_{k=1}^{5}4$.

With $n=5$: $\displaystyle\sum_{k=1}^5k^3 = \left[\frac{5(6)}2\right]^2 = 15^2=225$. $\ \displaystyle\sum_{k=1}^5k = \frac{5(6)}2=15$. $\ \displaystyle\sum_{k=1}^54=5(4)=20$.

$$3(225)-2(15)+20 = 675-30+20 = 665.$$

Practice 13

Evaluate $\displaystyle\sum_{k=1}^{5}k^3$ directly from the formula, as a warm-up check.

Solution

$$\sum_{k=1}^{5}k^3 = \left[\frac{5(6)}2\right]^2 = 15^2 = 225.$$

Solution

Split: $\displaystyle\sum_{k=1}^{6}(4k^3-2k+1) = 4\sum_{k=1}^6k^3 - 2\sum_{k=1}^6k + \sum_{k=1}^61$.

With $n=6$: $\displaystyle\sum_{k=1}^6k^3=\left[\frac{6(7)}2\right]^2=21^2=441$. $\ \displaystyle\sum_{k=1}^6k=\frac{6(7)}2=21$. $\ \displaystyle\sum_{k=1}^61=6$.

$$4(441)-2(21)+6 = 1764-42+6 = 1728.$$

The formulas only apply to $k$, $k^2$, $k^3$ individually, not to $(k+2)^3$ directly, so expand the binomial first:

$$(k+2)^3 = k^3+6k^2+12k+8.$$

Now split and apply all three formulas with $n=10$:

$\displaystyle\sum_{k=1}^{10}k^3 = \left[\frac{10(11)}2\right]^2 = 55^2=3025$.

$\displaystyle\sum_{k=1}^{10}k^2 = \frac{10(11)(21)}6 = 385$.

$\displaystyle\sum_{k=1}^{10}k = \frac{10(11)}2=55$.

$\displaystyle\sum_{k=1}^{10}8 = 10(8)=80$.

With $n=4$: $\displaystyle\sum_{k=1}^4k^2 = \frac{4(5)(9)}6=30$. $\ \displaystyle\sum_{k=1}^4k=\frac{4(5)}2=10$. $\ \displaystyle\sum_{k=1}^41=4$.

Practice 17

Find a simplified formula, in terms of $n$, for $\displaystyle\sum_{k=1}^{n}(6k^2+4k)$.

Solution

$$\sum_{k=1}^{n}(6k^2+4k) = 6\cdot\frac{n(n+1)(2n+1)}6 + 4\cdot\frac{n(n+1)}2 = n(n+1)(2n+1) + 2n(n+1).$$

The sum of the first $n$ even numbers is $\displaystyle\sum_{k=1}^{n}2k = 2\sum_{k=1}^{n}k = 2\cdot\frac{n(n+1)}2 = n(n+1)$.

4 Double summation notation

A double sum, $\displaystyle\sum_{i}\sum_{j}a_{ij}$, adds up a two-index quantity $a_{ij}$ over every combination of $i$ and $j$ in their respective ranges. It's read from the inside out: for each fixed value of the outer index, evaluate the entire inner sum first (treating the outer index as a constant while doing so), then add up those results as the outer index runs through its own range. This is exactly the structure you'll need for summing over the rows and columns of a matrix.

Key idea

$$\sum_{i=1}^{m}\sum_{j=1}^{n}a_{ij} = \sum_{i=1}^{m}\left(\sum_{j=1}^{n}a_{ij}\right) = \underbrace{(a_{11}+a_{12}+\cdots+a_{1n})}_{i=1} + \underbrace{(a_{21}+a_{22}+\cdots+a_{2n})}_{i=2} + \cdots$$

For a finite double sum like this, the two $\Sigma$'s can always be swapped without changing the total, $\sum_i\sum_j a_{ij} = \sum_j\sum_i a_{ij}$, every term $a_{ij}$ gets added exactly once either way, only the order in which they're grouped differs.

A double sum separates into a product of two ordinary single sums exactly when the summand factors as $a_{ij}=f(i)g(j)$: $\displaystyle\sum_i\sum_j f(i)g(j) = \left(\sum_if(i)\right)\left(\sum_jg(j)\right)$.

Worked example

Evaluate $\displaystyle\sum_{i=1}^{2}\sum_{j=1}^{3}(i+j)$, and verify the order can be swapped without changing the answer.

Inner sum first, for each fixed $i$: at $i=1$, $\displaystyle\sum_{j=1}^{3}(1+j) = (1+1)+(1+2)+(1+3)=2+3+4=9$. At $i=2$, $\displaystyle\sum_{j=1}^3(2+j)=(2+1)+(2+2)+(2+3)=3+4+5=12$.

Outer sum: $9+12=21$.

Now swap the order, summing over $i$ first for each fixed $j$: at $j=1$, $\displaystyle\sum_{i=1}^2(i+1)=2+3=5$. At $j=2$: $3+4=7$. At $j=3$: $4+5=9$.

Outer sum: $5+7+9=21$.

Both orders give $21$, confirming the swap rule from the key idea.

Practice 19

Evaluate $\displaystyle\sum_{i=1}^{2}\sum_{j=1}^{2}ij$, using the fact that the summand $ij$ factors as $f(i)g(j)=i\cdot j$.

Solution

Since the summand separates: $\displaystyle\sum_{i=1}^2\sum_{j=1}^2 ij = \left(\sum_{i=1}^2i\right)\left(\sum_{j=1}^2j\right) = (1+2)(1+2) = 3\times3=9$.

Practice 20

Evaluate $\displaystyle\sum_{i=1}^{3}\sum_{j=1}^{2}(2i-j)$.

Solution

Inner sum for each $i$: $\displaystyle\sum_{j=1}^{2}(2i-j) = (2i-1)+(2i-2) = 4i-3$.

Outer sum: $\displaystyle\sum_{i=1}^{3}(4i-3) = (4-3)+(8-3)+(12-3) = 1+5+9=15$.

Practice 21

Evaluate $\displaystyle\sum_{i=1}^{3}\sum_{j=1}^{3}1$.

Solution

Every combination of $i\in\{1,2,3\}$ and $j\in\{1,2,3\}$ contributes exactly $1$, and there are $3\times3=9$ such combinations. $$\sum_{i=1}^3\sum_{j=1}^31 = 9.$$

This is the double-sum analogue of Practice 5's counting trap, a double sum of the constant $1$ simply counts the total number of $(i,j)$ pairs.

Practice 22

Given $x_1=2,\ x_2=5,\ y_1=3,\ y_2=-2,\ y_3=4$, evaluate $\displaystyle\sum_{i=1}^{2}\sum_{j=1}^{3}x_iy_j$.

Solution

The summand $x_iy_j$ separates (it's a function of $i$ times a function of $j$), so:

$$\sum_{i=1}^{2}\sum_{j=1}^{3}x_iy_j = \left(\sum_{i=1}^{2}x_i\right)\left(\sum_{j=1}^{3}y_j\right) = (2+5)(3-2+4) = 7\times5 = 35.$$

Practice 23

Explain precisely when a double sum $\displaystyle\sum_i\sum_ja_{ij}$ can be separated into a product of two single sums, and give an example of a summand $a_{ij}$ that cannot be separated this way.

Solution

Separation requires the summand to factor cleanly into a piece depending only on $i$ times a piece depending only on $j$, i.e.\ $a_{ij}=f(i)g(j)$, with no mixing of the two indices inside a single additive term. The worked example's $a_{ij}=i+j$ is a clear example that does not separate: it's a sum of an $i$-piece and a $j$-piece, not a product, and indeed $\left(\sum_i i\right)+\left(\sum_j j\right)$ would give a completely different (and wrong) number than the correctly computed $21$. As a rule of thumb, sums of terms that mix both indices additively (like $i+j$, or $i^2+j$) generally don't separate, while products of single-index pieces (like $ij$, or $i^2j^3$) always do.

5 The Kronecker delta

The Kronecker delta $\delta_{ij}$ is a two-index quantity that is remarkably simple to define, but turns out to be one of the most useful pieces of notation in all of linear algebra, precisely because of how it behaves inside sums (Sections 6 and 7).

Key idea

$$\delta_{ij} = \begin{cases}1 & \text{if } i=j \\ 0 & \text{if } i\neq j\end{cases}$$

Immediate consequences worth noting: $\delta_{ij}=\delta_{ji}$ (it's symmetric, since $i=j$ and $j=i$ are the same condition), and $\delta_{ii}=1$ for every single value of $i$ (an index matched against itself always agrees).

Worked example

Evaluate $\delta_{23}$, $\delta_{55}$, and $\displaystyle\sum_{i=1}^{4}\delta_{ii}$.

$\delta_{23}$: indices $2$ and $3$ don't match, so $\delta_{23}=0$.

$\delta_{55}$: indices match ($5=5$), so $\delta_{55}=1$.

$\displaystyle\sum_{i=1}^{4}\delta_{ii}$: every single term in this sum has matching indices by construction ($\delta_{11},\delta_{22},\delta_{33},\delta_{44}$), so every term equals $1$, four terms total:

$$\sum_{i=1}^{4}\delta_{ii} = 1+1+1+1 = 4.$$

Practice 24

Evaluate $\delta_{12}$ and $\delta_{44}$.

Solution

$\delta_{12}=0$ (indices don't match). $\ \delta_{44}=1$ (indices match).

Practice 25

Evaluate $\displaystyle\sum_{i=1}^{5}\delta_{ii}$.

Solution

Five matching-index terms, each equal to $1$: $\displaystyle\sum_{i=1}^5\delta_{ii} = 5$.

Practice 26

Evaluate $\displaystyle\sum_{i=1}^{4}\delta_{i3}$.

Solution

As $i$ runs through $1,2,3,4$, only $i=3$ makes $\delta_{i3}$ equal to $1$ (matching the fixed second index $3$); every other term is $0$.

$$\sum_{i=1}^{4}\delta_{i3} = 0+0+1+0 = 1.$$

This single-nonzero-term pattern, most of a sum vanishing except for one surviving term, is the entire idea behind Section 6's sifting property, worth noticing here first in its simplest possible form.

Practice 27

Is $\delta_{ij}=\delta_{ji}$ always true? Justify your answer directly from the definition.

Solution

Yes, always. $\delta_{ij}=1$ exactly when $i=j$, and $\delta_{ji}=1$ exactly when $j=i$, but "$i=j$" and "$j=i$" are the identical condition (equality doesn't care about the order it's written in). So $\delta_{ij}$ and $\delta_{ji}$ are $1$ under exactly the same circumstances, and $0$ under exactly the same circumstances too, meaning they're always equal. This symmetry is what made Practice 6.2 (evaluating $\delta_{3k}$ instead of $\delta_{k3}$) give the identical answer later.

Practice 28

Write out the entries of the $3\times3$ identity matrix $I$ using $I_{ij}=\delta_{ij}$ notation, by evaluating $\delta_{ij}$ for every combination of $i,j\in\{1,2,3\}$.

Solution

$I_{11}=\delta_{11}=1,\ I_{12}=\delta_{12}=0,\ I_{13}=\delta_{13}=0$, and similarly for each row, every diagonal entry ($i=j$) is $1$, every off-diagonal entry ($i\neq j$) is $0$:

$$I = \begin{pmatrix}1&0&0\\0&1&0\\0&0&1\end{pmatrix}$$

This is precisely why $I_{ij}=\delta_{ij}$ is the standard way to write the identity matrix in index notation, the Kronecker delta's definition is, word for word, the defining property of an identity matrix's entries.

Practice 29

Evaluate $\displaystyle\sum_{i=1}^{3}\sum_{j=1}^{3}\delta_{ij}$.

Solution

Referring back to Practice 28's identity matrix: this double sum simply adds up every entry of a $3\times3$ identity matrix. Only the $3$ diagonal entries are $1$; the $6$ off-diagonal entries are all $0$.

$$\sum_{i=1}^3\sum_{j=1}^3\delta_{ij} = 3.$$

In general, $\displaystyle\sum_{i=1}^n\sum_{j=1}^n\delta_{ij}=n$, this is exactly the trace of the $n\times n$ identity matrix, revisited properly in Section 8.

6 The delta's sifting property

The single most important thing the Kronecker delta does, and the reason it's worth learning at all rather than just being a curiosity, is what happens when it appears inside a sum alongside another indexed quantity. It sifts, or picks out, exactly one term from the entire sum, and silently deletes every other term.

Key idea: the sifting property

$$\sum_{i}\delta_{ij}\,x_i = x_j$$

(provided $j$ falls within the range being summed over). In words: summing $\delta_{ij}x_i$ over $i$ replaces $i$ with $j$ everywhere else in the summand, then removes the sum entirely, since only the $i=j$ term survives.

Why this works

Every term in $\sum_i\delta_{ij}x_i$ has the factor $\delta_{ij}$ attached, and by definition $\delta_{ij}=0$ for every value of $i$ except $i=j$. So every term in the sum is forced to be $0$, except the single term where $i=j$, which is $\delta_{jj}\,x_j = 1\cdot x_j = x_j$. Adding up "$x_j$, plus a pile of zeros" just gives $x_j$ back. The delta hasn't done any real arithmetic here, it's acted purely as a substitution device, replacing the index $i$ with $j$ and collapsing an entire sum down to one surviving term.

Worked example

Evaluate $\displaystyle\sum_{k=1}^{3}\delta_{k2}\,x_k$.

By the sifting property, this replaces $k$ with $2$ in $x_k$, and the sum vanishes:

$$\sum_{k=1}^{3}\delta_{k2}\,x_k = x_2.$$

Written out as a check: at $k=1$, $\delta_{12}x_1=0$. At $k=2$, $\delta_{22}x_2=x_2$. At $k=3$, $\delta_{32}x_3=0$. Total: $x_2$, matching.

Solution

Replace $i$ with $2$ in $i^2$: $\displaystyle\sum_{i=1}^{3}\delta_{i2}i^2 = 2^2=4$.

Practice 35

Explain, without using the word "sift," what $\displaystyle\sum_i\delta_{ij}x_i$ does to a list of numbers $x_1,x_2,\dots,x_n$, thought of as the components of a vector $x$.

Solution

It extracts a single component from the vector, specifically the $j$th one, and discards the rest entirely. Every entry of the vector gets multiplied by either $0$ (deleting it from the total) or $1$ (keeping it exactly as-is), and the delta's second index $j$ determines which single entry survives that process. In effect, $\sum_i\delta_{ij}x_i$ is a formula for "go and read off $x_j$," expressed using sum notation instead of ordinary indexing brackets.

7 Delta contraction in double sums

The sifting property from Section 6 becomes even more useful once two Kronecker deltas, or a delta alongside a genuine double sum, are involved. The core technique never changes: work through the sums one at a time, from the inside out, and let each delta's sifting property collapse one layer at a time.

Key idea

To evaluate $\displaystyle\sum_i\sum_j(\text{stuff involving }\delta_{ij}\text{, or }\delta\text{'s of other index pairs})$: pick whichever index has a delta pinning it directly to a fixed number first, substitute that value in everywhere, and repeat with the remaining sum. A particularly important special case, two deltas sharing one common index, contracts down to a single new delta:

$$\sum_{j}\delta_{ij}\delta_{jk} = \delta_{ik}.$$

Worked example

Evaluate $\displaystyle\sum_{i=1}^{3}\sum_{j=1}^{3}\delta_{ij}\,a_ib_j$.

Work the inner sum first, over $j$, treating $i$ as fixed: $\displaystyle\sum_{j=1}^{3}\delta_{ij}a_ib_j$. Since $a_i$ doesn't depend on $j$, pull it out: $a_i\displaystyle\sum_{j=1}^{3}\delta_{ij}b_j$. By the sifting property (Section 6), this inner sum collapses to $a_i\,b_i$.

Now the outer sum, over $i$, is just an ordinary single sum: $\displaystyle\sum_{i=1}^{3}a_ib_i$.

$$\sum_{i=1}^{3}\sum_{j=1}^{3}\delta_{ij}a_ib_j = \sum_{i=1}^{3}a_ib_i.$$

This particular identity is worth remembering on its own: a double sum with a single $\delta_{ij}$ ties the two indices together, turning what looked like a double sum into an ordinary single sum, exactly the dot-product formula revisited in Section 8.

Practice 36 TUTORIAL

Evaluate $\displaystyle\sum_{i=1}^{4}\sum_{j=1}^{4}\delta_{ij}\,\delta_{j3}\,x_i\,x_3$ (treat $x_3$ as a fixed constant throughout, not itself indexed by $i$ or $j$).

Solution

Work the inner sum over $j$ first, for fixed $i$, pulling out everything that doesn't depend on $j$ ($x_i$ and $x_3$ are both constants here):

$$x_i\,x_3\sum_{j=1}^{4}\delta_{ij}\delta_{j3}.$$

By the sifting property, $\displaystyle\sum_j\delta_{ij}\delta_{j3}$ replaces $j$ with $3$ inside $\delta_{ij}$, giving $\delta_{i3}$.

So the whole expression reduces to $\displaystyle\sum_{i=1}^{4}x_i\,x_3\,\delta_{i3}$. Now sift over $i$: replace $i$ with $3$ in $x_i$.

$$\sum_{i=1}^{4}\sum_{j=1}^{4}\delta_{ij}\delta_{j3}x_ix_3 = x_3\cdot x_3 = x_3^2.$$

Practice 37 TUTORIAL

Evaluate $\displaystyle\sum_{k=1}^{4}\sum_{m=1}^{2}\delta_{km}\,\delta_{k3}$ (notice the two sums have different ranges, $k$ up to $4$ but $m$ only up to $2$).

Solution

Work the inner sum over $m$ first, for fixed $k$: $\delta_{k3}$ doesn't depend on $m$, pull it out: $\delta_{k3}\displaystyle\sum_{m=1}^{2}\delta_{km}$.

Since $m$ only ranges over $\{1,2\}$, the sum $\displaystyle\sum_{m=1}^{2}\delta_{km}$ equals $1$ if $k\in\{1,2\}$ (a matching $m$ exists in range), and equals $0$ if $k=3$ or $k=4$ (no matching $m$ falls inside the range $1$ to $2$).

So the full summand is $\delta_{k3}\times\big[1\text{ if }k\in\{1,2\},\text{ else }0\big]$. But $\delta_{k3}$ is only nonzero at $k=3$, and at $k=3$ the bracket is $0$ (since $3\notin\{1,2\}$). The two conditions can never both hold at once.

$$\sum_{k=1}^{4}\sum_{m=1}^{2}\delta_{km}\delta_{k3} = 0.$$

A useful lesson: whenever two sums run over mismatched ranges, always check whether the target index a delta is trying to "sift toward" actually falls inside every relevant range, not just one of them.

Practice 38 TUTORIAL

Evaluate $\displaystyle\sum_{j=1}^{4}\sum_{i=1}^{4}\delta_{i3}\,\delta_{ij}$.

Solution

Work the inner sum over $i$ first (for fixed $j$): $\displaystyle\sum_{i=1}^{4}\delta_{i3}\delta_{ij}$. By the sifting property, replace $i$ with $3$ inside $\delta_{ij}$ (since $\delta_{i3}$ forces $i=3$, and $3$ falls within the range $1$ to $4$):

$$\sum_{i=1}^{4}\delta_{i3}\delta_{ij} = \delta_{3j}.$$

Now the outer sum over $j$: $\displaystyle\sum_{j=1}^{4}\delta_{3j}$. This equals $1$ exactly once, at $j=3$ (which does fall within the range $1$ to $4$).

$$\sum_{j=1}^{4}\sum_{i=1}^{4}\delta_{i3}\delta_{ij} = 1.$$

Practice 39 TUTORIAL

Evaluate $\displaystyle\sum_{j=1}^{3}\sum_{i=1}^{3}(3i+j)\,\delta_{ij}$.

Solution

Work the inner sum over $i$ first, for fixed $j$: by the sifting property, $\displaystyle\sum_{i=1}^{3}(3i+j)\delta_{ij}$ replaces $i$ with $j$ throughout the expression $(3i+j)$:

$$\sum_{i=1}^{3}(3i+j)\delta_{ij} = 3j+j = 4j.$$

Now the outer sum over $j$: $\displaystyle\sum_{j=1}^{3}4j = 4(1+2+3) = 4(6) = 24.$

Practice 40

Using $n=3$, verify the general identity $\displaystyle\sum_{j}\delta_{ij}\delta_{jk}=\delta_{ik}$ from the key idea by evaluating both sides for the specific case $i=1,\ k=1$, and separately for $i=1,\ k=2$.

Solution

Case $i=1,k=1$: $\displaystyle\sum_{j=1}^{3}\delta_{1j}\delta_{j1}$. Only $j=1$ makes $\delta_{1j}$ nonzero, giving $\delta_{11}\delta_{11}=1\times1=1$. Right side: $\delta_{ik}=\delta_{11}=1$. Match.

Case $i=1,k=2$: $\displaystyle\sum_{j=1}^{3}\delta_{1j}\delta_{j2}$. Only $j=1$ makes $\delta_{1j}$ nonzero, giving $\delta_{11}\delta_{12}=1\times0=0$. Right side: $\delta_{ik}=\delta_{12}=0$. Match.

Both cases confirm the identity: contracting two Kronecker deltas over a shared index produces a single new delta joining the two remaining (unshared) indices, exactly the matrix statement "$I\times I=I$" written out in index notation, revisited directly in Section 8.

Practice 41

Given $a_1=3,\ a_2=-2,\ a_3=5$, evaluate $\displaystyle\sum_{i=1}^{3}\sum_{j=1}^{3}\delta_{ij}\,a_ia_j$.

Solution

Using the worked example's identity, $\displaystyle\sum_i\sum_j\delta_{ij}a_ia_j = \sum_ia_i^2$ (the delta ties the two indices together, turning the double sum into an ordinary sum of squares):

$$\sum_{i=1}^{3}a_i^2 = 3^2+(-2)^2+5^2 = 9+4+25 = 38.$$

8 Applications to matrices and vectors

Everything in this lesson exists to support exactly this section. Matrix algebra, once you move past writing out full grids of numbers, is done almost entirely using indices, sums, and the Kronecker delta, and every technique from Sections 1 through 7 shows up directly.

Key idea: the index-notation dictionary

Identity matrix: $I_{ij}=\delta_{ij}$.

Matrix multiplication: the $(i,j)$ entry of $AB$ is $\displaystyle(AB)_{ij} = \sum_{k}A_{ik}B_{kj}$ (row $i$ of $A$ dotted with column $j$ of $B$).

Trace: $\displaystyle\operatorname{tr}(A) = \sum_iA_{ii} = \sum_i\sum_jA_{ij}\delta_{ij}$ (adding just the diagonal entries).

Dot product: $\displaystyle\mathbf{a}\cdot\mathbf{b} = \sum_ia_ib_i = \sum_i\sum_j\delta_{ij}a_ib_j$.

Identity acting on anything: $\displaystyle\sum_j\delta_{ij}v_j=v_i$ and $\displaystyle\sum_j\delta_{ij}A_{jk}=A_{ik}$, both direct restatements of Section 6's sifting property, this is the index-notation proof that multiplying by the identity matrix changes nothing.

Worked example

Show, using index notation, that multiplying any vector $\mathbf v$ by the identity matrix leaves it unchanged, then verify numerically with $\mathbf v=(9,-4,2)$.

In index notation, the $i$th component of $I\mathbf v$ is $\displaystyle(I\mathbf v)_i = \sum_jI_{ij}v_j = \sum_j\delta_{ij}v_j$.

By the sifting property (Section 6), this collapses immediately: $\displaystyle\sum_j\delta_{ij}v_j = v_i$.

So $(I\mathbf v)_i=v_i$ for every $i$, meaning $I\mathbf v=\mathbf v$ exactly, proven from the delta's definition alone, no matrix diagram required.

Numeric check with $\mathbf v=(9,-4,2)$: $(I\mathbf v)_1=\sum_j\delta_{1j}v_j=v_1=9$. $\ (I\mathbf v)_2=v_2=-4$. $\ (I\mathbf v)_3=v_3=2$.

$I\mathbf v = (9,-4,2) = \mathbf v$, confirmed.

Practice 42

Given the $3\times3$ matrix $A=\begin{pmatrix}2&3&1\\4&5&6\\7&8&9\end{pmatrix}$, evaluate the trace two ways: (i) directly, $\sum_iA_{ii}$, and (ii) via the double sum $\sum_i\sum_jA_{ij}\delta_{ij}$. Confirm they match.

Solution

Direct: $\operatorname{tr}(A) = A_{11}+A_{22}+A_{33} = 2+5+9=16$.

Via delta: by the worked example's dot-product-style identity, $\displaystyle\sum_i\sum_jA_{ij}\delta_{ij} = \sum_iA_{ii}$ (the delta ties $j$ to $i$, collapsing the double sum), which is the same computation: $16$.

Both give $16$, as they must, the delta double sum is just a more roundabout way of writing "add the diagonal entries."

Practice 43

Given $A=\begin{pmatrix}1&2\\3&4\end{pmatrix}$ and $B=\begin{pmatrix}5&6\\7&8\end{pmatrix}$, use the index formula $\displaystyle(AB)_{12}=\sum_kA_{1k}B_{k2}$ to find just the $(1,2)$ entry of $AB$, without computing the whole product matrix.

Solution

$$(AB)_{12} = \sum_{k=1}^{2}A_{1k}B_{k2} = A_{11}B_{12}+A_{12}B_{22} = 1(6)+2(8) = 6+16=22.$$

This matches ordinary row-times-column matrix multiplication exactly, row $1$ of $A$ is $(1,2)$, column $2$ of $B$ is $(6,8)$, and their dot product is $1(6)+2(8)=22$; the index formula is simply that same rule written with $\Sigma$-notation instead of a diagram.

Practice 44

Given $\mathbf a=(2,-1,3)$ and $\mathbf b=(4,5,-2)$, evaluate $\mathbf a\cdot\mathbf b$ two ways: (i) directly, $\sum_ia_ib_i$, and (ii) via $\sum_i\sum_j\delta_{ij}a_ib_j$.

Solution

Direct: $\mathbf a\cdot\mathbf b = 2(4)+(-1)(5)+3(-2) = 8-5-6=-3$.

Via delta: $\displaystyle\sum_i\sum_j\delta_{ij}a_ib_j$ collapses, by Practice 41's identity, to $\displaystyle\sum_ia_ib_i$, the identical single sum, giving $-3$ again.

This confirms the dot product formula in the key idea is not a new definition, it's the ordinary dot product $\sum_ia_ib_i$, just dressed up with an unnecessary (but harmless) $\delta_{ij}$, useful mainly because it shows the dot product and matrix trace share the exact same underlying delta-contraction structure.

Practice 45

Explain, referring back to the definition in Section 5, precisely why $I_{ij}=\delta_{ij}$ is the correct index-notation formula for an identity matrix of any size, not just $3\times3$.

Solution

An identity matrix, by definition, has $1$ on every diagonal entry and $0$ everywhere else, regardless of how large it is. $\delta_{ij}$, by its own definition, equals $1$ precisely when $i=j$ (the diagonal condition) and $0$ whenever $i\neq j$ (the off-diagonal condition). These are word-for-word the same rule, so setting $I_{ij}=\delta_{ij}$ reproduces the correct identity matrix for whatever range $i$ and $j$ happen to run over, $2\times2$, $3\times3$, or $100\times100$, the formula itself never needs to change with the size.

Practice 46

Using the sifting property, simplify $\displaystyle\sum_{j}\delta_{ij}A_{jk}$ to a single indexed quantity (no sum), and explain what matrix statement this represents.

Solution

By the sifting property, summing over $j$ replaces $j$ with $i$ inside $A_{jk}$:

$$\sum_j\delta_{ij}A_{jk} = A_{ik}.$$

Recognise the left side as exactly the matrix-multiplication formula from the key idea, $(IA)_{ik} = \sum_j I_{ij}A_{jk} = \sum_j\delta_{ij}A_{jk}$. So this identity is the index-notation proof of $IA=A$, multiplying any matrix by the identity matrix (on the left) leaves it completely unchanged, mirroring the worked example's vector version exactly.

Practice 47

Tie together the whole lesson: explain, in a few sentences, how Section 7's identity $\sum_j\delta_{ij}\delta_{jk}=\delta_{ik}$ and Practice 46's identity $\sum_j\delta_{ij}A_{jk}=A_{ik}$ are really the same underlying fact, applied to two different matrices.

Solution

Since $I_{jk}=\delta_{jk}$ (Section 5), Practice 46's identity $\sum_j\delta_{ij}A_{jk}=A_{ik}$ is really the statement $(IA)_{ik}=A_{ik}$, "identity times $A$ equals $A$." If you now substitute $A=I$ itself into that very same identity, $A_{jk}$ becomes $\delta_{jk}$, and the statement becomes $\sum_j\delta_{ij}\delta_{jk}=\delta_{ik}$, precisely Section 7's contraction identity. In other words, Section 7's two-delta identity isn't a separate, unrelated fact to memorise, it's the single special case of "multiplying by the identity matrix does nothing" applied to the situation where the matrix being multiplied happens to be the identity matrix itself, $I\times I=I$, written out one entry at a time.
