Topic 1
Intuitive Limits
What a limit is — and what it is not
A limit is not about what happens at a point — it is about what a function approaches as the input gets close to that point. This distinction is subtle but crucial. The function does not even need to be defined at the point in question. Everything in calculus — derivatives, integrals, continuity — is defined using limits.
DefinitionThe limit — formal statement▶

We write \(\displaystyle\lim_{x\to a} f(x) = L\) and say "the limit of f(x) as x approaches a equals L" if:

\[ \text{As } x \text{ gets arbitrarily close to } a \text{ (but } x\neq a \text{), } f(x) \text{ gets arbitrarily close to } L. \]

The formal (\(\varepsilon\)-\(\delta\)) definition: \(\lim_{x\to a}f(x)=L\) means for every \(\varepsilon>0\) there exists \(\delta>0\) such that \(0<|x-a|<\delta \Rightarrow |f(x)-L|<\varepsilon\).

For this course, we use the intuitive version: the limit is the value the function approaches, not necessarily the value it takes.

Key point: \(\lim_{x\to a}f(x)\) depends entirely on the behaviour of f near a, not at a. The function may be undefined at a, or may have a different value than the limit there.
PatternThree ways to evaluate a limit intuitively▶

1. Numerically — build a table of values approaching from both sides.

2. Graphically — trace the graph from both sides and see where it heads.

3. Algebraically — substitute, simplify, or use limit laws (Topic 2).

The limit exists if and only if the left-hand limit equals the right-hand limit (Topic 3). Always approach from both sides when evaluating numerically.
ExampleThree worked examples▶
Example 1 — Numerical approach: estimate \(\lim_{x\to2}(x^2-1)\)

Example 2 — A function undefined at the point

Consider \(f(x) = \dfrac{x^2-4}{x-2}\). This is undefined at x=2 (0/0). But:

\[ \frac{x^2-4}{x-2} = \frac{(x-2)(x+2)}{x-2} = x+2 \quad (x \neq 2) \]

As x approaches 2, f(x) approaches 4. The limit exists even though f(2) does not.

ANSWER \(\lim_{x\to2}\dfrac{x^2-4}{x-2} = 4\)
Example 3 — Limit vs function value

Define \(g(x) = \begin{cases}x+3 & x\neq1 \\ 7 & x=1\end{cases}\)

As x approaches 1: \(g(x) \to 1+3 = 4\). But \(g(1) = 7\).

KEY POINT \(\lim_{x\to1}g(x) = 4\), but \(g(1) = 7\). The limit and the function value are different.
NoteWhat the limit is not▶
Watch out: \(\lim_{x\to a}f(x)\) is NOT necessarily equal to \(f(a)\). They are equal only when f is continuous at a. Do not assume they are the same without checking.
Watch out: if f(x) oscillates wildly near a (like \(\sin(1/x)\) near 0), the limit may not exist even if f is defined everywhere. A limit requires the function to settle toward a single value.

Topic 2
Limit Laws & Algebraic Evaluation
Computing limits systematically without building tables
Tables are slow and imprecise. The limit laws give you algebraic tools to evaluate limits exactly. The key skill is recognising when direct substitution works, and what to do when it produces the indeterminate form 0/0.
DefinitionThe limit laws▶

Suppose \(\lim_{x\to a}f(x)=L\) and \(\lim_{x\to a}g(x)=M\). Then:

\[ \lim_{x\to a}[f(x)+g(x)] = L+M \qquad \lim_{x\to a}[f(x)-g(x)] = L-M \]
\[ \lim_{x\to a}[c\cdot f(x)] = cL \qquad \lim_{x\to a}[f(x)\cdot g(x)] = LM \]
\[ \lim_{x\to a}\frac{f(x)}{g(x)} = \frac{L}{M} \quad (M\neq0) \qquad \lim_{x\to a}[f(x)]^n = L^n \]

Direct substitution: if f is a polynomial or rational function and a is in the domain, then \(\lim_{x\to a}f(x) = f(a)\).

PatternStrategy — three-step method▶
1
Try direct substitution. Substitute x=a. If the result is a real number, that is the limit.
2
Got 0/0? Factorise and cancel. Or rationalise the numerator/denominator. Or use a trig identity.
3
Got a/0 (a ≠ 0)? The limit does not exist — it may be ±∞ (vertical asymptote).
Result of substitutionInterpretationAction
Real number LLimit exists = LDone
\(0/0\)Indeterminate — need more workFactorise, rationalise, or L'Hôpital
\(k/0\), \(k\neq0\)Limit is ±∞ or DNECheck one-sided limits
\(\infty - \infty\)IndeterminateRearrange or L'Hôpital

Topic 3
One-Sided Limits
Left-hand and right-hand limits — and when they must agree
When a function behaves differently as you approach from the left versus the right, the two-sided limit may not exist. One-sided limits let you describe this behaviour precisely and are essential for analysing piecewise functions and identifying discontinuities.
DefinitionLeft-hand and right-hand limits▶
\[ \lim_{x\to a^-}f(x) = L \quad \text{(left-hand limit: approach from below)} \]
\[ \lim_{x\to a^+}f(x) = M \quad \text{(right-hand limit: approach from above)} \]

The two-sided limit exists if and only if both one-sided limits exist and are equal:

\[ \lim_{x\to a}f(x) = L \iff \lim_{x\to a^-}f(x) = L \text{ and } \lim_{x\to a^+}f(x) = L \]

If \(\lim_{x\to a^-}f(x) \neq \lim_{x\to a^+}f(x)\), the two-sided limit does not exist (DNE).

ExampleFour worked examples▶
Example 1 — Piecewise function

Let \(f(x)=\begin{cases}2x+1 & x<2\\ x^2-1 & x\geq2\end{cases}\). Find \(\lim_{x\to2}f(x)\).

1
LHL: \(\lim_{x\to2^-}(2x+1) = 5\)
2
RHL: \(\lim_{x\to2^+}(x^2-1) = 3\)
3
LHL \(\neq\) RHL → limit does not exist.
ANSWER DNE (jump discontinuity at x=2)
Example 2 — Absolute value: \(\lim_{x\to0}\dfrac{|x|}{x}\)
1
For \(x>0\): \(|x|/x = x/x = 1\). RHL = 1.
2
For \(x<0\): \(|x|/x = -x/x = -1\). LHL = −1.
ANSWER DNE (LHL = −1 ≠ RHL = 1)
Example 3 — Infinite one-sided limits: \(\lim_{x\to0^+}\dfrac{1}{x}\)

As \(x\to0^+\), \(1/x \to +\infty\). The right-hand limit is \(+\infty\).

As \(x\to0^-\), \(1/x \to -\infty\). The left-hand limit is \(-\infty\).

ANSWER \(\lim_{x\to0^+}\frac{1}{x}=+\infty\), \(\lim_{x\to0^-}\frac{1}{x}=-\infty\). Two-sided limit DNE.
Example 4 — Finding k so that a piecewise limit exists

Let \(g(x)=\begin{cases}kx+3 & x\leq1\\ x^2+2k & x>1\end{cases}\). Find k so that \(\lim_{x\to1}g(x)\) exists.

1
LHL: \(k(1)+3 = k+3\)
2
RHL: \(1+2k\)
3
Set equal: \(k+3=1+2k \Rightarrow k=2\)
ANSWER \(k=2\)
NoteInfinite limits and vertical asymptotes▶
When \(\lim_{x\to a^+}f(x)=+\infty\) or \(-\infty\), the line \(x=a\) is a vertical asymptote. The function grows without bound near that point.
Watch out: \(\lim_{x\to a}f(x)=\infty\) does NOT mean the limit exists. Infinity is not a real number. It means the function is unbounded near a. We say the limit "is infinite" or "does not exist".

Topic 4
Continuity
What it means for a function to have no breaks, holes, or jumps
Continuity is the formal statement that a function "can be drawn without lifting the pen". It requires three conditions. Understanding what breaks continuity — and what type of break it is — is essential for analysing function behaviour and applying calculus theorems.
DefinitionThe three conditions for continuity▶

A function \(f\) is continuous at \(x=a\) if ALL THREE of the following hold:

1
\(f(a)\) is defined (the function has a value at a).
2
\(\lim_{x\to a}f(x)\) exists (both one-sided limits exist and are equal).
3
\(\lim_{x\to a}f(x) = f(a)\) (the limit equals the function value).

If any condition fails, f is discontinuous at a. Three types:

TypeWhat failsGraph
Removable (hole)Condition 1 or 3Single missing or misplaced point
JumpCondition 2 (LHL ≠ RHL)Graph jumps to a different level
InfiniteCondition 2 (limit = ±∞)Vertical asymptote
Intermediate Value Theorem (IVT): if f is continuous on [a,b] and N is any value between f(a) and f(b), then there exists at least one c in (a,b) with f(c) = N. This guarantees roots exist in intervals where the function changes sign.
ExampleFour worked examples▶
Example 1 — Check all three conditions for \(f(x)=\dfrac{x^2-1}{x-1}\) at x=1
1
f(1): denominator = 0. f(1) is undefined. Condition 1 FAILS.
2
But \(\lim_{x\to1}\frac{(x-1)(x+1)}{x-1} = \lim_{x\to1}(x+1) = 2\). Limit exists.
ANSWER Removable discontinuity at x=1. The "hole" can be patched by defining f(1)=2.
Example 2 — Check continuity of \(f(x)=\begin{cases}x+2 & x<1\\ 5 & x=1\\ 3x-1 & x>1\end{cases}\) at x=1
1
f(1) = 5. ✓ (Condition 1)
2
LHL = 1+2 = 3. RHL = 3(1)−1 = 2. 3≠2 → limit DNE. Condition 2 FAILS.
ANSWER Jump discontinuity at x=1.
Example 3 — IVT application: show \(x^3+x-1=0\) has a root in (0,1)
1
Let \(f(x)=x^3+x-1\). f is continuous everywhere (polynomial).
2
\(f(0) = -1 < 0\) and \(f(1) = 1+1-1 = 1 > 0\).
3
Since f(0)<0
ANSWER The equation has at least one root in (0,1). ✓
Example 4 — Find a so that \(f(x)=\begin{cases}ax^2-1 & x\leq2\\ 3x+a & x>2\end{cases}\) is continuous at x=2
1
Continuity requires LHL = RHL = f(2).
2
f(2) = LHL = 4a−1. RHL = 6+a.
3
\(4a-1=6+a \Rightarrow 3a=7 \Rightarrow a=7/3\)
ANSWER \(a = \dfrac{7}{3}\)
NoteWhich functions are always continuous?▶
The following are continuous on their entire domain: polynomials, rational functions (except where denominator=0), root functions, trig functions, exponential functions, logarithmic functions. Their compositions and combinations (sum, product, quotient, composition) are also continuous wherever defined.
Watch out: "continuous on its domain" does not mean "continuous everywhere". \(f(x)=1/x\) is continuous on its domain (all \(x\neq0\)) but has an infinite discontinuity at \(x=0\).

Topic 5
Limits at Infinity
What happens to a function as x grows without bound
As x grows very large (positive or negative), functions can approach a finite value (horizontal asymptote), grow without bound, or oscillate. Knowing which behaviour occurs — and computing the limiting value — is essential for curve sketching and understanding long-term behaviour.
DefinitionHorizontal asymptotes and infinite limits▶
\[ \lim_{x\to+\infty}f(x) = L \quad\text{means f(x) approaches L as x grows positively without bound} \]
\[ \lim_{x\to-\infty}f(x) = M \quad\text{means f(x) approaches M as x grows negatively without bound} \]

If \(\lim_{x\to\pm\infty}f(x)=L\), then the line \(y=L\) is a horizontal asymptote.

Key facts:

\[ \lim_{x\to\pm\infty}\frac{1}{x^n} = 0 \quad(n>0) \qquad \lim_{x\to+\infty}e^x = +\infty \qquad \lim_{x\to-\infty}e^x = 0 \]
PatternRational functions — divide by the highest power▶

For rational functions \(\dfrac{p(x)}{q(x)}\) as \(x\to\pm\infty\):

Divide every term by the highest power of x in the denominator. Everything with x in the denominator → 0.

Degree of numerator vs denominatorLimit at ±∞Asymptote?
deg(p) < deg(q)0\(y=0\)
deg(p) = deg(q)ratio of leading coefficients\(y=\frac{a_n}{b_n}\)
deg(p) > deg(q)±∞None (oblique asymptote)
ExampleFour worked examples▶
Example 1 — \(\lim_{x\to\infty}\dfrac{3x^2-2x+1}{5x^2+4}\)
1
Same degree (2). Divide numerator and denominator by \(x^2\):
2
\(\dfrac{3-2/x+1/x^2}{5+4/x^2} \xrightarrow{x\to\infty} \dfrac{3}{5}\)
ANSWER \(\dfrac{3}{5}\)   (horizontal asymptote \(y=3/5\))
Example 2 — \(\lim_{x\to\infty}\dfrac{4x+1}{x^2-3}\)
1
Degree of numerator (1) < degree of denominator (2). Divide by \(x^2\):
2
\(\dfrac{4/x+1/x^2}{1-3/x^2}\to\dfrac{0}{1}=0\)
ANSWER 0   (horizontal asymptote \(y=0\))
Example 3 — \(\lim_{x\to\infty}(\sqrt{x^2+3x}-x)\)
1
Rationalise: multiply by \(\dfrac{\sqrt{x^2+3x}+x}{\sqrt{x^2+3x}+x}\)
2
\(= \dfrac{(x^2+3x)-x^2}{\sqrt{x^2+3x}+x} = \dfrac{3x}{\sqrt{x^2+3x}+x}\)
3
Divide by x (positive): \(= \dfrac{3}{\sqrt{1+3/x}+1} \to \dfrac{3}{1+1} = \dfrac{3}{2}\)
ANSWER \(\dfrac{3}{2}\)
Example 4 — End behaviour of \(f(x)=\dfrac{2x^3-1}{x+2}\)

deg numerator (3) > deg denominator (1) → limit is ±∞. No horizontal asymptote. The function has an oblique asymptote (found by polynomial long division).

ANSWER \(\lim_{x\to\pm\infty}f(x)=\pm\infty\). No horizontal asymptote.
NoteDividing by x — sign matters▶
Watch out: when computing \(\lim_{x\to-\infty}\), dividing by x introduces a sign change if x is negative. Specifically, \(\sqrt{x^2} = |x| = x\) for x>0 but \(\sqrt{x^2}=|x|=-x\) for x<0. This flips a sign and is the most common error in limits at \(-\infty\).

Topic 6
Special Limits
The squeeze theorem, trig limits, and the exponential limit
Some limits cannot be evaluated by direct substitution or simple algebra. Two powerful tools — the Squeeze Theorem and a handful of fundamental limits — handle these cases. The trig limit \(\lim_{x\to0}\frac{\sin x}{x}=1\) and the exponential limit \(\lim_{n\to\infty}(1+1/n)^n=e\) are foundational results used throughout calculus.
DefinitionThe Squeeze Theorem and key special limits▶

Squeeze Theorem: if \(g(x)\leq f(x)\leq h(x)\) for all x near a, and \(\lim_{x\to a}g(x)=\lim_{x\to a}h(x)=L\), then \(\lim_{x\to a}f(x)=L\).

Fundamental limits to memorise:

\[ \lim_{x\to0}\frac{\sin x}{x} = 1 \qquad \lim_{x\to0}\frac{1-\cos x}{x} = 0 \qquad \lim_{x\to0}\frac{\tan x}{x} = 1 \]
\[ \lim_{n\to\infty}\!\left(1+\frac{1}{n}\right)^n = e \qquad \lim_{x\to0}(1+x)^{1/x} = e \]
\[ \lim_{x\to0}\frac{e^x-1}{x} = 1 \qquad \lim_{x\to0}\frac{\ln(1+x)}{x} = 1 \]
ExampleFour worked examples▶
Example 1 — Squeeze: \(\lim_{x\to0}x^2\sin(1/x)\)
1
We know \(-1\leq\sin(1/x)\leq1\) for all x≠0.
2
Multiply by \(x^2\geq0\): \(-x^2\leq x^2\sin(1/x)\leq x^2\)
3
Both \(-x^2\to0\) and \(x^2\to0\). By squeeze: limit = 0.
ANSWER 0
Example 2 — Using \(\lim_{x\to0}\frac{\sin x}{x}=1\): find \(\lim_{x\to0}\dfrac{\sin(5x)}{3x}\)
1
Rewrite: \(\dfrac{\sin(5x)}{3x} = \dfrac{5}{3}\cdot\dfrac{\sin(5x)}{5x}\)
2
As \(x\to0\), \(5x\to0\), so \(\frac{\sin(5x)}{5x}\to1\)
3
Limit = \(\frac{5}{3}\times1 = \frac{5}{3}\)
ANSWER \(\dfrac{5}{3}\)
Example 3 — Limit involving e: \(\lim_{x\to0}\dfrac{e^{3x}-1}{x}\)
1
Rewrite: \(\dfrac{e^{3x}-1}{x} = 3\cdot\dfrac{e^{3x}-1}{3x}\)
2
As \(x\to0\), \(3x\to0\), so \(\frac{e^{3x}-1}{3x}\to1\)
ANSWER 3

Topic 7
L'Hôpital's Rule
Resolving indeterminate forms using derivatives
When direct substitution gives 0/0 or ∞/∞, algebraic manipulation is not always practical. L'Hôpital's Rule provides an elegant shortcut: differentiate the numerator and denominator separately, then take the limit again. It requires knowing derivatives (Guide 2), but is introduced here as a limit technique.
DefinitionL'Hôpital's Rule — statement and conditions▶

If \(\lim_{x\to a}\frac{f(x)}{g(x)}\) produces \(\frac{0}{0}\) or \(\frac{\infty}{\infty}\), and if \(f\) and \(g\) are differentiable near \(a\), then:

\[ \lim_{x\to a}\frac{f(x)}{g(x)} = \lim_{x\to a}\frac{f'(x)}{g'(x)} \]

provided the limit on the right exists (or is ±∞).

Critical conditions: L'Hôpital only applies to \(0/0\) or \(\infty/\infty\) forms. Do NOT apply it to other situations (like \(0/\infty\) or \(3/0\)). Check the form before applying.
Other indeterminate forms: \(0\cdot\infty\), \(\infty-\infty\), \(0^0\), \(\infty^0\), \(1^\infty\) — these must be rewritten as \(0/0\) or \(\infty/\infty\) before applying L'Hôpital.
ExampleFive worked examples▶
Example 1 — Basic 0/0: \(\lim_{x\to0}\dfrac{e^x-1}{x}\)
1
Form: 0/0. Apply L'Hôpital: differentiate top and bottom.
2
\(\lim_{x\to0}\dfrac{e^x}{1} = e^0 = 1\)
ANSWER 1
Example 2 — ∞/∞: \(\lim_{x\to\infty}\dfrac{\ln x}{x}\)
1
Form: ∞/∞. L'Hôpital: \(\dfrac{d/dx(\ln x)}{d/dx(x)} = \dfrac{1/x}{1} = \dfrac{1}{x}\)
2
\(\lim_{x\to\infty}\dfrac{1}{x} = 0\)
ANSWER 0   (logarithm grows slower than any power of x)
Example 3 — Apply twice: \(\lim_{x\to0}\dfrac{1-\cos x}{x^2}\)
1
Form: 0/0. L'Hôpital once: \(\dfrac{\sin x}{2x}\). Still 0/0.
2
L'Hôpital again: \(\dfrac{\cos x}{2}\to\dfrac{1}{2}\)
ANSWER \(\dfrac{1}{2}\)
Example 4 — 0·∞ form: \(\lim_{x\to0^+}x\ln x\)
1
Rewrite as \(\dfrac{\ln x}{1/x}\) (now ∞/∞ form).
2
L'Hôpital: \(\dfrac{1/x}{-1/x^2} = -x\to0\)
ANSWER 0
Example 5 — \(\lim_{x\to\infty}\dfrac{x^2}{e^x}\)
1
∞/∞. L'Hôpital: \(\dfrac{2x}{e^x}\). Still ∞/∞.
2
L'Hôpital again: \(\dfrac{2}{e^x}\to0\)
ANSWER 0   (exponential growth dominates any polynomial)
NoteCommon L'Hôpital mistakes▶
Watch out: L'Hôpital differentiates the numerator and denominator separately — it is NOT the quotient rule. \(\frac{d}{dx}\frac{f}{g}\neq\frac{f'}{g'}\). L'Hôpital says \(\lim\frac{f}{g}=\lim\frac{f'}{g'}\) (under the correct conditions).
Watch out: always verify the form is 0/0 or ∞/∞ before applying. Applying L'Hôpital to a limit that is not indeterminate gives a wrong answer.
