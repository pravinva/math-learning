Topic 1
Trig Ratios & Exact Values
SOH CAH TOA — and the exact values every student must know cold
Trigonometry starts in a right-angled triangle. The three ratios — sine, cosine, tangent — relate the angles to the sides. Exact values for 30°, 45°, and 60° appear in almost every trig question at this level. You need these in your memory, not derived each time.
DefinitionThe three trig ratios in a right triangle▶

For an angle \(\theta\) in a right-angled triangle, label the sides as opposite (O), adjacent (A), and hypotenuse (H):

\[ \sin\theta = \frac{\text{Opposite}}{\text{Hypotenuse}} \qquad \cos\theta = \frac{\text{Adjacent}}{\text{Hypotenuse}} \qquad \tan\theta = \frac{\text{Opposite}}{\text{Adjacent}} \]

Memory aid: SOH CAH TOA. The hypotenuse is always the longest side, opposite the right angle.

Three reciprocal ratios also exist:

\[ \csc\theta = \frac{1}{\sin\theta} \qquad \sec\theta = \frac{1}{\cos\theta} \qquad \cot\theta = \frac{1}{\tan\theta} \]

Exact values — memorise all of these:

Angle\(\sin\)\(\cos\)\(\tan\)
0°  (0)010
30°  (\(\pi/6\))\(\frac{1}{2}\)\(\frac{\sqrt{3}}{2}\)\(\frac{1}{\sqrt{3}}\)
45°  (\(\pi/4\))\(\frac{1}{\sqrt{2}}\)\(\frac{1}{\sqrt{2}}\)1
60°  (\(\pi/3\))\(\frac{\sqrt{3}}{2}\)\(\frac{1}{2}\)\(\sqrt{3}\)
90°  (\(\pi/2\))10undefined
Memory trick for sin: \(\sin 0°, \sin 30°, \sin 45°, \sin 60°, \sin 90°\) are \(\frac{\sqrt{0}}{2}, \frac{\sqrt{1}}{2}, \frac{\sqrt{2}}{2}, \frac{\sqrt{3}}{2}, \frac{\sqrt{4}}{2}\). The pattern under the root goes 0,1,2,3,4. Cos is the reverse order.

To find an angle from two sides: identify which ratio applies, then apply the inverse function. Always check the answer makes sense (between 0° and 90° in a right triangle).

ExampleFour worked examples▶
Example 1 — Find the side opposite 35° in a triangle with hypotenuse 12 cm
1
Know: angle 35°, hypotenuse 12. Want: opposite. Use \(\sin\).
2
\(\text{opp} = 12\sin 35° = 12 \times 0.5736 \approx 6.88\text{ cm}\)
ANSWER 6.88 cm
Example 2 — Find angle \(\theta\) if opposite = 7, hypotenuse = 10
1
\(\sin\theta = \frac{7}{10} = 0.7\)
2
\(\theta = \sin^{-1}(0.7) \approx 44.4°\)
ANSWER \(\theta \approx 44.4°\)
Example 3 — Use exact values: find the hypotenuse of a 30–60–90 triangle with opposite side (to 60°) equal to \(\sqrt{3}\)
1
\(\sin 60° = \frac{\sqrt{3}}{2} = \frac{\text{opp}}{H} = \frac{\sqrt{3}}{H}\)
2
\(H = \frac{\sqrt{3}}{\sin 60°} = \frac{\sqrt{3}}{\sqrt{3}/2} = 2\)
ANSWER Hypotenuse = 2
Example 4 — Exact value computation: evaluate \(\sin 45° \cos 30° + \cos 45° \sin 30°\)
1
\(= \frac{1}{\sqrt{2}}\cdot\frac{\sqrt{3}}{2} + \frac{1}{\sqrt{2}}\cdot\frac{1}{2} = \frac{\sqrt{3}}{2\sqrt{2}} + \frac{1}{2\sqrt{2}} = \frac{\sqrt{3}+1}{2\sqrt{2}}\)
2
Rationalise: \(= \frac{(\sqrt{3}+1)\sqrt{2}}{4} = \frac{\sqrt{6}+\sqrt{2}}{4}\)
ANSWER \(\dfrac{\sqrt{6}+\sqrt{2}}{4}\)   (which is \(\sin 75°\) — notice this!)
NoteCommon mistakes with trig ratios▶
Watch out: the hypotenuse is always opposite the right angle — it is always the longest side. Do not confuse "opposite" and "adjacent" — they change depending on which angle you are working with.
Watch out: \(\sin^{-1}(x)\) means arcsin(x), not \(\frac{1}{\sin x}\). These are completely different things. When finding an angle from a ratio, use the inverse function on your calculator.
The exact values for 30° and 60° come from an equilateral triangle with side 2 (split in half). The exact values for 45° come from a right isoceles triangle with legs of length 1. Drawing these triangles is the best way to reconstruct exact values in an exam if you forget them.

Topic 2
The Unit Circle & Radians
Extending trig beyond right triangles — to all angles
Right-triangle trig only works for angles between 0° and 90°. To handle angles like 120°, 270°, or even negative angles, we need the unit circle. It extends sine and cosine to any angle — and explains why trig functions have negative values in certain quadrants.
DefinitionRadians and the unit circle▶

Radians: one radian is the angle subtended at the centre of a circle by an arc equal in length to the radius.

\[ \pi \text{ rad} = 180° \qquad 1\text{ rad} = \frac{180°}{\pi} \approx 57.3° \]

Converting: degrees → radians: multiply by \(\dfrac{\pi}{180}\). Radians → degrees: multiply by \(\dfrac{180}{\pi}\).

The unit circle has radius 1, centred at the origin. For any angle \(\theta\) measured anticlockwise from the positive x-axis, the point \(P\) on the circle has coordinates:

\[ P = (\cos\theta,\; \sin\theta) \]

This defines \(\cos\theta\) and \(\sin\theta\) for any angle — not just those in a right triangle.

PatternCAST diagram — signs in each quadrant▶

The sign of each trig function depends on which quadrant the angle falls in:

Quadrant 2 (90°–180°)
Sin positive
cos, tan negative
Quadrant 1 (0°–90°)
All positive
Quadrant 3 (180°–270°)
Tan positive
sin, cos negative
Quadrant 4 (270°–360°)
Cos positive
sin, tan negative

Memory: A S T C reading anticlockwise from Q1, or: "All Students Take Calculus".

Reference angle: to find the exact value of any angle, find its reference angle (the acute angle to the nearest x-axis) and apply the correct sign from CAST.

Arc length and sector area formulas (in radians only): \(s = r\theta\) and \(A = \frac{1}{2}r^2\theta\). These only work in radians — if given degrees, convert first.

Topic 3
Graphs of Trig Functions
Amplitude, period, phase shift — reading and writing transformed trig graphs
Trig graphs repeat — they are periodic. Understanding the four parameters (a, b, c, d) in \(y = a\sin(bx+c)+d\) lets you sketch any transformed trig graph and write the equation of a graph from its shape. This is a core exam topic.
DefinitionThe general form and its four parameters▶
\[ y = a\sin(bx + c) + d \]
ParameterNameEffectFormula
\(a\)AmplitudeVertical stretch. If \(a<0\), reflected in x-axis.Amplitude = \(|a|\)
\(b\)Angular frequencyHorizontal compression. Larger b → shorter period.Period = \(\dfrac{2\pi}{|b|}\)
\(c\)Phase shiftHorizontal shift. Moves graph left (if +) or right (if −).Shift = \(-\dfrac{c}{b}\)
\(d\)Vertical shiftMoves midline up (if +) or down (if −).Midline at \(y = d\)

Range: from \(d - |a|\) to \(d + |a|\).

Basic graphs to know:

FunctionPeriodRangeKey feature
\(y = \sin x\)\(2\pi\)\([-1,1]\)Starts at 0, rises first
\(y = \cos x\)\(2\pi\)\([-1,1]\)Starts at 1, same shape as sin shifted left π/2
\(y = \tan x\)\(\pi\)All realsVertical asymptotes at \(x = \frac{\pi}{2}+n\pi\)
PatternHow to read a graph and write its equation▶
1
Find a: amplitude = (max − min)/2. Sign: positive if starts going up (sin) or starts at max (cos).
2
Find d: midline = (max + min)/2.
3
Find b: read the period from the graph. Then \(b = 2\pi/\text{period}\).
4
Find c: identify the phase shift. If the sin graph starts rising at \(x = p\), then \(-c/b = p\), so \(c = -bp\).
The phase shift is \(-c/b\) (positive means shift right). Be careful: \(\sin(2x - \pi/3)\) has shift \(\pi/6\) to the right, not \(\pi/3\). Always factor out b first: \(\sin\!\left(2\!\left(x - \frac{\pi}{6}\right)\right)\).
ExampleFour worked examples▶
Example 1 — State amplitude, period, phase shift, midline for \(y = 3\sin(2x - \pi) + 1\)
1
Amplitude: \(|a| = 3\)
2
Period: \(\frac{2\pi}{|b|} = \frac{2\pi}{2} = \pi\)
3
Phase shift: \(-\frac{c}{b} = -\frac{-\pi}{2} = \frac{\pi}{2}\) right
4
Midline: \(y = 1\). Range: \([-2, 4]\)
ANSWER Amp = 3, period = π, shift π/2 right, midline y = 1
Example 2 — Sketch \(y = 2\cos(x) - 1\): describe key features
1
Start: basic \(\cos x\) shifted down 1 and stretched vertically by 2.
2
Maximum: \(y = 2(1)-1 = 1\) at \(x = 0, 2\pi, \ldots\)
3
Minimum: \(y = 2(-1)-1 = -3\) at \(x = \pi, 3\pi, \ldots\)
4
Midline: \(y = -1\). Period: \(2\pi\). Amplitude: 2.
KEY POINTS Max 1, min −3, midline y=−1, period 2π
Example 3 — Write the equation of a sine graph with amplitude 4, period \(\pi\), shifted right \(\frac{\pi}{4}\), no vertical shift
1
a = 4. Period = π → \(b = 2\pi/\pi = 2\).
2
Phase shift π/4 right → \(-c/b = \pi/4 \Rightarrow c = -\pi/2\)
ANSWER \(y = 4\sin\!\left(2x - \frac{\pi}{2}\right)\)
Example 4 — Find the period and asymptotes of \(y = \tan(3x)\)
1
Period of \(\tan(bx) = \pi/|b| = \pi/3\)
2
Asymptotes where \(3x = \pi/2 + n\pi\), i.e. \(x = \pi/6 + n\pi/3\)
ANSWER Period = π/3. Asymptotes at \(x = \dfrac{\pi}{6} + \dfrac{n\pi}{3}\)
NoteGraph transformation traps▶
Watch out: \(y = \sin(2x - \pi)\) is NOT shifted by \(\pi\). The shift is \(-\pi/b = -\pi/2\)... wait, shift = \(+\pi/2\) right. Always factor: \(\sin\!\left(2\left(x - \frac{\pi}{2}\right)\right)\). The shift is π/2 right.
Watch out: the tan graph has period \(\pi\), not \(2\pi\). Students often try to use \(2\pi/b\) for tan — the formula is \(\pi/b\) for tan.
To sketch: plot 5 key points — start (phase shift), first quarter (max or zero), half period (zero), three-quarters (min or zero), full period (back to start). These 5 points define one complete cycle.

Topic 4
Solving Trig Equations
Finding all solutions — not just the first one the calculator gives
The calculator's \(\sin^{-1}\) button gives only one answer. But trig equations almost always have multiple solutions within a given range because the functions repeat. The systematic method — CAST diagram plus symmetry — ensures you find every solution.
DefinitionPrincipal value and all solutions▶

The principal value is the one angle in the standard range that your calculator returns:

\[ \sin^{-1}: \left[-\frac{\pi}{2}, \frac{\pi}{2}\right] \qquad \cos^{-1}: [0, \pi] \qquad \tan^{-1}: \left(-\frac{\pi}{2}, \frac{\pi}{2}\right) \]

To find all solutions in \([0°, 360°]\) or \([0, 2\pi]\):

1
Find the reference angle: the positive acute angle whose trig value has the same magnitude.
2
Use CAST to determine which quadrants give the correct sign.
3
Write the angle in each valid quadrant using the reference angle.
4
Check all answers are within the required range.

General solution (all possible solutions, not just one cycle):

\[ \sin\theta = k: \quad \theta = \sin^{-1}k + 360°n \quad\text{or}\quad \theta = 180° - \sin^{-1}k + 360°n \]
\[ \cos\theta = k: \quad \theta = \pm\cos^{-1}k + 360°n \]
\[ \tan\theta = k: \quad \theta = \tan^{-1}k + 180°n \]
PatternSymmetry relationships — the fast way to find the second solution▶
EquationFirst solutionSecond solution in \([0°, 360°]\)
\(\sin\theta = +k\)\(\theta_1 = \sin^{-1}k\) (Q1)\(\theta_2 = 180° - \theta_1\) (Q2)
\(\sin\theta = -k\)\(\theta_1 = 180° + \sin^{-1}k\) (Q3)\(\theta_2 = 360° - \sin^{-1}k\) (Q4)
\(\cos\theta = +k\)\(\theta_1 = \cos^{-1}k\) (Q1)\(\theta_2 = 360° - \theta_1\) (Q4)
\(\cos\theta = -k\)\(\theta_1 = 180° - \cos^{-1}k\) (Q2)\(\theta_2 = 180° + \cos^{-1}k\) (Q3)
\(\tan\theta = k\)\(\theta_1 = \tan^{-1}k\)\(\theta_2 = \theta_1 + 180°\)
ExampleFive worked examples▶
Example 1 — Solve \(\sin\theta = 0.6\) for \(\theta \in [0°, 360°]\)
1
Reference: \(\sin^{-1}(0.6) \approx 36.87°\)

Example 2 — Solve \(\cos\theta = -\frac{1}{2}\) for \(\theta \in [0°, 360°]\) exactly

Example 3 — Solve \(2\sin\theta - 1 = 0\) for \(\theta \in [0, 2\pi]\)

NoteThe most common errors in trig equations▶
Watch out: never cancel \(\sin\theta\) or \(\cos\theta\) from both sides (you lose solutions). Example: \(\sin\theta\cos\theta = \sin\theta\) — do NOT divide by \(\sin\theta\). Rearrange to \(\sin\theta(\cos\theta-1)=0\) and solve each factor.
Watch out: when solving \(\sin(2\theta)=k\), double the range first. If \(\theta\in[0°,360°]\), then \(2\theta\in[0°,720°]\) — you must find all solutions in the doubled range, then halve.
Quadratic-type trig equations: treat \(\sin\theta\) or \(\cos\theta\) as a single variable. Factorise or use the quadratic formula. Remember: \(|\sin\theta|\leq1\) and \(|\cos\theta|\leq1\) — reject any "solution" where the value is outside [−1,1].

Topic 5
Trig Identities
Fundamental relationships that are always true — and how to prove new ones
Identities are equations that are true for every value of the angle. They are not equations to solve — they are relationships you use to simplify or transform expressions. The Pythagorean identity is the most important and appears everywhere.
DefinitionThe core identities▶

Pythagorean identities (derived from \(x^2+y^2=1\) on the unit circle):

\[ \sin^2\theta + \cos^2\theta = 1 \]
\[ 1 + \tan^2\theta = \sec^2\theta \qquad (\text{divide first by }\cos^2\theta) \]
\[ \cot^2\theta + 1 = \csc^2\theta \qquad (\text{divide first by }\sin^2\theta) \]

Quotient identity:

\[ \tan\theta = \frac{\sin\theta}{\cos\theta} \qquad \cot\theta = \frac{\cos\theta}{\sin\theta} \]

Rearrangements of the Pythagorean identity — memorise all three forms:

\[ \sin^2\theta = 1 - \cos^2\theta \qquad \cos^2\theta = 1 - \sin^2\theta \]
PatternHow to prove an identity▶
1
Work on one side only (usually the more complicated side). Never move terms across the equals sign.
2
Convert to sin and cos — replace tan, sec, csc, cot with sin/cos expressions.
3
Look for opportunities to use \(\sin^2+\cos^2=1\), factorise, or combine fractions.
4
Aim for the other side — every step should move you toward the target expression.
You cannot prove an identity by assuming it is true. Do not cross-multiply or move terms across = sign. Work on one side only and transform it into the other side.
ExampleFive identity proofs worked in full▶
Example 1 — Prove \(\dfrac{\sin\theta}{\tan\theta} = \cos\theta\)
LHS
\(\dfrac{\sin\theta}{\tan\theta} = \dfrac{\sin\theta}{\sin\theta/\cos\theta} = \sin\theta \cdot \dfrac{\cos\theta}{\sin\theta} = \cos\theta\) = RHS ✓
Example 2 — Prove \(\sin^2\theta\tan^2\theta = \tan^2\theta - \sin^2\theta\)
RHS
\(\tan^2\theta - \sin^2\theta = \dfrac{\sin^2\theta}{\cos^2\theta} - \sin^2\theta = \sin^2\theta\!\left(\dfrac{1}{\cos^2\theta}-1\right) = \sin^2\theta \cdot \dfrac{1-\cos^2\theta}{\cos^2\theta}\)
\(= \sin^2\theta \cdot \dfrac{\sin^2\theta}{\cos^2\theta} = \sin^2\theta\tan^2\theta\) = LHS ✓
Example 3 — Prove \(\dfrac{1-\cos\theta}{1+\cos\theta} = (\csc\theta - \cot\theta)^2\)
RHS
\(\left(\dfrac{1}{\sin\theta} - \dfrac{\cos\theta}{\sin\theta}\right)^2 = \left(\dfrac{1-\cos\theta}{\sin\theta}\right)^2 = \dfrac{(1-\cos\theta)^2}{\sin^2\theta}\)
\(= \dfrac{(1-\cos\theta)^2}{1-\cos^2\theta} = \dfrac{(1-\cos\theta)^2}{(1-\cos\theta)(1+\cos\theta)} = \dfrac{1-\cos\theta}{1+\cos\theta}\) = LHS ✓
Example 4 — Simplify \(\dfrac{\cos^2\theta - 1}{\cos\theta}\)
1
\(\cos^2\theta - 1 = -(1-\cos^2\theta) = -\sin^2\theta\)
2
\(\dfrac{-\sin^2\theta}{\cos\theta} = -\sin\theta\cdot\dfrac{\sin\theta}{\cos\theta} = -\sin\theta\tan\theta\)
ANSWER \(-\sin\theta\tan\theta\)
Example 5 — Given \(\sin\theta = \frac{3}{5}\) and \(\theta\) is acute, find \(\cos\theta\) and \(\tan\theta\)
1

NoteIdentity vs equation — and what not to do in a proof▶
An identity is true for all values of θ. An equation is true for specific values. When asked to "prove" or "show", you are proving an identity — work on one side only.
Illegal proof moves: squaring both sides, cross-multiplying, adding the same thing to both sides. All of these assume the result is true before you've shown it.
Watch out: \(\sin^2\theta\) means \((\sin\theta)^2\), not \(\sin(\theta^2)\). These are very different. Write brackets when unsure.

Topic 6
Sine & Cosine Rules
Solving non-right triangles — the complete toolkit
Right-triangle trig only works when there is a right angle. The sine rule and cosine rule handle any triangle. Knowing which rule to use — and when the ambiguous case applies — is what separates good students from great ones.
DefinitionBoth rules and the area formula▶

Sine Rule — for any triangle with sides a, b, c opposite angles A, B, C:

\[ \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} \]

Cosine Rule:

\[ a^2 = b^2 + c^2 - 2bc\cos A \]

Rearranged to find an angle:

\[ \cos A = \frac{b^2 + c^2 - a^2}{2bc} \]

Area of a triangle:

\[ \text{Area} = \frac{1}{2}ab\sin C \]
GivenUse
Two angles + one side (AAS or ASA)Sine Rule
Two sides + angle opposite one of them (SSA)Sine Rule (check ambiguous case)
Two sides + included angle (SAS)Cosine Rule
Three sides (SSS)Cosine Rule
PatternThe ambiguous case (SSA)▶

When given two sides and an angle NOT between them (SSA), there may be two valid triangles. When \(\sin B = k\) and you are finding angle B:

1
Find \(B_1 = \sin^{-1}(k)\) (acute angle).
2
Check if \(B_2 = 180° - B_1\) is valid: it is valid if \(A + B_2 < 180°\).
3
If valid, both triangles exist and you must give both solutions.
The ambiguous case only arises with the Sine Rule when you are finding an angle, not a side. The cosine rule, when finding an angle, gives a unique answer (cosine is 1-to-1 on [0°,180°]).

Example 1 — Sine Rule: find side a given A=40°, B=75°, b=12 cm

NoteRule selection and the most common exam traps▶
Watch out: always label the triangle consistently. Side a is opposite angle A, side b opposite angle B. If you mix these up, the rule gives wrong answers.
Watch out: in the cosine rule, \(a^2 = b^2 + c^2 - 2bc\cos A\). Students often forget to halve the last term or apply the wrong sign. The formula for angle is always \(\cos A = (b^2+c^2-a^2)/(2bc)\).
If you get \(\cos A > 1\) or \(\cos A < -1\) when using the cosine rule, you have made an arithmetic error — no real angle exists outside this range.

Topic 7
Compound & Double Angle Formulas
Expanding sin(A±B), cos(A±B), and their double-angle special cases
The compound angle formulas let you find exact values of angles like 75° or 15° that are not in the standard exact value table. They also appear constantly in proofs and in solving equations. The double angle formulas are just special cases when A = B — they are worth knowing separately because they appear so often.
DefinitionAll the compound and double angle formulas▶

Compound angle formulas:

\[ \sin(A+B) = \sin A\cos B + \cos A\sin B \]
\[ \sin(A-B) = \sin A\cos B - \cos A\sin B \]
\[ \cos(A+B) = \cos A\cos B - \sin A\sin B \]
\[ \cos(A-B) = \cos A\cos B + \sin A\sin B \]
\[ \tan(A\pm B) = \frac{\tan A \pm \tan B}{1 \mp \tan A\tan B} \]

Double angle formulas (set B = A):

\[ \sin 2A = 2\sin A\cos A \]
\[ \cos 2A = \cos^2 A - \sin^2 A = 2\cos^2 A - 1 = 1 - 2\sin^2 A \]
\[ \tan 2A = \frac{2\tan A}{1 - \tan^2 A} \]
The three forms of cos 2A are all equal — choose whichever is most convenient. If the target has only cos terms, use \(2\cos^2A-1\). Only sin? Use \(1-2\sin^2A\). Both? Use \(\cos^2A-\sin^2A\).
PatternWhen and how to use compound angle formulas▶
Use caseStrategy
Exact value of e.g. sin 75°Write as sin(45°+30°) and expand
Prove an identity involving 2AReplace 2A using double angle formula
Solve an equation with mixed sin A and sin 2AReplace sin 2A = 2sinAcosA, then factorise
Integration or differentiation (later)Use \(\cos^2A = \frac{1+\cos2A}{2}\) and \(\sin^2A = \frac{1-\cos2A}{2}\)

The half-angle forms rearranged from cos2A:

\[ \cos^2 A = \frac{1+\cos 2A}{2} \qquad \sin^2 A = \frac{1-\cos 2A}{2} \]

Topic 8
Inverse Trig Functions
arcsin, arccos, arctan — and why domain restrictions matter
Sine, cosine, and tangent are not one-to-one functions — the same output can come from many different inputs. To define an inverse, we must restrict the domain. Getting the domain and range of each inverse function correct is essential — it determines which angle your calculator returns and why.
DefinitionThe three inverse functions and their restricted domains▶
FunctionInput (domain)Output (range)What it gives
\(y = \arcsin x = \sin^{-1}x\)\([-1, 1]\)\(\left[-\frac{\pi}{2}, \frac{\pi}{2}\right]\)Angle in Q4, Q1 (or Q−1 to Q1)
\(y = \arccos x = \cos^{-1}x\)\([-1, 1]\)\([0, \pi]\)Angle in Q1, Q2
\(y = \arctan x = \tan^{-1}x\)All reals\(\left(-\frac{\pi}{2}, \frac{\pi}{2}\right)\)Angle in Q4, Q1 (open endpoints)
These restricted ranges mean the inverse trig functions always return an angle in those ranges — even if the original equation has solutions elsewhere. To find all solutions, use the principal value and apply CAST/symmetry (Topic 4).

Important identities:

\[ \sin(\arcsin x) = x \qquad \arcsin(\sin\theta) = \theta \text{ only if } \theta\in\left[-\frac{\pi}{2},\frac{\pi}{2}\right] \]
\[ \arcsin x + \arccos x = \frac{\pi}{2} \qquad \text{for } x\in[-1,1] \]
PatternEvaluating and using inverse trig expressions▶
1
To evaluate \(\arcsin(k)\): find the angle in \([-\pi/2, \pi/2]\) whose sine is k.
2
To evaluate \(\sin(\arccos(x))\): let \(\theta = \arccos(x)\), so \(\cos\theta = x\). Draw a right triangle, find the opposite side using Pythagoras, then read off the required ratio.
3
To find all solutions of \(\sin\theta = k\) in \([0°,360°]\): get the principal value from the inverse function, then apply CAST to find the second solution.

Apply tan to both sides. Use the tan addition formula on the left.
▶
FULL SOLUTION
1
Let \(\alpha = \arctan(1/2)\), \(\beta = \arctan(1/3)\). Both in \((0, \pi/2)\).
2
\(\tan(\alpha+\beta) = \dfrac{\tan\alpha+\tan\beta}{1-\tan\alpha\tan\beta} = \dfrac{\frac{1}{2}+\frac{1}{3}}{1-\frac{1}{2}\cdot\frac{1}{3}} = \dfrac{\frac{5}{6}}{\frac{5}{6}} = 1\)
3
\(\tan(\alpha+\beta)=1\) and \(\alpha+\beta\in(0,\pi)\). Since \(\tan(\pi/4)=1\), we get \(\alpha+\beta=\pi/4\) ✓
