Core Method (Teach This First)

1) Scale the Domain Before Solving

When the equation has a transformed argument, solve for the whole argument first (call it \(\theta\)). Adjust the domain boundaries to match that argument window.

  • If equation uses \(2x\), original \(0\le x\le2\pi\) becomes \(0\le2x\le4\pi\) (two full rotations).
  • If equation uses \(x/2\), original \(0\le x\le2\pi\) becomes \(0\le x/2\le\pi\) (half-speed argument).
  • Negative windows (for example \(-2\pi\le x\le2\pi\)) include clockwise (negative) and counterclockwise (positive) angles.

2) Solve the Whole Argument \(\theta\), Not \(x\), First

For \(\sin(Bx+C)=k\), set \(\theta=Bx+C\), solve trig values for \(\theta\) inside the transformed domain, then back-solve to \(x\).

3) Isolate the Trig Function First (Amplitude/Vertical Shift)

For general form \(A\sin(B(x-C))+D=k\), first isolate:

\[ \sin(B(x-C))=\frac{k-D}{A} \]

Only then transform domain boundaries for \(B(x-C)\).

4) Back-Solve at the End

After collecting all valid \(\theta\)-solutions in the transformed window, undo the shifts and scale to find \(x\).

Common error: dividing by \(B\) too early and missing valid angle repeats. Always solve in \(\theta\)-space first.

Domain Adjustment Master Table (use every time)

Start with original domain in \(x\), then transform both boundaries exactly the same way as the inside argument.

  • If argument is \(2x\): \(x\in[a,b]\Rightarrow 2x\in[2a,2b]\). Example: \(0\le x\le2\pi\Rightarrow 0\le2x\le4\pi\) (double sweep).
  • If argument is \(3x\): \(x\in[a,b]\Rightarrow 3x\in[3a,3b]\). Example: \(0\le x\le2\pi\Rightarrow 0\le3x\le6\pi\) (triple sweep).
  • If argument is \(x/2\): \(x\in[a,b]\Rightarrow x/2\in[a/2,b/2]\). Example: \(0\le x\le2\pi\Rightarrow 0\le x/2\le\pi\) (half sweep).
  • If argument is \(Bx+C\): do exactly the same operations to the boundaries in order (multiply by \(B\), then add/subtract \(C\)).
  • If argument is \(B(x-C)\): easiest is expand to \(Bx-BC\), then transform boundaries with those same operations.
  • Negative domains: keep negatives; do not force to positive. Example: \(-2\pi\le x\le2\pi\Rightarrow -4\pi\le2x\le4\pi\).

Five-line routine for transformed equations

  • Line 1: Isolate trig function: \(A\sin(\cdot)+D=k\Rightarrow \sin(\cdot)=\frac{k-D}{A}\).
  • Line 2: Define \(\theta=\) whole argument.
  • Line 3: Transform domain boundaries into \(\theta\)-window.
  • Line 4: Solve trig equation for all \(\theta\) in that window.
  • Line 5: Substitute back \(\theta=\) argument and solve for \(x\).

Worked Example 1 (Classic)

Solve: \(\sin(2x)=\frac12\), for \(0\le x\le2\pi\).

  • Scale domain first: from \(0\le x\le2\pi\), multiply by 2 to match the inside argument: \(0\le2x\le4\pi\). Define \(\theta=2x\), so working window is \(0\le\theta\le4\pi\).
  • Find reference angle: \(\alpha=\sin^{-1}(1/2)=\pi/6\).
  • Choose quadrants by sign: sine is positive, so in one cycle solutions are QI and QII: \[ \theta=\alpha=\frac{\pi}{6},\qquad \theta=\pi-\alpha=\frac{5\pi}{6}. \]
  • Why add \(2\pi\)? Because sine has period \(2\pi\). We must include the second cycle (from \(2\pi\) to \(4\pi\)) since our transformed window reaches \(4\pi\). Add \(2\pi\) to each first-cycle angle: \[ \frac{\pi}{6}+2\pi=\frac{13\pi}{6},\qquad \frac{5\pi}{6}+2\pi=\frac{17\pi}{6}. \]
  • Do we subtract \(2\pi\) here? No. Subtracting would move angles into negative values, outside \(0\le\theta\le4\pi\). Subtracting \(2\pi\) is used only when transformed window includes negatives.
  • Back-solve to \(x\): divide every \(\theta\)-solution by 2.

\[ x=\frac{\pi}{12},\ \frac{5\pi}{12},\ \frac{13\pi}{12},\ \frac{17\pi}{12} \]

Worked Example 2 (Amplitude + Shift)

Solve: \(2\sin\left(2x-\frac{\pi}{3}\right)+1=2\), for \(0\le x\le\pi\).

  • Isolate the trig part first: \(2\sin(\cdot)+1=2\Rightarrow \sin\left(2x-\frac{\pi}{3}\right)=\frac12\). Let \(\theta=2x-\frac{\pi}{3}\).
  • Transform the domain carefully: start \(0\le x\le\pi\). Multiply by 2: \(0\le2x\le2\pi\). Subtract \(\pi/3\): \(-\pi/3\le2x-\pi/3\le5\pi/3\). So \(-\pi/3\le\theta\le5\pi/3\).
  • Reference angle and quadrants: \(\alpha=\sin^{-1}(1/2)=\pi/6\). Since sine is positive, candidate families are \[ \theta=\alpha+2k\pi,\qquad \theta=\pi-\alpha+2k\pi. \]
  • Filter by transformed window: \[ -\frac{\pi}{3}\le\theta\le\frac{5\pi}{3}. \] For \(k=0\), we get \(\theta=\pi/6,\ 5\pi/6\), both valid. If we subtract \(2\pi\), angles become \(-11\pi/6,\ -7\pi/6\), too small (outside lower bound \(-\pi/3\)). If we add \(2\pi\), angles become \(13\pi/6,\ 17\pi/6\), too large (outside upper bound \(5\pi/3\)). So only \(\theta=\pi/6,\ 5\pi/6\) survive.
  • Back-solve: \[ 2x-\frac{\pi}{3}=\frac{\pi}{6}\Rightarrow x=\frac{\pi}{4},\qquad 2x-\frac{\pi}{3}=\frac{5\pi}{6}\Rightarrow x=\frac{7\pi}{12} \]
Final: \(x=\frac{\pi}{4},\ \frac{7\pi}{12}\).

High-Volume Point-Choosing Practice (Graphing Coordinates Only)

This section is only about picking accurate plotting points from the equation before drawing the curve. For each question, use:

  • midline \(y=D\), max \(D+|A|\), min \(D-|A|\)
  • period \(T=\frac{2\pi}{|B|}\)
  • step size \(T/4\)
  • phase shift start \(x=h\) from \(B(x-h)\)
  • 5-point pattern: cosine \(=\) peak, midline, trough, midline, peak; sine \(=\) midline, peak, midline, trough, midline.

Quick self-check rubric

For each drill, check these 5 things:

  • Did you compute correct period \(T\)?
  • Did you use step size \(T/4\)?
  • Did you start from correct phase-shift point \(x=h\)?
  • Did you apply sine/cosine 5-point y-pattern correctly?
  • If \(A<0\), did you reflect vertically (swap peak/trough behavior)?

Q10Solve \(3\cos\left(2\left(x-\frac{\pi}{5}\right)\right)-1=-2\), \(0\le x\le2\pi\)

Answers + Graphs (Click to Expand)

Answer 1\(x=\frac{\pi}{12},\frac{5\pi}{12},\frac{13\pi}{12},\frac{17\pi}{12}\)

Set \(\theta=2x\), so \(0\le\theta\le4\pi\). For \(\sin\theta=1/2\), reference angle is \(\pi/6\), and sine-positive quadrants are QI/QII: \(\theta=\pi/6,\ 5\pi/6\) in first cycle. Since window continues to \(4\pi\), add \(2\pi\): \(\theta=13\pi/6,\ 17\pi/6\). (No subtract \(2\pi\) because window has no negatives.) Divide all by 2 to get \(x\).

Answer 2\(x=\frac{\pi}{4},\frac{7\pi}{12}\)

After isolating, solve \(\sin\theta=1/2\) with \(\theta=2x-\pi/3\) in \(-\pi/3\le\theta\le5\pi/3\). Use QI/QII template: \(\theta=\pi/6+2k\pi\), \(\theta=5\pi/6+2k\pi\). Only \(k=0\) values fit. \(k=-1\) (subtract \(2\pi\)) is too negative; \(k=1\) (add \(2\pi\)) exceeds \(5\pi/3\). Then back-solve each linear equation for \(x\).

Answer 3\(x=0,\frac{\pi}{3},\pi\)

Set \(\theta=2x-\pi/3\), so \(-\pi/3\le\theta\le5\pi/3\). For \(\cos\theta=1/2\), reference angle \(\alpha=\pi/3\), cosine-positive quadrants are QI and QIV: \(\theta=\pi/3\) and \(\theta=2\pi-\pi/3=5\pi/3\), plus boundary \(\theta=-\pi/3\) also satisfies cosine \(=1/2\). These three \(\theta\)-values are in-window; then solve each for \(x\).

Answer 4\(x=\frac{\pi}{18},\frac{5\pi}{18},\frac{13\pi}{18},\frac{17\pi}{18},\frac{25\pi}{18},\frac{29\pi}{18}\)

Let \(\theta=3x+\pi/6\), transformed window \(\pi/6\le\theta\le37\pi/6\). Solve \(\cos\theta=1/2\): base angles \(\pi/3\) (QI) and \(5\pi/3\) (QIV). Because window spans multiple cycles, add \(2\pi\) repeatedly until upper bound is exceeded: \[ \theta\in\left\{\frac{\pi}{3},\frac{5\pi}{3},\frac{7\pi}{3},\frac{11\pi}{3},\frac{13\pi}{3},\frac{17\pi}{3}\right\}. \] Then solve \(3x+\pi/6=\theta\) for each.

Answer 5\(x=\frac{\pi}{4},\frac{3\pi}{4},\frac{5\pi}{4},\frac{7\pi}{4}\)

Set \(\phi=2x-\pi/4\). Tangent period is \(\pi\), so \[ \tan\phi=1\Rightarrow \phi=\frac{\pi}{4}+k\pi. \] This already encodes “add/subtract period” via integer \(k\). Substitute back and keep only \(x\in[0,2\pi]\).

Answer 6\(x=\frac{5\pi}{3},3\pi,\frac{17\pi}{3}\)

Set \(\theta=x/2-\pi/3\), so \(-\pi/3\le\theta\le5\pi/3\). For \(\sin\theta=-1/2\), reference angle \(\pi/6\), sine-negative quadrants are QIII/QIV: \[ \theta=\frac{7\pi}{6},\ \frac{11\pi}{6}. \] Also \(\theta=-\pi/6\) (subtract \(2\pi\) from \(11\pi/6\)) lies in window and is valid. Back-solve each \(\theta\) to get 3 \(x\)-solutions.

Answer 7\(x=\frac{\pi}{3},\pi,\frac{7\pi}{3}\)

After isolating, let \(\theta=x-\pi/6\), and use \(\sin\theta=3/4\), with \(\alpha=\sin^{-1}(3/4)\). Positive sine gives QI/QII families: \[ \theta=\alpha+2k\pi,\quad \theta=\pi-\alpha+2k\pi. \] Convert each to \(x=\theta+\pi/6\), then keep only values in \(-\pi\le x\le2\pi\).

Answer 8\(x=-2\pi,\ -\frac{\pi}{2},\ 0,\ \frac{3\pi}{2},\ 2\pi\)

Set \(\theta=x/2+\pi/4\). Transformed window becomes \(-3\pi/4\le\theta\le5\pi/4\). For \(\cos\theta=\sqrt2/2\), base angles are \(\pm\pi/4\). Then add/subtract \(2\pi\) only if still inside the window. Keep valid \(\theta\), then recover \(x=2(\theta-\pi/4)\).

Answer 9\(x=-\frac{3\pi}{2},\ \frac{\pi}{2}\)

\(\tan(x/2)=1\Rightarrow x/2=\pi/4+k\pi\). This already includes all repeats because tangent repeats every \(\pi\) in its argument. Multiply by 2: \[ x=\frac{\pi}{2}+2k\pi. \] Check \(-2\pi\le x\le2\pi\): only \(k=-1,0\) survive.

Answer 10\(x\approx1.194,\ 1.948,\ 4.336,\ 5.089\)

Let \(\theta=2(x-\pi/5)\). Then \(\cos\theta=-1/3\), with reference angle \(\alpha=\cos^{-1}(1/3)\). Cosine is negative in QII/QIII, so one-cycle solutions are \[ \theta=\pi-\alpha,\quad \theta=\pi+\alpha. \] Because the transformed window spans two cycles, add \(2\pi\) to get the second-cycle pair. Then solve \(x=\theta/2+\pi/5\), giving the 4 values shown.

Tip: For every transformed equation, write the argument as \(\theta\), transform the domain in \(\theta\), solve in \(\theta\), then back-solve to \(x\).
