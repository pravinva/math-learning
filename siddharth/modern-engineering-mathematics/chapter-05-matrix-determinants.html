Matrix Algebra and Determinants

Think of matrices as machines: input vector goes in, transformed vector comes out. Determinants tell whether the machine squashes area/volume to zero. This page walks slowly and carefully with lots of examples and practice.

5.2 Matrix Basics and Operations

Order, addition, scalar multiplication, matrix multiplication, and interpretation as transformations.

Motivation first

Real engineering systems often use many simultaneous equations. Matrices package these equations into a compact, computable structure: \(A\mathbf x=\mathbf b\).

$$A=\begin{bmatrix}a&b\\c&d\end{bmatrix},\quad A\begin{bmatrix}x\\y\end{bmatrix}= \begin{bmatrix}ax+by\\cx+dy\end{bmatrix}$$

Worked Example

\(A=\begin{bmatrix}2&1\\-1&3\end{bmatrix},\ \mathbf x=\begin{bmatrix}4\\2\end{bmatrix}\Rightarrow A\mathbf x=\begin{bmatrix}10\\2\end{bmatrix}\).

Read it row-by-row: first row gives first output equation, second row gives second output equation.

5.3 Determinants

Determinant as scale factor and invertibility test.
$$\det\begin{bmatrix}a&b\\c&d\end{bmatrix}=ad-bc$$
Geometric meaning: for 2×2 matrices, \(|\det(A)|\) is area scale of transformed shapes. If \(\det(A)=0\), area collapses to a line, so inverse does not exist.
Determinant signMeaning
\(\det(A)>0\)Orientation preserved
\(\det(A)<0\)Orientation flipped (reflection included)
\(\det(A)=0\)Singular, no unique inverse
Worked Example

\(A=\begin{bmatrix}3&2\\5&4\end{bmatrix}\Rightarrow \det(A)=12-10=2\). Area doubles (up to orientation), and inverse exists.

Subtopic Practice (8) — Determinants

  1. \(\det\begin{bmatrix}2&1\\4&3\end{bmatrix}\).
    Show solution
    \(6-4=2\).
  2. \(\det\begin{bmatrix}1&2\\2&4\end{bmatrix}\).
    Show solution
    \(4-4=0\).
  3. For what \(k\) is \(\begin{bmatrix}1&k\\2&4\end{bmatrix}\) singular?
    Show solution
    \(4-2k=0\Rightarrow k=2\).
  4. \(\det\begin{bmatrix}3&0&1\\2&1&4\\0&2&5\end{bmatrix}\).
    Show solution
    Expand row1: \(3(1\cdot5-4\cdot2)+1(2\cdot2-1\cdot0)=-9+4=-5\).
  5. If \(\det(A)\neq0\), inverse exists?
    Show solution
    Yes.
  6. If \(\det(A)=0\), system \(Ax=b\) unique?
    Show solution
    Not guaranteed; no unique inverse.
  7. Area scale for \(A\) with \(\det(A)=-3\)?
    Show solution
    Scale factor \(3\), orientation flips.
  8. Compute \(\det(I_3)\).
    Show solution
    1.

5.4–5.5 Inverse and Solving Linear Equations

Use inverse matrix and elimination methods; know when each method is practical.
$$A^{-1}=\frac{1}{ad-bc}\begin{bmatrix}d&-b\\-c&a\end{bmatrix}\quad(\det A\neq0)$$

Method selection for weak students

  • 2×2 system: inverse method is fast.
  • 3×3 and larger: elimination is usually safer.
  • Computers: LU/iterative methods for large sparse systems.

Step-by-step Example

Solve \(\begin{cases}2x+y=7\\x-3y=-8\end{cases}\). From first, \(y=7-2x\). Substitute: \(x-3(7-2x)=-8\Rightarrow7x=13\Rightarrow x=13/7\), then \(y=23/7\).

Common mistake: forgetting to check determinant/non-singular condition before using inverse formula.

5.6–5.7 Rank and Eigenvalues

Rank tells independent information count; eigenvalues tell natural scaling directions.
$$A\mathbf v=\lambda\mathbf v,\qquad \det(A-\lambda I)=0$$

Intuition ladder

  • Rank: how many independent rows/columns remain after elimination.
  • Eigenvector: direction unchanged by matrix machine.
  • Eigenvalue: stretch/shrink factor on that special direction.
Worked Example

\(A=\begin{bmatrix}2&0\\0&3\end{bmatrix}\): x-axis and y-axis stay in same direction, with scaling 2 and 3. So eigenpairs are \((\lambda_1=2,\mathbf e_1)\), \((\lambda_2=3,\mathbf e_2)\).

Subtopic Practice (8) — Rank and Eigenvalues

  1. Rank of \(I_3\).
    Show solution
    3.
  2. Rank of \(\begin{bmatrix}1&2\\2&4\end{bmatrix}\).
    Show solution
    1.
  3. Characteristic equation of \(\begin{bmatrix}2&0\\0&5\end{bmatrix}\).
    Show solution
    \((2-\lambda)(5-\lambda)=0\).
  4. Eigenvalues of previous matrix.
    Show solution
    2 and 5.
  5. Eigenvector for \(\lambda=2\) in previous matrix.
    Show solution
    Any nonzero multiple of \((1,0)\).
  6. Eigenvalues of \(\begin{bmatrix}1&0\\0&1\end{bmatrix}\).
    Show solution
    1, 1.
  7. If \(\lambda=0\) is eigenvalue, \(\det(A)=?\)
    Show solution
    0.
  8. Physical meaning of eigenvectors in systems?
    Show solution
    Natural mode directions that only scale, not rotate.

Solved Examples Bank (8)

Extra solved examples for slower revision and confidence building.
  1. Compute \(AB\), where \(A=\begin{bmatrix}2&1\\0&-1\end{bmatrix},B=\begin{bmatrix}3&4\\5&6\end{bmatrix}\).
    Show full solution
    \(AB=\begin{bmatrix}11&14\\-5&-6\end{bmatrix}\).
  2. Find \(\det\begin{bmatrix}4&7\\2&5\end{bmatrix}\).
    Show full solution
    \(\det=20-14=6\).
  3. Find inverse of \(\begin{bmatrix}4&7\\2&5\end{bmatrix}\).
    Show full solution
    \(A^{-1}=\frac{1}{6}\begin{bmatrix}5&-7\\-2&4\end{bmatrix}\).
  4. Solve \(3x+2y=7,\ x-y=1\).
    Show full solution
    From second \(x=1+y\). Substitute: \(3+5y=7\Rightarrow y=4/5\), \(x=9/5\).
  5. Determine rank of \(\begin{bmatrix}1&2&3\\2&4&6\\1&1&1\end{bmatrix}\).
    Show full solution
    Row2 = 2*Row1, but Row3 independent of Row1. Rank = 2.
  6. Find characteristic equation of \(\begin{bmatrix}1&2\\2&1\end{bmatrix}\).
    Show full solution
    \(\det\begin{bmatrix}1-\lambda&2\\2&1-\lambda\end{bmatrix}= (1-\lambda)^2-4=0\).
  7. Find eigenvalues of previous matrix.
    Show full solution
    \((1-\lambda)^2=4\Rightarrow \lambda=3,-1\).
  8. Check if \((1,1)\) is eigenvector for \(\begin{bmatrix}2&1\\1&2\end{bmatrix}\).
    Show full solution
    \(A(1,1)=(3,3)=3(1,1)\), so yes, eigenvalue \(3\).

  1. \(\det\begin{bmatrix}5&2\\1&3\end{bmatrix}\).
    Show answer
    \(15-2=13\).
  2. For which \(k\), matrix \(\begin{bmatrix}1&2\\k&4\end{bmatrix}\) is singular?

  3. Find inverse of \(\begin{bmatrix}2&1\\3&2\end{bmatrix}\).

  4. State meaning of \(\det(A)=0\).
    Show answer

  5. \(\det\begin{bmatrix}1&0&2\\2&1&3\\0&1&4\end{bmatrix}\).
    Show answer

Set C — Linear systems and rank (8)

  1. Solve \(x+y=5,\ 2x-y=1\).
    Show answer
    \(x=2,\ y=3\).
  2. Classify: \(x+y=2,\ 2x+2y=4\).
    Show answer
    Infinitely many solutions (same line).
  3. Classify: \(x+y=2,\ 2x+2y=5\).
    Show answer
    No solution (parallel inconsistent lines).
  4. Rank of identity \(I_3\)?
    Show answer
    3.
  5. Rank of \(\begin{bmatrix}1&2\\2&4\end{bmatrix}\)?
    Show answer
    1 (second row multiple of first).
  6. What does full rank mean for square \(n\times n\)?
    Show answer
    Rank \(=n\), invertible, unique solution for every \(b\).
  7. Solve \(x-2y=4,\ 3x+y=5\).
    Show answer
    From first \(x=4+2y\). Then \(12+7y=5\Rightarrow y=-1,\ x=2\).

    Show answer
    At most rank 3.

Set D — Eigenvalues and applications (8)

  1. Eigenvalues of \(\begin{bmatrix}4&0\\0&-1\end{bmatrix}\).
    Show answer
    \(\lambda=4,-1\).
  2. Find eigenvalues of \(\begin{bmatrix}2&1\\1&2\end{bmatrix}\).
    Show answer
    \(\det\begin{bmatrix}2-\lambda&1\\1&2-\lambda\end{bmatrix}=0\Rightarrow(2-\lambda)^2-1=0\), so \(\lambda=3,1\).
  3. For \(\lambda=3\) in previous matrix, give one eigenvector.
    Show answer
    \((1,1)\) works.
  4. Interpret negative eigenvalue physically.
    Show answer
    Direction flips plus scaling by absolute value.
  5. Why eigenvalues matter in vibration/spring systems?
    Show answer
    They relate to natural modes and frequencies.
  6. Steady-state heat equations in layered materials form what math object?
    Show answer
    A linear system \(A\mathbf x=\mathbf b\).
  7. Eigenvalues of identity matrix \(I_3\)?
    Show answer
    All eigenvalues are 1 (triple).
  8. If \(\lambda=0\) is an eigenvalue, what can you infer about \(A\)?
    Show answer
    \(\det(A)=0\), so matrix is singular.
