7.2 Sequences and Pattern Building

Arithmetic, geometric, and general nth-term thinking.

Motivation first

A sequence is a list produced by a rule. In engineering, sampled sensor signals, iterative algorithms, and error updates are all sequences.

$$a_n=a_1+(n-1)d\quad\text{(arithmetic)},\qquad a_n=a_1r^{n-1}\quad\text{(geometric)}$$

Step-by-step Example

Sequence: \(5,8,11,\dots\). Difference is +3, so arithmetic with \(a_1=5,d=3\). Thus \(a_n=5+3(n-1)=3n+2\).

7.3 Finite Sequences and Series

Converting term patterns to compact summation formulas.
$$S_n=\frac{n}{2}\bigl(2a_1+(n-1)d\bigr),\qquad S_n=a_1\frac{1-r^n}{1-r}\ (r\neq1)$$
Engineers use finite series for total cost, sampled energy, algorithm step counts, and cumulative error estimates.

Worked Example

Find \(1+3+5+\dots+19\). This is arithmetic: \(a_1=1,d=2,a_n=19\Rightarrow n=10\). Then \(S_{10}=10(1+19)/2=100\).

Subtopic Practice (8) — Finite Series

  1. Find \(S_{20}\) for arithmetic sequence \(2,5,8,\dots\).
    Show solution
    \(a_{20}=59\), \(S_{20}=20(2+59)/2=610\).
  2. Find \(1+2+\cdots+40\).
    Show solution
    \(40\cdot41/2=820\).
  3. Find \(S_8\) for geometric \(a_1=3,r=2\).
    Show solution
    \(3(2^8-1)=765\).
  4. Find \(S_6\) for geometric \(a_1=10,r=0.5\).
    Show solution
    \(10\frac{1-0.5^6}{1-0.5}=19.6875\).
  5. Find number of terms if \(a_1=4,d=3,a_n=46\).
    Show solution
    \(46=4+(n-1)3\Rightarrow n=15\).
  6. Find \(2+4+8+\dots+1024\).
    Show solution
    \(a_1=2,r=2,n=10\), \(S=2(2^{10}-1)=2046\).
  7. Evaluate \(\sum_{k=1}^{10}(2k-1)\).
    Show solution
    First 10 odd numbers sum to \(10^2=100\).
  8. Find \(\sum_{k=1}^{6}3k\).
    Show solution
    \(3\sum k=3(21)=63\).

7.4 Recurrence Relations

Generating sequences from previous values (discrete dynamics).

Simple model idea

A recurrence is like a "next-step rule". Example: \(a_{n+1}=a_n+3\) or \(a_{n+1}=0.8a_n+2\). These are used in control loops and iterative numerical methods.

$$a_{n+1}=pa_n+q,\quad a_0\text{ given}$$

p

a0
Worked Example

Given \(a_{n+1}=2a_n,\ a_0=3\): terms are \(3,6,12,24,\dots\). This is geometric with ratio 2, so \(a_n=3\cdot2^n\).

Common pitfall: forgetting index alignment (starting from \(a_0\) vs \(a_1\)).

Subtopic Practice (8) — Recurrence

  1. \(a_{n+1}=a_n+4,\ a_0=1\): find \(a_5\).
    Show solution
    \(a_5=1+5(4)=21\).
  2. \(a_{n+1}=3a_n,\ a_0=2\): find \(a_4\).
    Show solution
    \(2\cdot3^4=162\).
  3. \(a_{n+1}=0.5a_n,\ a_0=16\): find \(a_6\).
    Show solution
    \(16(0.5)^6=0.25\).
  4. \(a_{n+1}=a_n+2n,\ a_1=1\): find \(a_2,a_3\).
    Show solution
    \(a_2=3,\ a_3=7\).
  5. For \(a_{n+1}=0.8a_n+1\), fixed point value?
    Show solution
    Set \(a=0.8a+1\Rightarrow a=5\).
  6. \(a_{n+1}=a_n-1,\ a_0=3\). Find first term \(<0\).
    Show solution
    Terms: 3,2,1,0,-1. First below 0 at \(a_4\).
  7. If \(a_{n+1}=a_n\), what does sequence do?
    Show solution
    Constant sequence.
  8. \(a_{n+1}=-a_n,\ a_0=2\): describe behavior.
    Show solution
    Alternates \(2,-2,2,-2,\dots\).

7.5–7.7 Limits, Infinite Series, and Power Series

Convergence ideas and practical test selection.
$$\lim_{n\to\infty}a_n=L,\qquad \sum_{n=0}^{\infty}ar^n=\frac{a}{1-r}\ (|r|<1)$$

Intuition ladder

  • Sequence convergence asks: do terms settle to one value?
  • Series convergence asks: do partial sums settle?
  • Power series adds variable \(x\): convergence depends on radius \(R\).
In engineering, infinite series give tractable approximations for signals, response functions, and numerical solutions.
Worked Example

\(\sum_{n=0}^{\infty}3(0.4)^n\) converges since \(|r|<1\). Sum is \(3/(1-0.4)=5\).

Subtopic Practice (8) — Limits and Infinite Series

  1. Find \(\lim_{n\to\infty}\frac{5n+1}{2n+3}\).
    Show solution
    \(\frac{5}{2}\).
  2. Does \(a_n=(-1)^n\) converge?
    Show solution
    No, oscillates.
  3. Find \(\sum_{n=0}^{\infty}4(1/3)^n\).
    Show solution
    \(4/(1-1/3)=6\).
  4. Does \(\sum_{n=1}^{\infty}\frac{1}{n}\) converge?
    Show solution
    No (harmonic diverges).
  5. For geometric series, what condition guarantees convergence?
    Show solution
    \(|r|<1\).
  6. Find \(\lim_{n\to\infty}(0.8)^n\).
    Show solution
    0.
  7. Find radius of convergence of \(\sum x^n\).
    Show solution
    \(R=1\).
  8. Find \(\sum_{n=1}^{\infty}\frac{1}{2^n}\).
    Show solution
    1.

Set D — Limits and Infinite Series (8)

  1. \(\lim_{n\to\infty}\frac{7n-2}{3n+9}\).
    Show answer
    \(7/3\).
  2. \(\lim_{n\to\infty}\frac{1}{n^2}\).
    Show answer
    0.
  3. \(\sum_{n=0}^{\infty}5(0.2)^n\).
    Show answer
    \(5/0.8=6.25\).
  4. Does \(\sum_{n=1}^{\infty}(-1)^n\) converge?
    Show answer
    No (oscillatory partial sums).
  5. Find \(\sum_{n=1}^{\infty}\frac{3}{4^n}\).
    Show answer
    \(=3\cdot\frac{1/4}{1-1/4}=1\).
  6. Radius of convergence for \(\sum (2x)^n\).
    Show answer
    \(|2x|<1\Rightarrow R=1/2\).
  7. Does \(\sum_{n=1}^{\infty}\frac{1}{n^2}\) converge?
    Show answer
    Yes.
  8. For geometric with \(r=-0.5\), converge or diverge?
    Show answer
    Converges since \(|r|<1\).
